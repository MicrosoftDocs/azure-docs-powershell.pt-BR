# Módulo AZS. backup. admin
## Descritivo
Versão prévia do módulo da operadora de backup AzureStack.  Este módulo permite que os operadores monitorem e gerenciem backups.

## Cmdlets. AZS. backup. admin
### [Get-AzsBackup](Get-AzsBackup.md)
Retorna um backup de um local com base em nome.

### [Get-AzsBackupConfiguration](Get-AzsBackupConfiguration.md)
Retorna a lista de configurações de backup.

### [New-AzsEncryptionKeyBase64](New-AzsEncryptionKeyBase64.md)
Gerar chave de criptografia para backups de infraestrutura.

### [Restore-AzsBackup](Restore-AzsBackup.md)
Restaurar um backup.

### [Set-AzsBackupConfiguration](Set-AzsBackupConfiguration.md)
Defina a configuração de backup no local especificado.

### [Start-AzsBackup](Start-AzsBackup.md)
Fazer backup de um local específico.
