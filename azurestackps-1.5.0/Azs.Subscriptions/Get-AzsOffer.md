# Get-AzsOffer

## Sinopse
Obtenha a lista de ofertas públicas.

## SYNTAX

```
Get-AzsOffer [[-Skip] <Int32>] [[-Top] <Int32>] [<CommonParameters>]
```

## DESCRITIVO
Obtenha a lista de ofertas públicas.

## EXEMPLOS

### --------------------------EXEMPLO 1--------------------------
```
Get-AzsOffer | fl
```

Obtenha a lista de ofertas públicas.
