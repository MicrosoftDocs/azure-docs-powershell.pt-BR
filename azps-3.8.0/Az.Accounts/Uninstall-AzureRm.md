# Uninstall-AzureRm

## Sinopse
Remove todos os módulos AzureRm de um computador.

## SYNTAX

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
Remove todos os módulos AzureRm de um computador.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Uninstall-AzureRm
```

Executar esse comando removerá todos os módulos AzureRm da máquina para a versão do PowerShell em que o cmdlet é executado.
