# Remove-AzSqlDatabaseFailoverGroup

## Sinopse
Remove um grupo de failover do banco de dados SQL do Azure.

## SYNTAX

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
Esse comando Remove o grupo de failover com o nome especificado, deixando todos os bancos de dados e relações de replicação intactos. O ponto de extremidade de escuta será cancelado do DNS.
O servidor primário do grupo de failover deve ser usado para executar o comando.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Remover um grupo de failover.
