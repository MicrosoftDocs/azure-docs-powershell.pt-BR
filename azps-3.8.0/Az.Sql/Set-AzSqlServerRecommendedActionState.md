# Set-AzSqlServerRecommendedActionState

## Sinopse
Atualiza o estado de uma ação recomendada do Azure SQL Server.

## SYNTAX

```
Set-AzSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzSqlServerRecommendedActionState** atualiza o estado de uma ação recomendada do Azure SQL Server.
Esse cmdlet aplica, reverte ou descarta a ação recomendada com base no novo estado.
