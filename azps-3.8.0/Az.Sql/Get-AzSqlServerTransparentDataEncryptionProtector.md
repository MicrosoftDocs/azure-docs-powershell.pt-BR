# Get-AzSqlServerTransparentDataEncryptionProtector

## Sinopse
Obtém o protetor de criptografia de dados transparente (TDE)

## SYNTAX

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Get-AzSqlServerTransparentDataEncryptionProtector Obtém informações sobre o protetor TDE.

## EXEMPLOS

### Exemplo 1: obter o protetor de criptografia de dados transparente (TDE)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Esse comando obtém o protetor TDE do servidor chamado ContosoServer no grupo de recursos chamado ContosoResourceGroup.
