# Get-AzComputeResourceSku

## Sinopse
Listar todas as SKUs do recurso de computação

## SYNTAX

```
Get-AzComputeResourceSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Listar todas as SKUs do recurso de computação

## EXEMPLOS

### Exemplo 1
```
PS C:\> PS C:\> Get-AzComputeResourceSku | where {$_.Locations.Contains("westus")};
```

Listar todas as SKUs do recurso de computação na região oeste dos EUA
