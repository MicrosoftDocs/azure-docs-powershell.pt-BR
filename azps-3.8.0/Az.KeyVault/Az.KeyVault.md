# Módulo AZ. keyvault
## Descritivo
Este tópico exibe os tópicos da ajuda para os cmdlets do Azure Key Vault.

## Cmdlets AZ. keyvault
### [Add-AzKeyVaultCertificate](Add-AzKeyVaultCertificate.md)
Adiciona um certificado a um cofre de chaves.

### [Add-AzKeyVaultCertificateContact](Add-AzKeyVaultCertificateContact.md)
Adiciona um contato para notificações de certificado.

### [Add-AzKeyVaultKey](Add-AzKeyVaultKey.md)
Cria uma chave em um cofre de chaves ou importa uma chave para um cofre de chaves.

### [Add-AzKeyVaultManagedStorageAccount](Add-AzKeyVaultManagedStorageAccount.md)
Adiciona uma conta de armazenamento do Azure existente ao cofre de chaves especificado para que as chaves sejam gerenciadas pelo serviço do cofre de chaves.

### [Add-AzKeyVaultNetworkRule](Add-AzKeyVaultNetworkRule.md)
Adiciona uma regra para restringir o acesso a um cofre de chaves baseado no endereço de Internet do cliente.

### [Backup-AzKeyVaultCertificate](Backup-AzKeyVaultCertificate.md)
Faz backup de um certificado em um cofre de chaves.

### [Backup-AzKeyVaultKey](Backup-AzKeyVaultKey.md)
Faz o backup de uma chave em um cofre de chaves.

### [Backup-AzKeyVaultManagedStorageAccount](Backup-AzKeyVaultManagedStorageAccount.md)
Faz backup de uma conta de armazenamento gerenciado pelo keyvault.

### [Backup-AzKeyVaultSecret](Backup-AzKeyVaultSecret.md)
Faz o backup de um segredo em um cofre de chaves.

### [Get-AzKeyVault](Get-AzKeyVault.md)
Obtém cofres de chave.

### [Get-AzKeyVaultCertificate](Get-AzKeyVaultCertificate.md)
Obtém um certificado de um cofre de chaves.

### [Get-AzKeyVaultCertificateContact](Get-AzKeyVaultCertificateContact.md)
Obtém contatos que são registrados para notificações de certificado para um cofre de chaves.

### [Get-AzKeyVaultCertificateIssuer](Get-AzKeyVaultCertificateIssuer.md)
Obtém um emissor de certificado para um cofre de chaves.

### [Get-AzKeyVaultCertificateOperation](Get-AzKeyVaultCertificateOperation.md)
Obtém o status de uma operação de certificado.

### [Get-AzKeyVaultCertificatePolicy](Get-AzKeyVaultCertificatePolicy.md)
Obtém a política para um certificado em um cofre de chaves.

### [Get-AzKeyVaultKey](Get-AzKeyVaultKey.md)
Obtém chaves do cofre de chaves.

### [Get-AzKeyVaultManagedStorageAccount](Get-AzKeyVaultManagedStorageAccount.md)
Obtém contas de armazenamento do Azure gerenciadas do Key Vault.

### [Get-AzKeyVaultManagedStorageSasDefinition](Get-AzKeyVaultManagedStorageSasDefinition.md)
Obtém definições de SAS de armazenamento gerenciado do Key Vault.

### [Get-AzKeyVaultSecret](Get-AzKeyVaultSecret.md)
Obtém os segredos em um cofre de chaves.

### [Import-AzKeyVaultCertificate](Import-AzKeyVaultCertificate.md)
Importa um certificado para um cofre de chaves.

### [New-AzKeyVault](New-AzKeyVault.md)

### [New-AzKeyVaultCertificateAdministratorDetail](New-AzKeyVaultCertificateAdministratorDetail.md)

### [New-AzKeyVaultCertificateOrganizationDetail](New-AzKeyVaultCertificateOrganizationDetail.md)

### [New-AzKeyVaultCertificatePolicy](New-AzKeyVaultCertificatePolicy.md)
