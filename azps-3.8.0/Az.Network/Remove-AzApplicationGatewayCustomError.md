# Remove-AzApplicationGatewayCustomError

## Sinopse
Remove um erro personalizado de um gateway de aplicativo.

## SYNTAX

```
Remove-AzApplicationGatewayCustomError -StatusCode <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayCustomError** remove um erro personalizado de um Application Gateway.

## EXEMPLOS

### Exemplo 1: Remove o erro personalizado de um aplicativo gateway
```
PS C:\> $updatedgateway = Remove-AzApplicationGatewayCustomError -ApplicationGateway $appgw -StatusCode HttpStatus502
```

Esse comando Remove o erro personalizado do código de status http 502 da $appgw do gateway do aplicativo e retorna o gateway atualizado.
