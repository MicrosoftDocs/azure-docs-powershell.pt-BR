# Update-AzP2sVpnGateway

## Sinopse
Atualize um P2SVpnGateway existente em VirtualHub para conectividade com o ponto para o site.

## SYNTAX

### ByP2SVpnGatewayNameNoVpnServerConfigurationUpdate (padrão)
```
Update-AzP2sVpnGateway -ResourceGroupName <String> -Name <String> [-VpnClientAddressPool <String[]>]
 [-VpnGatewayScaleUnit <UInt32>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Update-AzP2sVpnGateway** permite que você atualize um P2SVpnGateway existente em VirtualHub com novos VpnClientAddressPool ou novo VpnServerConfiguration ou alterar VpnGatewayScaleUnit.
