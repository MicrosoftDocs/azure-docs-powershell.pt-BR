# Set-AzApplicationGatewayAutoscaleConfiguration

## Sinopse
Atualiza a configuração de autoescala de um gateway de aplicativo.

## SYNTAX

```
Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> -MinCapacity <Int32>
 [-MaxCapacity <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzApplicationGatewayAutoscaleConfiguration** modifica a configuração de autoescala existente de um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw -MinCapacity 5
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

O primeiro comando obtém o gateway do aplicativo e o armazena em $gw variável.
O segundo comando atualiza a configuração de autoescala do gateway do aplicativo.
O terceiro comando atualiza o Application Gateway no Azure.
