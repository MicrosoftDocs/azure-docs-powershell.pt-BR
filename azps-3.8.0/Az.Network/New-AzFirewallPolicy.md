# New-AzFirewallPolicy

## Sinopse
Cria uma nova política de firewall do Azure

## SYNTAX

```
New-AzFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String> [-ThreatIntelMode <String>]
 [-BasePolicy <String>] [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzFirewallPolicy** cria uma política de firewall do Azure.

## EXEMPLOS

### 1. criar uma política vazia
```powershell
PS C:\> New-AzFirewallPolicy -Name fp1 -ResourceGroupName TestRg
```

Este exemplo cria uma política de firewall do Azure

### 2. criar uma política vazia com o modo ThreatIntel
```powershell
PS C:\> New-AzFirewallPolicy -Name fp1 -ResourceGroupName TestRg -ThreatIntelMode "Deny"
```

Este exemplo cria uma política de firewall do Azure com um modo de ameaça Intel
