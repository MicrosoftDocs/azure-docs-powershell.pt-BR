# Add-AzDelegation

## Sinopse
Adiciona uma delegação a uma sub-rede.

## SYNTAX

```
Add-AzDelegation -Name <String> -ServiceName <String> -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzDelegation** adiciona uma delegação de serviço a uma sub-rede do Azure.

## EXEMPLOS

### 1: adicionando uma delegação
```powershell
PS C:\> $vnet = Get-AzVirtualNetwork -Name "myVNet" -ResourceGroupName "myResourceGroup"
PS C:\> $subnet = Get-AzVirtualNetworkSubnetConfig -Name "mySubnet" -VirtualNetwork $vnet
PS C:\> $subnet = Add-AzDelegation -Name "myDelegation" -ServiceName "Microsoft.Sql/servers" -Subnet $subnet
PS C:\> Set-AzVirtualNetwork $vnet
```

O primeiro comando recupera a rede virtual na qual a sub-rede se encontra. O segundo comando isola a sub-rede de interesse-aquela que você deseja delegar. O terceiro comando adiciona uma delegação à sub-rede. Esse exemplo específico seria útil quando você quiser habilitar o SQL para criar pontos de extremidade de interface nessa sub-rede. O comando final envia a sub-rede atualizada ao servidor para realmente atualizar sua sub-rede.
