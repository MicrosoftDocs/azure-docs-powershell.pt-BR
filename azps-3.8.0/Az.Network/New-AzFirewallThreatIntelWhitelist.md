# New-AzFirewallThreatIntelWhitelist

## Sinopse
Criar uma nova lista branca de inteligência de ameaças para o Firewall do Azure

## SYNTAX

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzFirewallThreatIntelWhitelist** cria um objeto de lista branca da Intel com ameaças, que pode ser usado ao criar ou configurar um firewall do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

Este exemplo cria uma lista branca da Intel com ameaças contendo uma lista branca do FQDN de duas entradas e uma lista de endereços IP de duas entradas
