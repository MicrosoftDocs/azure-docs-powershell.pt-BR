# Remove-AzBatchAccount

## Sinopse
Remove uma conta em lotes.

## SYNTAX

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzBatchAccount** remove uma conta em lotes do Azure.
Esse cmdlet o avisa antes de remover uma conta, a menos que você especifique o parâmetro *Force* .

## EXEMPLOS

### Exemplo 1: remover uma conta em lotes
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

Esse comando Remove a conta em lotes chamada pfuller.
Esse comando solicita confirmação antes de excluir a conta.
