# Stop-AzManagementGroupDeployment

## Sinopse
Cancelar uma implantação em execução em um grupo de gerenciamento

## SYNTAX

### StopByDeploymentName (padrão)
```
Stop-AzManagementGroupDeployment [-ManagementGroupId] <String> [-Name] <String> [-PassThru]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzManagementGroupDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzManagementGroupDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzManagementGroupDeployment** cancela uma implantação que foi iniciada, mas não foi concluída em um grupo de gerenciamento.
Para interromper uma implantação, a implantação deve ter um estado de provisionamento incompleto, como provisionamento, e não um estado concluído, como provisionado ou com falha.

Para criar uma implantação em um grupo de gerenciamento, use o cmdlet New-AzManagementGroupDeployment.

## EXEMPLOS

### Exemplo 1
```
PS C:\>Stop-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01"
```

Esse comando cancela uma implantação em execução "deployment01" no grupo de gerenciamento "myMG".

### Exemplo 2
```
PS C:\>Get-AzManagementGroupDeployment -ManagementGroupId "myMG" -Name "deployment01" | Stop-AzManagementGroupDeployment
```

Este comando obtém a implantação "deployment01" no grupo de gerenciamento "myMG" e a cancela.
