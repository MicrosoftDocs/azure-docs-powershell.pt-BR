# Suspend-AzAnalysisServicesServer

## Sinopse
Suspende uma instância do servidor do Analysis Services

## SYNTAX

```
Suspend-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Suspend-AzAnalysisServicesServer suspende uma instância do servidor do Analysis Services

## EXEMPLOS

### Exemplo 1
```
PS C:\> Suspend-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Esse comando suspenderá um servidor ativo chamado TestServer no modo de teste do modo de Resource
