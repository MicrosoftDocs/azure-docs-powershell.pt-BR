# Restart-AzAnalysisServicesInstance

## Sinopse
Reinicia uma instância do servidor do Analysis Services no ambiente atualmente conectado, conforme especificado no comando Add-AzAnalysisServicesAccount

## SYNTAX

```
Restart-AzAnalysisServicesInstance [-Instance] <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Restart-AzAnalysisServicesInstance reinicia uma instância do servidor do Azure Analysis Services

## EXEMPLOS

### Exemplo 1
```
PS C:\>Restart-AzAnalysisServicesInstance
Instance: testserver
```

Esse comando irá reiniciar o servidor ' TestServer ' no ambiente especificado no comando Add-AzAnalysisServicesAccount
