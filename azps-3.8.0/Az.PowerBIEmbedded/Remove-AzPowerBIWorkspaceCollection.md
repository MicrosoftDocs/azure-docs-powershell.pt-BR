# Remove-AzPowerBIWorkspaceCollection

## Sinopse
Remove um conjunto do espaço de trabalho do Power BI.

## SYNTAX

```
Remove-AzPowerBIWorkspaceCollection [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzPowerBIWorkspaceCollection** remove uma coleção de espaço de trabalho do Power bi de sua assinatura do Azure e do grupo de recursos.

## EXEMPLOS

### Exemplo 1: remover uma coleção de espaço de trabalho
```
PS C:\>Remove-AzPowerBIWorkspaceCollection -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Esse comando Remove a coleção do espaço de trabalho chamado WCN11 no grupo de recursos especificado.
