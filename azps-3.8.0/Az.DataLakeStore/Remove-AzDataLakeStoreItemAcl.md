# Remove-AzDataLakeStoreItemAcl

## Sinopse
Limpa a ACL de um arquivo ou pasta no repositório data Lake.

## SYNTAX

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzDataLakeStoreItemAcl** limpa a ACL (lista de controle de acesso) de um arquivo ou pasta no repositório data Lake.

## EXEMPLOS

### Exemplo 1: remover a ACL de uma pasta
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Esse comando Remove a ACL do diretório raiz da conta ContosoADL.
