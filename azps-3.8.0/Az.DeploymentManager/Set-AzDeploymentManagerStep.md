# Set-AzDeploymentManagerStep

## Sinopse
Atualiza a etapa.

## SYNTAX

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzDeploymentManagerStep** atualiza uma etapa com o objeto Step especificado.
O cmdlet retorna o objeto Step atualizado.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

Esse comando atualiza uma etapa cujo nome e o meu nome do elemento de Resource correspondem às propriedades Name e ResourceGroupName do $stepObject, respectivamente.
A etapa seria atualizada para as propriedades definidas no $stepObject.
