# Set-AzCosmosDBTable

## Sinopse
Define a tabela CosmosDB.

## SYNTAX

### ByNameParameterSet (padrão)
```
Set-AzCosmosDBTable -ResourceGroupName <String> -AccountName <String> -Name <String> [-Throughput <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Set-AzCosmosDBTable -Name <String> [-Throughput <Int32>] -InputObject <PSDatabaseAccount>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzCosmosDBTable** cria uma nova tabela ou atualiza uma tabela existente, substituindo toda a tabela existente.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzCosmosDBTable -AccountName {account} -Name {tableName} -ResourceGroupName {rgName}

Name    Id    Resource
{name}  {id}  Microsoft.Azure.Commands.CosmosDB.Models.PSTableGetPropertiesResource
```

Objeto Resource contém as propriedades _rid, _ts, _ ETag da tabela.
