# New-AzIotCentralApp

## Sinopse
Cria um novo aplicativo Central de IoT.

## SYNTAX

```
New-AzIotCentralApp [-Subdomain] <String> [-DisplayName <String>] [-Template <String>] [-Sku <String>]
 [-Location <String>] [-Tag <Hashtable>] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Cria um novo aplicativo Central de IoT com as propriedades e os metadados fornecidos. Para obter uma introdução ao IoT central, consulte https://docs.microsoft.com/en-us/azure/iot-central/ .

## EXEMPLOS

### Exemplo 1 Crie um aplicativo Central simples de IoT.
```powershell
PS C:\> New-AzIotCentralApp -ResourceGroupName "MyResourceGroupName" -Name "MyAppResourceName" -Subdomain "MyAppSubdomain"
```

Exemplo de saída:

Crie um aplicativo da central do IoT na faixa de preços padrão ST2, na região do grupo de recursos.

### Exemplo 2 crie um aplicativo Central simples de IoT.
```powershell
PS C:\> New-AzIotCentralApp -ResourceGroupName "MyResourceGroupName" -Name "MyAppResourceName" -Subdomain "MyAppSubdomain" -Sku "ST2" -DisplayName "My Custom Display Name" -Template "iotc-default" -Location "westus"
```

Exemplo de saída:

Crie um aplicativo Central do IoT com o tipo de preço padrão ST2 na região ' oesteus ', com um nome de exibição personalizado, com base no modelo IOTC-padrão.
