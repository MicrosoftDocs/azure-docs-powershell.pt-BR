# Remove-AzRecoveryServicesAsrProtectionContainer

## Sinopse
Exclui o contêiner de proteção especificado da malha.

## SYNTAX

```
Remove-AzRecoveryServicesAsrProtectionContainer -InputObject <ASRProtectionContainer>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Remove-AzRecoveryServicesAsrProtectionContainer exclui o contêiner de proteção do Azure site Recovery especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzRecoveryServicesAsrProtectionContainer -Name xxxxx  -Fabric $fabric
PS C:\> Remove-AzRecoveryServicesAsrProtectionContainer -InputObject $protectionContainer
```

Inicia a exclusão do contêiner de proteção especificado e retorna o trabalho ASR usado para acompanhar a operação de remoção.
