# Remove-AzRedisCachePatchSchedule

## Sinopse
Remove o cronograma de correção.

## SYNTAX

```
Remove-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzRedisCachePatchSchedule** remove o cronograma do patch de um cache no Azure Redis cache.

## EXEMPLOS

### Exemplo 1: remover o cronograma do patch
```
PS C:\>Remove-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Esse comando Remove o cronograma do patch do cache chamado RedisCache06.
