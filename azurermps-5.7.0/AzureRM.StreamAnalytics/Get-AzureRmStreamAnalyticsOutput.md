# Get-AzureRmStreamAnalyticsOutput

## Sinopse
Obtém as saídas definidas em um trabalho ou saída de análise de fluxo especificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmStreamAnalyticsOutput** lista todas as saídas definidas em um trabalho de análise de fluxo especificado ou obtém informações sobre uma saída específica.

## EXEMPLOS

### EXEMPLO 1: obter informações sobre saídas de trabalho
```
PS C:\>Get-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Esse comando retorna informações sobre as saídas definidas no trabalho StreamingJob.

### EXEMPLO 2: obter informações sobre uma saída de trabalho específica
```
PS C:\>Get-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Esse comando retorna informações sobre a saída chamada output definida no Job StreamingJob.
