# Set-AzureRmCdnProfile

## Sinopse
Atualiza um perfil CDN.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmCdnProfile** atualiza um perfil da CDN (rede de distribuição de conteúdo) do Azure.
