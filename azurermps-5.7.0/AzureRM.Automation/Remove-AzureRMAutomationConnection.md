# Remove-AzureRmAutomationConnection

## Sinopse
Remove uma conexão de automação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmAutomationConnection** remove uma conexão da automação do Azure.

## EXEMPLOS

### Exemplo 1: remover uma conexão
```
PS C:\>Remove-AzureRmAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Esse comando Remove um certificado denominado ContosoConnection na conta de automação chamada Contoso17.
