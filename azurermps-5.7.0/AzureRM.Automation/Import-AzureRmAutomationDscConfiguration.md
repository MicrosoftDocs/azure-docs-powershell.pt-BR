# Import-AzureRmAutomationDscConfiguration

## Sinopse
Importa uma configuração DSC para automação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Import-AzureRmAutomationDscConfiguration -SourcePath <String> [-Tags <IDictionary>] [-Description <String>]
 [-Published] [-Force] [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Import-AzureRmAutomationDscConfiguration** importa uma configuração DSC (configuração de estado desejado APS) para a automação do Azure.
Especifique o caminho de um script APS que contém uma única configuração DSC.

## EXEMPLOS

### Exemplo 1: importar uma configuração DSC para automação
```
PS C:\>Import-AzureRmAutomationDscConfiguration -AutomationAccountName "Contoso17"-ResourceGroupName "ResourceGroup01" -SourcePath "C:\DSC\client.ps1" -Force
```

Esse comando importa a configuração DSC do arquivo chamado client.ps1 para a conta de automação chamada Contoso17. O comando especifica o parâmetro *Force* . Se houver uma configuração DSC existente, esse comando a substituirá.
