# Start-AzureRmAutomationRunbook

## Sinopse
Inicia um trabalho de runbook.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### ByAsynchronousReturnJob (padrão)
```
Start-AzureRmAutomationRunbook [-Name] <String> [-Parameters <IDictionary>] [-RunOn <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### BySynchronousReturnJobOutput
```
Start-AzureRmAutomationRunbook [-Name] <String> [-Parameters <IDictionary>] [-RunOn <String>] [-Wait]
 [-MaxWaitSeconds <Int32>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Start-AzureRmAutomationRunbook** inicia um trabalho do runbook de automação do Azure.
Especifique a ID ou o nome de um runbook.

## EXEMPLOS

### Exemplo 1: iniciar um trabalho de runbook
```
PS C:\>Start-AzureRmAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01"
```

Esse comando inicia um trabalho de runbook para o runbook chamado Runbk01 na conta de automação do Azure chamada Contoso17.

### Exemplo 2: iniciar um trabalho do runbook e esperar pelos resultados
```
Start-AzureRmAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01" -MaxWaitSeconds 1000 -Wait
```

Esse comando inicia um trabalho de runbook para o runbook chamado Runbk01 na conta de automação do Azure chamada Contoso17.
Esse comando especifica o parâmetro _Wait_ .
Portanto, retorna os resultados após a conclusão do trabalho.
O cmdlet aguarda até 1000 segundos para os resultados.

## EXIBE

### Microsoft. Azure. Commands. Automation. Model. Job
Esse cmdlet retorna um objeto de **trabalho** , a menos que você especifique o parâmetro _Wait_ .

Se você não especificar _Wait_ , o PowerShell do Azure retorna um objeto de **trabalho** imediatamente.
Se você especificar _Wait_ , o PowerShell do Azure concluirá o trabalho e, em seguida, retornará o resultado.
O resultado não é um objeto de **trabalho** .
