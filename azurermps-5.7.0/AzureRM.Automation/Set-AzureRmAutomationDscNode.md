# Set-AzureRmAutomationDscNode

## Sinopse
Modifica a configuração do nó para a qual um nó DSC está mapeado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmAutomationDscNode** modifica uma configuração de nó DSC (configuração de estado desejado APS).
A automação do Azure armazena a configuração do nó DSC como um documento de configuração do MOF (formato de objeto gerenciado).

## EXEMPLOS

### Exemplo 1: modificar o mapeamento de configuração do nó
```
PS C:\>Set-AzureRmAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Esse comando atribui a configuração de nó chamada contoso. NodeConfiguration01 ao nó que tem o GUID especificado.
