# Stop-AzureRmAutomationJob

## Sinopse
Interrompe um trabalho de automação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmAutomationJob** interrompe um trabalho de automação do Azure.
Especifique um trabalho de automação em execução.

## EXEMPLOS

### Exemplo 1: parar um trabalho
```
PS C:\>Stop-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Esse comando para o trabalho que tem a ID especificada.
