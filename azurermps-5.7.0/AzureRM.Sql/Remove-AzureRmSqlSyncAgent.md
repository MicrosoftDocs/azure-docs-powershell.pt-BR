# Remove-AzureRmSqlSyncAgent

## Sinopse
Remove um agente de sincronização do SQL do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmSqlSyncAgent** remove um agente de sincronização do SQL do Azure.

## EXEMPLOS

### Exemplo 1: remover um agente de sincronização
```
PS C:\>Remove-AzureRmSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Esse comando Remove o agente SQL Sync do Azure chamado syncAgent01.
