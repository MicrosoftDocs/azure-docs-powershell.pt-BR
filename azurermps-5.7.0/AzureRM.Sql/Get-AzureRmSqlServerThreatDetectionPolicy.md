# Get-AzureRmSqlServerThreatDetectionPolicy

## Sinopse
Obtém a política de detecção de ameaças para um servidor.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlServerThreatDetectionPolicy -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSqlServerThreatDetectionPolicy** Obtém a política de detecção de ameaças de um SQL Server do Azure.
Para usar esse cmdlet, especifique os parâmetros *ResourceGroupName* e *nomedoservidor* para identificar o servidor para o qual esse cmdlet obtém a política.

## EXEMPLOS

### Exemplo 1: obter a política de detecção de ameaças para um servidor
```
PS C:\>Get-AzureRmSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Este comando obtém a política de detecção de ameaças para um servidor chamado Server01.
O servidor é atribuído à ResourceGroup11 do grupo de recursos.
