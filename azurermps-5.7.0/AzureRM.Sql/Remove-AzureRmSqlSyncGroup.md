# Remove-AzureRmSqlSyncGroup

## Sinopse
Remove um grupo de sincronização do banco de dados SQL do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmSqlSyncGroup** remove um grupo de sincronização do banco de dados SQL do Azure.

## EXEMPLOS

### Exemplo 1: remover um grupo de sincronização
```
PS C:\>Remove-AzureRmSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Esse comando Remove o grupo de sincronização do banco de dados SQL do Azure chamado syncGroup01.
