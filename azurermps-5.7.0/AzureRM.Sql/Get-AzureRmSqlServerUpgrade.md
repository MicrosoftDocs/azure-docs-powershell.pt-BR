# Get-AzureRmSqlServerUpgrade

## Sinopse
Obtém o status de uma atualização do servidor de banco de dados SQL do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSqlServerUpgrade** Obtém o status de uma atualização do servidor de banco de dados SQL do Azure.

## EXEMPLOS

### Exemplo 1: obter o status de uma atualização
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Esse comando obtém o status de uma atualização do servidor chamado Server01 no grupo de recursos chamado ResourceGroup01.
