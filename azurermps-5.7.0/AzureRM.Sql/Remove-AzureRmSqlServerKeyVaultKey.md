# Remove-AzureRmSqlServerKeyVaultKey

## Sinopse
Remove uma chave do cofre de chaves de um SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Remove-AzureRmSqlServerKeyVaultKey remove a chave do cofre de chaves do SQL Server especificado.

Observe que as permissões do SQL Server para o cofre da chave não são alteradas.
Para alterar permissões, use set-AzureRmKeyVaultAccessPolicy.

Observe que esse cmdlet não faz alterações no cofre de chaves.
Para remover uma chave do cofre de chaves, use remove-AzureKeyVaultKey.

## EXEMPLOS

### Exemplo 1: remover uma chave do cofre de chaves
```
PS C:\> Remove-AzureRmSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Este comando Remove a chave do cofre de chaves com ID ' https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 ' do servidor especificado.
