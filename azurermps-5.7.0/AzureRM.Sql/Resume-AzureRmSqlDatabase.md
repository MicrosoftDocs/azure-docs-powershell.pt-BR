# Resume-AzureRmSqlDatabase

## Sinopse
Retoma um banco de dados de data warehouse do SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Resume-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **resume-AzureRmSqlDatabase** retoma um banco de dados de SQL data warehouse do Azure.

## EXEMPLOS

### Exemplo 1: retoma um banco de dados de SQL data warehouse do Azure
```
PS C:\>Resume-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando retoma um banco de dados suspenso do Azure SQL data warehouse.
