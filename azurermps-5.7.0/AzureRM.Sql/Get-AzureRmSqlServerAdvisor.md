# Get-AzureRmSqlServerAdvisor

## Sinopse
Obtém um ou mais conselheiros para um SQL Server do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSqlServerAdvisor** Obtém um ou mais conselheiros do SQL Server do Azure para um SQL Server do Azure.
