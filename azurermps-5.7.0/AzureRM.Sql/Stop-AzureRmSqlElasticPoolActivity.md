# Stop-AzureRmSqlElasticPoolActivity

## Sinopse
Cancela a operação de atualização assíncrona em um pool elástico.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmSqlElasticPoolActivity** cancela a operação de atualização assíncrona em um pool elástico.

## EXEMPLOS

### Exemplo 1: cancelar a operação de atualização assíncrona em um pool elástico
```
PS C:\> Stop-AzureRmSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -OperationId af97005d-9243-4f8a-844e-402d1cc855f5

ErrorCode       :

```

Esse comando cancela a operação de atualizações assíncronas no pool elástico.
