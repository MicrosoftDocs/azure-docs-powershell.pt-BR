# Set-AzureRmSqlDatabaseGeoBackupPolicy

## Sinopse
Define uma política de backup geográfico do banco de dados.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmSqlDatabaseGeoBackupPolicy** define a política de backup geográfico registrada em um banco de dados.
Este é um recurso de backup do Azure usado para definir a política de armazenamento de backup.
