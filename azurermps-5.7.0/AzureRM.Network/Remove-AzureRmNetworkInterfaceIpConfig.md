# Remove-AzureRmNetworkInterfaceIpConfig

## Sinopse
Remove uma configuração de IP de interface de rede de uma interface de rede.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmNetworkInterfaceIpConfig -Name <String> -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmNetworkInterfaceIpConfig** remove uma configuração de IP de interface de rede de uma interface de rede do Azure.

## EXEMPLOS

### 1: excluir uma configuração de IP de uma interface de rede
```
$nic = Get-AzureRmNetworkInterface -Name mynic -ResourceGroupName myrg

Remove-AzureRmNetworkInterfaceIpConfig -Name IPConfig-1 -NetworkInterface $nic
```

O primeiro comando obtém uma interface de rede chamada mynic e a armazena na variável $nic. O segundo comando Remove a configuração de IP chamada IPConfig-1 associada a essa interface de rede.
