# Remove-AzureRmPublicIpAddress

## Sinopse
Remove um endereço IP público.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmPublicIpAddress** remove um endereço IP público do Azure.

## EXEMPLOS

### 1: remover um recurso de endereço IP público
```
Remove-AzureRmPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Este comando Remove o recurso de endereço IP público chamado $publicIpName no $rgName do grupo de recursos.
