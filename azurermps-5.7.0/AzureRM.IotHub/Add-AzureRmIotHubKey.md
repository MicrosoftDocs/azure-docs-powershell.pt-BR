# Add-AzureRmIotHubKey

## Sinopse
Cria uma chave IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Add-AzureRmIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String> [-PrimaryKey <String>]
 [-SecondaryKey <String>] -Rights <PSAccessRights> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Cria uma chave para o IotHub fornecido.
KeyNames não são exclusivos e precisam ser gerenciados com cuidado.

## EXEMPLOS

### Exemplo 1 adicionar uma chave a um IotHub
```
PS C:\> Add-AzureRmIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "newkey" -PrimaryKey "primarykey" -SecondaryKey "secondarykey" -Rights RegistryRead
```

Cria uma chave chamada "myKey" para o iothub "myiothub" com permissões de RegistryRead.
