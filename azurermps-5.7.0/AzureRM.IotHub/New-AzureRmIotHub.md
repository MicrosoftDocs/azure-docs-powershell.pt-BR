# New-AzureRmIotHub

## Sinopse
Cria um novo IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmIotHub -ResourceGroupName <String> -Name <String> -SkuName <PSIotHubSku> -Units <Int64>
 -Location <String> [-Properties <PSIotHubInputProperties>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Cria um novo IotHub.
Você pode criar o IotHub com as propriedades padrão ou especificar o proerties de entrada.

## EXEMPLOS

### Exemplo 1 criar uma nova IotHub com propriedades padrão
```
PS C:\> New-AzureRmIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName "S1" -Units 1 -Location "northeurope"
```

Cria um novo IotHub chamado "myiothub" do SKU "S1", a capacidade 1 e o local "northeurope".

### Exemplo 2 crie uma nova IotHub com o MaxDeliveryCount da fila CloudtoDevice definida como 20
```
PS C:\> New-AzureRmIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub" -SkuName "S1" -Units 1 -Location "northeurope" -Properties $properties
```

Cria um novo IotHub chamado "myiothub" do SKU "S1", a capacidade 1 e o local "northeurope" com propriedades de entrada avançadas representadas por $properties.
