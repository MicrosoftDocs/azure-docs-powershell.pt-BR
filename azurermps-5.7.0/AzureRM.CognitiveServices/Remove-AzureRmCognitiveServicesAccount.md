# Remove-AzureRmCognitiveServicesAccount

## Sinopse
Exclui uma conta de serviços cognitiva.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmCognitiveServicesAccount** exclui a conta de serviços cognitivas especificada.
