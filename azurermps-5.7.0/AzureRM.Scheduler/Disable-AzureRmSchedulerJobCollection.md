# Disable-AzureRmSchedulerJobCollection

## Sinopse
Desabilita uma coleção de trabalhos.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Disable-AzureRmSchedulerJobCollection -ResourceGroupName <String> -JobCollectionName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Disable-AzureRmSchedulerJobCollection** desabilita uma coleção de trabalhos no Agendador do Azure.
