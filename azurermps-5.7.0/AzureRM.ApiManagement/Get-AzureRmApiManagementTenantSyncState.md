# Get-AzureRmApiManagementTenantSyncState

## Sinopse
Obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório do git.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmApiManagementTenantSyncState -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmApiManagementTenantSyncState** Obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório do git.

## EXEMPLOS

### Exemplo 1: obter o status da sincronização mais recente
```
PS C:\>$apimContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzureRmApiManagementTenantSyncState -Context $apimContext 
```

Esse comando obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório do git.
