# Get-AzureRmApiManagementTenantGitAccess

## Sinopse
Obtém a configuração de acesso ao git para um locatário.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmApiManagementTenantGitAccess -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmApiManagementTenantGitAccess** Obtém a configuração de acesso ao git para um locatário.

## EXEMPLOS

### Exemplo 1: obter configuração de acesso ao locatário
```
PS C:\>$apimContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzureRmApiManagementTenantGitAccess -Context $apimContext 
```

Este comando obtém a configuração de acesso ao git para o contexto especificado.
