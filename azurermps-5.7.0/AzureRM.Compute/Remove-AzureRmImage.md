# Remove-AzureRmImage

## Sinopse
Remove uma imagem.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmImage [-ResourceGroupName] <String> [-ImageName] <String> [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmImage** remove uma imagem..

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzureRmImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' -Force;
```

Este comando Remove a imagem chamada "Image01" no grupo de recursos "ResourceGroup01".
