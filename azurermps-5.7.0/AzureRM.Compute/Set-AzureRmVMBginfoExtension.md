# Set-AzureRmVMBginfoExtension

## Sinopse
Adiciona a extensão do BGInfo a uma máquina virtual.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmVMBginfoExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name <String>]
 [-TypeHandlerVersion <String>] [-Location <String>] [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmVMBGInfoExtension** adiciona a extensão BGInfo a uma máquina virtual.

## EXEMPLOS

### Exemplo 1: adicionar a extensão do BGInfo a uma máquina virtual
```
PS C:\> Set-AzureRmVMBgInfoExtension -ResourceGroupName "ContosoRG" -VMName "ContosoVM"
```
Esse comando adiciona a extensão BGInfo a uma máquina virtual nomeada ContosoVM na ContosoRG do grupo de recursos.

### Exemplo 2: adicionar uma versão específica da extensão do BGInfo a uma máquina virtual
```
PS C:\> Set-AzureRmVMBgInfoExtension -ResourceGroupName "ContosoRG" -VMName "ContosoVM" -Name "ExtensionName" -TypeHandlerVersion "2.1" -Location "West Europe"
```

Esse comando adiciona a extensão BGInfo à máquina virtual chamada ContosoVM.
O comando especifica o grupo de recursos e o local da máquina virtual.
O comando especifica o nome e a versão da extensão.
