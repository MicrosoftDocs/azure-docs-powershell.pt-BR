# Remove-AzureRmDataLakeStoreFirewallRule

## Sinopse
Remove a regra de firewall especificada no repositório data Lake especificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmDataLakeStoreFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmDataLakeStoreFirewallRule** remove a regra de firewall especificada no repositório data Lake especificado.

## EXEMPLOS

### Exemplo 1: remover uma regra de firewall de uma conta
```
PS C:\> Remove-AzureRmDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule
```

Remove a regra de firewall "MyFirewallRule" da conta "ContosoADL"
