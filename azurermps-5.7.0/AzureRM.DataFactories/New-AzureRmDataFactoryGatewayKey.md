# New-AzureRmDataFactoryGatewayKey

## Sinopse
Cria uma chave de gateway para um Azure data Factory. Esse cmdlet é preterido e você deve usar **New-AzureRmDataFactoryGatewayAuthKey** em vez disso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### ByFactoryName (padrão)
```
New-AzureRmDataFactoryGatewayKey [-DataFactoryName] <String> [-GatewayName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
New-AzureRmDataFactoryGatewayKey [-DataFactory] <PSDataFactory> [-GatewayName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmDataFactoryGatewayKey** cria uma chave de gateway para um gateway de fábrica de dados do Azure especificado.
Você registra o gateway com um serviço na nuvem usando essa chave. Esse cmdlet é preterido e você deve usar **New-AzureRmDataFactoryGatewayAuthKey** em vez disso.

## EXEMPLOS

### Exemplo 1: criar uma chave de gateway
```
PS C:\>New-AzureRmDataFactoryGatewayKey -ResourceGroupName "ADF" -GatewayName "ContosoGateway" -DataFactoryName "WikiADF" | Format-List
GatewayKey : ADF#40cbb3d9-2736-4794-a8a6-e6b839b4894f@a2d875ce-c9d7-4b8b-ad65-dd3ebbb9a940@8c0d1801-e863-44af-82e6-fb2f0c00f2ae@xz#Y9R0NhAeH3u7wgnrJyiWj4Y/QIhH4fFilIdzZgwsVQA=
```

Esse comando cria uma chave de gateway para o gateway de fábrica de dados chamado ContosoGateway e, em seguida, passa a chave de gateway para o cmdlet Format-List usando o operador pipeline.
Para obter mais informações, digite `Get-Help Format-List` .
