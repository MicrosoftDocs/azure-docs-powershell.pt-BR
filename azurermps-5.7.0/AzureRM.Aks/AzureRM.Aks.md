# Módulo AzureRM. AKs
## Descritivo
Comandos para interagir com os clusters gerenciados do kubernetes.

## Cmdlets AzureRM. AKs
### [Get-AzureRmAks](Get-AzureRmAks.md)
Liste os clusters gerenciados do kubernetes.

### [Import-AzureRmAksCredential](Import-AzureRmAksCredential.md)
Importar e mesclar a configuração do Kubectl para um cluster kubernetes gerenciado.

### [New-AzureRmAks](New-AzureRmAks.md)
Crie um novo cluster gerenciado kubernetes.

### [Remove-AzureRmAks](Remove-AzureRmAks.md)
Excluir um cluster do kubernetes gerenciado.

### [Set-AzureRmAks](Set-AzureRmAks.md)
Atualize ou crie um cluster do kubernetes gerenciado.

### [Start-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
Crie um encapsulamento SSH Kubectl para o painel do cluster gerenciado.

### [Parar-AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
Interrompa o encapsulamento SSH Kubectl criado em Start-AzureRmKubernetesDashboard.
