# Módulo AzureRM. LocationBasedServices
## Descritivo
Este tópico exibe ajuda para os cmdlets de serviços baseados no Azure Location.

## Cmdlets AzureRM. LocationBasedServices
### [Get-AzureRmLocationBasedServicesAccount](Get-AzureRmLocationBasedServicesAccount.md)
Obtém a conta.

### [Get-AzureRmLocationBasedServicesAccountKey](Get-AzureRmLocationBasedServicesAccountKey.md)
Obtém as chaves de API para uma conta. Essas chaves são o mecanismo de autenticação usado em chamadas subsequentes para serviços baseados no Azure Location.

### [New-AzureRmLocationBasedServicesAccount](New-AzureRmLocationBasedServicesAccount.md)
Cria uma conta de serviços com base em local.

### [New-AzureRmLocationBasedServicesAccountKey](New-AzureRmLocationBasedServicesAccountKey.md)
Regenera a chave de uma conta.

### [Remove-AzureRmLocationBasedServicesAccount](Remove-AzureRmLocationBasedServicesAccount.md)
Exclui uma conta de serviços com base em localização.
