# Módulo AzureRM. ManagementPartner
## Descritivo
Cmdlets de parceiros de gerenciamento para o Azure Resource Manager.

## Cmdlets AzureRM. ManagementPartner
### [Get-AzureRmManagementPartner](Get-AzureRmManagementPartner.md)
Obtém a ID da MPN (Microsoft Partner Network) do usuário ou entidade de serviço atual autenticado. 

### [New-AzureRmManagementPartner](New-AzureRmManagementPartner.md)
Associa uma ID do Microsoft Partner Network (MPN) ao usuário ou entidade de serviço atual autenticado.

### [Remove-AzureRmManagementPartner](Remove-AzureRmManagementPartner.md)
Exclua a ID da MPN (Microsoft Partner Network) do usuário ou entidade de serviço atual autenticado.

### [Update-AzureRmManagementPartner](Update-AzureRmManagementPartner.md)
Atualiza a ID da MPN (Microsoft Partner Network) do usuário ou entidade de serviço atual autenticado.
