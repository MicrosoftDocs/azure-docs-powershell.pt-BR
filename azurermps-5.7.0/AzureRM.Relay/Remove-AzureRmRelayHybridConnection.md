# Remove-AzureRmRelayHybridConnection

## Sinopse
Remove o HybridConnection do namespace HybridConnection especificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmRelayHybridConnection** remove o HybridConnection do namespace de retransmissão especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Remove o HybridConnection `TestHybridConnection` do namespace `TestNameSpace-Relay1` .
