# New-AzureRmUserAssignedIdentity

## Sinopse
Cria uma nova identidade atribuída ao usuário ou atualiza uma identidade atribuída pelo usuário existente.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmUserAssignedIdentity** cria uma nova identidade atribuída ao usuário. Quando usado com uma identidade já existente, ele atualizou a identidade.
Para adicionar marcas do Azure Resource Manager à identidade, use o cmdlet Set-AzureRmResource.
