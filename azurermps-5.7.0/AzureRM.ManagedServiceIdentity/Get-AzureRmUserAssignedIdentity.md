# Get-AzureRmUserAssignedIdentity

## Sinopse
Obtém identidade/identidade atribuídas pelo usuário.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### SuscriptionParameterSet (padrão)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
**Get-AzureRmUserAssignedIdentity** Obtém identidades atribuídas pelo usuário existente.
