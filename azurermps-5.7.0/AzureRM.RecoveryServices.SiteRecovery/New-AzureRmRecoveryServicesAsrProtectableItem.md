# New-AzureRmRecoveryServicesAsrProtectableItem

## Sinopse
Adicione (descubra) um servidor físico à lista de itens que protegem.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer>
 -FriendlyName <String> -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O **New-AzureRmRecoveryServicesAsrProtectableItem** adiciona um novo item de proteção à lista de itens que protegem os itens que protegem em um contêiner de proteção em uma malha ASR (aplicável somente para o tipo de malha do VMware).

## EXEMPLOS

### Exemplo 1
```
PS C:\> New-AzureRmRecoveryServicesAsrProtectableItem ProtectionContainer $pc -Name $name -IPAddress $ipaddresss -OSType $OsType
```

Adicionar ou descobrir o novo serviço de recuperação do ProtectableItem.
