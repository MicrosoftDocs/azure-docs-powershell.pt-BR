# Get-AzureRmSiteRecoveryVaultSettings

## Sinopse
Obtém informações de configurações para o cofre de recuperação do site atual.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSiteRecoveryVaultSettings** Obtém informações de configuração relacionadas ao cofre atual do Azure site Recovery.
