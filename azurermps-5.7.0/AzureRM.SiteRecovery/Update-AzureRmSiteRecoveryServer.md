# Update-AzureRmSiteRecoveryServer

## Sinopse
Atualiza um servidor de recuperação de site.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Update-AzureRmSiteRecoveryServer -Server <ASRServer> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Update-AzureRmSiteRecoveryServer** atualiza um servidor do Azure site Recovery.
