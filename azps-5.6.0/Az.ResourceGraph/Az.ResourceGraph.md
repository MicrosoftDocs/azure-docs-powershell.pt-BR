# Módulo Az.ResourceGraph
## Descrição
Microsoft Azure PowerShell: cmdlets resourceGraph

## Az.ResourceGraph Cmdlets
### [Get-AzResourceGraphQuery](Get-AzResourceGraphQuery.md)
Obter uma única consulta de gráfico pelo resourceName.

### [New-AzResourceGraphQuery](New-AzResourceGraphQuery.md)
Crie uma nova consulta gráfica.

### [Remove-AzResourceGraphQuery](Remove-AzResourceGraphQuery.md)
Exclua uma consulta gráfica.

### [Search-AzGraph](Search-AzGraph.md)
Consulta os recursos gerenciados pelo Gerenciador de Recursos do Azure.

### [Update-AzResourceGraphQuery](Update-AzResourceGraphQuery.md)
Atualiza uma consulta gráfica que já foi adicionada.
