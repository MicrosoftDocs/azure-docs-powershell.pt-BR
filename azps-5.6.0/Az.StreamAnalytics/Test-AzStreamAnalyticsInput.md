# Test-AzStreamAnalyticsInput

## SYNOPSIS
Testa o status da conexão de uma entrada.

## SINTAXE

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Test-AzStreamAnalyticsInput** testa a capacidade do Stream Analytics de se conectar a uma entrada.

## EXEMPLOS

### Exemplo 1: Testar o status de conexão de um fluxo de entrada
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Isso testa o status de conexão da entrada EntryStream no StreamingJob.
