# New-AzStreamAnalyticsFunction

## SYNOPSIS
Cria ou substitui uma função em um trabalho do Stream Analytics.

## SINTAXE

```
New-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzStreamAnalyticsFunction** cria uma função em um trabalho do Azure Stream Analytics ou substitui uma função existente.
Defina a função em um arquivo JSON (Notação de Objeto JavaScript).
Você pode especificar o nome da função usando o parâmetro *Name* ou no arquivo .json.
Se você especificar o nome de ambas as maneiras, os nomes deverão corresponder.
Para substituir uma função existente, especifique o nome da função existente.

## EXEMPLOS

### Exemplo 1: Criar uma função Stream Analytics
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

Este comando cria uma função a partir do arquivo Function07.json.
O nome da função é armazenado no arquivo .json.

### Exemplo 2: Criar uma função de Análise de Fluxo chamada ScoreTweet
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

Este comando cria uma função no trabalho chamado ScoreTweet.

### Exemplo 3: Substituir uma função de Análise de Fluxo
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

Este comando substitui a definição da função existente chamada ScoreTweet pela definição em Function22.json.
