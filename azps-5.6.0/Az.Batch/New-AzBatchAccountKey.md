# New-AzBatchAccountKey

## SYNOPSIS
Regenera uma chave de uma conta Batch.

## SINTAXE

```
New-AzBatchAccountKey [-AccountName] <String> [-ResourceGroupName <String>] -KeyType <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzBatchAccountKey** regenera a chave primária ou secundária de uma conta batch do Azure.
O cmdlet retorna um **objeto BatchAccountContext** que tem suas propriedades **PrimaryAccountKey** e **SecondaryAccountKey** atuais.

## EXEMPLOS

### Exemplo 1: Regenerar a chave da conta primária em uma conta Batch
```
PS C:\>New-AzBatchAccountKey -AccountName "pfuller" -KeyType "Primary"
AccountName                  : pfuller
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Esse comando regenera a chave da conta primária na conta Batch chamada pfuller.
