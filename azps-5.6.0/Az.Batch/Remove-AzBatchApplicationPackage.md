# Remove-AzBatchApplicationPackage

## SYNOPSIS
Exclui um registro de pacote de aplicativo e o arquivo binário.

## SINTAXE

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzBatchApplicationPackage** exclui um registro de pacote de aplicativo e o arquivo binário de uma conta do Lote do Azure.

## EXEMPLOS

### Exemplo 1: Excluir um pacote de aplicativo de uma conta batch
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

Este comando exclui a versão 1.0 do aplicativo Litware da conta ContosoBatchGroup.
O comando exclui o registro do pacote e o blob que contém o arquivo binário do pacote.
