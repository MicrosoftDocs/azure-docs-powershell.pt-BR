# Remove-AzBatchAccount

## SYNOPSIS
Remove uma conta Batch.

## SINTAXE

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzBatchAccount** remove uma conta do Lote do Azure.
Este cmdlet solicita que você remova uma conta, a menos que você especifique o *parâmetro Force.*

## EXEMPLOS

### Exemplo 1: Remover uma conta em lote
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

Este comando remove a conta Batch chamada pfuller.
Este comando solicita a confirmação antes de excluir a conta.
