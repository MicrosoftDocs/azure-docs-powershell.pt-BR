# Update-AzServiceFabricReliability

## SYNOPSIS
Atualize a camada de confiabilidade do tipo de nó principal em um cluster.

## SINTAXE

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Use **Update-AzServiceFabricReliability** para atualizar a confiabilidade do tipo de nó principal em um cluster.

## EXEMPLOS

### Exemplo 1
```powershell
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Este comando altera a camada de confiabilidade do tipo de nó principal para silver.

### Exemplo 2

Atualize a camada de confiabilidade do tipo de nó principal em um cluster. (gerado automaticamente)

<!-- Aladdin Generated Example -->
```powershell
Update-AzServiceFabricReliability -AutoAddNode -Name 'Contoso01SFCluster' -ReliabilityLevel None -ResourceGroupName 'Group1'
```
