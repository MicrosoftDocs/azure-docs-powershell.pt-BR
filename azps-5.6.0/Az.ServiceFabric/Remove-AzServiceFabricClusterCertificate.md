# Remove-AzServiceFabricClusterCertificate

## SYNOPSIS
Remova um certificado de cluster de ser usado para segurança de cluster.

## SINTAXE

```
Remove-AzServiceFabricClusterCertificate -Thumbprint <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Use **Remove-AzServiceFabricClusterCertificate** para remover um certificado de cluster do cluster, desde que haja outro certificado válido que já está em uso no cluster.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzServiceFabricClusterCertificate -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -Thumbprint '5F3660C715EBBDA31DB1FFDCF508302348DE8E7A
```

Este comando remove o certificado com a impressão digital 5F3660C715EBBDA31DB1FFDCF508302348DE8E7A de ser usada para segurança de cluster.
