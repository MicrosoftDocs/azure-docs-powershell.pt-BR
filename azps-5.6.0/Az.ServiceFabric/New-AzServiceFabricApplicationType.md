# New-AzServiceFabricApplicationType

## SYNOPSIS
Crie um novo tipo de aplicativo de malha de serviço no grupo de recursos especificado e cluster.

## SINTAXE

```
New-AzServiceFabricApplicationType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet cria um novo tipo de aplicativo de malha de serviço sob o grupo de recursos especificado e o cluster.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $resourceGroupName = "testRG"
PS C:\> $clusterName = "testCluster"
PS C:\> $appTypeName = "testAppType"
PS C:\> $appType = New-AzServiceFabricApplicationType -ResourceGroupName $resourceGroupName -ClusterName $clusterName -Name $appTypeName -Verbose
```

Este exemplo criará um novo tipo de aplicativo "testAppType" no grupo de recursos e cluster especificados.
