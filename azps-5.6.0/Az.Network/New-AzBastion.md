## DESCRIPTION
Cria um recurso de bastião. Isso precisará de um Endereço Ip Público e uma VirtualNetwork. Deve haver uma sub-rede com o nome AzureBastionSubnet nesta VirtualNetwork.O Endereço Ip Púbico deve ser criado com o padrão Sku.
