# Remove-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
Remove mapeamentos de caminho de URL para um pool de servidores de back-end.

## SINTAXE

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayUrlPathMapConfig** remove mapeamentos de caminho de URL para um pool de servidores back-end.

## EXEMPLOS

### Exemplo 1: Remover um mapeamento de caminho de URL de um gateway de aplicativo
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway $appgw -Name "map01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

O primeiro comando obtém o gateway de aplicativo chamado appGwName e armazena o resultado na $appgw variável.
O segundo comando remove o mapeamento de caminho da URL chamado map01 do gateway de aplicativo.
O terceiro comando atualiza o gateway de aplicativo.
