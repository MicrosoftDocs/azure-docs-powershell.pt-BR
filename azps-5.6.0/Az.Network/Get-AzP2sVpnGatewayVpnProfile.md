# Get-AzP2sVpnGatewayVpnProfile

## SYNOPSIS
Gera e retorna uma URL do SAS para o cliente baixar o perfil vpn para a configuração de cliente ponto a site para ter conectividade ponto a site com P2SVpnGateway.

## SINTAXE

### ByP2SVpnGatewayName (Padrão)
```
Get-AzP2sVpnGatewayVpnProfile [-Name <String>] -ResourceGroupName <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Get-AzP2sVpnGatewayVpnProfile -InputObject <PSP2SVpnGateway> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Get-AzP2sVpnGatewayVpnProfile -ResourceId <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzP2sVpnGatewayVpnProfile** permite que você gere e receba uma URL do SAS para o cliente baixar o perfil vpn para a configuração de cliente ponto a site para ter conectividade ponto a site com p2SVpnGateway. Apontar para a instalação do cliente do site usando esse perfil vpn pode se conectar somente a esse P2SVpnGateway específico.
