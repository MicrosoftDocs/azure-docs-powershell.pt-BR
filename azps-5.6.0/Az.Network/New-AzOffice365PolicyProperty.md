# New-AzOffice365PolicyProperty

## SYNOPSIS
Defina uma nova política de quebra de tráfego do Office 365 a ser usada com um site de Dispositivo Virtual.

## SINTAXE

```
New-AzOffice365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O New-AzOffice365PolicyProperties define uma política de quebra do Office 365 que deve ser usada com um site de Dispositivo Virtual. 

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $o365Policy = New-AzOffice365PolicyProperty -Allow -Optimize 
```

Crie o objeto de política de quebra de tráfego do Office 365 a ser usado com comandos de site do Dispositivo Virtual.
