# Get-AzExpressRouteCircuitRouteTableSummary

## SYNOPSIS
Obtém um resumo da tabela de rota de um circuito ExpressRoute.

## SINTAXE

```
Get-AzExpressRouteCircuitRouteTableSummary -ResourceGroupName <String> -ExpressRouteCircuitName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzExpressRouteCircuitRouteTableSummary** recupera um resumo das informações do vizinho BGP para um contexto de roteamento específico. Essas informações são úteis para determinar por quanto tempo um contexto de roteamento foi estabelecido e o número de prefixos de rota anunciados pelo roteador de paração.

## EXEMPLOS

### Exemplo 1: Exibir o resumo da rota para o caminho principal
```
Get-AzExpressRouteCircuitRouteTableSummary -ResourceGroupName $RG -ExpressRouteCircuitName $CircuitName -DevicePath 'Primary'
```
