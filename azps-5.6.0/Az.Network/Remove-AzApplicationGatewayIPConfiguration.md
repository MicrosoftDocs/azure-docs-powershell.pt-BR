# Remove-AzApplicationGatewayIPConfiguration

## SYNOPSIS
Remove uma configuração IP de um gateway de aplicativo.

## SINTAXE

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayIPConfiguration** remove uma configuração IP de um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: Remover uma configuração IP de um gateway de aplicativo do Azure
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na $AppGw variável.
O segundo comando remove a configuração IP denominada Subnet02 do gateway de aplicativo armazenado em $AppGw.
