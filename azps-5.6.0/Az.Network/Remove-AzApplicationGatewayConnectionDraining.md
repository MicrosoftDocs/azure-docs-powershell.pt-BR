# Remove-AzApplicationGatewayConnectionDraining

## SYNOPSIS
Remove a configuração de esvaziamento de conexão de um objeto de configurações HTTP de back-end.

## SINTAXE

```
Remove-AzApplicationGatewayConnectionDraining -BackendHttpSettings <PSApplicationGatewayBackendHttpSettings>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayConnectionDraining** remove a configuração de esvaziamento de conexão de um objeto de configurações HTTP back-end.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Settings  = Get-AzApplicationGatewayBackendHttpSettings -Name "Settings01" -ApplicationGateway $AppGw
PS C:\> Remove-AzApplicationGatewayConnectionDraining -BackendHttpSettings $Settings
```

O primeiro comando obtém o gateway de aplicativo chamado ApplicationGateway01 no grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando obtém as configurações HTTP de back-end denominadas Settings01 para $AppGw e armazena as configurações na variável $Settings back-end.
O último comando remove a configuração de esvaziamento de conexão das configurações HTTP de back-end armazenadas $Settings.
