# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
Cria uma porta front-end para um gateway de aplicativo.

## SINTAXE

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzApplicationGatewayFrontendPort** cria uma porta front-end para um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: Exemplo1: Criar uma porta front-end
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

Este comando cria uma porta front-end chamada FrontEndPort01 na porta 80 e armazena o resultado na variável chamada $FrontEndPort.
