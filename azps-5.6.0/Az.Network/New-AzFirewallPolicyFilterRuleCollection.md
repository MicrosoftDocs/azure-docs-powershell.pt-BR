# New-AzFirewallPolicyFilterRuleCollection

## SYNOPSIS
Criar uma nova coleção de regras de filtro de política de firewall do Azure

## SINTAXE

```
New-AzFirewallPolicyFilterRuleCollection -Name <String> -Priority <UInt32> -Rule <PSAzureFirewallPolicyRule[]>
 -ActionType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzFirewallPolicyFilterRuleCollection** cria uma coleção de regras filter para uma Política de Firewall do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzFirewallPolicyFilterRuleCollection -Name FR1 -Priority 400 -Rule $appRule1 ,$appRule2 -ActionType "Allow"
```

Este exemplo cria uma regra Filter com 2 condições de regra
