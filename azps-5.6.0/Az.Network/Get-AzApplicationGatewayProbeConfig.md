# Get-AzApplicationGatewayProbeConfig

## SYNOPSIS
Obtém uma configuração de sonda de saúde existente de um Gateway de Aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayProbeConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzApplicationGatewayProbeConfig cmdlet obtém uma configuração de sonda de saúde existente de um Gateway de Aplicativo.

## EXEMPLOS

### Exemplo 1: Obter uma sonda existente de um gateway de aplicativo
```
PS C:\>Get-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe02"
```

Este comando obtém a sonda de saúde chamada Probe02 do gateway de aplicativo chamado Gateway.
