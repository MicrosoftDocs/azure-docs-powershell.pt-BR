# Remove-AzExpressRoutePort

## SYNOPSIS
Remove um ExpressRoutePort.

## SINTAXE

### ResourceNameParameterSet (Padrão)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzExpressRoutePort** remove um ExpressRoutePort.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Remove $PortName recurso ExpressRoutePort no $rg de recursos em sua assinatura.

### Exemplo 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Remove o recurso ExpressRoutePort em InputObject.

### Exemplo 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Remove o recurso ExpressRoutePort com ResourceId $id.
