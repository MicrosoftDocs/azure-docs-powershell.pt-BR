# Get-AzNetworkUsage

## SYNOPSIS
Lista os usos de rede para uma assinatura

## SINTAXE

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzNetworkUsage cmdlet obtém limites e uso atual para recursos de rede.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzNetworkUsage -Location westcentralus

CurrentValue : 6
Limit        : 50

CurrentValue : 1
Limit        : 20

ResourceType : Network Security Groups
CurrentValue : 2
Limit        : 100

ResourceType : Public IP Addresses

Limit        : 60

ResourceType : Network Interfaces
CurrentValue : 1
Limit        : 300

CurrentValue : 1
Limit        : 100

CurrentValue : 0
Limit        : 1000
```

Obtém dados de uso de recursos na região westcentralus
