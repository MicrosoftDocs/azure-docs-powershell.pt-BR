# Get-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Obtém a Configuração de Escala Automática do Gateway de Aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApplicationGatewayAutoscaleConfiguration obtém** Configuração de Escala Automática do Gateway de Aplicativos.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

O primeiro comando obtém o gateway do aplicativo e o armazena $gw variável.
O segundo comando extrai a configuração de escala automática do gateway de aplicativo.
