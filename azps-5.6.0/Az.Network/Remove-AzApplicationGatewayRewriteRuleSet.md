# Remove-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Remove um conjunto de regras de regra de regra de regra de um gateway de aplicativo.

## SINTAXE

```
Remove-AzApplicationGatewayRewriteRuleSet -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayRewriteRuleSet** remove um conjunto de regras de reescrita de um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "RuleSet02"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na $AppGw variável.
O segundo comando remove o conjunto de regras de regra de reescrita denominado RuleSet02 do gateway de aplicativo armazenado em $AppGw.
