# Get-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Obtém uma configuração de regra NAT de entrada para um balanceador de carga.

## SINTAXE

```
Get-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzLoadBalancerInboundNatRuleConfig** obtém uma ou mais regras de conversão de endereço de rede de entrada (NAT) em um balanceador de carga do Azure.

## EXEMPLOS

### Exemplo 1: Obter uma configuração de regra NAT de entrada
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerInboundNatRuleConfig -Name "MyInboundNatRule1" -LoadBalancer $slb
```

O primeiro comando obtém o balanceador de carga chamado MyLoadBalancer e o armazena na variável $slb.
O segundo comando obtém a regra NAT associada chamada MyInboundNatRule1 do balanceador de carga no $slb.
