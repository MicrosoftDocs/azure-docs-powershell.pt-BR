# Add-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Adiciona uma cadeia de certificados de AC de cliente confiável a um gateway de aplicativo.

## SINTAXE

```
Add-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Add-AzApplicationGatewayTrustedClientCertificate** adiciona uma cadeia de certificados DE cliente confiável a um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClient = New-AzApplicationGatewayTrustedClientCertificate -Name "ClientCert" -CertificateFile "C:\clientCAChain.cer"
PS C:\> $AppGw = Add-AzApplicationGatewaySslProfile -Name $sslProfileName -ApplicationGateway $gw -TrustedClientCertificates $trustedClient
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

O primeiro comando obtém o gateway do aplicativo e o armazena $gw variável. O segundo comando cria uma nova cadeia de certificados de AC de cliente confiável que está tomando o caminho do certificado ca do cliente como entrada. O terceiro comando cria um perfil SSL usando certificado de cliente confiável. O quarto comando atualiza o Gateway de Aplicativos.
