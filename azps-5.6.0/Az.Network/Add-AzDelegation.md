# Add-AzDelegation

## SYNOPSIS
Adiciona uma delegação a uma sub-rede.

## SINTAXE

```
Add-AzDelegation -Name <String> -ServiceName <String> -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Add-AzDelegation** adiciona uma delegação de serviço a uma sub-rede do Azure.

## EXEMPLOS

### Exemplo 1: Adicionando uma delegação
```powershell
PS C:\> $vnet = Get-AzVirtualNetwork -Name "myVNet" -ResourceGroupName "myResourceGroup"
PS C:\> $subnet = Get-AzVirtualNetworkSubnetConfig -Name "mySubnet" -VirtualNetwork $vnet
PS C:\> $subnet = Add-AzDelegation -Name "myDelegation" -ServiceName "Microsoft.Sql/servers" -Subnet $subnet
PS C:\> Set-AzVirtualNetwork $vnet
```

O primeiro comando recupera a rede virtual na qual a sub-rede está. O segundo comando isola a sub-rede de interesse - a que você deseja delegar. O terceiro comando adiciona uma delegação à sub-rede. Este exemplo específico seria útil quando você deseja habilitar SQL criar pontos de extremidade da interface nesta sub-rede. O comando final envia a sub-rede atualizada para o servidor para atualizar sua sub-rede.
