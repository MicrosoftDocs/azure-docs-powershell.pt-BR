# Get-AzRouteTable

## SYNOPSIS
Obtém tabelas de rota.

## SINTAXE

### NoExpand (Padrão)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Expandir
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzRouteTable** obtém tabelas de rota do Azure.
Você pode obter uma tabela de rota única ou obter todas as tabelas de rota em um grupo de recursos ou em sua assinatura.
