# Get-AzLoadBalancerInboundNatPoolConfig

## SYNOPSIS
Obtém uma ou mais configurações de pool NAT de entrada de um balanceador de carga.

## SINTAXE

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzLoadBalancerInboundNatPoolConfig** obtém uma ou mais configurações de pool NAT de entrada de um balanceador de carga.

## EXEMPLOS

### 1: Obter
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
