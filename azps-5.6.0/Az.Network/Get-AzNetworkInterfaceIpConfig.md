# Get-AzNetworkInterfaceIpConfig

## SYNOPSIS
Obtém uma configuração IP de interface de rede para uma interface de rede.

## SINTAXE

```
Get-AzNetworkInterfaceIpConfig [-Name <String>] -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzNetworkInterfaceIPConfig obtém** uma configuração IP de interface de rede de uma interface de rede do Azure.

## EXEMPLOS

### 1: Obter uma configuração IP de uma interface de rede
```
$nic1 = Get-AzNetworkInterface -Name mynic -ResourceGroupName $myrg
Get-AzNetworkInterfaceIpConfig -Name ipconfig1 -NetworkInterface $nic1
```

O primeiro comando obtém uma interface de rede existente chamada mynic e a armazena na variável $nic 1. O segundo comando obtém a configuração IP chamada ipconfig1 dessa interface de rede.
