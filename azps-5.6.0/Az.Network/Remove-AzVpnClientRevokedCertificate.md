# Remove-AzVpnClientRevokedCertificate

## SYNOPSIS
Remove um certificado de revogação de cliente VPN.

## SINTAXE

```
Remove-AzVpnClientRevokedCertificate -VpnClientRevokedCertificateName <String>
 -VirtualNetworkGatewayName <String> -ResourceGroupName <String> -Thumbprint <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzVpnClientRevokedCertificate** remove um certificado de revogação de cliente de um gateway de rede virtual.
Certificados de revogação de cliente impedem que os computadores cliente usem o certificado especificado para autenticação.
Se você remover um certificado de revogação do cliente, os computadores clientes poderão usar o certificado proibido anteriormente para fazer uma conexão vpn (rede virtual privada).

## EXEMPLOS

### Exemplo 1: Remover um certificado de revogação de cliente de um gateway de rede virtual
```powershell
PS C:\>Remove-AzVpnClientRevokedCertificate -VirtualNetworkGatewayName "ContosoVirtualNetwork" -ResourceGroupName"ContosoResourceGroup" -VpnClientRevokedCertificateName "ContosoRevokedClientCertificate" -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3"
```

Este comando remove um certificado de revogação de cliente de um gateway de rede virtual chamado ContosoVirtualNetwork.
Para remover um certificado de revogação de cliente, você deve especificar o nome do certificado e a impressão digital do certificado.
