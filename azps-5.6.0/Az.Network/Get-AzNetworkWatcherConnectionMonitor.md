# Get-AzNetworkWatcherConnectionMonitor

## SYNOPSIS
Retorna o monitor de conexão com o nome especificado ou a lista de monitores de conexão

## SINTAXE

### SetByName (Padrão)
```
Get-AzNetworkWatcherConnectionMonitor -NetworkWatcherName <String> -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Get-AzNetworkWatcherConnectionMonitor -NetworkWatcher <PSNetworkWatcher> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByLocation
```
Get-AzNetworkWatcherConnectionMonitor -Location <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResourceId
```
Get-AzNetworkWatcherConnectionMonitor -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O Get-AzNetworkWatcherConnectionMonitor cmdlet retorna o monitor de conexão com o nome/resourceId especificado ou a lista de monitores de conexão correspondentes ao watcher/local de rede especificado.
