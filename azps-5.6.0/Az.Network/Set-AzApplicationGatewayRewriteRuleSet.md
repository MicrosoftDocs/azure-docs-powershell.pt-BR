# Set-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Modifica um conjunto de regras de regra de reescrita para um gateway de aplicativo.

## SINTAXE

```
Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway <PSApplicationGateway> -Name <String>
 -RewriteRule <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSApplicationGatewayRewriteRule]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzApplicationGatewayRewriteRuleSet** modifica uma regra de roteamento de solicitação.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "ruleset1" -RewriteRule $rule
```

O primeiro comando obtém o gateway de aplicativo chamado ApplicationGateway01 e o armazena na variável $AppGw de aplicativos.
O segundo comando modifica o conjunto de regras de regra de regra de reescrita para o gateway de aplicativo usar regras de reescrita especificadas na variável $rule de reescrita.
