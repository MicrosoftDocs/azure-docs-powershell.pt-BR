# Get-AzVpnClientIpsecParameter

## SYNOPSIS
Obtém os parâmetros ipsec vpn definidos no Virtual Network Gateway for Point to site connections.

## SINTAXE

```
Get-AzVpnClientIpsecParameter [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Gateway de Rede Virtual é o objeto que representa seu gateway no Azure.
O cmdlet **Get-AzVpnClientIpsecParameter** retorna o objeto de seus parâmetros ipsec vpn definidos no gateway no Azure com base no Nome do Gateway e no Nome do Grupo de Recursos.

## EXEMPLOS

### Exemplo 1: obtém os parâmetros ipsec vpn definidos no Virtual Network Gateway for Point to site connections.
```powershell
PS C:\> $VpnClientIPsecParameters = Get-AzVpnClientIpsecParameter -Name myGateway -ResourceGroupName myRG
```

Retorna o objeto dos parâmetros ipsec vpn definidos no Gateway de Rede Virtual com o nome "myGateway" dentro do grupo de recursos "myRG"
