# Get-AzVirtualNetworkTap

## SYNOPSIS
Obtém um toque de rede virtual

## SINTAXE

### ListParameterSet (Padrão)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzVirtualNetworkTap** obtém um toque de rede virtual do Azure ou uma lista de toques de rede virtual do Azure em um grupo de recursos.

## EXEMPLOS

### Exemplo 1: Obter um toque de rede virtual
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Este comando obtém uma referência de toque virtualNetwork para determinado "VirtualTap1" em "ResourceGroup1".

### Exemplo 2: Obter todos os toques de rede virtual usando filtragem
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Este comando obtém todas as referências de toque virtualNetwork que começam com "VirtualTap".
