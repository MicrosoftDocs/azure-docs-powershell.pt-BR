# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## SYNOPSIS
Este commandlet retorna uma lista de marcas de dispositivos VPN com suporte, modelos e versões de firmware.

## SINTAXE

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Este commandlet retorna uma lista de marcas de dispositivos VPN com suporte, modelos e versões de firmware.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

Retorna a lista de marcas de dispositivos VPN com suporte, modelos e versões de firmware:
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
