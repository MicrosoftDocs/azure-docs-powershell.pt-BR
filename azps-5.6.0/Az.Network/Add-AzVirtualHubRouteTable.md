# Add-AzVirtualHubRouteTable

## SYNOPSIS
Cria um recurso de Tabela de Rota de Hub Virtual que é filho do VirtualHub.

## SINTAXE

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O recurso de tabela de rota de hub virtual contém a lista de rotas e uma lista de conexões às quais ele pode ser anexado e é usado para rotear o tráfego em um Hub Virtual.
