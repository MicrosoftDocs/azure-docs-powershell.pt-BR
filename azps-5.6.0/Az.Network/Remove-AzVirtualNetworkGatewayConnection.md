# Remove-AzVirtualNetworkGatewayConnection

## SYNOPSIS
Exclui uma conexão de gateway de rede virtual

## SINTAXE

```
Remove-AzVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
A Conexão de Gateway de Rede Virtual é o objeto que representa o túnel IPsec (Site para Site ou Vnet-para-Vnet) conectado ao Gateway de Rede Virtual no Azure.
O cmdlet **Remove-AzVirtualNetworkGatewayConnection** exclui o objeto de sua conexão com base em Nome e Nome do Grupo de Recursos.

## EXEMPLOS

### Exemplo 1: Excluir uma conexão de gateway de rede virtual
```powershell
Remove-AzVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

Exclui o objeto da Conexão de Gateway de Rede Virtual com o nome "myTunnel" dentro do grupo de recursos "myRG"
