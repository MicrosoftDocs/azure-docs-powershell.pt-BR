# Remove-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Remove o objeto de cadeia de certificados ca do cliente confiável de um gateway de aplicativo.

## SINTAXE

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayTrustedClientCertificate** remove o objeto de cadeia de certificados de CA do cliente confiável de um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na $gw variável. O segundo comando remove a cadeia de certificados de AC de cliente confiável chamada "TrustedClientCertificate01" do gateway de aplicativo armazenado em $gw.
