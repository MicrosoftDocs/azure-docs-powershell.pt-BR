# Get-AzVirtualNetworkGatewayLearnedRoute

## SYNOPSIS
Lista as rotas aprendidas por um gateway de rede virtual do Azure

## SINTAXE

```
Get-AzVirtualNetworkGatewayLearnedRoute -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Enumera rotas aprendidas por um gateway de rede virtual do Azure de várias fontes. Isso inclui rotas aprendidas sobre BGP, bem como rotas estáticas. 

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzVirtualNetworkGatewayLearnedRoute -ResourceGroupName resourceGroup -VirtualNetworkGatewayname gatewayName

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.1.0.0/16
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       :
LocalAddress : 10.1.0.254
Network      : 10.0.0.254/32
NextHop      :
Origin       : Network
SourcePeer   : 10.1.0.254
Weight       : 32768

AsPath       : 65515
LocalAddress : 10.1.0.254
Network      : 10.0.0.0/16
NextHop      : 10.0.0.254
Origin       : EBgp
SourcePeer   : 10.0.0.254
Weight       : 32768
```

Para o gateway de rede virtual do Azure chamado gatewayname no resourceGroup do grupo de recursos, recupera rotas que o gateway do Azure conhece. O gateway de rede virtual do Azure nesse caso tem duas rotas estáticas (10.1.0.0/16 e 10.0.0.254/32), bem como uma rota aprendida sobre bgp (10.0.0.0/16).
