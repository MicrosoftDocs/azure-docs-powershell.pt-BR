# New-AzFirewallHubPublicIpAddress

## SYNOPSIS
Ip público assoicado ao firewall no hub virtual

## SINTAXE

```
New-AzFirewallHubPublicIpAddress [-Count <Int32>] [-Addresses <PSAzureFirewallPublicIpAddress[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Ip público assoicado ao firewall no hub virtual

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzFirewallHubPublicIpAddress -Count 2
```

Isso criará dois ips públicos no firewall anexado ao hub virtual. Isso criará o endereço ip no back-end. Não podemos fornecer os ipaddresses explicitamente para um novo firewall.

### Exemplo 2
```powershell
PS C:\> $publicIp1 = New-AzFirewallPublicIpAddress -Address 10.2.3.4
PS C:\> $publicIp2 = New-AzFirewallPublicIpAddress -Address 20.56.37.46
PS C:\> New-AzFirewallHubPublicIpAddress -Count 3 -Addresses $publicIp1, $publicIp2
```

Isso criará um novo ip público no firewall mantendo $publicIp 1, $publicIp 2 que já existem no firewall.
