# Add-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Adiciona uma configuração de regra NAT de entrada a um balanceador de carga.

## SINTAXE

### SetByResource (Padrão)
```
Add-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfiguration <PSFrontendIPConfiguration>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByResourceId
```
Add-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>]
 [-FrontendPort <Int32>] [-BackendPort <Int32>] [-IdleTimeoutInMinutes <Int32>] [-EnableFloatingIP]
 [-EnableTcpReset] [-FrontendIpConfigurationId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Add-AzLoadBalancerInboundNatRuleConfig** adiciona uma configuração de regra nat (conversão de endereço de rede de entrada) a um balanceador de carga do Azure.

## EXEMPLOS

### Exemplo 1: Adicionar uma configuração de regra NAT de entrada a um balanceador de carga
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerInboundNatRuleConfig -Name "NewNatRule" -FrontendIPConfiguration $slb.FrontendIpConfigurations[0] -Protocol "Tcp" -FrontendPort 3350 -BackendPort 3350 -EnableFloatingIP
PS C:\> $slb | Set-AzLoadBalancer
```

O primeiro comando obtém o balanceador de carga chamado MyloadBalancer e o armazena na variável $slb.
O segundo comando usa o operador de pipeline para passar o balanceador de carga no $slb para **Add-AzLoadBalancerInboundNatRuleConfig**, que adiciona uma configuração de regra NAT de entrada ao balanceador de carga.
O último comando define a configuração como loadbalancer, se você não executar Set-AzLoadBalancer, suas alterações não serão aplicadas ao loadbalancer.
