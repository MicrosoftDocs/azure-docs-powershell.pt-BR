# Get-AzExpressRouteCircuit

## SYNOPSIS
Obtém um circuito do Azure ExpressRoute do Azure.

## SINTAXE

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzExpressRouteCircuit** é usado para recuperar um objeto de circuito ExpressRoute da sua assinatura. O objeto circuit retornado pode ser usado como entrada para outros cmdlets que operam em circuitos ExpressRoute.
