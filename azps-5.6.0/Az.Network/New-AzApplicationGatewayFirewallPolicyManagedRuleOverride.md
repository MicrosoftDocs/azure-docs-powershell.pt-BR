# New-AzApplicationGatewayFirewallPolicyManagedRuleOverride

## SYNOPSIS
Cria uma entrada managedRuleOverride para entrada RuleGroupOverrideGroup.

## SINTAXE

```
New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId <String> [-State <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O **New-AzApplicationGatewayFirewallPolicyManagedRuleOverride** cria uma entrada ruleOverride.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> ruleOverrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleOverride -RuleId $ruleId -State Disabled
```

Cria uma entrada ruleOverride com RuleId como $ruleId e Estado como Desabilitado.
