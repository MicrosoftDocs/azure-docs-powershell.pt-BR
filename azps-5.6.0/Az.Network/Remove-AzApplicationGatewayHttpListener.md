# Remove-AzApplicationGatewayHttpListener

## SYNOPSIS
Remove um ouvinte HTTP de um gateway de aplicativo.

## SINTAXE

```
Remove-AzApplicationGatewayHttpListener -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayHttpListener** remove um ouvinte HTTP de um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: Remover um ouvinte HTTP do gateway de aplicativo
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayHttpListener -ApplicationGateway $AppGw -Name "Listener02"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na $AppGw variável.
O segundo comando remove o ouvinte HTTP chamado Listener02 do gateway de aplicativo armazenado $AppGw.
