# Get-AzExpressRouteServiceProvider

## SYNOPSIS
Obtém uma lista de provedores de serviços ExpressRoute e seus atributos.

## SINTAXE

```
Get-AzExpressRouteServiceProvider [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzExpressRouteServiceProvider** recupera uma lista de provedores de serviços ExpressRoute e seus atributos. O atributo inclui opções de localização e largura de banda.

## EXEMPLOS

### Exemplo 1: Obter uma lista de provedor de serviços com locais em "Vale do Silício"
```
Get-AzExpressRouteServiceProvider |
   Where-Object PeeringLocations -Contains "Silicon Valley" |
   Select-Object Name
```
