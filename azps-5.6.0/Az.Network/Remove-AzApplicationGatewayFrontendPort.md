# Remove-AzApplicationGatewayFrontendPort

## SYNOPSIS
Remove uma porta front-end de um gateway de aplicativo.

## SINTAXE

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApplicationGatewayFrontendPort** remove uma porta front-end de um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo: remover uma porta front-end de um gateway de aplicativo
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e armazena o gateway $AppGw variável.
O segundo comando remove a porta chamada FrontEndPort02 do gateway de aplicativo.
