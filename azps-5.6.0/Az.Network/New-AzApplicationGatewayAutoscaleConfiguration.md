# New-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Cria uma Configuração de Escala Automática para o Gateway de Aplicativo.

## SINTAXE

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzApplicationGatewayAutoscaleConfiguration** cria a Configuração de Escala Automática para um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

O primeiro comando cria uma configuração de escala automática com capacidade mínima 3.
O segundo comando cria um gateway de aplicativo com a configuração de escala automática.
