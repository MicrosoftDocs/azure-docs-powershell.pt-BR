# Remove-AzPrivateEndpoint

## SYNOPSIS
Remove um ponto de extremidade privado.

## SINTAXE

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzPrivateEndpoint** remove um ponto de extremidade privado. 

## EXEMPLOS

### Exemplo 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Este exemplo remove um ponto de extremidade privado chamado MyPrivateEndpoint1.
