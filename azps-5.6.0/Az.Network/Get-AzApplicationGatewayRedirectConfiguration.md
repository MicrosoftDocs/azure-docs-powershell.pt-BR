# Get-AzApplicationGatewayRedirectConfiguration

## SYNOPSIS
Obtém uma configuração de redirecionamento existente de um Gateway de Aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayRedirectConfiguration [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApplicationGatewayRedirectConfiguration** obtém uma configuração de redirecionamento existente de um Gateway de Aplicativo.

## EXEMPLOS

### Exemplo 1
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $RedirectConfig = Get-AzApplicationGatewayRedirectConfiguration -Name "Redirect01" -ApplicationGateway $AppGW
```

O primeiro comando obtém o Gateway de Aplicativo chamado ApplicationGateway01 e armazena o resultado na variável chamada $AppGW.
O segundo comando obtém a configuração de redirecionamento chamada Redirect01 do Gateway de Aplicativo armazenado na variável denominada $AppGW.
