# Get-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Obtém um certificado de autenticação para um gateway de aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApplicationGatewayAuthenticationCertificate** obtém um certificado de autenticação para um gateway de aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: Obter um certificado de autenticação especificado
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $cert = Get-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -ApplicationGateway $appgw
```

O primeiro comando obtém o gateway de aplicativo chamado appGwName e o armazena na variável $appgw de aplicativos.
O segundo comando obtém o certificado de autenticação chamado cert01 e o armazena na variável $cert de autenticação.
