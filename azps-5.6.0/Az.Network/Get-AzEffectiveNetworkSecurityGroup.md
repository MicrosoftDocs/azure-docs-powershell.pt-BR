# Get-AzEffectiveNetworkSecurityGroup

## SYNOPSIS
Obtém o grupo de segurança de rede efetivo de uma interface de rede.

## SINTAXE

```
Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzEffectiveNetworkSecurityGroup** retorna o grupo de segurança de rede efetivo que é aplicado em uma interface de rede.

## EXEMPLOS

### Exemplo 1: Obter o grupo de segurança de rede efetivo em uma interface de rede
```
PS C:\>Get-AzEffectiveNetworkSecurityGroup -NetworkInterfaceName "MyNetworkInterface" -ResourceGroupName "myResourceGroup"
```

Este comando obtém todas as regras de segurança de rede efetivas associadas à interface de rede chamada MyNetworkInterface no grupo de recursos chamado myResourceGroup.
