# Get-AzApplicationGatewayFrontendPort

## SYNOPSIS
Obtém a porta front-end de um gateway de aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayFrontendPort [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApplicationGatewayFrontendPort** obtém a porta front-end de um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1: Obter uma porta front-end especificada
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndPort = Get-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando obtém a porta front-end chamada FrontEndPort01 $AppGw e a armazena na variável $FrontEndPort.

### Exemplo 2: Obter uma lista de portas front-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndPorts = Get-AzApplicationGatewayFrontendPort  -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando obtém uma lista das portas front-end $AppGw e as armazena na variável $FrontEndPorts.
