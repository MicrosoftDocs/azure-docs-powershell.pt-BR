# New-AzPublicIpTag

## SYNOPSIS
Cria uma marca IP.

## SINTAXE

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzPublicIpTag** cria uma marca IP.

## EXEMPLOS

### Exemplo 1: Criar uma nova marca IP
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

Este comando cria uma nova Marca IP com o Tagtype como "FirstPartyUsage" e marca como "/Sql". Isso é usado na criação publicIpAddress com essas marcas específicas para alocação.
