# Remove-AzVirtualNetworkTap

## SYNOPSIS
Remove um toque de rede virtual.

## SINTAXE

### RemoveByNameParameterSet (Padrão)
```
Remove-AzVirtualNetworkTap -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzVirtualNetworkTap -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzVirtualNetworkTap -InputObject <PSVirtualNetworkTap> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzVirtualNetworkTap** remove um toque de rede virtual do Azure.

## EXEMPLOS

### Exemplo 1: Remover um toque de rede virtual
```
PS C:\>Remove-AzNetworkInterface -Name "VirtualNetworkTap1" -ResourceGroup "ResourceGroup1"
```

Este comando remove o VirtualNetworkTap1 no grupo de recursos ResourceGroup1.
Como o *parâmetro Force* não é usado, o usuário será solicitado a confirmar essa ação.
