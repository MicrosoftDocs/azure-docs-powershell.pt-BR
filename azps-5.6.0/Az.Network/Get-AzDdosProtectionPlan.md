# Get-AzDdosProtectionPlan

## SYNOPSIS
Obtém um plano de proteção DDoS.

## SINTAXE

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzDdosProtectionPlan cmdlet obtém um plano de proteção DDoS.

## EXEMPLOS

Nesse caso, precisamos especificar os atributos **ResourceGroupName** e **Name,** que correspondem ao grupo de recursos e ao nome do plano de proteção DDoS, respectivamente.
