# Get-AzExpressRouteCircuitRouteTable

## SYNOPSIS
Obtém uma tabela de rota de um circuito ExpressRoute.

## SINTAXE

```
Get-AzExpressRouteCircuitRouteTable -ResourceGroupName <String> -ExpressRouteCircuitName <String>
 -PeeringType <String> -DevicePath <DevicePathEnum> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzExpressRouteCircuitRouteTable** recupera uma tabela de rota detalhada de um circuito ExpressRoute. A tabela de rota mostrará todas as rotas ou poderá ser filtrada para mostrar rotas para um tipo de paração específico. Você pode usar a tabela de rota para validar sua configuração de par e conectividade.

## EXEMPLOS

### Exemplo 1: Exibir a tabela de rota para o caminho principal
```
Get-AzExpressRouteCircuitRouteTable -ResourceGroupName $RG -ExpressRouteCircuitName $CircuitName -DevicePath 'Primary'
```
