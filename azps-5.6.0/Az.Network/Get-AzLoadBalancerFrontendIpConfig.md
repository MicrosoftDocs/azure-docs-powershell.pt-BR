# Get-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Obtém uma configuração de IP front-end em um balanceador de carga.

## SINTAXE

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzLoadBalancerFrontendIpConfig** obtém uma configuração de IP front-end ou uma lista de configurações de IP front-end em um balanceador de carga.

## EXEMPLOS

### Exemplo 1: obter uma configuração de IP front-end em um balanceador de carga
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

O primeiro comando obtém o balanceador de carga chamado MyLoadBalancer e o armazena na variável $slb.
O segundo comando obtém a configuração de IP front-end associada a esse balanceador de carga.
