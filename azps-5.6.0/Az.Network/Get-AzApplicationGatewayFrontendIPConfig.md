# Get-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Obtém a configuração de IP front-end de um gateway de aplicativo.

## SINTAXE

```
Get-AzApplicationGatewayFrontendIPConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApplicationGatewayFrontendIPConfig obtém** a configuração de IP front-end de um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1: Obter uma configuração IP front-end especificada
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIP= Get-AzApplicationGatewayFrontendIPConfig -Name "FrontEndIP01" -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando obtém a configuração de IP front-end chamada FrontEndIP01 $AppGw e armazena-a na variável $FrontEndIP front-end.

### Exemplo 2: Obter uma lista de configurações de IP front-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIPs= Get-AzApplicationGatewayFrontendIPConfig  -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando obtém uma lista das configurações de IP front-end $AppGw o armazena na variável $FrontEndIPs front-end.
