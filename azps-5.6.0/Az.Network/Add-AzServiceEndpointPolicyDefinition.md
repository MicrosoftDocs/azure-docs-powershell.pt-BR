# Add-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Adiciona uma definição de política de ponto de extremidade de serviço a uma política especificada.

## SINTAXE

```
Add-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Add-AzServiceEndpointPolicyDefinition** adiciona uma definição de política de ponto de extremidade de serviço à política.

## EXEMPLOS

### Exemplo 1: atualiza uma definição de política de ponto de extremidade de serviço em uma política de ponto de extremidade de serviço
```
$policydef= New-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ResourceGroupName "ResourceGroup01" -Service "Microsoft.Storage" -ServiceResources "subscriptions/sub1" -Description "New Definition"
```

Este comando atualizou a definição da política de ponto de extremidade de serviço com o nome ServiceEndpointPolicyDefinition1, serviço Microsoft.Storage, assinaturas/sub1 de recursos de serviço e descrição "Nova Definição" que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $policydef.
