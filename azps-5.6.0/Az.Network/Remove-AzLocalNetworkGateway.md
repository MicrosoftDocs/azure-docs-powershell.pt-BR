# Remove-AzLocalNetworkGateway

## SYNOPSIS
Exclui um Gateway de Rede Local

## SINTAXE

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O Gateway de Rede Local é o objeto que representa seu dispositivo VPN local.
O cmdlet **Remove-AzLocalNetworkGateway** exclui o objeto que representa seu gateway local com base em Nome e Nome do Grupo de Recursos.

## EXEMPLOS

### Exemplo 1: Excluir um Gateway de Rede Local
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Exclui o objeto do Gateway de Rede Local com o nome "myLocalGW" dentro do grupo de recursos "myRG" Observação: primeiro, exclua todas as conexões com o Gateway de Rede Local usando o cmdlet **Remove-AzVirtualNetworkGatewayConnection.**
