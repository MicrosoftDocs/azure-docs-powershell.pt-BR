# Set-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Atualiza um Certificado Raiz Confiável de um gateway de aplicativo.

## SINTAXE

```
Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzApplicationGatewayTrustedRootCertificate** modifica o certificado raiz confiável existente de um Gateway de Aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\rootCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Cenários de exemplo acima mostram como atualizar um certificado raiz confiável existente quando um certificado raiz é rolado.
O primeiro comando obtém um gateway de aplicativo e o armazena na $gw variável.
O segundo comando modifica o certificado raiz confiável existente com um novo certificado raiz.
O terceiro comando atualiza o gateway de aplicativo no Azure.
