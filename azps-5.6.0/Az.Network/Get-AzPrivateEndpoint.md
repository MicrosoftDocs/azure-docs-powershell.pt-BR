# Get-AzPrivateEndpoint

## SYNOPSIS
Obter um ponto de extremidade privado

## SINTAXE

### NoExpand (Padrão)
```
Get-AzPrivateEndpoint [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Expandir
```
Get-AzPrivateEndpoint -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzPrivateEndpoint** obtém um ou mais pontos de extremidade privados.
