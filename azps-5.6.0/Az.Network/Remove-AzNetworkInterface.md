# Remove-AzNetworkInterface

## SYNOPSIS
Remove uma interface de rede.

## SINTAXE

```
Remove-AzNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzNetworkInterface** remove uma interface de rede do Azure.

## EXEMPLOS

### Exemplo 1: Remover uma interface de rede
```
PS C:\>Remove-AzNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

Este comando remove a interface de rede NetworkInterface1 no grupo de recursos ResourceGroup1.
Como o *parâmetro Force* não é usado, o usuário será solicitado a confirmar essa ação.

### Exemplo 2: Remover uma interface de rede
```
PS C:\>Get-AzNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzNetworkInterface -Force
```

Este comando remove todas as interfaces de rede no grupo de recursos ResourceGroup1.
Como o *parâmetro Force* é usado, o usuário não é solicitado a confirmar.
