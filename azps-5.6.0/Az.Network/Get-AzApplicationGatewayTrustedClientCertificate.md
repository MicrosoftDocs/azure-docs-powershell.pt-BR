# Get-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Obtém a cadeia de certificados de AC de cliente confiável com um nome específico do Gateway de Aplicativos.

## SINTAXE

```
Get-AzApplicationGatewayTrustedClientCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzApplicationGatewayTrustedClientCertificate cmdlet obtém a cadeia de certificados de AC do cliente confiável com um nome específico do Gateway de Aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClientCert = Get-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName
```

O primeiro comando obtém o Gateway de Aplicativo e o armazena $gw variável. O segundo comando obtém a cadeia de certificados de AC de cliente confiável com um nome especificado do Gateway de Aplicativo.
