# Get-AzEventHubNamespace

## SYNOPSIS
Obtém os detalhes de um namespace hubs de eventos ou obtém uma lista de todos os namespaces de Hubs de Eventos na assinatura atual do Azure.

## SINTAXE

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzEventHubNamespace cmdlet obtém os detalhes de um namespace de Hubs de Eventos especificado ou uma lista de todos os namespaces de Hubs de Eventos na assinatura atual do Azure.
Se o nome do namespace for fornecido, os detalhes de um único namespace de Hubs de Eventos serão retornados.
Se o nome do namespace não for fornecido, uma lista de namespaces será retornada.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzEventHubNamespace -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName

Name                   : MyNamespaceName
Id                     : /subscriptions/{subscriptionId}/resourceGroups/Default-EventHub-WestCentralUS/providers/Microsoft.EventHub/namespaces/MyNamespaceName
ResourceGroupName      : Default-EventHub-WestCentralUS
Location               : West US
Sku                    : Name : Standard , Capacity : 1 , Tier : Standard
Tags                   :
ProvisioningState      : Succeeded
Status                 : Active
CreatedAt              : 5/24/2019 12:47:27 AM
UpdatedAt              : 5/24/2019 12:48:14 AM
ServiceBusEndpoint     : #########
Enabled                : True
KafkaEnabled           : True
IsAutoInflateEnabled   : True
MaximumThroughputUnits : 10
```

Obtém os detalhes do namespace \` MyNamespaceName \` no grupo de recursos \` MyResourceGroupName \` .
