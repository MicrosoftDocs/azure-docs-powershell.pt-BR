# Get-AzEventHubClustersAvailableRegion

## SYNOPSIS
Obtém os detalhes do cluster Eventhub único ou a lista de clusters no determinado Grupo de Recursos

## SINTAXE

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
A Get-AzEventHubClustersAvailableRegion de cmdlets de regiões onde dedicados estão disponíveis para criar.

## EXEMPLOS

### Exemplo 1
