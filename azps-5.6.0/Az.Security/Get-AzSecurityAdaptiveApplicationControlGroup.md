# Get-AzSecurityAdaptiveApplicationControlGroup

## SYNOPSIS
Obtém um grupo de VM/servidor de controle de aplicativo.

## SINTAXE

### SubscriptionScope (Padrão)
```
Get-AzSecurityAdaptiveApplicationControlGroup  -GroupName <String> -AscLocation <String> [-SubscriptionId <String>]
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Controles de aplicativo adaptáveis são calculados automaticamente pelo Centro de Segurança do Azure, use este cmdlet para obter um grupo de VM/servidor de controle de aplicativos.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzSecurityAdaptiveApplicationControlGroup -GroupName GROUP1 -AscLocation centralus
Id         : /subscriptions/3eeab341-f466-499c-a8be-85427e154baf7612f869/providers/Microsoft.Security/locations/centralus/applicationWhitelistings/GROUP1
Name       : GROUP1
Type       : Microsoft.Security/applicationWhitelistings
Location   : centralus
Properties : Microsoft.Azure.Commands.SecurityCenter.Models.AdaptiveApplicationControls.PSSecurityAdaptiveApplicationControlsProperties
```
Obtém um grupo de VM/servidor de controle de aplicativo.
