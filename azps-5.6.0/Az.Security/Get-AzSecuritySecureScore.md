# Get-AzSecuritySecureScore

## SYNOPSIS
Obtém pontuações seguras de segurança e seus resultados em uma assinatura

## SINTAXE

### SubscriptionScope (Padrão)
```
Get-AzSecuritySecureScore [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecuritySecureScore -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzSecuritySecureScore
```

Obtém todas as pontuações seguras de segurança em uma assinatura
