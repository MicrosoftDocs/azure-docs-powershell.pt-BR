# Set-AzSecurityWorkspaceSetting

## SYNOPSIS
Atualiza as configurações de espaço de trabalho da assinatura.

## SINTAXE

```
Set-AzSecurityWorkspaceSetting -Name <String> -Scope <String> -WorkspaceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Atualiza as configurações de espaço de trabalho da assinatura.
O espaço de trabalho configurado manterá os dados de segurança coletados pelo agente agente do Azure Log Analytics instalado em VMs dentro dessa assinatura.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzSecurityWorkspaceSetting -Name "default" -Scope "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869" -WorkspaceId  "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/resourcegroups/mainws/providers/microsoft.operationalinsights/workspaces/securityuserws"

Id                                                                                                         Name    WorkspaceId 
--                                                                                                         ----    ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/workspaceSettings/default default /...
```

Define o espaço de trabalho "securityuserws" para manter todos os dados de segurança coletados pelo agente do Azure Log Analytics.
