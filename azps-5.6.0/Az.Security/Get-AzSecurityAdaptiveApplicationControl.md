# Get-AzSecurityAdaptiveApplicationControl

## SYNOPSIS
Obtém uma lista de grupos de VM/servidor de controle de aplicativos para a assinatura.

## SINTAXE

### SubscriptionScope (Padrão)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Controles de aplicativo adaptáveis são calculados automaticamente pelo Centro de Segurança do Azure, use este cmdlet para obter uma lista de recursos controles de aplicativo adaptáveis no escopo de uma assinatura.
