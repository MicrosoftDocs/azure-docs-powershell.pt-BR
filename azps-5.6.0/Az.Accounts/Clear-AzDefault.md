# Clear-AzDefault

## SYNOPSIS
Limpa os padrões definidos pelo usuário no contexto atual.

## SINTAXE

```
Clear-AzDefault [-ResourceGroup] [-PassThru] [-Force] [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O Clear-AzDefault cmdlet remove os padrões definidos pelo usuário, dependendo dos parâmetros de comutadores especificados pelo usuário.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Clear-AzDefault
```

Este comando remove todos os padrões definidos pelo usuário no contexto atual.

### Exemplo 2
```powershell
PS C:\> Clear-AzDefault -ResourceGroup
```

Este comando remove o grupo de recursos padrão definido pelo usuário no contexto atual.
