# Get-AzDefault

## SYNOPSIS
Obter os padrões definidos pelo usuário no contexto atual.

## SINTAXE

```
Get-AzDefault [-ResourceGroup] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzDefault cmdlet obtém o Grupo de Recursos que o usuário definiu como padrão no contexto atual.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzDefault

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Este comando retorna os padrões atuais se houver padrões definidos ou retorna nada se nenhum padrão for definido.

### Exemplo 2
```
PS C:\> Get-AzDefault -ResourceGroup

Id         : /subscriptions/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/resourceGroups/myResourceGroup
Name       : myResourceGroup
Properties : Microsoft.Azure.Management.Internal.Resources.Models.ResourceGroupProperties
Location   : eastus
ManagedBy  :
Tags       :
```

Este comando retornará o Grupo de Recursos padrão atual se houver um conjunto padrão ou retornará nada se nenhum padrão for definido.
