# Get-AzTrafficManagerProfile

## SYNOPSIS
Obtém um perfil do Gerenciador de Tráfego.

## SINTAXE

### ResourceGroupParameterSet
```
Get-AzTrafficManagerProfile [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzTrafficManagerProfile [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzTrafficManagerProfile** obtém um perfil do Gerenciador de Tráfego do Azure e retorna um objeto que representa esse perfil.
Especifique um perfil pelo nome e nome do grupo de recursos.

Você pode modificar esse objeto localmente e aplicar alterações ao perfil usando o cmdlet Set-AzTrafficManagerProfile.

## EXEMPLOS

### Exemplo 1: Obter um perfil
```
PS C:\>Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
```

Este comando obtém o perfil chamado ContosoProfile em ResourceGroup11.
