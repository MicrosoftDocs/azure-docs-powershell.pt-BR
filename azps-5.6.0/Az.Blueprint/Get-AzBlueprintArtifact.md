# Get-AzBlueprintArtifact

## SYNOPSIS
Recupere artefatos de uma definição de projeto.

## SINTAXE

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Recupere artefatos de uma definição de projeto. Se uma versão de definição de projeto não for especificada, a versão de rascunho será recuperada. No caso em que não houver versão de rascunho, a definição de projeto publicada mais recente será retornada.
