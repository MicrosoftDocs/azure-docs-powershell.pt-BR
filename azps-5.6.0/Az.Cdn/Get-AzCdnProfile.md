# Get-AzCdnProfile

## SYNOPSIS
Obtém um perfil cdn.

## SINTAXE

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzCdnProfile** obtém um perfil cdn (Rede de Distribuição de Conteúdo) do Azure e suas informações relacionadas.
