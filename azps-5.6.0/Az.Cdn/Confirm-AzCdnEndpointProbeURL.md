# Confirm-AzCdnEndpointProbeURL

## SYNOPSIS
Valida uma URL de sonda.

## SINTAXE

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Confirm-AzCdnEndpointProbeURL** confirma se a URL da sonda fornecida pode ser usada para aceleração dinâmica do site.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Valida a url da sonda " http://www.bing.com/images "
