# Set-AzCdnProfile

## SYNOPSIS
Atualiza um perfil de CDN.

## SINTAXE

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzCdnProfile** atualiza um perfil cdn (Rede de Distribuição de Conteúdo) do Azure.
