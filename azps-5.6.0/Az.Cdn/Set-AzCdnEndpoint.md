# Set-AzCdnEndpoint

## SYNOPSIS
Atualiza um ponto de extremidade cdn.

## SINTAXE

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzCdnEndpoint** atualiza um ponto de extremidade cdn (Rede de Distribuição de Conteúdo) do Azure.
