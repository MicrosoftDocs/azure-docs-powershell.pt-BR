# Get-AzCdnProfileSsoUrl

## SYNOPSIS
Obtém a URL de entrada única de um perfil de CDN.

## SINTAXE

### ByFieldsParameterSet (Padrão)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzCdnProfileSsoUrl** obtém a URL de entrada única do perfil CDN (Rede de Entrega de Conteúdo) do Azure.
Essa URL permite que os usuários se conectem a um portal complementar e usem recursos adicionais da CDN.
