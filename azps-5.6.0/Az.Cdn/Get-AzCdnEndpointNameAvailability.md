# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Obtém o status de disponibilidade do ponto de extremidade cdn.

## SINTAXE

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzCdnEndpointNameAvailability** obtém o status de disponibilidade do ponto de extremidade cdn (Rede de Distribuição de Conteúdo) do Azure.
