# Módulo Az.NetAppFiles
## Descrição
Os tópicos desta seção documentam os cmdlets do Azure PowerShell para Arquivos do Azure NetApp na estrutura do Gerenciador de Recursos do Azure (ARM). Os cmdlets existem no namespace Microsoft.Azure.Commands.NetAppFiles.

## Az.NetAppFiles Cmdlets
### [Approve-AzNetAppFilesReplication](Approve-AzNetAppFilesReplication.md)
Aprovar/autorizar a conexão de replicação no volume de origem

### [Get-AzNetAppFilesAccount](Get-AzNetAppFilesAccount.md)
Obtém detalhes de uma conta do Azure NetApp Files (ANF).

### [Get-AzNetAppFilesActiveDirectory](Get-AzNetAppFilesActiveDirectory.md)
Obtém detalhes de uma configuração do Azure NetApp Files (ANF) Active Directory.

### [Get-AzNetAppFilesBackup](Get-AzNetAppFilesBackup.md)
Obtém detalhes de um backup do Azure NetApp Files (ANF).

### [Get-AzNetAppFilesBackupPolicy](Get-AzNetAppFilesBackupPolicy.md)
Obtém detalhes de uma Política de Backup de Arquivos do Azure NetApp (ANF).

### [Get-AzNetAppFilesPool](Get-AzNetAppFilesPool.md)
Obtém detalhes de um pool de Arquivos do Azure NetApp (ANF).

### [Get-AzNetAppFilesReplicationStatus](Get-AzNetAppFilesReplicationStatus.md)
Obter o status da replicação

### [Get-AzNetAppFilesSnapshot](Get-AzNetAppFilesSnapshot.md)
Obtém detalhes de um instantâneo do Azure NetApp Files (ANF).

### [Get-AzNetAppFilesSnapshotPolicy](Get-AzNetAppFilesSnapshotPolicy.md)
Obtém detalhes de uma política de instantâneo do Azure NetApp Files (ANF).

### [Get-AzNetAppFilesVault](Get-AzNetAppFilesVault.md)

### [Get-AzNetAppFilesVolume](Get-AzNetAppFilesVolume.md)

### [New-AzNetAppFilesBackupPolicy](New-AzNetAppFilesBackupPolicy.md)
Cria uma nova política de backup arquivos do Azure NetApp (ANF) para uma conta ANF.

### [New-AzNetAppFilesPool](New-AzNetAppFilesPool.md)
Cria um novo pool de Arquivos do Azure NetApp (ANF).

### [New-AzNetAppFilesSnapshot](New-AzNetAppFilesSnapshot.md)
