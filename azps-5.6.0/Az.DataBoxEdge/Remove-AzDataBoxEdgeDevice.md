# Remove-AzDataBoxEdgeDevice

## SYNOPSIS
Remove um dispositivo de Borda da Caixa de Dados.

## SINTAXE

### DeleteByNameParameterSet (Padrão)
```
Remove-AzDataBoxEdgeDevice [-ResourceGroupName] <String> [-Name] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzDataBoxEdgeDevice -ResourceId <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-PassThru]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzDataBoxEdgeDevice -InputObject <PSDataBoxEdgeDevice> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzDataBoxEdgeDevice** remove a configuração de um dispositivo de Borda da Caixa de Dados.
Observe que o dispositivo só pode ser excluído depois que você tiver feito o pedido e antes de o dispositivo ser preparado pela Microsoft para envio.

Consulte a documentação sobre Como excluir o recurso antes de usar este [cmdlet](https://docs.microsoft.com/azure/databox-online/data-box-edge-return-device#delete-the-resource)

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Remove-AzDataBoxEdgeDevice ResourceGroupName resourceGroupName -Name deviceName
```
