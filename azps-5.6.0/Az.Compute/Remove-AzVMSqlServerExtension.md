# Remove-AzVMSqlServerExtension

## SYNOPSIS
Remove uma SQL Server de uma máquina virtual.

## SINTAXE

```
Remove-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzVMSqlServerExtension** remove uma extensão do AzureSQL Server de uma máquina virtual.

## EXEMPLOS

### Exemplo 1: Remover uma extensão SQL Server de dados
```
PS C:\> Remove-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22" -Name "SqlIaaSAgent"
```

Este comando remove uma extensão SQL Server da máquina virtual chamada ContosoVM22.
