# Get-AzVMCustomScriptExtension

## SYNOPSIS
Obtém informações sobre uma extensão de script personalizada.

## SINTAXE

```
Get-AzVMCustomScriptExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzVMCustomScriptExtension** obtém informações sobre uma Extensão de Máquina Virtual de script personalizada em uma máquina virtual.

## EXEMPLOS

### Exemplo 1: Obter uma extensão de script personalizada
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript"
```

Este comando obtém a extensão de script personalizada chamada ContosoCustomScript para a máquina virtual chamada VirtualMachine07.

### Exemplo 2: Obter a exibição de instância de uma extensão de script personalizada
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript" -Status
```

Este comando obtém a exibição de instância da extensão de script personalizada chamada ContosoCustomScript para a máquina virtual chamada VirtualMachine07.
