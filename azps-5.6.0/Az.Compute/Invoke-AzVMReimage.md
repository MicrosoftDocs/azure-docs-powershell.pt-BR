# Invoke-AzVMReimage

## SYNOPSIS
Reimage uma máquina virtual do Azure.

## SINTAXE

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Invoke-AzVMReimage** reimages an Azure virtual machine.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Este comando reimaja a máquina virtual chamada VirtualMachine07 em ResourceGroup11.
