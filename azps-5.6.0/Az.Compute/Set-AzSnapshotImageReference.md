# Set-AzSnapshotImageReference

## SYNOPSIS
Define as propriedades de referência da imagem em um objeto instantâneo.

## SINTAXE

```
Set-AzSnapshotImageReference [-Snapshot] <PSSnapshot> [[-Id] <String>] [[-Lun] <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzSnapshotImageReference** define as propriedades de referência da imagem em um objeto instantâneo.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $snapshotconfig = New-AzSnapshotConfig -SnapshotSizeGB 10 -AccountType PremiumLRS -OsType Windows -CreateOption FromImage;
PS C:\> $image = '/subscriptions/0000000-0000-0000-0000-000000000000/resourceGroups/ResourceGroup01/providers/Microsoft.Compute/images/TestImage123';        
PS C:\> $snapshotconfig = Set-AzSnapshotImageReference -Snapshot $snapshotconfig -Id $image -Lun 0;
PS C:\> New-AzSnapshot -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01' -Snapshot $snapshotconfig;
```

O primeiro comando cria um objeto instantâneo local com tamanho de 10 GB Premium_LRS de conta de armazenamento.  Ele também define o tipo de sistema operacional do Windows.
O segundo comando define a ID da imagem e o número da unidade lógica 0 para o objeto instantâneo.
O último comando pega o objeto instantâneo e cria um instantâneo com o nome 'Instantâneo01' no grupo de recursos 'ResourceGroup01'.
