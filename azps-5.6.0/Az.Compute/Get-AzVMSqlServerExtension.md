# Get-AzVMSqlServerExtension

## SYNOPSIS
Obtém as configurações de uma SQL Server em uma máquina virtual.

## SINTAXE

```
Get-AzVMSqlServerExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzVMSqlServerExtension** obtém as configurações da infraestrutura SQL Server como um agente de serviço (IaaS) em uma máquina virtual.

## EXEMPLOS

### Exemplo 1: Obter as configurações de uma extensão SQL Server em uma máquina virtual
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Este comando obtém as configurações da extensão SQL Server em uma máquina virtual chamada ContosoVM07.

### Exemplo 2: Obter as configurações usando o pipeline
```
PS C:\> Get-AzVM -ServiceName "Service08" -Name "ContosoVM22" | Get-AzVMSqlServerExtension
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Este comando obtém a máquina virtual chamada ContosoVM22 no service Service08 usando o cmdlet Get-AzVM.
O comando passa os resultados para o cmdlet atual usando o operador de pipeline.
O comando atual obtém as configurações do SQL Server Agente IaaS nessa máquina virtual.

### Exemplo 3: Obter as configurações de uma versão SQL Server específica
```
PS C:\> Get-AzVMSqlServerExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM07" -Version "1.0"
ExtensionName        : SqlIaaSAgent
Publisher            : Microsoft.SqlServer.Management
Version              : 1.0
State                : Enable
RoleName             : VMName
AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Este comando obtém as configurações da versão 1.0 da extensão SQL Server em uma máquina virtual chamada ContosoVM07.
