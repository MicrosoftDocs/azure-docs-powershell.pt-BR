# Remove-AzVM

## SYNOPSIS
Remove uma máquina virtual do Azure.

## SINTAXE

### ResourceGroupNameParameterSetName (Padrão)
```
Remove-AzVM [-Name] <String> [-Force] [-NoWait] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Remove-AzVM [-Force] [-NoWait] [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzVM** remove uma máquina virtual do Azure.

## EXEMPLOS

### Exemplo 1: Remover uma máquina virtual
```
PS C:\> Remove-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Este comando remove a máquina virtual chamada VirtualMachine07 no grupo de recursos ResourceGroup11.
