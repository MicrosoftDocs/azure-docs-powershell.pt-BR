# Start-AzVmss

## SYNOPSIS
Inicia o VMSS ou um conjunto de máquinas virtuais no VMSS.

## SINTAXE

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Start-AzVmss** inicia todas as máquinas virtuais dentro do Conjunto de Escala de Máquina Virtual (VMSS) ou um conjunto de máquinas virtuais.
Você pode usar o *parâmetro InstanceId* para selecionar um conjunto de máquinas virtuais.

## EXEMPLOS

### Exemplo 1: Iniciar um conjunto específico de máquinas virtuais no VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

Este comando inicia um conjunto específico de máquinas virtuais especificadas pela matriz de cadeia de caracteres de ID de instância que pertencem à VMSS chamada ContosoVMSS.

### Exemplo 2: Iniciar todas as máquinas virtuais no VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

Este comando inicia todas as máquinas virtuais que pertencem à VMSS chamada ContosoVMSS.
