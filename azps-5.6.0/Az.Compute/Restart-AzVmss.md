# Restart-AzVmss

## SYNOPSIS
Reinicia o VMSS ou uma máquina virtual dentro do VMSS.

## SINTAXE

```
Restart-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Restart-AzVmss** reinicia o Conjunto de Escala de Máquina Virtual (VMSS).
Esse cmdlet também pode ser usado para reiniciar uma máquina virtual específica dentro do VMSS usando o *parâmetro InstanceId.*

## EXEMPLOS

### Exemplo 1: Reiniciar o VMSS
```
PS C:\> Restart-AzVmss -ResourceGroupName "Group001" -VMScaleSetName "VMSS001";
```

Este comando reinicia o VMSS chamado VMSS001 que pertence ao grupo de recursos chamado Group001.

### Exemplo 2: reiniciar uma máquina virtual específica dentro do VMSS
```
PS C:\> Restart-AzVmss -ResourceGroupName "Group004" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Este comando reinicia uma máquina virtual que tem a ID da instância 1 no VMSS chamado VMSS001 que pertence ao grupo de recursos chamado Group001.
