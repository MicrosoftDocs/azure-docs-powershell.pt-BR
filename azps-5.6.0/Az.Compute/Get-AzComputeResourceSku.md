# Get-AzComputeResourceSku

## SYNOPSIS
Listar todos os Skus de recurso de computação

## SINTAXE

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Listar todos os Skus de recurso de computação

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzComputeResourceSku "westus";
```

Listar todos os skus de recurso de computação na região do Oeste dos EUA
