# Start-AzVmssRollingOSUpgrade

## SYNOPSIS
Inicia uma atualização de rolagem para mover todas as instâncias de conjunto de escala de máquina virtual para a versão mais recente disponível do sistema operacional de imagem da plataforma.

## SINTAXE

```
Start-AzVmssRollingOSUpgrade [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Inicia uma atualização de rolagem para mover todas as instâncias de conjunto de escala de máquina virtual para a versão mais recente disponível do sistema operacional de imagem da plataforma.
As instâncias que já estão executando a versão mais recente do sistema operacional disponível não são afetadas.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Start-AzVmssRollingOSUpgrade -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Este comando inicia uma atualização de rolagem de todas as instâncias de vm do conjunto de escala de VM "VMSS001" no grupo de recursos "Group001".
