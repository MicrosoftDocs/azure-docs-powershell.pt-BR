# Get-AzVMImagePublisher

## SYNOPSIS
Obtém os editores VMImage.

## SINTAXE

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzVMImagePublisher** obtém os editores VMImage.

## EXEMPLOS

### Exemplo 1: Obter editores VMImage para uma região
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

Este comando obtém os editores de instâncias do VMImage para a região central dos EUA no perfil do Azure.
