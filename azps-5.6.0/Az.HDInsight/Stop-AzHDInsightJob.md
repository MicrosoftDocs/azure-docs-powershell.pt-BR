# Stop-AzHDInsightJob

## SYNOPSIS
Interrompe um trabalho em execução especificado em um cluster.

## SINTAXE

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Stop-AzHDInsightJob** interrompe um trabalho em execução especificado em um cluster HDInsight do Azure.

## EXEMPLOS

### Exemplo 1: Parar um trabalho no cluster especificado
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Este comando interrompe um trabalho no cluster chamado your-hadoop-001.
