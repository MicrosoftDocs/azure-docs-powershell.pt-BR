# Disable-AzHDInsightMonitoring

## SYNOPSIS
Desabilita o monitoramento em um cluster HDInsight e os logs relevantes param de fluir para o espaço de trabalho de monitoramento especificado durante a habilitação.

## SINTAXE

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Disable-AzHDInsightMonitoring** desabilita o monitoramento em um cluster HDInsight do Azure.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

O monitoramento será desabilitado no cluster HDInsight e os logs relevantes param de fluir para o espaço de trabalho de monitoramento.

### Exemplo 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

O monitoramento será desabilitado no cluster HDInsight e os logs relevantes param de fluir para o espaço de trabalho de monitoramento.
