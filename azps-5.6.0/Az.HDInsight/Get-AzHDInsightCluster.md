# Get-AzHDInsightCluster

## SYNOPSIS
Obtém e lista todos os clusters HDInsight do Azure associados à assinatura atual ou a um grupo de recursos especificado ou recupera um cluster específico.

## SINTAXE

```
Get-AzHDInsightCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzHDInsightCluster** lista os clusters de serviço HDInsight do Azure para a assinatura atual.
Use o *parâmetro ClusterName* para obter detalhes de um cluster específico.

## EXEMPLOS

### Exemplo 1: Listar todos os clusters HDInsight do Azure
```
PS C:\>Get-AzHDInsightCluster
```

Este comando lista todos os clusters HDInsight do Azure.
