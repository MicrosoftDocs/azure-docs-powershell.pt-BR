# Get-AzHDInsightPersistedScriptAction

## SYNOPSIS
Obtém as ações de script persistentes para um cluster e as lista em ordem cronológica ou obtém detalhes de uma ação de script persistente especificada.

## SINTAXE

```
Get-AzHDInsightPersistedScriptAction [-ClusterName] <String> [[-Name] <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzHDInsightPersistedScriptAction** obtém as ações de script persistentes para um cluster HDInsight do Azure e as lista em ordem cronológica ou obtém detalhes de uma ação de script persistente especificada.

## EXEMPLOS

### Exemplo 1: Obter as ações de script persistentes em um cluster
```
PS C:\>Get-AzHDInsightPersistedScriptAction -ClusterName "your-hadoop-001"
```

Esse comando obtém ações de script persistentes no cluster chamado your-hadoop-001.
