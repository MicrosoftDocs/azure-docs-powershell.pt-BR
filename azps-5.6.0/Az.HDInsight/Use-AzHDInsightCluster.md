# Use-AzHDInsightCluster

## SYNOPSIS
Seleciona um cluster a ser usado com o Invoke-RmAzureHDInsightHiveJob cmdlet.

## SINTAXE

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Use-AzHDInsightCluster** seleciona o cluster HDInsight do Azure para o cmdlet Invoke-AzHDInsightHiveJob a ser usado para enviar trabalhos hive.

## EXEMPLOS

### Exemplo 1: Selecione um cluster para envio de consulta hive
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Este comando seleciona um cluster para um envio de consulta hive.
