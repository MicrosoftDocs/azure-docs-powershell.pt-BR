# Remove-AzHDInsightPersistedScriptAction

## SYNOPSIS
Remove uma ação de script persistente de um cluster HDInsight.

## SINTAXE

```
Remove-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzHDInsightPersistedScriptAction** remove uma ação de script persistente da lista de ações de script persistentes do cluster HDInsight especificado.
O script removido não será mais executado quando o cluster for ampliado.

## EXEMPLOS

### Exemplo 1: Remover uma ação de script da lista de ações de script persistentes em um cluster
```
PS C:\>Remove-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Este comando remove a ação de script chamada Scriptaction da lista de ações de script persistentes no cluster especificado.
