# Start-AzHDInsightJob

## SYNOPSIS
Inicia um trabalho definido do Azure HDInsight em um cluster especificado.

## SINTAXE

```
Start-AzHDInsightJob [-ClusterName] <String> [-JobDefinition] <AzureHDInsightJobDefinition>
 [-HttpCredential] <PSCredential> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Start-AzHDInsightJob** inicia um trabalho DEFINIDO do Azure HDInsight em um cluster especificado.
Pode ser um trabalho MapReduce, um trabalho de Streaming MapReduce, um trabalho de Hive ou um trabalho de Leião.

## EXEMPLOS

### Exemplo 1: Iniciar um trabalho no cluster especificado
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Este comando inicia um trabalho no cluster chamado your-hadoop-001.
