# Test-AzAnalysisServicesServer

## SYNOPSIS
Testa a existência de uma instância do servidor do Analysis Services

## SINTAXE

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Test-AzAnalysisServicesServer cmdlet testa a existência de uma instância do servidor do Analysis Services

## EXEMPLOS

### Exemplo 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Este comando testará se há um servidor chamado testserver no grupo de recursos testgroup
