# Restart-AzAnalysisServicesInstance

## SYNOPSIS
Reinicia uma instância do servidor do Analysis Services no ambiente conectado no momento, conforme especificado no Add-AzAnalysisServicesAccount comando

## SINTAXE

```
Restart-AzAnalysisServicesInstance [-Instance] <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O Restart-AzAnalysisServicesInstance cmdlet reinicia uma instância do servidor do Azure Analysis Services

## EXEMPLOS

### Exemplo 1
```
PS C:\>Restart-AzAnalysisServicesInstance
Instance: testserver
```

Este comando reiniciará o servidor "testserver" no ambiente especificado no comando Add-AzAnalysisServicesAccount servidor
