# Get-AzAnalysisServicesServer

## SYNOPSIS
Obtém os detalhes de um servidor do Analysis Services.

## SINTAXE

```
Get-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzAnalysisServicesServer cmdlet obtém os detalhes de um servidor do Analysis Services.

## EXEMPLOS

### Exemplo 1
```
PS C:\>Get-AzAnalysisServicesServer -ResourceGroupName "ResourceGroup03"
```

Este comando obtém todos os servidores do Azure Analysis Services no grupo de recursos chamado ResourceGroup03.

### Exemplo 2: Obter um servidor
```
PS C:\>Get-AzAnalysisServicesServer -ResourceGroupName "ResourceGroup03" -Name "testserver"
```

Este comando obtém o servidor do Azure Analysis Services chamado testserver no grupo de recursos chamado ResourceGroup03.
