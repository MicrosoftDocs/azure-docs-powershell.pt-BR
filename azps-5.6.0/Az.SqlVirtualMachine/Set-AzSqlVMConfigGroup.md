# Set-AzSqlVMConfigGroup

## SYNOPSIS
De definir as informações relativas a um grupo de máquinas virtuais sql em uma configuração de máquina virtual sql.

## SINTAXE

```
Set-AzSqlVMConfigGroup [-SqlVM] <AzureSqlVMModel> [-SqlVMGroup] <AzureSqlVMGroupModel>
 -ClusterOperatorAccountPassword <SecureString> -SqlServiceAccountPassword <SecureString>
 [-ClusterBootstrapAccountPassword <SecureString>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O Set-AzSqlVMConfigGroup cmdlet definiu as informações necessárias para ingressar em um grupo de máquinas virtuais sql para uma configuração de máquina virtual sql.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $config = Set-AzSqlVMConfigGroup -SqlVM $config -SqlVMGroup $group -ClusterOperatorAccountPassword 'password' -SqlServiceAccountPassword 'password'
```

Atualize as informações de grupo de uma configuração de máquina virtual sql.
