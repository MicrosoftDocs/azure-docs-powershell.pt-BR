# Get-AzRedisCacheKey

## SYNOPSIS
Obtém as chaves de acesso para um Cache Redis.

## SINTAXE

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzRedisCacheKey** obtém as chaves de acesso para um Cache do Azure Redis.

## EXEMPLOS

### Exemplo 1: Obter as chaves de acesso para um Cache Redis
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Este comando obtém as chaves de acesso chamadas MyCacheKey.
