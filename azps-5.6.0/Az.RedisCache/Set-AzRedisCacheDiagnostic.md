# Set-AzRedisCacheDiagnostic

## SYNOPSIS
Habilita diagnósticos em um Cache do Azure Redis.

## SINTAXE

```
Set-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String> -StorageAccountId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzRedisCacheDiagnostic** habilita diagnósticos para um Cache do Azure Redis.

## EXEMPLOS

### Exemplo 1: Habilitar diagnósticos
```
PS C:\>Set-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -StorageAccountId "/subscriptions/fffff139-aaaa-bbbb-cccc-21f21f35806e/resourcegroups/myresourcegroup/providers/Microsoft.Storage/storageAccounts/mystorageaccount"
```

Este comando habilita diagnósticos para um cache do Azure Redis.
Este comando habilita o diagnóstico ou atualiza a conta de armazenamento de todos os Caches do Azure Redis na mesma região da assinatura.
