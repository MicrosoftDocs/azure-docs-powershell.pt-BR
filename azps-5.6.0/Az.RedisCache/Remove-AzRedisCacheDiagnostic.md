# Remove-AzRedisCacheDiagnostic

## SYNOPSIS
Desabilita os diagnósticos em um Cache do Azure Redis.

## SINTAXE

```
Remove-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzRedisCacheDiagnostic** desabilita os diagnósticos em um Cache do Azure Redis.

## EXEMPLOS

### Exemplo 1: Desabilitar diagnósticos
```
PS C:\>Remove-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Este comando desabilita os diagnósticos no Cache do Azure Redis especificado.
Isso desabilita o diagnóstico de todos os Caches do Azure Redis na mesma região da assinatura.
