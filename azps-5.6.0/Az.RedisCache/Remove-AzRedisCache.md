# Remove-AzRedisCache

## SYNOPSIS
Remove um Cache Redis.

## SINTAXE

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzRedisCache** remove um Cache do Azure Redis.

## EXEMPLOS

### Exemplo 1: Remover um Cache Redis e retornar o resultado
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Este comando remove um Cache Redis e exibe se a operação foi bem-sucedida.

### Exemplo 2: Remover um Cache Redis e não exibir o resultado
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Este comando remove um Cache Redis.
Como o *parâmetro PassThru* não é especificado, o resultado da operação não é exibido.
