# Get-AzPeeringServiceCountry

## SYNOPSIS
Lista os países disponíveis para o serviço de peering.

## SINTAXE

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Lista os países disponíveis para o serviço de peering.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista de países disponíveis para o serviço de peering.
