# Get-AzLegacyPeering

## SYNOPSIS
Usado para converter recursos de peering herdados em recursos do Azure Resource Management (ARM). 

## SINTAXE

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O comando é usado para exibir recursos de peering herdados que você deve convertê-los em Recursos de Gerenciamento de Recursos do Azure (ARM).
