# Get-AzManagementPartner

## SYNOPSIS
Obtém a ID do MPN (Microsoft Partner Network) do usuário ou entidade de serviço autenticado atual.

## SINTAXE

```
Get-AzManagementPartner [[-PartnerId] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Obtém a ID do MPN (Microsoft Partner Network) do usuário ou entidade de serviço autenticado atual.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzManagementPartner
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Obter a ID do parceiro de gerenciamento atual

### Exemplo 2
```powershell
PS C:\> Get-AzManagementPartner -PartnerId 4977985
PartnerId   : 4977985
PartnerName : Test_Test_DPORTest
TenantId    : 1b1121dd-6900-412a-af73-e8d44f81e1c1
ObjectId    : aa67f786-0552-423e-8849-244ed12bf581
State       : Active
```

Obter a ID do parceiro de gerenciamento atual usando uma id de parceiro, se eles não corresponderem, ele falhará
