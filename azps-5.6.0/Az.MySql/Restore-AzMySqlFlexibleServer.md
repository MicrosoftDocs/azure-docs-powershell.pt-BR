# Restore-AzMySqlFlexibleServer

## SYNOPSIS
Restaurar um servidor de um backup existente

## SINTAXE

```
Restore-AzMySqlFlexibleServer -Name <String> -ResourceGroupName <String> -InputObject <IServerAutoGenerated>
 -RestorePointInTime <DateTime> [-SubscriptionId <String>] [-Zone <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Restaurar um servidor de um backup existente

## EXEMPLOS

### Exemplo 1: Restaurar o servidor MySql usando a Restauração pointintime
```powershell
PS C:\> $restorePointInTime = (Get-Date).AddMinutes(-10)
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | Restore-AzMySqlFlexibleServer -Name mysql-test-restore -ResourceGroupName PowershellMySqlTest -RestorePointInTime $restorePointInTime 

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName   SkuTier       
----               -------- ------------------ ------- ----------------------- -------   -------       
mysql-test-restore eastus   mysql_test         5.7     10240                   Standard_D2ds_v4 GeneralPurpose
```

Esses cmdlets restauram o servidor MySql usando a Restauração PointInTime.

## NOTES

ALIASES

PROPRIEDADES DE PARÂMETRO COMPLEXO

Para criar os parâmetros descritos abaixo, construa uma tabela de hash contendo as propriedades apropriadas. Para obter informações sobre tabelas de hash, execute Get-Help about_Hash_Tables.


INPUTOBJECT <IServerAutoGenerated> : O objeto do servidor de origem a ser restaurado.
  - `Location <String>`: A localização geográfica onde o recurso mora
  - `[Tag <ITrackedResourceTags>]`: Marcas de recurso.
    - `[(Any) <String>]`: Isso indica que qualquer propriedade pode ser adicionada a esse objeto.
  - `[AdministratorLogin <String>]`: O nome de logon do administrador de um servidor. Só pode ser especificado quando o servidor está sendo criado (e é necessário para a criação).
  - `[AdministratorLoginPassword <SecureString>]`: A senha do logon do administrador (necessário para a criação do servidor).
  - `[AvailabilityZone <String>]`: informações da zona de disponibilidade do servidor.
  - `[CreateMode <CreateMode?>]`: O modo para criar um novo servidor MySQL.
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`: ID de recurso do braço de sub-rede delegado.
  - `[HaEnabled <HaEnabledEnum?>]`: Habilitar HA ou não para um servidor.
  - `[IdentityType <ResourceIdentityType?>]`: O tipo de identidade.
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`: Status mostrando se a criptografia de infraestrutura habilitada pelo servidor.
  - `[MaintenanceWindowCustomWindow <String>]`: indica se a janela personalizada está habilitada ou desabilitada
  - `[MaintenanceWindowDayOfWeek <Int32?>]`: dia da semana para janela de manutenção
  - `[MaintenanceWindowStartHour <Int32?>]`: hora de início da janela de manutenção
  - `[MaintenanceWindowStartMinute <Int32?>]`: minuto inicial para a janela de manutenção
  - `[PropertiesTag <IServerPropertiesTags>]`: Metadados específicos do aplicativo na forma de pares de valores-chave.
    - `[(Any) <String>]`: Isso indica que qualquer propriedade pode ser adicionada a esse objeto.
  - `[ReplicationRole <String>]`: a função de replicação.
  - `[RestorePointInTime <DateTime?>]`: Tempo de criação de ponto de restauração (formato ISO8601), especificando a hora de restauração.
  - `[SkuName <String>]`: O nome do sku, por exemplo, Standard_D32s_v3.
  - `[SkuTier <SkuTier?>]`: A camada da SKU específica, por exemplo, GeneralPurpose.
  - `[SourceServerId <String>]`: A id do servidor MySQL de origem.
  - `[SslEnforcement <SslEnforcementEnum?>]`: Habilitar a imposição ssl ou não quando se conectar ao servidor.
  - `[StorageProfileBackupRetentionDay <Int32?>]`: Dias de retenção de backup para o servidor.
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`: Habilitar o Crescimento Automático de Armazenamento.
  - `[StorageProfileStorageIop <Int32?>]`: IOPS de armazenamento para um servidor.
  - `[StorageProfileStorageMb <Int32?>]`: Armazenamento máximo permitido para um servidor.
  - `[Version <ServerVersion?>]`: Versão do servidor.
