# Get-AzKeyVaultCertificateContact

## SYNOPSIS
Obtém contatos registrados para notificações de certificado para um cofre de chaves.

## SINTAXE

### VaultName (Padrão)
```
Get-AzKeyVaultCertificateContact [-VaultName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByInputObject
```
Get-AzKeyVaultCertificateContact [-InputObject] <PSKeyVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceId
```
Get-AzKeyVaultCertificateContact [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzKeyVaultCertificateContact** obtém contatos registrados para notificações de certificado para um cofre de chaves no Azure Key Vault.

## EXEMPLOS

### Exemplo 1: Obter todos os contatos de certificado
```powershell
PS C:\> $Contacts = Get-AzKeyVaultCertificateContact -VaultName "Contoso"

Email                   VaultName
-----                   ---------
username@microsoft.com  Contoso
username1@microsoft.com Contoso
```

Esse comando obtém todos os contatos dos objetos de certificado no cofre de chaves contoso e os armazena na variável $Contacts.
