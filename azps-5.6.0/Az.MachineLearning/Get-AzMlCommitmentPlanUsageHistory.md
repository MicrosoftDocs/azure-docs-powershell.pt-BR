# Get-AzMlCommitmentPlanUsageHistory

## SYNOPSIS
Recupera informações de histórico de uso para um plano de compromisso especificado.

## SINTAXE

```
Get-AzMlCommitmentPlanUsageHistory -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Recupera informações de histórico de uso para um plano de compromisso especificado, incluindo recursos usados e recursos restantes dentro do plano.

## EXEMPLOS

### Exemplo 1: obter histórico de uso para um plano de compromisso específico
```
Get-AzMlCommitmentPlanUsageHistory -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName"
```
