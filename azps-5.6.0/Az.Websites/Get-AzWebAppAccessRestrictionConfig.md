# Get-AzWebAppAccessRestrictionConfig

## SYNOPSIS
Obtém a configuração de Restiction do Access para um Aplicativo Web do Azure.

## SINTAXE

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzWebAppAccessRestrictionConfig obtém** a config Restrição de Acesso sobre um Aplicativo Web do Azure.

## EXEMPLOS

### Exemplo 1: Obter uma Configuração de Restrição de Acesso a Aplicativo Web de um grupo de recursos
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Este comando obtém a configuração de restrição de acesso de um Aplicativo Web chamado ContosoSite que pertence ao grupo de recursos Default-Web-WestUS.
