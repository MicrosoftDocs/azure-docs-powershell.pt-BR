# Enter-AzWebAppContainerPSSession

## SYNOPSIS
Abre uma sessão remota do PowerShell no contêiner do Windows especificado em um determinado site ou slot e determinado grupo de recursos

## SINTAXE

### S1 (Padrão)
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [[-SlotName] <String>] [-ResourceGroupName] <String>
 [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Enter-AzWebAppContainerPSSession [-PassThru] [-Force] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
abre uma sessão remota do PowerShell no contêiner do Windows especificado em um determinado site ou slot e determinado grupo de recursos

## EXEMPLOS

### Exemplo 1
```
PS C:\> Enter-AzWebAppContainerPSSession -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Este comando abre uma sessão remota do PowerShell no aplicativo de contêiner do Windows ContosoASP
