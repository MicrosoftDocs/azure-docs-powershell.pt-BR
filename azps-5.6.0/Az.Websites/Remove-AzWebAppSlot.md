# Remove-AzWebAppSlot

## SYNOPSIS

## SINTAXE

### S1
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzWebAppSlot [-Force] [-AsJob] [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzWebAppSlot** remove um Slot do Aplicativo Web do Azure fornecido para o grupo de recursos e o nome do Aplicativo Web.
Esse cmdlet, por padrão, também remove todos os slots e métricas.

## EXEMPLOS

### Exemplo 1: Remover um slot de aplicativo Web
```
PS C:\>Remove-AzWebAppSlot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite" -Slot "Slot001"
```

Este comando remove o Slot chamado Slot001 associado ao Web App ContosoSite que pertence ao grupo de recursos chamado Default-Web-WestUS.
