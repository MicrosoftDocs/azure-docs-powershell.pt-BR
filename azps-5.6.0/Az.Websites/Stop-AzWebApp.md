# Stop-AzWebApp

## SYNOPSIS
Interrompe um Aplicativo Web do Azure.

## SINTAXE

### S1
```
Stop-AzWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Stop-AzWebApp [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Stop-AzWebApp** interrompe um Aplicativo Web do Azure.

## EXEMPLOS

### Exemplo 1: Parar um aplicativo Web
```
PS C:\>Stop-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp"
```

Este comando interrompe o Aplicativo Web chamado ContosoWebApp que pertence ao grupo de recursos chamado Default-Web-WestUS.
