# Get-AzWebAppBackup

## SYNOPSIS

## SINTAXE

### FromResourceName
```
Get-AzWebAppBackup [-BackupId] <String> [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FromWebApp
```
Get-AzWebAppBackup [-BackupId] <String> [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzWebAppBackup** obtém o backup especificado de um Aplicativo Web do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\>Get-AzWebAppBackup -ResourceGroupName "Default-Web-WestUS" -Name "WebAppStandard" -BackupId "12345"
```

Este comando obtém o backup com a ID "12345" do Aplicativo Web chamado WebAppStandard que pertence ao grupo de recursos Default-Web-WestUS.
