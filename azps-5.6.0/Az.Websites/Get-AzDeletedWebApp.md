# Get-AzDeletedWebApp

## SYNOPSIS
Obtém aplicativos Web excluídos na assinatura.

## SINTAXE

```
Get-AzDeletedWebApp [[-ResourceGroupName] <String>] [[-Name] <String>] [[-Slot] <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzDeletedWebApp** retorna todos os aplicativos Web excluídos na assinatura. Os aplicativos excluídos podem, opcionalmente, ser filtrados por grupo de recursos, nome e slot. Pode haver mais de um aplicativo excluído com o mesmo nome e grupo de recursos. Verifique o DeletionTime para distinguir aplicativos excluídos que compartilham o mesmo nome.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzDeletedWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Este comando obtém os aplicativos excluídos chamados ContosoSite pertencentes ao grupo de recursos Default-Web-WestUS.
