# Remove-AzAppServicePlan

## SYNOPSIS
Remove um plano de Serviço de Aplicativo do Azure.

## SINTAXE

### S1
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-AppServicePlan] <PSAppServicePlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzAppServicePlan** remove um plano de Serviço de Aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: Remover um plano de Serviço de Aplicativo
```
PS C:\>Remove-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Este comando remove o plano de Serviço de Aplicativo do Azure chamado ContosoASP que pertence ao grupo de recursos chamado Default-Web-WestUS.
