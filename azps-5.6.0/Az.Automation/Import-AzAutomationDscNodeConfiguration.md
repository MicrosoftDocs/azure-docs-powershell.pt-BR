# Import-AzAutomationDscNodeConfiguration

## SYNOPSIS
Importa um documento MOF como uma configuração de nó DSC na Automação.

## SINTAXE

```
Import-AzAutomationDscNodeConfiguration -Path <String> -ConfigurationName <String> [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-IncrementNodeConfigurationBuild] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Import-AzAutomationDscConfiguration** importa um documento de configuração do Formato de Objeto Gerenciado (MOF) para a Automação do Azure como uma configuração de nó DSC (Configuração de Estado Desejado).
Especifique o caminho de um arquivo .mof.

## EXEMPLOS

### Exemplo 1: Importar uma configuração de nó DSC para Automação
```
PS C:\>Import-AzAutomationDscNodeConfiguration -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -ConfigurationName "ContosoConfiguration" -Path "C:\DSC\webserver.mof" -Force
```

Este comando importa uma configuração de nó DSC do arquivo chamado webserver.mof para a conta de automação chamada Contoso17, sob a configuração do DSC ContosoConfiguration.
O comando especifica o parâmetro *Force.*
Se houver uma configuração de nó DSC existente chamada ContosoConfiguration.webserver, esse comando a substituirá.

### Exemplo 2: Importe uma configuração de nó DSC para Automação e crie uma nova versão de com build e não sobrescreva NodeConfiguration existente.
```
PS C:\>Import-AzAutomationDscNodeConfiguration -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -ConfigurationName "ContosoConfiguration" -Path "C:\DSC\webserver.mof" -IncrementNodeConfigurationBuild
```

Este comando importa uma configuração de nó DSC do arquivo chamado webserver.mof para a conta de automação chamada Contoso17, sob a configuração do DSC ContosoConfiguration.
O comando especifica o parâmetro *Force.*
Se houver uma configuração de nó DSC existente chamada ContosoConfiguration.webserver, este comando adiciona uma nova versão de com build com o nome ContosoConfiguration[2].webserver.
