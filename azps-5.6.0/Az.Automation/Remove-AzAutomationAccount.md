# Remove-AzAutomationAccount

## SYNOPSIS
Remove uma conta de automação.

## SINTAXE

```
Remove-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzAutomationAccount** remove uma conta de Automação do Azure de um grupo de recursos.
Para obter mais informações sobre contas de automação, consulte New-AzAutomationAccount cmdlet.

## EXEMPLOS

### Exemplo 1: Remover uma conta de automação
```
PS C:\>Remove-AzAutomationAccount -Name "ContosoAutomationAccount" -Force -ResourceGroupName "ResourceGroup01"
```

Este comando remove uma conta de automação chamada ContosoAutomationAccount sem solicitar a validação do usuário.
