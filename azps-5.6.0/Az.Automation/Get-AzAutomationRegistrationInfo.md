# Get-AzAutomationRegistrationInfo

## SYNOPSIS
Obtém informações de registro para a integração de um nó DSC ou de um trabalhador híbrido para Automação.

## SINTAXE

```
Get-AzAutomationRegistrationInfo [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzAutomationRegistrationInfo** obtém o ponto de extremidade e as chaves necessárias para integrar um nó DSC (Configuração de Estado Desejado) ou um trabalhador híbrido em uma conta de Automação do Azure.

## EXEMPLOS

### Exemplo 1: Obter informações de registro
```
PS C:\>Get-AzAutomationRegistrationInfo -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Este comando obtém as informações de registro da conta de Automação chamada AutomationAccount01 no Grupo de Recursos chamado ResourceGroup01.
