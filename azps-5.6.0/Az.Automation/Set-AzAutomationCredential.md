# Set-AzAutomationCredential

## SYNOPSIS
Modifica uma credencial de automação.

## SINTAXE

```
Set-AzAutomationCredential [-Name] <String> [-Description <String>] [-Value <PSCredential>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzAutomationCredential** modifica uma credencial como um **objeto PSCredential** no Azure Automation.

## EXEMPLOS

### Exemplo 1: atualizar uma credencial
```
PS C:\>$User = "Contoso\DChew"
PS C:\> $Password = ConvertTo-SecureString "Password" -AsPlainText -Force
PS C:\> $Credential = New-Object -TypeName System.Management.Automation.PSCredential -ArgumentList $User, $Password
PS C:\> Set-AzAutomationCredential -AutomationAccountName "Contoso17" -Name "ContosoCredential" -ResourceGroupName "ResourceGroup01" -Value $Credential
```

O primeiro comando atribui um nome de usuário à variável $User de usuário.
O segundo comando converte uma senha de texto simples em uma cadeia de caracteres segura usando o cmdlet ConvertTo-SecureString de texto.
O comando armazena esse objeto na variável $Password.
O terceiro comando cria uma credencial com base $User e $Password e a armazena na variável $Credential.
O comando final modifica a credencial de automação chamada ContosoCredential para usar a credencial em $Credential.
