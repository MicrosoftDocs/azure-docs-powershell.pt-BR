# Get-AzAutomationDscCompilationJobOutput

## SYNOPSIS
Obtém os fluxos de log de um trabalho de compilação de DSC de automação.

## SINTAXE

```
Get-AzAutomationDscCompilationJobOutput [-Id] <Guid> [-Stream <CompilationJobStreamType>]
 [-StartTime <DateTimeOffset>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzAutomationDscCompilationJobOutput** obtém os registros de fluxo de um trabalho de compilação de DSC (Configuração de Estado Desejado) do APS na Automação do Azure.

## EXEMPLOS

### Exemplo 1: Obter os logs para um trabalho de compilação DSC
```
PS C:\>$Jobs = Get-AzAutomationDscCompilationJob -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
PS C:\> $Jobs[0] | Get-AzAutomationDscCompilationJobOutput -Stream "Any"
```

O primeiro comando obtém os trabalhos de compilação na conta de automação chamada Contoso17 usando o cmdlet Get-AzAutomationDscCompilationJob de compilação.
O comando armazena esses objetos na variável $Jobs.
O segundo comando obtém a saída de trabalho de compilação para qualquer fluxo para o primeiro membro da matriz $Jobs de compilação.
