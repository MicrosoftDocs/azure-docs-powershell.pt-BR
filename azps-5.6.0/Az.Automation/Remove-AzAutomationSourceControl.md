# Remove-AzAutomationSourceControl

## SYNOPSIS
Remove um controle de origem da Automação do Azure.

## SINTAXE

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O Remove-AzAutomationSourceControl cmdlet remove um controle de origem da Automação do Azure.

## EXEMPLOS

### Exemplo 1
Este comando remove o controle de origem de automação chamado VSTSNative na conta chamada devAccount.
Este comando especifica o *parâmetro Force.* Portanto, ele não solicita a confirmação.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
