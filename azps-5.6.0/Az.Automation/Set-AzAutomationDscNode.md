# Set-AzAutomationDscNode

## SYNOPSIS
Modifica a configuração do nó para a que um nó DSC é mapeado.

## SINTAXE

```
Set-AzAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzAutomationDscNode** modifica uma configuração de nó DSC (Configuração de Estado Desejado) do APS.
A Automação do Azure armazena a configuração do nó DSC como um documento de configuração do Formato de Objeto Gerenciado (MOF).

## EXEMPLOS

### Exemplo 1: Modificar o mapeamento de configuração do nó
```
PS C:\>Set-AzAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Este comando atribui a configuração do nó chamada Contoso.NodeConfiguration01 ao nó que tem o GUID especificado.
