# Get-AzAutomationSourceControlSyncJobOutput

## SYNOPSIS
Obtém a saída de um trabalho de sincronização de controle de origem de automação do Azure.

## SINTAXE

```
Get-AzAutomationSourceControlSyncJobOutput -SourceControlName <String> -JobId <Guid>
 [-Stream <SourceControlSyncJobStreamType>] [-StreamId <String>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzAutomationSourceControlSyncJobOutput** obtém a saída para um trabalho de sincronização de controle de origem de automação do Azure Automation.

## EXEMPLOS

### Exemplo 1
Este comando obtém a saída do trabalho de sincronização de controle de origem com a id 08d6d266-27b6-463c-beea-bc48a67ace15 para o controle de origem VSTSNative.

PS C:\> Get-AzAutomationSourceControlSyncJobOutput -ResourceGroupName "rg1" `
                                                        -AutomationAccountName "devAccount" `

                                                        -Stream Output | ForEach-Object {$_.summary}
