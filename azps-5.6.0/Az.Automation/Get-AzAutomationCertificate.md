# Get-AzAutomationCertificate

## SYNOPSIS
Obtém certificados de automação.

## SINTAXE

### ByAll (Padrão)
```
Get-AzAutomationCertificate [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByCertificateName
```
Get-AzAutomationCertificate [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzAutomationCertificate** obtém um ou mais certificados de Automação do Azure.
Por padrão, esse cmdlet obtém todos os certificados.
Especifique o nome de um certificado para obter um certificado específico.

## EXEMPLOS

### Exemplo 1: Obter todos os certificados
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17"
```

Este comando obtém metadados para todos os certificados na conta de automação chamada Contoso17.

### Exemplo 2: Obter um certificado
```
PS C:\>Get-AzAutomationCertificate -ResourceGroupName "ResourceGroup07" -AutomationAccountName "Contoso17" -Name "ContosoCertificate"
```

Este comando obtém metadados para o certificado chamado ContosoCertificate.
