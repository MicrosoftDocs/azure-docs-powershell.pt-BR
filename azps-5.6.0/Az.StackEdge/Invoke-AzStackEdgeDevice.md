# Invoke-AzStackEdgeDevice

## SYNOPSIS
Invoca ações específicas no dispositivo.

## DESCRIPTION
O cmdlet **Invoke-AzStackEdgeDevice** invoca ações para verificar, baixar e instalar as atualizações no dispositivo de Borda de Pilha. Uma verificação automática é executado no dispositivo diariamente, você pode acionar a verificação explicitamente executando este cmdlet.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### Exemplo 2
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### Exemplo 3
```powershell
PS C:\> Invoke-AzStackEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
