# Get-AzDataShareSubscription

## SYNOPSIS
Obtém informações sobre a assinatura de compartilhamento na conta de compartilhamento de dados.

## SINTAXE

### ByFieldsParameterSet (Padrão)
```
Get-AzDataShareSubscription -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareSubscription -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzDataShareSubscription** fornece informações sobre a assinatura de compartilhamento em uma conta de compartilhamento de dados. Se o nome da assinatura de compartilhamento for fornecido, o cmdlet dará informações sobre a assinatura de compartilhamento específica. Caso contrário, o cmdlet fornece uma lista de assinaturas de compartilhamento na conta de compartilhamento de dados.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzDataShareSubscription -ResourceGroupName "ADS" -AccountName "WikiAds" -Name "AdsShareSubscription"

ShareSender             : adsprovider@microsoft.com
ShareSenderCompanyName  : ADS Test

```

Este comando fornece informações sobre o compartilhamento de assinatura AdsShareSubscription na conta de compartilhamento de dados WikiAds.
