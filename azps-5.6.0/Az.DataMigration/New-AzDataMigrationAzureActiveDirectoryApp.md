# New-AzDataMigrationAzureActiveDirectoryApp

## SYNOPSIS
Crie uma nova instância Detalhes do Aplicativo DataMigration do Azure ActiveDirectory.

## SINTAXE

```
New-AzDataMigrationAzureActiveDirectoryApp -ApplicationId <String> -AppKey <SecureString>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Crie uma nova instância Detalhes do Aplicativo DataMigration do Azure ActiveDirectory.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $secpasswd = ConvertTo-SecureString "Your Secret Key Here" -AsPlainText -Force
C:\> New-AzDmsAadApp -ApplicationId "Your AppId/Service Principal ID here" -AppKey $secpasswd
```
ApplicationId : "Sua Id da Entidade de Serviço/AppId aqui" AppKey : System.Security.SecureString TenantId : "Id do locatário"
