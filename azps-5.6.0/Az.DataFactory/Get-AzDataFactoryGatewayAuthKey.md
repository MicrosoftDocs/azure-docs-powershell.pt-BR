# Get-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Obtém a chave de auth do gateway para uma Fábrica de Dados do Azure.

## SINTAXE

### ByFactoryName (Padrão)
```
Get-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzDataFactoryGatewayAuthKey** obtém a chave de entrada de gateway para um gateway especificado da Fábrica de Dados do Azure.
Você registra o gateway com um serviço de nuvem usando essa chave1 ou chave2 dessa tecla de auth.

## EXEMPLOS

### Exemplo 1: obtém a tecla de auth de um gateway
```
PS C:\> Get-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF

       gGHqUg1THLqA=
Key2 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@kFXxBdFCEBeL7LPB3hA3LqLd1uNFbyv
       YmWxtV4WD3JQ=
```

Este comando obtém a chave de entrada de gateway para o gateway de fábrica de dados chamado MyGateway.
