# New-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Cria uma chave de auth para um Gateway de Fábrica de Dados do Azure.

## SINTAXE

### ByFactoryName (Padrão)
```
New-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String> [-KeyName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzDataFactoryGatewayAuthKey** cria a chave de auth do gateway para um gateway especificado da Fábrica de Dados do Azure.
Você registra o gateway com um serviço de nuvem usando essa chave.

## EXEMPLOS

### Exemplo 1: cria uma chave de auth de gateway para Key1
```
PS C:\> New-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF -KeyName key1
Key1 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@BH0EV9hu/o2IYGQzfYYD203XhdS6Tty
       fkYwYFbG6wBU=
Key2 :
```

Este comando cria uma chave de auth de gateway de Key1 para o gateway de fábrica de dados chamado MyGateway.
