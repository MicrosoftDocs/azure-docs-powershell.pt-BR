# Módulo Az.DataBox
## Descrição
Módulo para Criar, Cancelar, Excluir e Buscar Trabalhos de Caixa de Dados

## Az.DataBox Cmdlets
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Obtém as credenciais de caixa de dados de um trabalho específico

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Obtém informações sobre Trabalhos de Caixa de Dados

### [New-AzDataBoxJob](New-AzDataBoxJob.md)
Cria um novo trabalho de caixa de dados usando os parâmetros especificados

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Exclui o trabalho da caixa de dados

### [Stop-AzDataBoxJob](Stop-AzDataBoxJob.md)
Cancela um trabalho de caixa de dados em andamento se o trabalho estiver em estado cancelável.
