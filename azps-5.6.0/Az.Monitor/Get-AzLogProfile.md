# Get-AzLogProfile

## SYNOPSIS
Obtém um perfil de log.

## SINTAXE

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzLogProfile** obtém um perfil de log.
