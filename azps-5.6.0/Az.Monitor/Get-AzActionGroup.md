# Get-AzActionGroup

## SYNOPSIS
Obtém grupos de ações.

## SINTAXE

### BySubscriptionOrResourceGroup (Padrão)
```
Get-AzActionGroup [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzActionGroup -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzActionGroup** obtém um ou mais grupos de ações.

## EXEMPLOS

### Exemplo 1: Obter um grupo de ações por ID de assinatura
```
PS C:\>Get-AzActionGroup
```

Este comando lista todo o grupo de ações da assinatura atual.

### Exemplo 2: Obter grupos de ações para o grupo de recursos determinado
```
PS C:\>Get-AzActionGroup -ResourceGroup "Default-activityLogAlerts"
```

Este comando lista grupos de ações para o grupo de recursos determinado.

### Exemplo 3: Obter um grupo de ações.
```
PS C:\>Get-AzActionGroup -ResourceGroup "Default-activityLogAlerts" -Name "actionGroup1"
```

Este comando lista um grupo de ações (uma lista com um único elemento).
