# New-AzActionGroup

## SYNOPSIS
Cria um objeto de referência ActionGroup na memória.

## SINTAXE

```
New-AzActionGroup -ActionGroupId <String>
 [-WebhookProperty <System.Collections.Generic.Dictionary`2[System.String,System.String]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzActionGroup** cria um objeto de referência do grupo de ações na memória.

## EXEMPLOS

### Exemplo 1: Criar um objeto de referência de grupo de ações na memória
```
PS C:\>$dict = New-Object "System.Collections.Generic.Dictionary``2[System.String,System.String]"
PS C:\>$dict.Add('key1', 'value1')
PS C:\>$actionGrp1 = New-AzActionGroup -ActionGroupId 'actiongr1' -WebhookProperty $dict
```
