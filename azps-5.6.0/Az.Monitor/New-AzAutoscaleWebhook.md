# New-AzAutoscaleWebhook

## SYNOPSIS
Cria um webhook de escala automática.

## SINTAXE

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzAutoscaleWebhook** cria um webhook de escala automática.

## EXEMPLOS

### Exemplo 1

Cria um webhook de escala automática. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
