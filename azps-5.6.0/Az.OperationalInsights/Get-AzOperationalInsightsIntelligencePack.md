# Get-AzOperationalInsightsIntelligencePack

## SYNOPSIS
Obtém os Pacotes de Inteligência disponíveis.

## SINTAXE

```
Get-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzOperationalInsightsIntelligencePack** obtém os Pacotes de Inteligência disponíveis.

## EXEMPLOS

### Exemplo 1: Obter Pacotes de Inteligência
```
PS C:\>Get-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Este comando obtém os Pacotes de Inteligência disponíveis.
