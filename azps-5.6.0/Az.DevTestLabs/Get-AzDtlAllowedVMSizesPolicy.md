# Get-AzDtlAllowedVMSizesPolicy

## SYNOPSIS
Obtém a política de tamanhos de máquina virtual permitido de um laboratório em Laboratórios DevTest.

## SINTAXE

```
Get-AzDtlAllowedVMSizesPolicy [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzDtlAllowedVMSizesPolicy** obtém a política de tamanhos de máquina virtual permitidos, que permite especificar uma lista de tamanhos de máquina virtual permitidos no laboratório.
O cmdlet retorna o status habilitado ou desabilitado da política e uma lista de todos os tamanhos de máquina virtual permitidos que você definiu na política especificada.
