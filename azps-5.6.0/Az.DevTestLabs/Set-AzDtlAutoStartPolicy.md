# Set-AzDtlAutoStartPolicy

## SYNOPSIS
Define a política de início automático de um laboratório em DevTest Labs.

## SINTAXE

### Habilitar (Padrão)
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Desabilitar
```
Set-AzDtlAutoStartPolicy [[-Time] <DateTime>] [[-Days] <DayOfWeek[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzDtlAutoStartPolicy** define a política de início automático de um laboratório, que permite que as máquinas virtuais de laboratório sejam agendadas para início automático.
O cmdlet usa o grupo de recursos especificado e o nome do laboratório para definir a política.

## EXEMPLOS

### Exemplo 1

Define a política de início automático de um laboratório em DevTest Labs. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoStartPolicy -Days Sunday -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
