# Set-AzPolicyDefinition

## SYNOPSIS
Modifica uma definição de política.

## DESCRIPTION
O cmdlet **Set-AzPolicyDefinition** modifica uma definição de política.

## EXEMPLOS

### Exemplo 1: atualizar a descrição de uma definição de política
```
PS C:\> $PolicyDefinition = Get-AzPolicyDefinition -Name 'VMPolicyDefinition'
PS C:\> Set-AzPolicyDefinition -Id $PolicyDefinition.ResourceId -Description 'Updated policy to not allow virtual machine creation'
```

O primeiro comando obtém uma definição de política chamada VMPolicyDefinition usando o cmdlet Get-AzPolicyDefinition de segurança.
O comando armazena esse objeto na $PolicyDefinition variável.
O segundo comando atualiza a descrição da definição de política identificada pela **propriedade ResourceId** do $PolicyDefinition.

### Exemplo 2: atualizar o modo de uma definição de política
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Mode 'All'
```

Este comando atualiza a definição de política chamada VMPolicyDefinition usando o cmdlet Set-AzPolicyDefinition para definir sua propriedade mode como 'All'.

### Exemplo 3: Atualizar os metadados de uma definição de política
```
PS C:\> Set-AzPolicyDefinition -Name 'VMPolicyDefinition' -Metadata '{"category":"Virtual Machine"}'
```
