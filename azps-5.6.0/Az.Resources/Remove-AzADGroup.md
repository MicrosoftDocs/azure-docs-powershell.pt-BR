# Remove-AzADGroup

## SYNOPSIS
Exclui um grupo do Active Directory.

## SINTAXE

### ObjectIdParameterSet (Padrão)
```
Remove-AzADGroup -ObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Remove-AzADGroup -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzADGroup -InputObject <PSADGroup> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Exclui um grupo do Active Directory.

## EXEMPLOS

### Exemplo 1: Remover um grupo por id de objeto

```powershell
PS C:\> Remove-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Remove o grupo com a id do objeto '85F89C90-780E-4AA6-9F4F-6F268D322EEE' do locatário.

### Exemplo 2: Remover um grupo canalização

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzADGroup
```

Obtém o grupo com a id do objeto '85F89C90-780E-4AA6-9F4F-6F268D322EEE' e canalizará Remove-AzADGroup remover o grupo do locatário.
