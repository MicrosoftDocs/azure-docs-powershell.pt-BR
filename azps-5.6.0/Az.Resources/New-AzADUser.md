# New-AzADUser

## SYNOPSIS
Cria um novo usuário do Active Directory.

## SINTAXE

```
New-AzADUser -DisplayName <String> -UserPrincipalName <String> -Password <SecureString> [-ImmutableId <String>]
 -MailNickname <String> [-ForceChangePasswordNextLogin] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cria um novo usuário do Active Directory (conta de trabalho/escola também popularmente conhecida como org-id).
Para obter mais informações: https://msdn.microsoft.com/en-us/library/azure/ad/graph/api/users-operations#CreateUser

## EXEMPLOS

### Exemplo 1: Criar um novo usuário do AD
```powershell
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> New-AzADUser -DisplayName "MyDisplayName" -UserPrincipalName "myemail@domain.com" -Password $SecureStringPassword -MailNickname "MyMailNickName"
```

Cria um novo usuário do AD com o nome "MyDisplayName" e o nome principal do usuário myemail@domain.com " em um locatário.
