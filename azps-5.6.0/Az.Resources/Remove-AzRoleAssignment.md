# Remove-AzRoleAssignment

## SYNOPSIS
Remove uma atribuição de função para a entidade de entidade especificada que é atribuída a uma função específica em um escopo específico.

## DESCRIPTION
Use o Remove-AzRoleAssignment para revogar o acesso a qualquer entidade principal em determinado escopo e função.
O objeto da atribuição, ou seja, a entidade deve ser especificada.
A entidade pode ser um usuário (use parâmetros SignInName ou ObjectId para identificar um usuário), grupo de segurança (use o parâmetro ObjectId para identificar um grupo) ou entidade de serviço (use parâmetros ServicePrincipalName ou ObjectId para identificar um ServicePrincipal.
A função à que a entidade é atribuída DEVE ser especificada usando o parâmetro RoleDefinitionName.
O escopo da atribuição PODE ser especificado e, se não for especificado, o padrão é o escopo da assinatura, ou seja, ele tentará excluir uma atribuição para a entidade e a função especificadas no escopo da assinatura.
O escopo da atribuição pode ser especificado usando um dos seguintes parâmetros.
a.
Escopo - Este é o escopo totalmente qualificado começando com \<subscriptionId\> /subscriptions/b.
ResourceGroupName - Nome de qualquer grupo de recursos sob a assinatura.
c.
ResourceName, ResourceType, ResourceGroupName e (opcionalmente) ParentResource - Identifica um determinado recurso sob a assinatura.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzRoleAssignment -ResourceGroupName rg1 -SignInName john.doe@contoso.com -RoleDefinitionName Reader
```

Remove uma atribuição de função para quem é atribuído à john.doe@contoso.com função Leitor no escopo do grupo de recursos rg1.

### Exemplo 2
```
PS C:\> Remove-AzRoleAssignment -ObjectId 36f81fc3-b00f-48cd-8218-3879f51ff39f -RoleDefinitionName Reader
```

Remove a atribuição de função para a entidade de grupo identificada pela ObjectId e atribuída à função Leitor.
O padrão é usar a assinatura atual como escopo para encontrar a atribuição a ser excluída.

### Exemplo 3
```
PS C:\> $roleassignment = Get-AzRoleAssignment |Select-Object -First 1 -Wait
PS C:\> Remove-AzRoleAssignment -InputObject $roleassignment
```

Remove o primeiro objeto de atribuição de função que é buscado do Get-AzRoleAssignment commandlet.
