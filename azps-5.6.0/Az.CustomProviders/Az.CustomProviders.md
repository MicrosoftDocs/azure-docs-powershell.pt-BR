# Módulo Az.CustomProviders
## Descrição
Microsoft Azure PowerShell: cmdlets de provedores personalizados

## Az.CustomProviders Cmdlets
### [Get-AzCustomProvider](Get-AzCustomProvider.md)
Obtém o manifesto do provedor de recursos personalizado.

### [Get-AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
Obter uma associação.

### [New-AzCustomProvider](New-AzCustomProvider.md)
Cria ou atualiza o provedor de recursos personalizado.

### [New-AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
Criar ou atualizar uma associação.

### [Remove-AzCustomProvider](Remove-AzCustomProvider.md)
Exclui o provedor de recursos personalizado.

### [Remove-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
Excluir uma associação.

### [Update-AzCustomProvider](Update-AzCustomProvider.md)
Atualiza um provedor de recursos personalizados existente.
O único valor que pode ser atualizado por meio de PATCH atualmente são as marcas.
