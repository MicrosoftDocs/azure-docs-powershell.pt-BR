# Get-AzMapsAccountKey

## SYNOPSIS
Obtém as chaves da API de uma conta.
Essas chaves são o mecanismo de autenticação usado em chamadas subsequentes para mapas do Azure.

## SINTAXE

### NameParameterSet (Padrão)
```
Get-AzMapsAccountKey [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### InputObjectParameterSet
```
Get-AzMapsAccountKey [-InputObject <PSMapsAccount>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzMapsAccountKey [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Get-AzMapsAccountKey cmdlet obtém as chaves da API de uma conta provisionada do Azure Maps.
Uma conta do Azure Maps tem duas chaves de API: Primária e Secundária.
As teclas habilitam a interação com o ponto de extremidade da conta do Azure Maps.
Use New-AzMapsAccountKey (New-AzMapsAccountKey.md)para regenerar uma chave.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzMapsAccountKey -ResourceGroupName MyResourceGroup -Name MyAccount

PrimaryKey                                  SecondaryKey
----------                                  ------------
******************************************* *******************************************
```

Retorna as chaves da conta chamada MyAccount no grupo de recursos MyResourceGroup.

### Exemplo 2
```powershell
PS C:\> Get-AzMapsAccountKey -ResourceId /subscriptions/21a9967a-e8a9-4656-a70b-96ff1c4d05a0/resourceGroups/MyResourceGroup/providers/Microsoft.Maps/accounts/MyAccount

PrimaryKey                                  SecondaryKey
----------                                  ------------
******************************************* *******************************************
```

Retorna as chaves da Conta de Mapas do Azure especificada.
