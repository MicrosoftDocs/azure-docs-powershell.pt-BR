# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Obtém um ou mais Consultores para um banco de dados do Azure SQL.

## SINTAXE

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabaseAdvisor** obtém um ou mais Consultores de Banco de Dados do Azure SQL para um banco de dados do Azure SQL.
