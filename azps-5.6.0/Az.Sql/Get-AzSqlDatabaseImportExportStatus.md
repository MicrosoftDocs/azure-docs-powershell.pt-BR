# Get-AzSqlDatabaseImportExportStatus

## SYNOPSIS
Obtém os detalhes de uma importação ou exportação de um banco de dados do Azure SQL.

## SINTAXE

```
Get-AzSqlDatabaseImportExportStatus [-OperationStatusLink] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabaseImportExportStatus** obtém detalhes de uma importação de arquivo bacpac de uma conta de armazenamento para um banco de dados do Azure SQL ou uma exportação de um banco de dados do Azure SQL como um arquivo de bacpac para uma conta de armazenamento.
Esse cmdlet também é suportado pelo serviço SQL Server Banco de Dados de Extensão no Azure.

## EXEMPLOS

### Exemplo 1: Obter o status de importação e exportação de um banco SQL banco de dados
```
PS C:\>Get-AzSqlDatabaseImportExportStatus -OperationStatusLink "https://management.contoso.com/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/resource01/providers/Microsoft.Sql/servers/server01/databases/database01/importExportOperationResults/00000000-000-0000-0000-000000000000?api-version=2014-04-01"
OperationStatusLink : 
ErrorMessage        : 
LastModifiedTime    : 4/15/2016 10:16:14 PM
QueuedTime          : 4/15/2016 10:16:13 PM
StatusMessage       : Running, Progress = 5.00 %
Status              : InProgress
```

Esse comando obtém o status de uma solicitação de importação ou exportação para um banco de dados na URL especificada.
