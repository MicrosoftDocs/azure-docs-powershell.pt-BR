# Get-AzSqlServerDisasterRecoveryConfigurationActivity

## SYNOPSIS
Obtém atividade para uma configuração de recuperação do sistema do servidor de banco de dados.

## SINTAXE

```
Get-AzSqlServerDisasterRecoveryConfigurationActivity [-ServerName] <String>
 -ServerDisasterRecoveryConfigurationName <String> [-OperationId <Guid>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlServerDisasterRecoveryConfigurationActivity** obtém atividade para uma configuração de recuperação do sistema SQL servidor de banco de dados.
