# Get-AzSqlElasticPoolDatabase

## SYNOPSIS
Obtém bancos de dados elásticas em um pool elástica e seus valores de propriedade.

## SINTAXE

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlElasticPoolDatabase** obtém bancos de dados elásticas em um pool elástica e seus valores de propriedade.
Você pode especificar o nome de um banco de dados elástica no Banco de Dados do Azure SQL para ver os valores de propriedade apenas para esse banco de dados.

## EXEMPLOS

### Exemplo 1: Obter todos os bancos de dados em um pool elástica
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Este comando obtém todos os bancos de dados em um pool elástica chamado ElasticPool01.

### Exemplo 2: Obter todos os bancos de dados em um pool elástica usando a filtragem
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Esse comando obtém todos os bancos de dados em um pool elástica chamado ElasticPool01 que começa com "Database".
