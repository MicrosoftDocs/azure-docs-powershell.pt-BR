# Get-AzSqlDatabaseRestorePoint

## SYNOPSIS
Recupera os pontos de restauração distintos dos quais um Data Warehouse SQL pode ser restaurado.

## SINTAXE

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabaseRestorePoint** recupera os pontos de restauração distintos dos que um Data Warehouse do Azure SQL pode ser restaurado.
Para um banco de dados do Azure SQL, a janela de restauração é contínua.
Isso significa que qualquer ponto no tempo no período de retenção de backup do banco de dados pode ser usado como um ponto de restauração.
Esse cmdlet também é suportado pelo serviço SQL Server Banco de Dados de Extensão no Azure.

## EXEMPLOS

### Exemplo 1: Obter todos os pontos de restauração
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

Este comando retorna todos os pontos de restauração disponíveis para o banco de dados SQL do Azure chamado Database01.
