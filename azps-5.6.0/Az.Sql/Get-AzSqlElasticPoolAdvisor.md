# Get-AzSqlElasticPoolAdvisor

## SYNOPSIS
Obtém um ou mais Consultores para um Pool Elástica do Azure SQL.

## SINTAXE

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlElasticPoolAdvisor** obtém um ou mais Azure SQL Elastic Pool Advisors para um Pool Elastic do Azure SQL Elastic.
