# Set-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Modifica uma configuração de recuperação de servidor de banco de dados.

## SINTAXE

```
Set-AzSqlServerDisasterRecoveryConfiguration -VirtualEndpointName <String> [-Failover] [-AllowDataLoss]
 [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzSqlServerDisasterRecoveryConfiguration** modifica uma configuração de recuperação SQL servidor de banco de dados.
