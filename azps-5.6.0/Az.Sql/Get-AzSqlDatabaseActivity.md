# Get-AzSqlDatabaseActivity

## SYNOPSIS
Obtém o status das operações do banco de dados.

## SINTAXE

```
Get-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabaseActivity** obtém o status das operações de banco de dados no Banco de Dados do Azure SQL.

## EXEMPLOS

### Exemplo 1: Obter status para todas as SQL banco de dados
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Este comando retorna o status de operação de todas as SQL de banco de dados em um pool elástica chamado ElasticPool01.

### Exemplo 2: Obter status para todas as operações SQL Banco de Dados
```
PS C:\>Get-AzSqlDatabaseActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Este comando retorna o status de todas as SQL de banco de dados em um banco de dados.
