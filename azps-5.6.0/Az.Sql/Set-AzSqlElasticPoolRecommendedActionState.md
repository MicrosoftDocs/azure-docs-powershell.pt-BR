# Set-AzSqlElasticPoolRecommendedActionState

## SYNOPSIS
Atualiza o estado de uma ação recomendada SQL Pool Elastic do Azure.

## SINTAXE

```
Set-AzSqlElasticPoolRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzSqlElasticPoolRecommendedActionState** atualiza o estado de uma ação recomendada do Pool Elástico do Azure SQL.
Este cmdlet aplica uma ação recomendada, revertida ou descartada com base no novo estado.
