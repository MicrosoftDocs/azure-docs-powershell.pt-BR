# New-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Cria uma regra de rede virtual do Azure SQL Server. 

## SINTAXE

```
New-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cria uma regra de rede virtual do Azure SQL Server. As Regras de Rede Virtual são usadas para conectar o SQL Server do Azure a uma Rede Virtual específica para restringir o acesso no SQL Server do Azure para estar disponível apenas na Rede Virtual. 

## EXEMPLOS

### Exemplo 1
```
PS C:\> $virtualNetworkRule = New-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Cria uma regra de SQL Server de rede virtual do Azure
