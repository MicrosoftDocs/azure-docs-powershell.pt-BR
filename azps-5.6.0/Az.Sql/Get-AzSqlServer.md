# Get-AzSqlServer

## SYNOPSIS
Retorna informações sobre SQL de banco de dados.

## SINTAXE

```
Get-AzSqlServer [[-ResourceGroupName] <String>] [[-ServerName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlServer** retorna informações sobre um ou mais servidores do Azure SQL Banco de Dados.
Especifique o nome de um servidor para ver informações apenas para esse servidor.

## EXEMPLOS

### Exemplo 1: Obter todas as instâncias de SQL Server atribuídas a um grupo de recursos
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01"

ServerName               : server01
Location                 : Central US

SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net

ResourceGroupName        : resourcegroup01
ServerName               : server02
Location                 : West US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server02.database.windows.net
```

Este comando obtém informações sobre todos os servidores de banco de dados do Azure SQL atribuídos ao grupo de recursos ResourceGroup01.

### Exemplo 2: obter informações sobre um servidor de banco de dados do Azure SQL Banco de Dados

Este comando obtém informações sobre o servidor de banco de dados do Azure SQL chamado Server01.
