# Get-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Obtém uma configuração de recuperação do sistema do servidor de banco de dados.

## SINTAXE

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlServerDisasterRecoveryConfiguration** obtém uma configuração SQL recuperação do sistema do servidor de banco de dados.
