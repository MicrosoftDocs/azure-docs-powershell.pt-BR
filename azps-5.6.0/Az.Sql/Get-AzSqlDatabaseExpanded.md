# Get-AzSqlDatabaseExpanded

## SYNOPSIS
Obtém um banco de dados e seus valores de propriedade expandida.

## SINTAXE

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabaseExpanded** obtém um banco de dados e seus valores de propriedade expandidos.
Esse cmdlet também é suportado pelo serviço SQL Server Banco de Dados de Extensão no Azure.

## EXEMPLOS

### Exemplo 1: Obter objeto database que tenha informações de consultor de camada de serviço
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Este comando retorna o banco de dados que tem propriedades expandidas que contêm as informações do consultor de camada de serviço.

### Exemplo 2: Listar objetos de banco de dados usando filtragem
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Este comando retorna o objeto de banco de dados expandido para todos os bancos de dados no Server01 que começam com "Database".
