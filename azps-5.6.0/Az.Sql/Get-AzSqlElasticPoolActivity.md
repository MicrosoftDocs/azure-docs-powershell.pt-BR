# Get-AzSqlElasticPoolActivity

## SYNOPSIS
Obtém o status das operações em um pool elástica.

## SINTAXE

```
Get-AzSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlElasticPoolActivity** obtém o status das operações em um pool elástica para um banco de dados do Azure SQL.
Você pode ver o status das atualizações de configuração e criação de pool.

## EXEMPLOS

### Exemplo 1: Obter o status das operações para um pool elástica
```
PS C:\>Get-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Este comando obtém o status das operações do pool elástica chamado ElasticPool01.
