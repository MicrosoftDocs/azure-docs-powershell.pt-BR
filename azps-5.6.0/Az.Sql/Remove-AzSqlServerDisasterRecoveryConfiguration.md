# Remove-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Remove uma configuração de recuperação do sistema SQL servidor de banco de dados.

## SINTAXE

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzSqlServerDisasterRecoveryConfiguration** remove uma configuração SQL recuperação do sistema do servidor de banco de dados.
