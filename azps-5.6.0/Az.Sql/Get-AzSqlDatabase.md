# Get-AzSqlDatabase

## SYNOPSIS
Obtém um ou mais bancos de dados.

## SINTAXE

```
Get-AzSqlDatabase [[-DatabaseName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlDatabase** obtém um ou mais bancos de dados do Azure SQL de um Servidor de Banco de Dados do Azure SQL.
Esse cmdlet também é suportado pelo serviço SQL Server Banco de Dados de Extensão no Azure.
