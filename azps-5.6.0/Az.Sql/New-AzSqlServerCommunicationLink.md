# New-AzSqlServerCommunicationLink

## SYNOPSIS
Cria um link de comunicação para transações de banco de dados elásticas entre dois SQL de banco de dados.

## SINTAXE

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzSqlServerCommunicationLink** cria um link de comunicação para transações de banco de dados elásticas entre dois servidores lógicos no Banco de Dados do Azure SQL.
As transações de banco de dados elásticas podem abranger bancos de dados em qualquer um dos servidores emparelhados.
Você pode criar mais de um link em um servidor.
Portanto, transações de banco de dados elásticas podem abranger um número maior de servidores.

## EXEMPLOS

### Exemplo 1: Criar um link de comunicação
```
PS C:\>New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Este comando cria um link chamado Link01 entre ContosoServer17 e ContosoServer02.
