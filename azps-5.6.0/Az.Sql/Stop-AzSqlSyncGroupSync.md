# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Interrompe uma sincronização de grupo de sincronização.

## SINTAXE

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Stop-AzSqlSyncGroupSync** interrompe uma sincronização de grupo de sincronização.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Este comando interrompe a sincronização que está em andamento para o mysg do grupo de sincronização.
