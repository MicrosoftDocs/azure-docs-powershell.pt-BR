# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Remove um Grupo de Failover SQL Banco de Dados do Azure.

## SINTAXE

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Este comando remove o Grupo de Failover com o nome especificado, deixando todos os bancos de dados e relações de replicação intactas. O ponto de extremidade do ouvinte será não registro no DNS.
O servidor principal do Grupo de Failover deve ser usado para executar o comando.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Remover um Grupo de Failover.
