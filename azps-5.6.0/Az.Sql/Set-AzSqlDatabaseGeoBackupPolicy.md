# Set-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Define uma política de backup geo-geográfica do banco de dados.

## SINTAXE

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzSqlDatabaseGeoBackupPolicy** define a política de backup geográfica registrada em um banco de dados.
Este é um recurso de Backup do Azure usado para definir a política de armazenamento de backup.
