# Resume-AzSqlDatabase

## SYNOPSIS
Retoma um banco de dados SQL Data Warehouse.

## SINTAXE

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Resume-AzSqlDatabase** retoma um banco de dados do Azure SQL Data Warehouse.

## EXEMPLOS

### Exemplo 1: retoma um banco de dados do Azure SQL Data Warehouse
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Este comando retoma um banco de dados suspenso do Azure SQL Data Warehouse.
