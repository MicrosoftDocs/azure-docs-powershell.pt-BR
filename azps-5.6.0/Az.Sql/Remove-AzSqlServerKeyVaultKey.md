# Remove-AzSqlServerKeyVaultKey

## SYNOPSIS
Remove uma chave do Cofre de Chaves de um SQL servidor.

## SINTAXE

```
Remove-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O Remove-AzSqlServerKeyVaultKey cmdlet remove a tecla Key Vault do servidor SQL especificado.
Observe que SQL permissões do servidor para o cofre da chave não são alteradas.
Para alterar permissões, use Set-AzKeyVaultAccessPolicy.
Observe que este cmdlet não faz alterações no Cofre de Chaves.
Para remover uma chave do Cofre de Chaves, use Remove-AzKeyVaultKey.

## EXEMPLOS

### Exemplo 1: Remover uma chave do Cofre de Chaves
```
PS C:\> Remove-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Este comando remove a tecla Key Vault com Id ' https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 ' do servidor especificado.
