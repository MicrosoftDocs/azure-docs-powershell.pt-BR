# Get-AzSqlServerUpgradeHint

## SYNOPSIS
Obtém dicas de camada de preços para atualizar um servidor de banco de dados do Azure SQL Banco de Dados.

## SINTAXE

```
Get-AzSqlServerUpgradeHint [-ServerName] <String> [-ExcludeElasticPools <Boolean>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSqlServerUpgradeHint** obtém dicas de camada de preços para atualizar um servidor de banco de dados do Azure SQL.
As dicas podem conter o pool de banco de dados elástica e dicas de banco de dados autônomos.
Os bancos de dados que ainda estão em camadas de preços da Web e do Business têm uma dica para atualizar para as novas camadas de preços Básica, Padrão ou Premium ou para entrar no pool de banco de dados elástica.
Este cmdlet retorna dicas para todos os bancos de dados hospedados no servidor especificado.

## EXEMPLOS

### Exemplo 1: Obter recomendações combinadas
```
PS C:\>Get-AzSqlServerUpgradeHint -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ElasticPools Databases           
------------ ---------           
{}           {database01, database02}
```

Este comando recebe recomendações combinadas para todos os bancos de dados em um servidor chamado Server01.
