# Update-AzSqlSyncSchema

## SYNOPSIS
Atualize o esquema de sincronização para um banco de dados de membro de sincronização ou um banco de dados de hub de sincronização.
Ele obterá o esquema de banco de dados mais recente do banco de dados real e o usará para atualizar o esquema armazenado em cache pelo banco de dados de metadados sync.
Se "SyncMemberName" for especificado, ele atualizará o esquema de banco de dados membro; caso não seja, ele atualizará o esquema de banco de dados do hub.

## SINTAXE

```
Update-AzSqlSyncSchema [-SyncGroupName] <String> [-SyncMemberName <String>] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Update-AzSqlSyncSchema** atualiza o esquema de sincronização para um banco de dados de membro de sincronização ou um banco de dados de hub de sincronização.

## EXEMPLOS

### Exemplo 1: atualizar o esquema de sincronização para um banco de dados de hub
```
PS C:\>Update-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01"
```

Este comando atualiza o esquema de sincronização do banco de dados de hub na sincronização do grupo de sincronizaçãoGroup01

### Exemplo 2: atualizar o esquema de sincronização de um banco de dados membro
```
PS C:\>Update-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01" -SyncMemberName "syncMember01"
```

Este comando atualiza o esquema de sincronização do banco de dados membro na sincronização de membros de sincronizaçãoMember01
