# New-AzCostManagementQueryComparisonExpressionObject

## SYNOPSIS
Criar um objeto na memória para QueryComparisonExpression

## SINTAXE

```
New-AzCostManagementQueryComparisonExpressionObject -Name <String> -Operator <OperatorType> -Value <String[]>
 [<CommonParameters>]
```

## DESCRIPTION
Criar um objeto na memória para QueryComparisonExpression

## EXEMPLOS

### Exemplo 1: Criar um objeto de expressão de comparação de consulta para exportação de gerenciamento de custos
```powershell
PS C:\> New-AzCostManagementQueryComparisonExpressionObject -Name 'ResourceLocation' -Value @('East US', 'West Europe')

Name             Operator Value
----             -------- -----
ResourceLocation In       {East US, West Europe}
```

Este comando cria um objeto de expressão de comparação de consulta para exportação de gerenciamento de custos.
