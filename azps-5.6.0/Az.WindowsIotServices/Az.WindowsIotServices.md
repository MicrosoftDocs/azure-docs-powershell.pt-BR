# Módulo Az.WindowsIotServices
## Descrição
Microsoft Azure PowerShell: cmdlets do WindowsIotServices

## Az.WindowsIotServices Cmdlets
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Obter os metadados não relacionados à segurança de um Serviço de Dispositivo IoT do Windows.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Crie ou atualize os metadados de um Serviço de Dispositivo IoT do Windows.
O padrão comum para modificar uma propriedade é recuperar os metadados e os metadados de segurança do Windows IoT Device Service e combiná-los com os valores modificados em um novo corpo para atualizar o Serviço de Dispositivo do Windows IoT.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Exclua um Serviço de Dispositivo windows IoT.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Atualiza os metadados de um Serviço de Dispositivo windows IoT.
O padrão comum para modificar uma propriedade é recuperar os metadados e os metadados de segurança do Windows IoT Device Service e combiná-los com os valores modificados em um novo corpo para atualizar o Serviço de Dispositivo do Windows IoT.
