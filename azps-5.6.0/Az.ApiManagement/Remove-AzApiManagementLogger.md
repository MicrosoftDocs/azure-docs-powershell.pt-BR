# Remove-AzApiManagementLogger

## SYNOPSIS
Remove um Api Management Logger.

## SINTAXE

```
Remove-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApiManagementLogger** remove um Azure API Management **Logger**.

## EXEMPLOS

### Exemplo 1: Remover um madeireiro
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

Este comando remove um madeireiro que tem o ID Logger123.
