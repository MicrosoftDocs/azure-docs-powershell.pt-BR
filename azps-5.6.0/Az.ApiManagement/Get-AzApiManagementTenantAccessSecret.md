# Get-AzApiManagementTenantAccessSecret

## SYNOPSIS
Obtém as chaves de configuração de acesso para um locatário.

## SINTAXE

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Obtém as chaves de configuração de acesso para um locatário.

## EXEMPLOS

### Exemplo 1: Obter chaves de configuração de acesso para locatários
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Este comando obtém as chaves de configuração de acesso do locatário para o contexto especificado.
