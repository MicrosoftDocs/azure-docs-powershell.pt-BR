# Get-AzApiManagementAuthorizationServer

## SYNOPSIS
Obtém um servidor de autorização de Gerenciamento de API.

## SINTAXE

### ContextParameterSet (Padrão)
```
Get-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServer [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApiManagementAuthorizationServer** obtém todos os servidores de autorização de Gerenciamento de API do Azure ou servidor de autorização especificado.
ClientSecret não será incluído nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementAuthorizationServerClientSecret**.

## EXEMPLOS

### Exemplo 1: Obter todos os servidores de autorização
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext
```

Este comando obtém todos os servidores de autorização de Gerenciamento de API.

### Exemplo 2: Obter um servidor de autorização especificado
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "0123456789"
```

Este comando obtém o servidor de autorização especificado.
