# Set-AzApiManagementNamedValue

## SYNOPSIS
Modifica um gerenciamento de API chamado Valor.

## SINTAXE

```
Set-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-Name <String>]
 [-Value <String>] [-Secret <Boolean>] [-Tag <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzApiManagementNamedValue** modifica um Valor nomeado de Gerenciamento de API do Azure.

## EXEMPLOS

### Exemplo 1: Alterar as marcas no valor nomeado
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -Tags $Tags -PassThru
```

O primeiro comando atribui dois valores à $Tags variável.
O segundo comando modifica o valor nomeado que tem a propriedade ID11.
O comando atribui as cadeias de caracteres $Tags como marcas no valor nomeado.

### Exemplo 2: Modificar o valor nomeado para ter um valor secreto
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property12" -Secret $True -PassThru
```

Este comando altera o valor nomeado a ser Criptografado.

### Exemplo 3

Modifica um gerenciamento de API chamado Valor. (gerado automaticamente)

```powershell
<!-- Aladdin Generated Example --> 
Set-AzApiManagementNamedValue -Context <PsApiManagementContext> -Name 'ContosoApi' -NamedValueId 'Property11' -Secret $true -Tag <String[]> -Value 'Property Value'
```
