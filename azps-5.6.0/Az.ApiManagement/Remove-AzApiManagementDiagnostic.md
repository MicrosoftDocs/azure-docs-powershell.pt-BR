# Remove-AzApiManagementDiagnostic

## SYNOPSIS
Remova a entidade Diagnostic do escopo global ou de nível de API.

## SINTAXE

### ByResourceIdParameterSet (Padrão)
```
Remove-AzApiManagementDiagnostic -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExpandParameterSetName
```
Remove-AzApiManagementDiagnostic -Context <PsApiManagementContext> [-ApiId <String>] -DiagnosticId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObjectParameterSet
```
Remove-AzApiManagementDiagnostic -InputObject <PsApiManagementDiagnostic> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzApiManagementDiagnostic** remove a entidade de diagnóstico especificada pelo escopo global ou `DiagnosticId` por um `ApiId` escopo

## EXEMPLOS

### Exemplo 1: Remover a entidade Diagnostic
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementDiagnostic -Context $apimContext -DiagnosticId "applicationinsights"
```

Este exemplo remove o diagnóstico `applicationinsights` do serviço de Gerenciamento de Api.
