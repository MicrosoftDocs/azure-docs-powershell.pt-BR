# Get-AzApiManagementTenantGitAccess

## SYNOPSIS
Obtém a configuração de acesso do Git para um locatário.

## SINTAXE

```
Get-AzApiManagementTenantGitAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApiManagementTenantGitAccess** obtém a configuração de acesso do Git para um locatário.
As chaves não serão incluídas nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementTenantGitAccessSecret**.

## EXEMPLOS

### Exemplo 1: Obter configuração de acesso de locatário
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantGitAccess -Context $apimContext
```
