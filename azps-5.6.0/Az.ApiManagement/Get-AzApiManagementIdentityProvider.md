# Get-AzApiManagementIdentityProvider

## SYNOPSIS
Obter os detalhes de configuração do provedor de identidade.

## SINTAXE

### AllIdentityProviders (Padrão)
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProviderByType
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Obter os detalhes de configuração do provedor de identidade.
ClientSecret não será incluído nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementIdentityProviderClientSecret**.

## EXEMPLOS

### Exemplo 1: Obter todos os provedores de identidade

```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementIdentityProvider -Context $apimContext
```

Obter toda a Configuração do provedor de identidade no serviço.

### Exemplo 2: Obter o Provedor de Identidade de Tipo AAD
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\> Get-AzApiManagementIdentityProvider -Context $apimContext -Type Aad
```
