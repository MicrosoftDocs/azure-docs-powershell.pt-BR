# Get-AzApiManagementTenantAccess

## SYNOPSIS
Obtém a configuração de acesso para um locatário.

## SINTAXE

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApiManagementTenantAccess** obtém a configuração de acesso de locatário para um locatário.
As chaves não serão incluídas nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementTenantAccessSecret**.

## EXEMPLOS

### Exemplo 1: Obter configuração de acesso de locatário
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Este comando obtém a configuração de acesso do locatário para o contexto especificado.
