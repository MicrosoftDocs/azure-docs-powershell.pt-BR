# Set-AzApiManagementGroup

## SYNOPSIS
Configura um grupo de gerenciamento de API.

## SINTAXE

```
Set-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzApiManagementGroup** configura um grupo de gerenciamento de API.

## EXEMPLOS

### Exemplo 1: Configurar um grupo de gerenciamento
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementGroup -Context $apimContext -GroupId "0001" -Description "Updated Management Group" -Name "Group0001"
```

Este comando configura um grupo de gerenciamento chamado Group0001.
