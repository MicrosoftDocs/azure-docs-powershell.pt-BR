# Save-AzApiManagementTenantGitConfiguration

## SYNOPSIS
Salva as alterações criando uma confirmação para a configuração atual.

## SINTAXE

```
Save-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Save-AzApiManagementTenantGitConfiguration** salva as alterações criando uma confirmação que contém o instantâneo de configuração atual em um branch no repositório.

## EXEMPLOS

### Exemplo 1: Salvar alterações na configuração
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Save-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Esse comando salva as alterações criando uma confirmação com o instantâneo de configuração atual para o branch especificado no repositório.
