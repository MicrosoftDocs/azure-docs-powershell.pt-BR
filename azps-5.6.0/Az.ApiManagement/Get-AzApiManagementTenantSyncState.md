# Get-AzApiManagementTenantSyncState

## SYNOPSIS
Obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório git.

## SINTAXE

```
Get-AzApiManagementTenantSyncState -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzApiManagementTenantSyncState** obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório do Git.

## EXEMPLOS

### Exemplo 1: Obter o status da sincronização mais recente
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantSyncState -Context $apimContext
```

Esse comando obtém o status da sincronização mais recente entre o banco de dados de configuração e o repositório git.
