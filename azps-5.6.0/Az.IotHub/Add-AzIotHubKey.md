# Add-AzIotHubKey

## SYNOPSIS
Cria uma chave IotHub.

## SINTAXE

### ResourceSet (Padrão)
```
Add-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String> [-PrimaryKey <String>]
 [-SecondaryKey <String>] -Rights <PSAccessRights> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Add-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-PrimaryKey <String>] [-SecondaryKey <String>]
 -Rights <PSAccessRights> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Cria uma chave para o IotHub fornecido.
KeyNames não são exclusivos e precisam ser gerenciados com cuidado.

## EXEMPLOS

### Exemplo 1 Adicionar uma chave a um IotHub
```
PS C:\> Add-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "newkey" -PrimaryKey "primarykey" -SecondaryKey "secondarykey" -Rights RegistryRead
```

Cria uma chave chamada "mykey" para o iothub "myiothub" com permissões registryRead.
