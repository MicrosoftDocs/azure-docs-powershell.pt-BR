# Invoke-AzIotHubDeploymentMetricsQuery

## SYNOPSIS
Invocar uma consulta métrica de implantação de Borda de IoT.

## SINTAXE

### ResourceSet (Padrão)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Avalie uma métrica personalizada de destino ou sistema definida em uma implantação de Borda IoT.
Há métricas de sistema pré-definidas que são calculadas pelo Hub de Iot e não podem ser personalizadas.
- "Direcionado" mostra os dispositivos de Borda de IoT que corresponderem à condição de direcionamento de implantação.
- "Aplicado" mostra os dispositivos de Borda de IoT direcionados que não são direcionados por outra implantação de prioridade mais alta.
- "Sucesso do Relatório" mostra os dispositivos de Borda de IoT que relataram que os módulos foram implantados com êxito.
- "Falha de Relatório" mostra os dispositivos de Borda de IoT que relataram que um ou mais módulos não foram implantados com êxito. 
  Para investigar ainda mais o erro, conecte-se remotamente a esses dispositivos e veja os arquivos de log.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Avalie a métrica "warningLimit" definida personalizada.

### Exemplo 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Avalie a métrica do sistema 'Reporting Success'.
