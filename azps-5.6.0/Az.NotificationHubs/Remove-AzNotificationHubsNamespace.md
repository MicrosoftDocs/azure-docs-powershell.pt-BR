# Remove-AzNotificationHubsNamespace

## SYNOPSIS
Remove um namespace do hub de notificação.

## SINTAXE

```
Remove-AzNotificationHubsNamespace [-ResourceGroup] <String> [-Namespace] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzNotificationHubsNamespace** remove um namespace de hub de notificação da sua implantação.
Namespaces são contêineres lógicos que ajudam você a organizar e gerenciar seus hubs de notificação.
O cmdlet **Remove-AzNotificationHubsNamespace** remove um namespace de hub de notificação da sua implantação.
Quando você executar esse cmdlet, o namespace especificado será excluído juntamente com todos os hubs de notificação associados a esse namespace.

## EXEMPLOS

### Exemplo 1: Remover um namespace de hub de notificação
```
PS C:\>Remove-AzNotificationHubsNamespace -Namespace "ContosoNamespace" -ResourceGroup "ContosoNotificationsGroup"
```

Este comando remove o namespace chamado ContosoNamespace.
Você deve especificar o grupo de recursos ao qual o namespace é atribuído.
