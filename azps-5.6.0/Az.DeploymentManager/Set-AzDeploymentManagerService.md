# Set-AzDeploymentManagerService

## SYNOPSIS
Atualiza o serviço.

## SINTAXE

```
Set-AzDeploymentManagerService [-InputObject] <PSServiceResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzDeploymentManagerService** atualiza um serviço com o objeto de serviço especificado.
O cmdlet retorna o objeto de serviço atualizado.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceObject
```

Este comando atualiza um serviço cujo nome, nome da topologia de serviço e ResourceGroup corresponderem às propriedades Name, ServiceTopologyName e ResourceGroupName do $serviceObject, respectivamente.
O serviço seria atualizado para as propriedades definidas no $serviceObject.
