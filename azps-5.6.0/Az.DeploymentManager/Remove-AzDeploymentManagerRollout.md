# Remove-AzDeploymentManagerRollout

## SYNOPSIS
Exclui a adoção.

## SINTAXE

### Interativo (Padrão)
```
Remove-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerRollout [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzDeploymentManagerRollout** exclui uma rolagem em um estado de terminal.
Especifique a distribuição pelo nome e o nome do grupo de recursos. Como alternativa, você pode fornecer o objeto Rollout ou ResourceId.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Este comando exclui um lançamento chamado ContosoRollout no ContosoResourceGroup.

### Exemplo 2: Excluir uma distribuição usando o identificador de recurso
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Este comando exclui um lançamento chamado ContosoRollout no ContosoResourceGroup.

### Exemplo 3: Excluir uma apostila usando o objeto de lançamento.
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Este comando exclui uma distribuição cujo nome e ResourceGroup corresponderem às propriedades Name e ResourceGroupName do $rolloutObject, respectivamente.
