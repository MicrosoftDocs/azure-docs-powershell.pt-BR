# Test-AzPowerBIEmbeddedCapacity

## SYNOPSIS
Testa a existência de uma instância da Capacidade Incorporada do PowerBI.

## SINTAXE

```
Test-AzPowerBIEmbeddedCapacity [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O Test-AzPowerBIEmbeddedCapacity cmdlet testa a existência de uma instância da Capacidade Incorporada do PowerBI

## EXEMPLOS

### Exemplo 1
```
PS C:\> Test-AzPowerBIEmbeddedCapacity -Name "testcapacity"
True
```

Este comando testará se há uma capacidade chamada testcapacity
