# Remove-AzStorageSyncService

## SYNOPSIS
Este comando excluirá o serviço de sincronização de armazenamento especificado.

## SINTAXE

### StringParameterSet (Padrão)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Este comando excluirá o serviço de sincronização de armazenamento especificado. Um serviço de sincronização de armazenamento só pode ser removido quando todos os grupos de sincronização contidos e servidores registrados são excluídos primeiro.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Este comando removerá o serviço de sincronização de armazenamento.
