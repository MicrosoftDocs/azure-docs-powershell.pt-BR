# Get-AzStorageSyncService

## SYNOPSIS
Este comando lista todos os serviços de sincronização de armazenamento dentro de um determinado escopo de grupo de assinatura/recurso.

## SINTAXE

### ParentStringParameterSet (Padrão)
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Este comando lista todos os serviços de sincronização de armazenamento dentro de um determinado escopo de grupo de assinatura/recurso. Ele também pode ser usado para listar os atributos de cada serviço de sincronização de armazenamento.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

Este comando lista todos os recursos do serviço de sincronização de armazenamento em um determinado grupo de recursos. Ele também pode ser usado para listar os atributos de cada serviço de sincronização de armazenamento. Especifique -StorageSyncServiceName para retornar um específico.
