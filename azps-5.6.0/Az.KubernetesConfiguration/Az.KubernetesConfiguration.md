# Módulo Az.KubernetesConfiguration
## Descrição
Microsoft Azure PowerShell: cmdlets KubernetesConfiguration

## Az.KubernetesConfiguration Cmdlets
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Obtém detalhes da Configuração de Controle de Origem.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Crie uma nova Configuração de Controle de Origem do Kubernetes.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Isso excluirá o arquivo YAML usado para configurar a configuração do controle Source, interrompendo assim a sincronização futura do repo de origem.
