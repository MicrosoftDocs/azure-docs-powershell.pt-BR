# Test-AzSynapseSparkPool

## SYNOPSIS
Verifica a existência de um pool de Spark de Análise de Synapse.

## SINTAXE

### TestByNameParameterSet (Padrão)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Test-AzSynapseSparkPool** verifica a existência de um pool de Spark de Análise de Synapse.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

Este comando verifica a existência do pool de Spark especificado.
