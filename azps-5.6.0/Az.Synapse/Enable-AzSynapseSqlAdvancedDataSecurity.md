# Enable-AzSynapseSqlAdvancedDataSecurity

## SYNOPSIS
Habilita a Segurança Avançada de Dados em um espaço de trabalho.

## SINTAXE

### EnableByNameParameterSet (Padrão)
```
Enable-AzSynapseSqlAdvancedDataSecurity [-ResourceGroupName <String>] -WorkspaceName <String>
 [-DoNotConfigureVulnerabilityAssessment] [-DeploymentName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### EnableByInputObjectParameterSet
```
Enable-AzSynapseSqlAdvancedDataSecurity -InputObject <PSSynapseWorkspace>
 [-DoNotConfigureVulnerabilityAssessment] [-DeploymentName <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### EnableByResourceIdParameterSet
```
Enable-AzSynapseSqlAdvancedDataSecurity -ResourceId <String> [-DoNotConfigureVulnerabilityAssessment]
 [-DeploymentName <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Enable-AzSynapseSqlAdvancedDataSecurity** habilita a Segurança Avançada de Dados em um espaço de trabalho. A Segurança Avançada de Dados é um pacote de segurança unificado que inclui Classificação de Dados, Avaliação de Vulnerabilidade e Proteção Avançada contra Ameaças para seu espaço de trabalho. (Uma nova conta de armazenamento será criada automaticamente para salvar avaliações de vulnerabilidade. Se uma conta de armazenamento tiver sido criada anteriormente para essa finalidade, ela será usada em vez disso)

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Enable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace
```

Este comando habilita o espaço de trabalho Segurança Avançada de Dados.

### Exemplo 2
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Enable-AzSynapseSqlAdvancedDataSecurity
```

Este comando habilita o espaço de trabalho Segurança Avançada de Dados por meio do pipeline.

### Exemplo 3
```powershell
PS C:\> Enable-AzSynapseSqlAdvancedDataSecurity -WorkspaceName ContosoWorkspace -DoNotConfigureVulnerabilityAssessment
```

Esse comando habilita o espaço de trabalho Segurança Avançada de Dados por meio do pipeline e não habilita automaticamente a Avaliação de Vulnerabilidade.
