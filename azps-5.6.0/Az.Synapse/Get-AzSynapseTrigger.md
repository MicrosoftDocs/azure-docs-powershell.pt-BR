# Get-AzSynapseTrigger

## SYNOPSIS
Obtém informações sobre gatilhos em um espaço de trabalho.

## SINTAXE

### GetByName (Padrão)
```
Get-AzSynapseTrigger -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzSynapseTrigger** obtém informações sobre gatilhos em um espaço de trabalho. Se você especificar o nome de um gatilho, o cmdlet obterá informações sobre esse gatilho. Se você não especificar um nome, o cmdlet obterá informações sobre todos os gatilhos no espaço de trabalho.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace
```

Obtém uma lista de todos os gatilhos que foram criados no espaço de trabalho ContosoWorkspace.

### Exemplo 2
```powershell
PS C:\> Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Obtém um único gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace.

### Exemplo 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseTrigger -Name ContosoTrigger
```

Obtém um único gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace por meio de pipeline.
