# Get-AzSynapsePipelineRun

## SYNOPSIS
Obtém informações sobre as executações de pipeline.

## SINTAXE

### GetByNameAndId (Padrão)
```
Get-AzSynapsePipelineRun -WorkspaceName <String> -PipelineRunId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameAndTime
```
Get-AzSynapsePipelineRun -WorkspaceName <String> -RunStartedAfter <DateTimeOffset>
 -RunStartedBefore <DateTimeOffset> [-PipelineName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObjectAndId
```
Get-AzSynapsePipelineRun -WorkspaceObject <PSSynapseWorkspace> -PipelineRunId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByObjectAndTime
```
Get-AzSynapsePipelineRun -WorkspaceObject <PSSynapseWorkspace> -RunStartedAfter <DateTimeOffset>
 -RunStartedBefore <DateTimeOffset> [-PipelineName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O **comando Get-AzSynapsePipelineRun** retorna informações sobre as executações para o pipeline especificado. Se PipelineRunId for especificado, ele mostrará detalhes para a executar com essa ID. Se PipelineRunId não for especificado, ele mostrará informações sobre todas as executações para os pipelines que aconteceram entre os valores de RunStartedAfter e RunStartedBefore.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzSynapsePipelineRun -WorkspaceName ContosoWorkspace -PipelineRunId "61eb095a-fe23-4591-8a97-fade6c65ca72"
```

Este comando obtém detalhes sobre o pipeline executado com a ID "61eb095a-fe23-4591-8a97-fade6c65ca72".
