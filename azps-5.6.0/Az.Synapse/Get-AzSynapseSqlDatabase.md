# Get-AzSynapseSqlDatabase

## SYNOPSIS
Obtém um banco de dados do Synapse Analytics SQL.

## SINTAXE

### GetByNameParameterSet (Padrão)
```
Get-AzSynapseSqlDatabase [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSqlDatabase [-Name <String>] -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlDatabase -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
[Esse recurso está em uma visualização limitada, inicialmente acessível apenas a determinadas assinaturas.] O cmdlet **Get-AzSynapseSqlDatabase** obtém informações sobre um banco de dados do Azure Synapse Analytics SQL.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzSynapseSqlDatabase -WorkspaceName ContosoWorkspace
```

Esse comando obtém todos os SQL de dados em um espaço de trabalho.

### Exemplo 2
```powershell
PS C:\> Get-AzSynapseSqlDatabase -WorkspaceName ContosoWorkspace -Name ContosoSqlDatabase
```

Este comando obtém o banco de dados SQL no espaço de trabalho ContosoWorkspace com o nome ContosoSqlDatabase.

### Exemplo 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseSqlDatabase
```

Esse comando obtém todos os SQL de dados em um espaço de trabalho por meio do pipeline.

### Exemplo 4
```powershell
PS C:\> Get-AzSynapseSqlDatabase -ResourceId "/subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace/sqlDatabases/ContosoSqlDatabase"
```

Esse comando obtém o banco SQL com a ID de recurso especificada.
