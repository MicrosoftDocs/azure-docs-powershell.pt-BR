# Get-AzCognitiveServicesAccountType

## SYNOPSIS
Obtém os Tipos de Conta de Serviços Cognitivos disponíveis.

## SINTAXE

### GetAccountTypes (Padrão)
```
Get-AzCognitiveServicesAccountType [-Location <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetAccountTypeWithName
```
Get-AzCognitiveServicesAccountType -TypeName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzCognitiveServicesAccountType** obtém os Tipos de Conta de Serviços Cognitivos disponíveis nesta assinatura.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountType
```

Obter a lista de tipos disponíveis.

### Exemplo 2
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -Location westus
```

Obter a lista de tipos disponíveis em westus.

### Exemplo 3
```powershell
PS C:\> Get-AzCognitiveServicesAccountType -TypeName Face

Face
```

Verifique se `Face` é um nome de tipo válido, o nome será retornado se for um nome válido.
