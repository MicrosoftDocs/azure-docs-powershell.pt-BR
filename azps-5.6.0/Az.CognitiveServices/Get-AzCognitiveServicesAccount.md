# Get-AzCognitiveServicesAccount

## SYNOPSIS
Obtém uma conta.

## SINTAXE

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzCognitiveServicesAccount** obtém as contas provisionadas dos Serviços Cognitivos no grupo de recursos especificado pelo *parâmetro ResourceGroupName.*
Se você não especificar o *parâmetro ResourceGroupName,* esse cmdlet obtém todas as contas dos Serviços Cognitivos para a assinatura atual.
