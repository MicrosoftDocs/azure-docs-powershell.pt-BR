# Start-AzRecoveryServicesAsrResynchronizeReplicationJob

## SYNOPSIS
Inicia a ressincronização de replicação.

## SINTAXE

### Padrão (Padrão)
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Start-AzRecoveryServicesAsrResynchronizeReplicationJob** inicia a ressincronização da replicação para o item protegido especificado se o protegido estiver em um estado necessário de ressincronização.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem $rpi
```

Inicia o trabalho para ressincronizar a replicação no item protegido de replicação passado.
