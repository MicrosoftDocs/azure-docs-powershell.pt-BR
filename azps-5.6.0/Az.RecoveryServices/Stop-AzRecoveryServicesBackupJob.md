# Stop-AzRecoveryServicesBackupJob

## SYNOPSIS
Cancela um trabalho em execução.

## SINTAXE

### JobFilterSet (Padrão)
```
Stop-AzRecoveryServicesBackupJob [-Job] <JobBase> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdFilterSet
```
Stop-AzRecoveryServicesBackupJob [-JobId] <String> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Stop-AzRecoveryServicesBackupJob** cancela um trabalho de Backup do Azure existente.
Use este cmdlet para interromper um trabalho que leva muito tempo e bloqueia outras atividades.
Você pode cancelar apenas tipos de trabalho de Backup e Restauração.
De definir o contexto do cofre usando o cmdlet Set-AzRecoveryServicesVaultContext antes de usar o cmdlet atual.

## EXEMPLOS

### Exemplo 1: Interromper um trabalho de backup
```
PS C:\>$Job = Get-AzRecoveryServicesBackupJob -Operation Backup
PS C:\> Stop-AzRecoveryServicesBackupJob -JobID $Job.InstanceId
```

O primeiro comando obtém um trabalho de backup e armazena o trabalho na variável $Job.
O último comando interrompe o trabalho especificando a ID de Instância do trabalho de backup $Job.
