# Get-AzRecoveryServicesAsrAlertSetting

## SYNOPSIS
Obtém as configurações de notificação de Recuperação de Site do Azure para o cofre.

## SINTAXE

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzRecoveryServicesAsrAlertSetting** obtém as configurações de notificação de Recuperação de Site do Azure para o cofre.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Obter Configuração de Alerta/Notificação para Recuperação de Site do Azure.
