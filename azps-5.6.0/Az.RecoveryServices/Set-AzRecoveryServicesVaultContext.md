# Set-AzRecoveryServicesVaultContext

## SYNOPSIS

Define o contexto do cofre.

## SINTAXE

```
Set-AzRecoveryServicesVaultContext -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION

O cmdlet **Set-AzRecoveryServicesVaultContext** define o contexto do cofre para os serviços de Recuperação de Site do Azure.

Aviso: este cmdlet está sendo preterido em uma versão futura de alteração de quebra. Não haverá nenhuma substituição para ele. Use o parâmetro -VaultId em todos os comandos dos Serviços de Recuperação em frente.

## EXEMPLOS

### Exemplo 1

```powershell
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> Set-AzRecoveryServicesVaultContext -Vault $vault
```

Define o contexto do cofre.
