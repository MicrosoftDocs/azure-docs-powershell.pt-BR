# Update-AzRecoveryServicesAsrProtectionDirection

## SYNOPSIS
Atualiza a direção de replicação para o item ou plano de recuperação protegido de replicação especificado. Usado para resguardar/reverter replicar um item replicado ou plano de recuperação com falha.

## DESCRIPTION
O cmdlet **Update-AzRecoveryServicesAsrProtectionDirection** atualiza a direção de replicação do objeto de Recuperação de Site especificado do Azure após a conclusão de uma operação de failover de confirmação.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -RecoveryPlan $RP -Direction PrimaryToRecovery
```

Inicie a operação de direção de atualização para o plano de recuperação especificado e retorna o objeto de trabalho ASR usado para rastrear a operação.

### Exemplo 2
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping -LogStorageAccountId $cacheStorageId `
 -ReplicationProtectedItem $rpi
```

Inicie a operação de direção de atualização para o item protegido de replicação especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e usando o armazenamento de cache (na mesma região que a VM).

### Exemplo 3
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping `
 -AzureToAzureDiskReplicationConfiguration $disk1,$disk2 -ReplicationProtectedItem  $rpi
```

Inicie a operação de direção de atualização para o item protegido de replicação especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e pela configuração de replicação de disco fornecida.

### Exemplo 4
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping `
 -AzureToAzureDiskReplicationConfiguration $disk1,$disk2 -ReplicationProtectedItem  $rpi `
 -DiskEncryptionVaultId  $DiskEncryptionVaultId -DiskEncryptionSecertUrl $DiskEncryptionSecertUrl `
 -KeyEncryptionVaultId $KeyEncryptionVaultId  -KeyEncryptionKeyUrl $KeyEncryptionKeyUrl
```

    
Inicie a operação de direção de atualização para o item protegido de replicação criptografada especificado na região do azure de destino definida pelo mapeamento de contêineres de proteção e pela configuração de replicação de disco fornecida.

### Exemplo 5
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping -LogStorageAccountId $cacheStorageId `
 -ReplicationProtectedItem $rpi -RecoveryProximityPlacementGroupId $ppg
```

Inicie a operação de direção de atualização para o item protegido de replicação especificado na região do azure de destino definida pelo mapeamento de contêineres de proteção e usando o armazenamento de cache (na mesma região que a VM) e o grupo de posicionamento de proximidade.
