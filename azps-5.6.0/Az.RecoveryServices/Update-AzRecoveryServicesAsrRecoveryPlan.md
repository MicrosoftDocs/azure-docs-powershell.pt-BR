# Update-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Atualiza o conteúdo de um plano de recuperação de site do Azure.

## SINTAXE

### ByRPObject (Padrão)
```
Update-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByRPFile
```
Update-AzRecoveryServicesAsrRecoveryPlan -Path <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Update-AzRecoveryServicesAsrRecoveryPlan** atualiza o conteúdo de um plano de recuperação usando o conteúdo do objeto do plano de recuperação ASR especificado ou do arquivo json de definição do plano de recuperação asR.

## EXEMPLOS

### Exemplo 1: atualizar um plano de recuperação
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Inicie a operação de atualização de um plano de recuperação usando o conteúdo do objeto de plano de recuperação ASR especificado e retorna o trabalho ASR usado para rastrear a operação.
