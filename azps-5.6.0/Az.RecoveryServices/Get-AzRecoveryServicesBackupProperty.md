# Get-AzRecoveryServicesBackupProperty

## SYNOPSIS
Obtém propriedades de backup.

## SINTAXE

```
Get-AzRecoveryServicesBackupProperty -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzRecoveryServicesBackupProperty** obtém propriedades de backup para um cofre dos Serviços de Recuperação.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzRecoveryServicesBackupProperty -Vault $vault
```

Obter a propriedade do cofre de backup para o cofre.
