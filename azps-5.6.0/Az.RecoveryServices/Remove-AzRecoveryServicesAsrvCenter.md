# Remove-AzRecoveryServicesAsrvCenter

## SYNOPSIS
Remove o servidor vCenter do fabric ASR e interrompe a descoberta de máquinas virtuais do servidor vCenter.

## SINTAXE

### Padrão (Padrão)
```
Remove-AzRecoveryServicesAsrvCenter -InputObject <ASRvCenter> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzRecoveryServicesAsrvCenter -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrvCenter -Fabric <ASRFabric> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzRecoveryServicesAsrvCenter** remove o servidor vCenter do tecido ASR e interrompe a descoberta de máquinas virtuais do servidor vCenter.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Remove-AzRecoveryServicesAsrvCenterServer -InputObject $vCenter
```

Remove o servidor vCenter do tecido ASR.

### Exemplo 2

Remove o servidor vCenter do fabric ASR e interrompe a descoberta de máquinas virtuais do servidor vCenter. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzRecoveryServicesAsrvCenter -Fabric $Fabric -Name 'V2VM'
```
