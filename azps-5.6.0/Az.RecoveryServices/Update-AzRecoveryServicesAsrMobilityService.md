# Update-AzRecoveryServicesAsrMobilityService

## SYNOPSIS
Atualizações do agente de serviço de mobilidade por push para máquinas protegidas.

## SINTAXE

```
Update-AzRecoveryServicesAsrMobilityService [-Account <ASRRunAsAccount>]
 -ReplicationProtectedItem <ASRReplicationProtectedItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Update-AzRecoveryServicesAsrMobilityService** tenta empurrar atualizações do agente de serviço de mobilidade para máquinas protegidas (se uma atualização estiver disponível.)

## EXEMPLOS

### Exemplo 1
```
PS C:\> Update-AzRecoveryServicesAsrMobilityService -ReplicationProtectedItem $rpi -Account $fabric.fabricSpecificDetails.RunAsAccounts[0]
```

Trabalho para rastrear o Agente de Serviço de Mobilidade do Item Protegido de Atualização.
