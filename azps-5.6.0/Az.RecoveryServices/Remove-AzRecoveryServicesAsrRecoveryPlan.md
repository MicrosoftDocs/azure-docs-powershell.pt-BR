# Remove-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Exclui o plano de recuperação ASR especificado do cofre dos Serviços de Recuperação.

## SINTAXE

### ByObject (Padrão)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzRecoveryServicesAsrRecoveryPlan** exclui o plano de recuperação especificado do cofre dos Serviços de Recuperação.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Inicia a exclusão do plano de recuperação especificado e retorna o trabalho ASR usado para rastrear a operação.
