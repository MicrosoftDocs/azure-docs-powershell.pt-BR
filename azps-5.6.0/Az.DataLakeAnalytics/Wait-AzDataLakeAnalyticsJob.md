# Wait-AzDataLakeAnalyticsJob

## SYNOPSIS
Aguarda a conclusão de um trabalho.

## SINTAXE

```
Wait-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [[-WaitIntervalInSeconds] <Int32>]
 [[-TimeoutInSeconds] <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Wait-AzDataLakeAnalyticsJob** aguarda a conclusão de um trabalho do Azure Data Lake Analytics.

## EXEMPLOS

### Exemplo 1: aguarde a conclusão de um trabalho
```
PS C:\>Wait-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

O comando a seguir aguarda a conclusão do trabalho com a ID especificada.
