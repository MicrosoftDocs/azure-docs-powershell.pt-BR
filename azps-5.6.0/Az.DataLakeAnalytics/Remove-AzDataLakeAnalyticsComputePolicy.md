# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Remove uma política de computação do Azure Data Lake Analytics especificada

## SINTAXE

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
A política de computação **Remove-AzDataLakeAnalyticsComputePolicy** remove uma política de computação do Azure Data Lake Analytics especificada.

## EXEMPLOS

### Exemplo 1: Remover uma política de computação
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Este comando remove a política de computação especificada com o nome 'myPolicy' na conta 'contosoadla'.
