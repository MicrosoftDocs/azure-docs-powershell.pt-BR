# Módulo Az.Communication
## Descrição
Microsoft Azure PowerShell: cmdlets de comunicação

## Az.Communication Cmdlets
### [Get-AzCommunicationService](Get-AzCommunicationService.md)
Obter o CommunicationService e suas propriedades.

### [Get-AzCommunicationServiceKey](Get-AzCommunicationServiceKey.md)
Obter as chaves de acesso do recurso CommunicationService.

### [New-AzCommunicationService](New-AzCommunicationService.md)
Crie um novo CommunicationService ou atualize um CommunicationService existente.

### [New-AzCommunicationServiceKey](New-AzCommunicationServiceKey.md)
Regenerar a chave de acesso do CommunicationService.
PrimaryKey e SecondaryKey não podem ser regenerados ao mesmo tempo.

### [Remove-AzCommunicationService](Remove-AzCommunicationService.md)
Operação para excluir um CommunicationService.

### [Set-AzCommunicationServiceNotificationHub](Set-AzCommunicationServiceNotificationHub.md)
Vincula um Hub de Notificação do Azure a esse serviço de comunicação.

### [Update-AzCommunicationService](Update-AzCommunicationService.md)
Operação para atualizar um CommunicationService existente.
