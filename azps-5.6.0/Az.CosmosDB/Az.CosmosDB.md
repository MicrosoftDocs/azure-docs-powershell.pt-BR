# Módulo Az.CosmosDB
## Descrição
Os tópicos desta seção documentam os cmdlets do Azure PowerShell para o Azure CosmosDB na estrutura do Gerenciador de Recursos do Azure (ARM). Os cmdlets existem no namespace Microsoft.Azure.Commands.CosmosDB.
