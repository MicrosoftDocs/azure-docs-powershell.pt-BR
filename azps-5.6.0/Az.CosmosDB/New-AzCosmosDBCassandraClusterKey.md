# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Cria uma nova Chave de Cluster de Cassandra do CosmosDB.

## SINTAXE

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O **New-AzCosmosDBCassandraClusterKey** cria uma nova Chave de Cluster de Cassandra de CosmosDB.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
