# New-AzCloudServiceVaultSecretGroupObject

## SYNOPSIS
Criar um objeto na memória para o Grupo Secreto do Vault

## SINTAXE

```
New-AzCloudServiceVaultSecretGroupObject [-CertificateUrl <String[]>] [-Id <String>] [<CommonParameters>]
```

## DESCRIPTION
Criar um objeto na memória para o Grupo Secreto

## EXEMPLOS

### Exemplo 1: Criar objeto de grupo secreto do cofre
```powershell
$keyVault = Get-AzKeyVault -VaultName 'ContosoKeyVault'
$certificate = Get-AzKeyVaultCertificate -VaultName 'ContosoKeyVault' -Name 'ContosoCert'
$secretGroup = New-AzCloudServiceVaultSecretGroupObject -Id $keyVault.ResourceId -CertificateUrl $certificate.SecretId
```

Este comando cria um objeto de grupo secreto do cofre que é usado para criar ou atualizar um serviço de nuvem.
Para obter mais detalhes, consulte New-AzCloudService.
