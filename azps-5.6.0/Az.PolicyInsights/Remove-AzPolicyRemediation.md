# Remove-AzPolicyRemediation

## SYNOPSIS
Exclui uma correção de política.

## SINTAXE

### ByName (Padrão)
```
Remove-AzPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzPolicyRemediation** exclui uma correção de política.

## EXEMPLOS

### Exemplo 1: Excluir uma correção de política no escopo do grupo de recursos
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Este comando exclui a correção chamada 'correção1' no grupo de recursos 'myRG'.

### Exemplo 2: Excluir uma correção de grupo de gerenciamento por meio de canalização
```
PS C:\> $remediation = Get-AzPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzPolicyRemediation -Confirm
```

Este comando exclui a correção denominada 'correção1' do grupo de gerenciamento 'mg1'. Um prompt de confirmação será apresentado antes de excluir o recurso.

### Exemplo 3: Cancelar e excluir uma correção de política
```
PS C:\> Remove-AzPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

Este comando exclui a correção chamada 'correção1' no grupo de recursos 'myRG'. Se a correção estiver em andamento, ela será cancelada antes de ser excluída.
