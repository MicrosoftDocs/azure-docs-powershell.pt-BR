# Remove-AzStorageCORSRule

## SYNOPSIS
Remove o CORS para um serviço de Armazenamento.

## SINTAXE

```
Remove-AzStorageCORSRule [-ServiceType] <StorageServiceType> [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzStorageCORSRule** remove o CORS (Compartilhamento de Recursos entre Origens) para um serviço de Armazenamento do Azure.
Este cmdlet exclui todas as regras do CORS em um tipo de serviço de Armazenamento.
Os tipos de serviços de armazenamento para este cmdlet são Blob, Table, Queue e File.

## EXEMPLOS

### Exemplo 1: Remover regras do CORS para o serviço blob
```
PS C:\>Remove-AzStorageCORSRule -ServiceType Blob
```

Este comando remove as regras do CORS para o tipo de serviço Blob.

### -ConcurrentTaskCount
Especifica o máximo de chamadas de rede simultâneas.
Você pode usar esse parâmetro para limitar a capacidade de limitar o uso de CPU local e largura de banda especificando o número máximo de chamadas de rede simultâneas.
O valor especificado é uma contagem absoluta e não é multiplicado pela contagem principal.
Esse parâmetro pode ajudar a reduzir problemas de conexão de rede em ambientes de baixa largura de banda, como 100 quilobits por segundo.
O valor padrão é 10.
