# Disable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Desabilite a política de retenção de exclusão para o serviço blob de armazenamento do Azure.

## SINTAXE

```
Disable-AzStorageDeleteRetentionPolicy [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Disable-AzStorageDeleteRetentionPolicy** desabilita a política de retenção de exclusão para o serviço blob de armazenamento do Azure.

## EXEMPLOS

### Exemplo 1: Desabilitar a política de retenção de exclusão para o serviço Blob
```
C:\PS>Disable-AzStorageDeleteRetentionPolicy
```

Este comando desabilita a política de retenção de exclusão para o serviço Blob.
