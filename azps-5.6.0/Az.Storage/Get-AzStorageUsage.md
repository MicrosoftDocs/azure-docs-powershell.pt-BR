# Get-AzStorageUsage

## SYNOPSIS
Obtém o uso de recursos de armazenamento da assinatura atual.

## SINTAXE

```
Get-AzStorageUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzStorageUsage** obtém o uso de recursos para o Armazenamento do Azure para a assinatura atual.

## EXEMPLOS

### Exemplo 1: Obter o uso de recursos de armazenamento do local especificado
```
PS C:\>Get-AzStorageUsage -Location 'West US'

LocalizedName : Storage Accounts
Name          : StorageAccounts
Unit          : Count
CurrentValue  : 18
Limit         : 250
```

Este comando obtém o uso de recursos de armazenamento do local especificado na assinatura atual.
