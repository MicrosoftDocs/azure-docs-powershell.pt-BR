# Set-AzStorageServiceLoggingProperty

## SYNOPSIS
Modifica o log dos serviços de Armazenamento do Azure.

## SINTAXE

```
Set-AzStorageServiceLoggingProperty [-ServiceType] <StorageServiceType> [-Version <Double>]
 [-RetentionDays <Int32>] [-LoggingOperations <LoggingOperations[]>] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzStorageServiceLoggingProperty** modifica o log dos serviços de Armazenamento do Azure.

## EXEMPLOS

### Exemplo 1: Modificar propriedades de registro em log para o serviço Blob
```
C:\PS>Set-AzStorageServiceLoggingProperty -ServiceType Blob -LoggingOperations Read,Write -PassThru -RetentionDays 10 -Version 1.0
```

Este comando modifica o registro em log da versão 1.0 para que o armazenamento blob inclua operações de leitura e gravação.
O registro em log do serviço de Armazenamento do Azure mantém entradas por 10 dias.
Como esse comando especifica o parâmetro *PassThru,* o comando exibe as propriedades de registro em log modificadas.
