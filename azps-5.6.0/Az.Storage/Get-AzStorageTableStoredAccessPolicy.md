# Get-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Obtém a política ou políticas de acesso armazenado para uma tabela de armazenamento do Azure.

## SINTAXE

```
Get-AzStorageTableStoredAccessPolicy [-Table] <String> [[-Policy] <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzStorageTableStoredAccessPolicy** lista a política de acesso armazenado ou as políticas de uma tabela de armazenamento do Azure.

## EXEMPLOS

### Exemplo 1: obter uma política de acesso armazenado em uma tabela de armazenamento
```
PS C:\>Get-AzStorageTableStoredAccessPolicy -Table "Table02" -Policy "Policy50"
```

Este comando obtém a política de acesso chamada Policy50 na tabela de armazenamento chamada Table02.

### Exemplo 2: Obter todas as políticas de acesso armazenadas em uma tabela de armazenamento
```
PS C:\>Get-AzStorageTableStoredAccessPolicy -Table "Table02"
```

Este comando obtém todas as políticas de acesso na tabela chamada Table02.
