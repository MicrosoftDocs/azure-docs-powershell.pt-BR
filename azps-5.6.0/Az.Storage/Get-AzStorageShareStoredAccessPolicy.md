# Get-AzStorageShareStoredAccessPolicy

## SYNOPSIS
Obtém políticas de acesso armazenadas para um compartilhamento de armazenamento.

## SINTAXE

```
Get-AzStorageShareStoredAccessPolicy [-ShareName] <String> [[-Policy] <String>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzStorageShareStoredAccessPolicy** obtém políticas de acesso armazenadas para um compartilhamento de Armazenamento do Azure.
Para obter uma política específica, especifique-a pelo nome.

## EXEMPLOS

### Exemplo 1: Obter uma política de acesso armazenado em um compartilhamento
```
PS C:\>Get-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Este comando obtém uma política de acesso armazenada chamada GeneralPolicy em ContosoShare.

### Exemplo 2: Obter todas as políticas de acesso armazenadas no compartilhamento
```
PS C:\>Get-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare"
```

Esse comando obtém todas as políticas de acesso armazenadas em ContosoShare.

### -ConcurrentTaskCount
Especifica o máximo de chamadas de rede simultâneas.
Você pode usar esse parâmetro para limitar a capacidade de limitar o uso de CPU local e largura de banda especificando o número máximo de chamadas de rede simultâneas.
O valor especificado é uma contagem absoluta e não é multiplicado pela contagem principal.
Esse parâmetro pode ajudar a reduzir problemas de conexão de rede em ambientes de baixa largura de banda, como 100 quilobits por segundo.
O valor padrão é 10.
