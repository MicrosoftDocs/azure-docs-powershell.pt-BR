# Get-AzMarketplaceTerms

## SYNOPSIS
Obter os termos de contrato para um determinado publisher id(Publisher), id(Product) e plan id(Name). O objeto terms retornado por este comando deve ser passado para Set-AzMarketplaceTerms aceitar os termos legais.

## SINTAXE

```
Get-AzMarketplaceTerms -Publisher <String> -Product <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzMarketplaceTerms** retorna termos para determinado id(Publisher), id(Product) de oferta e planejamento de id(Name).

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzMarketplaceTerms -Publisher "microsoft-ads" -Product "windows-data-science-vm" -Name "windows2016"
Publisher         : microsoft-ads
Product           : windows-data-science-vm
Plan              : windows2016
LicenseTextLink   : <LicenseTextLink>
PrivacyPolicyLink : <PrivacyPolicyLink>
Signature         : <Signature>
Accepted          : True
RetrieveDatetime  : <RetrieveDatetime>
```
