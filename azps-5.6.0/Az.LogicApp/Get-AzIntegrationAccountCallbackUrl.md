# Get-AzIntegrationAccountCallbackUrl

## SYNOPSIS
Obtém uma URL de retorno de chamada de conta de integração.

## SINTAXE

```
Get-AzIntegrationAccountCallbackUrl -ResourceGroupName <String> -Name <String> [-NotAfter <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Get-AzIntegrationAccountCallbackUrl** obtém uma URL de retorno de chamada de conta de integração de um grupo de recursos.
Este cmdlet retorna um **objeto CallbackUrl** que representa a URL de retorno de chamada da conta de integração.
Especifique o nome da conta de integração e o nome do grupo de recursos.
Este módulo dá suporte a parâmetros dinâmicos.
Para usar um parâmetro dinâmico, digite-o no comando.
Para descobrir os nomes dos parâmetros dinâmicos, digite um hífen (-) após o nome do cmdlet e pressione a tecla Tab repetidamente para passar pelos parâmetros disponíveis.
Se você omitir um parâmetro de modelo necessário, o cmdlet solicitará o valor.

## EXEMPLOS

### Exemplo 1: Obter uma URL de retorno de chamada de conta de integração
```
PS C:\>Get-AzIntegrationAccountCallbackUrl -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -NotAfter "03/25/2016 18:23:22"
CallBackUrl : https://<baseurl>/integrationAccounts/8811f0155a364b5e9618ba28f7180601?api-version=2015-08-01-preview&se=2016-03
              -25T18%3A23%3A22.0000000Z&sp=%2F%2Fread&sv=1.0&sig=<value>
```

Este comando obtém uma URL de retorno de chamada de conta de integração.
