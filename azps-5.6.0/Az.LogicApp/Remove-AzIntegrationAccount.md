# Remove-AzIntegrationAccount

## SYNOPSIS
Remove uma conta de integração.

## SINTAXE

```
Remove-AzIntegrationAccount -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Remove-AzIntegrationAccount** remove uma conta de integração de um grupo de recursos.
Especifique o nome da conta de integração e o nome do grupo de recursos.
Este módulo dá suporte a parâmetros dinâmicos.
Para usar um parâmetro dinâmico, digite-o no comando.
Para descobrir os nomes dos parâmetros dinâmicos, digite um hífen (-) após o nome do cmdlet e pressione a tecla Tab repetidamente para passar pelos parâmetros disponíveis.
Se você omitir um parâmetro de modelo necessário, o cmdlet solicitará o valor.

## EXEMPLOS

### Exemplo 1: Remover uma conta de integração
```
PS C:\>Remove-AzIntegrationAccount -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -Force
```

Este comando remove uma conta de integração chamada IntegrationAccount31.
