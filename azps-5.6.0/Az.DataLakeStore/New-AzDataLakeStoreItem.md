# New-AzDataLakeStoreItem

## SYNOPSIS
Cria um novo arquivo ou pasta no Data Lake Store.

## SINTAXE

```
New-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-Value] <Object>]
 [[-Encoding] <Encoding>] [-Folder] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **New-AzDataLakeStoreItem** cria um novo arquivo ou pasta no Data Lake Store.

## EXEMPLOS

### Exemplo 1: Criar um novo arquivo e uma nova pasta
```
PS C:\>New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFile.txt"
PS C:\> New-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/NewFolder" -Folder
```

O primeiro comando cria o arquivo NewFile.txt para a conta especificada.
O segundo comando cria a pasta NewFolder na pasta raiz.
