# Set-AzDataLakeStoreItemPermission

## SYNOPSIS
Modifica o octal de permissão de um arquivo ou pasta no Data Lake Store.

## SINTAXE

```
Set-AzDataLakeStoreItemPermission [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Permission] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
O cmdlet **Set-AzDataLakeStoreItemPermission** modifica o octal de permissão de um arquivo ou pasta no Data Lake Store.

## EXEMPLOS

### Exemplo 1: definir o octal de permissão para um item
```
PS C:\>Set-AzDataLakeStoreItemPermission -AccountName "ContosoADL" -Path "/file.txt" -Permission 0770
```

Este comando define o octal de permissão para um arquivo como 0770, que se traduz em limpar o bit grudado, definindo permissões de leitura/gravação/execução para o proprietário do arquivo, definindo permissões de leitura/gravação/execução para o grupo proprietário do arquivo e limpando permissões de leitura/gravação/execução para outras pessoas.
