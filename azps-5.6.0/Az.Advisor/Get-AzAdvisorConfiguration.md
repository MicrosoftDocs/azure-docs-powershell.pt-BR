# Get-AzAdvisorConfiguration

## SYNOPSIS
Obter as configurações do Azure Advisor para a assinatura ou grupo de recursos determinado.

## SINTAXE

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
As configurações associadas a uma assinatura têm dois tipos:

Configuração de nível de assinatura: só pode haver uma configuração desse tipo para uma assinatura. LowCpuThreshold e Exclude são a única propriedade desse tipo de configuração.

Configuração de nível resourceGroup: pode haver apenas uma configuração para cada ResourceGroup em uma assinatura. Exclude é a única propriedade desse tipo de configuração.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\>$data = Get-AzAdvisorConfiguration
Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}
Name       : {user_subscription}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}-{resourceGroupName}
Name       : {user_subscription}-{resourceGroupName}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

PS C:\>$data[0].Properties
AdditionalProperties :
Exclude              : False
LowCpuThreshold      : 20

PS C:\>$data[1].Properties

```
Recupera uma lista de configurações do Azure Advisor.
