# Get-AzSupportProblemClassification

## SYNOPSIS
Obter classificações de problemas para o serviço especificado.

## SINTAXE

### GetByNameParameterSet (Padrão)
```
Get-AzSupportProblemClassification -ServiceId <String> [-Id <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSupportProblemClassification [-Id <String>] -ServiceObject <PSSupportService>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Obtém a lista atual de classificação de problemas para um serviço do Azure. Você pode usar o GUID de classificação de problemas e serviço para criar um novo tíquete de suporte usando New-AzSupportTicket.

Sempre use os GUIDs de classificação de problemas e de serviço obtidos programaticamente. Essa prática garante que você tenha o conjunto mais recente de GUIDs de classificação de problemas e serviço para criação de tíquetes de suporte.
