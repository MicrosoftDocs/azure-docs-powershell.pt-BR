# Get-AzAksVersion

## SYNOPSIS
Listar a versão disponível para a criação de cluster kubernetes gerenciado.

## SINTAXE

```
Get-AzAksVersion -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Listar a versão disponível para a criação de cluster kubernetes gerenciado.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzAksVersion -Location westus
```
