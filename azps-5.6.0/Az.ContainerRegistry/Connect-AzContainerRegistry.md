# Connect-AzContainerRegistry

## SYNOPSIS
Faça logon em um registro de contêiner do azure.

## SINTAXE

### WithoutNameAndPasswordParameterSet (Default)
```
Connect-AzContainerRegistry -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### WithNameAndPasswordParameterSet
```
Connect-AzContainerRegistry -Name <String> -UserName <String> -Password <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIPTION
Faça logon em um registro de contêiner do azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName
```

Faça logon no ACR sem credenciais quando já entrar na conta do azure.

### Exemplo 2
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $RegistryName -Password $AdminPassWord
```

Faça logon no ACR com nome de usuário/senha do administrador quando o usuário do administrador estiver habilitado.

### Exemplo 3
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $ServicePrincipal -Password $ServicePrincipalPassword
```

Faça logon no ACR com a ID do aplicativo principal de serviço e a senha.
