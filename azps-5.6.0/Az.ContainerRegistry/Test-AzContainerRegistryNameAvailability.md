# Test-AzContainerRegistryNameAvailability

## SYNOPSIS
Verifica a disponibilidade de um nome de registro de contêiner.

## SINTAXE

```
Test-AzContainerRegistryNameAvailability [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIPTION
O Test-AzContainerRegistryNameAvailability cmdlet verifica se um nome de registro de contêiner é válido e disponível para uso.

## EXEMPLOS

### Exemplo 1: verifica a disponibilidade de um nome de registro de contêiner
```powershell
PS C:\>Test-AzContainerRegistryNameAvailability -Name 'SomeRegistryName'

NameAvailable Reason Message
------------- ------ -------
         True
```

Este comando verifica a disponibilidade do nome do registro de contêiner \` SomeRegistryName \` .
