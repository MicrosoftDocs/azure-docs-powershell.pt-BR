# Merge-AzReservation

## SYNOPSIS
Mescla dois `Reservation` s.

## SINTAXE

### CommandLine (Padrão)
```
Merge-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Merge-AzReservation -Reservation <PSReservation[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Mesclar o `Reservation` s especificado em um novo `Reservation` . Os dois `Reservation` s que estão sendo mesclados devem ter as mesmas propriedades.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Merge-AzReservation -ReservationOrderId "1111aaaa-b1b2-c0c2-d0d2-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111","11111111-0000-0000-0000-1111111111"
```

Mesclar os dois `Reservation` s especificados em um `Reservation`
