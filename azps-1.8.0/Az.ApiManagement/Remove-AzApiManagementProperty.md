# Remove-AzApiManagementProperty

## Sinopse
Remove uma propriedade de gerenciamento de API.

## SYNTAX

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApiManagementProperty** remove uma **Propriedade** de gerenciamento de API do Azure.

## EXEMPLOS

### Exemplo 1: remover uma propriedade
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Esse comando Remove a propriedade que tem a ID Property11.
