# Remove-AzKustoDatabase

## Sinopse
Exclui um banco de dados do Kusto existente.

## SYNTAX

### ByNameAndResourceGroup (padrão)
```
Remove-AzKustoDatabase [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoDatabase [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoDatabase [-InputObject] <PSKustoDatabase> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Exclui um banco de dados do Kusto existente.

## EXEMPLOS

### Exemplo 1-excluir um banco de dados do Kusto existente por nome

```
PS C:\> Remove-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase
```

O comando acima exclui o banco de dados Kusto chamado "mykustodatabase" no cluster "mykustocluster" localizado no grupo de recursos "testrg".

### Exemplo 2-excluir um banco de dados do Kusto existente por tubulação

```
PS C:\> Get-AzKustoDatabase -ResourceGroupName testrg -ClusterName mykustocluster -Name mykustodatabase | Remove-AzKustoDatabase
```

O comando acima Obtém o banco de dados Kusto chamado "mykustodatabase" no cluster "mykustocluster" localizado no grupo de recursos "testrg" usando o `Get-AzKustoDatabase` cmdlet e canaliza o resultado para `Remove-AzKustoDatabase` excluí-lo.
