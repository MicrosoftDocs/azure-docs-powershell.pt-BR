# Remove-AzRelayHybridConnection

## Sinopse
Remove o HybridConnection do namespace HybridConnection especificado.

## SYNTAX

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzRelayHybridConnection** remove o HybridConnection do namespace de retransmissão especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Remove o HybridConnection `TestHybridConnection` do namespace `TestNameSpace-Relay1` .
