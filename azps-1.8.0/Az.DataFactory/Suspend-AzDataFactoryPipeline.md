# Suspend-AzDataFactoryPipeline

## Sinopse
Suspende um pipeline no Azure data Factory.

## SYNTAX

### ByFactoryName (padrão)
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Suspend-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Suspend-AzDataFactoryPipeline** suspende um pipeline no Azure data Factory.
Você pode retomar o pipeline usando o cmdlet Resume-AzDataFactoryPipeline.

## EXEMPLOS

### Exemplo 1: suspender um pipeline
```
PS C:\>Suspend-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikiSample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to suspend pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Esse comando suspende o pipeline chamado DPWikiSample no data Factory chamado WikiADF.
O comando retorna um valor de $True.
