# Register-AzModule

## Sinopse
PARA uso interno somente – forneça suporte ao tempo de execução para cmdlets gerados pelo autorest

## SYNTAX

```
Register-AzModule [<CommonParameters>]
```

## DESCRITIVO
PARA uso interno somente – forneça suporte ao tempo de execução para cmdlets gerados pelo autorest

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Register-AzModule
```

Usado internamente por cmdlets gerados pelo autorest
