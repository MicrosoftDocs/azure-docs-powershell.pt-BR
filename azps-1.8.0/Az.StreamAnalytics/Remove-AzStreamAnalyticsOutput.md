# Remove-AzStreamAnalyticsOutput

## Sinopse
Exclui uma saída de um trabalho do Stream Analytics.

## SYNTAX

```
Remove-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzStreamAnalyticsOutput** exclui de forma assíncrona uma saída de um trabalho do Stream Analytics no Azure.

## EXEMPLOS

### EXEMPLO 1: remover uma saída de um trabalho
```
PS C:\>Remove-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Esse comando Remove a saída de saída no trabalho StreamingJob.
