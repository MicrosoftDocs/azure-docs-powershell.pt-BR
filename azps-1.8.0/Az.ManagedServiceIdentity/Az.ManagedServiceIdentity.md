# Módulo AZ. ManagedServiceIdentity
## Descritivo
Este tópico exibe os tópicos da ajuda para os cmdlets de identidade de serviço gerenciado do Azure.

## Cmdlets AZ. ManagedServiceIdentity
### [Get-AzUserAssignedIdentity](Get-AzUserAssignedIdentity.md)
Obtém identidade/identidade atribuídas pelo usuário.

### [New-AzUserAssignedIdentity](New-AzUserAssignedIdentity.md)
Cria uma nova identidade atribuída ao usuário ou atualiza uma identidade atribuída pelo usuário existente.

### [Remove-AzUserAssignedIdentity](Remove-AzUserAssignedIdentity.md)
Remove uma identidade atribuída pelo usuário.
