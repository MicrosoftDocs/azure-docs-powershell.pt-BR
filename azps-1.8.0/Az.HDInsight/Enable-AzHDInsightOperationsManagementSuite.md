# Enable-AzHDInsightOperationsManagementSuite

## Sinopse
Habilita o OMS (Operations Management Suite) em um cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho do OMS especificado durante a ativação.

## SYNTAX

```
Enable-AzHDInsightOperationsManagementSuite [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Enable-AzHDInsightOperationsManagementSuite** HABILITA o OMS (Operations Management Suite) em um cluster do Azure HDInsight.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

O serviço de gerenciamento de operações (OMS) será habilitado no cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho do OMS com a ID 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Exemplo 2
```
PS C:\> Enable-AzHDInsightOMS -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

O serviço de gerenciamento de operações (OMS) será habilitado no cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho do OMS com a ID 1d364e89-bb71-4503-aa3d-a23535aea7bd
