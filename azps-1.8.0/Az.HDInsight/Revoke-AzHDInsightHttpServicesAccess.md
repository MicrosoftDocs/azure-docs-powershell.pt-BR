# Revoke-AzHDInsightHttpServicesAccess

## Sinopse
Desabilita o acesso HTTP ao cluster.

## SYNTAX

```
Revoke-AzHDInsightHttpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **REVOKE-AzHDInsightHttpServicesAccess** desabilita o acesso http a um cluster do Azure HDInsight para os serviços Web ODBC, Ambari, Oozie e webHCatalog.

## EXEMPLOS

### Exemplo 1: desabilitar o acesso HTTP a um cluster
```
PS C:\>Revoke-AzHDInsightHttpServicesAccess `
       -ClusterName "your-hadoop_001"
```

Esse comando revoga o acesso HTTP ao cluster chamado seu hadoop_001.
