# Get-AzOperationalInsightsSchema

## Sinopse
Retorna o esquema associado a um espaço de trabalho.

## SYNTAX

```
Get-AzOperationalInsightsSchema [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzOperationalInsightsSchema** retorna o esquema associado ao espaço de trabalho especificado dentro desse grupo de recursos.

## EXEMPLOS

### Exemplo 1: obter os esquemas de um espaço de trabalho
```
PS C:\>Get-AzOperationalInsightsSchema -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Esse comando obtém os esquemas associados a um espaço de trabalho.
