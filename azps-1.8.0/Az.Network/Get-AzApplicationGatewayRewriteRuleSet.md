# Get-AzApplicationGatewayRewriteRuleSet

## Sinopse
Obtém o conjunto de regras de regravação de um gateway de aplicativo.

## SYNTAX

```
Get-AzApplicationGatewayRewriteRuleSet [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obtém o conjunto de regras de regravação de um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1: obter um conjunto de regras de regravação específico
```
PS C:\> $AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Rule = Get-AzApplicationGatewayRewriteRuleSet -Name "RuleSet01" -ApplicationGateway $AppGW
```

O primeiro comando obtém o gateway do aplicativo chamado ApplicationGateway01 e armazena o resultado na variável chamada $AppGW.
O segundo comando obtém o conjunto de regras Rewrite chamado RuleSet01 do gateway do aplicativo armazenado na variável chamada $AppGW.
