# Remove-AzRecoveryServicesAsrRecoveryPlan

## Sinopse
Exclui o plano de recuperação ASR especificado do cofre de serviços de recuperação.

## SYNTAX

### ByObject (padrão)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzRecoveryServicesAsrRecoveryPlan** exclui o plano de recuperação especificado do cofre de serviços de recuperação.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Inicia a exclusão do plano de recuperação especificado e retorna o trabalho ASR usado para acompanhar a operação.
