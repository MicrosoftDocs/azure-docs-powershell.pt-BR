# Test-AzAnalysisServicesServer

## Sinopse
Testa a existência de uma instância do servidor do Analysis Services

## SYNTAX

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Test-AzAnalysisServicesServer testa a existência de uma instância do servidor do Analysis Services

## EXEMPLOS

### Exemplo 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Esse comando testará se há um servidor chamado TestServer no botão de teste do MySource
