# Get-AzMlOpCluster

## Sinopse
Obtém um objeto de cluster operacional de operação.

## SYNTAX

### GetByName
```
Get-AzMlOpCluster -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceGroup
```
Get-AzMlOpCluster [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obtém um objeto de cluster operacional de operação por nome ou grupo de recursos ou por assinatura.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzMlOpCluster -ResourceGroupName my-group -Name my-cluster
```

Obtém um cluster de operação específico por nome.

### Exemplo 2
```
PS C:\> Get-AzMlOpCluster -ResourceGroupName my-group
```

Obtém todos os clusters de operação de funcionamento em um grupo de recursos.

### Exemplo 3
```
PS C:\> Get-AzMlOpCluster
```

Obtém todos os clusters de operação de funcionamento em uma assinatura.
