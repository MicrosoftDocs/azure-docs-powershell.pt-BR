# Get-AzSqlDatabaseExpanded

## Sinopse
Obtém um banco de dados e seus valores de propriedade expandida.

## SYNTAX

```
Get-AzSqlDatabaseExpanded [-ServerName] <String> [[-DatabaseName] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlDatabaseExpanded** Obtém um banco de dados e seus valores de propriedade expandida.
Esse cmdlet também é compatível com o serviço Stretch Database do SQL Server no Azure.

## EXEMPLOS

### Exemplo 1: obter objeto de banco de dados com informações do supervisor da camada de serviço
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando retorna o banco de dados que tem propriedades expandidas que contêm as informações do supervisor da camada de serviço.

### Exemplo 2: listar objetos do banco de dados usando filtragem
```
PS C:\> Get-AzSqlDatabaseExpanded -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database*"
```

Esse comando retorna um objeto de banco de dados expandido para todos os bancos de dados no Server01 que começam com "banco de dados".
