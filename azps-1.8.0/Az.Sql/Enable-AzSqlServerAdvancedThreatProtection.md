# Enable-AzSqlServerAdvancedThreatProtection

## Sinopse
Habilita a proteção avançada contra ameaças em um servidor.

## SYNTAX

```
Enable-AzSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Enable-AzSqlServerAdvancedThreatProtection** habilita a proteção avançada contra ameaças em um servidor.

## EXEMPLOS

### Exemplo 1-habilitar a proteção avançada contra ameaças do servidor
```powershell
PS C:\>  Enable-AzSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exemplo 2-habilitar a proteção avançada contra ameaças do servidor do recurso do servidor
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
