# Remove-AzSqlServerCommunicationLink

## Sinopse
Exclui um link de comunicação para transações de banco de dados elástico entre dois servidores.

## SYNTAX

```
Remove-AzSqlServerCommunicationLink [-LinkName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzSqlServerCommunicationLink** exclui um link de comunicação de servidor para servidor para transações de banco de dados elástico no banco de dados SQL do Azure.

## EXEMPLOS

### Exemplo 1: excluir um link de comunicação
```
PS C:\>Remove-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01"
```

Esse comando exclui um link de comunicação de servidor para servidor chamado Link01 em ContosoServer17.
