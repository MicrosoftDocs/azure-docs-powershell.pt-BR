# Get-AzSqlInstance

## Sinopse
Retorna informações sobre a instância de banco de dados gerenciado do Azure SQL.

## SYNTAX

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlInstance** retorna informações sobre uma ou mais instâncias gerenciadas do SQL do Azure.
Especifique o nome de uma instância para ver informações somente para essa instância.
