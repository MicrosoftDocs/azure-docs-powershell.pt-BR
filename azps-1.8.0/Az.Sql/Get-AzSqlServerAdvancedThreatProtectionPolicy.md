# Get-AzSqlServerAdvancedThreatProtectionPolicy

## Sinopse
Obtém uma política avançada de proteção contra ameaças de um servidor.

## SYNTAX

```
Get-AzSqlServerAdvancedThreatProtectionPolicy [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlServerAdvancedThreatProtectionPolicy** Retrives a política avançada de proteção contra ameaças de um servidor.

## EXEMPLOS

### Exemplo 1-obtém proteção avançada contra ameaças do servidor
```powershell
PS C:\>  Get-AzSqlServerAdvancedThreatProtectionPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exemplo 2-obtém proteção avançada contra ameaças do servidor do recurso do servidor
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Get-AzSqlServerAdvancedThreatProtectionPolicy

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
