# Get-AzOperationalInsightsCluster

## Sinopse
Obter ou listar clusters

## Sintaxe

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Obter ou listar clusters, agrupar listas em grupo de recursos quando "-ClusterName" não foi fornecido, listar clusters em assinatura quando "-ClusterName" e "ResourceGroupName" não foram fornecidos.

## Exemplos

### Exemplo 1
```powershell
Get-AzOperationalInsightsCluster -ResourceGroupName {rg-name} -ClusterName {cluster-name}

```
