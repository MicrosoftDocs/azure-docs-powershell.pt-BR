# Disable-AzOperationalInsightsIISLogCollection

## Sinopse
Interrompe a coleta de logs de IIS de computadores.

## Sintaxe

### ByWorkspaceName (Padrão)
```
Disable-AzOperationalInsightsIISLogCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsIISLogCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Disable-AzOperationalInsightsIISLogCollection** interrompe a coleção de logs do IIS (Serviços de Informações da Internet) de computadores conectados em um espaço de trabalho.
