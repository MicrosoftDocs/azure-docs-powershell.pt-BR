# Módulo Az.DedicatedHsm
## Descrição
Microsoft Azure PowerShell: cmdlets DedicatedHsm

## Az.DedicatedHsm Cmdlets
### [Get-AzDedicatedHsm](Get-AzDedicatedHsm.md)
Obtém o HSM dedicado do Azure especificado.

### [New-AzDedicatedHsm](New-AzDedicatedHsm.md)
Criar ou atualizar um HSM dedicado na assinatura especificada.

### [Remove-AzDedicatedHsm](Remove-AzDedicatedHsm.md)
Exclui o HSM dedicado do Azure especificado.

### [Update-AzDedicatedHsm](Update-AzDedicatedHsm.md)
Atualize um HSM dedicado na assinatura especificada.
