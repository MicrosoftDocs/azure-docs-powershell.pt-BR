# Set-AzAutomationVariable

## Sinopse
Modifica uma variável automação.

## Sintaxe

### UpdateVariableValue
```
Set-AzAutomationVariable [-Name] <String> -Encrypted <Boolean> -Value <Object> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### UpdateVariableDescription
```
Set-AzAutomationVariable [-Name] <String> -Description <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzAutomationVariable** modifica o valor ou a descrição de uma variável na Automação do Azure.
Para criptografar a variável, especifique o *parâmetro Criptografado.*
Não é possível modificar o estado criptografado de uma variável após a criação.
A *especificação criptografada* para uma variável existente, não criptografada.

## Exemplos

### Exemplo 1: Definir o valor de uma variável
```
PS C:\>Set-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -ResourceGroupName "ResourceGroup01" -Value "New Value" -Encrypted $False
```

Esse comando define um novo valor para a variável chamada StringVariable22 na conta de Automação do Azure chamada Contoso17.
