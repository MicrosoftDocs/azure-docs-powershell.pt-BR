# Get-AzAutomationHybridWorkerGroup

## Sinopse
Obtém grupos de trabalhadores de runbook híbridos.

## Sintaxe

### ByAll (Padrão)
```
Get-AzAutomationHybridWorkerGroup [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationHybridWorkerGroup [[-Name] <String>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzAutomationHybridWorkerGroup** obtém grupos de trabalhadores híbridos do runbook Azure Automation.
Para obter um grupo específico, especifique seu nome.

## Exemplos

### Exemplo 1: Obter todos os grupos de trabalhadores de runbook híbridos
```
PS C:\>Get-AzAutomationHybridWorkerGroup -ResourceGroupName "ResourceGroupName01" -AutomationAccountName "Contoso17"
```

Esse comando obtém todos os grupos de trabalhadores de runbook híbridos na conta automação chamada Contoso17.

### Exemplo 2: Obter um único grupo de trabalhadores de runbook híbrido
```
PS C:\>Get-AzAutomationHybridWorkerGroup -ResourceGroupName "ResourceGroupName01" -AutomationAccountName "Contoso17" -Name "HybridRunbookWorkerGroup01"
```

Esse comando obtém o grupo de trabalhadores de runbook híbrido chamado HybridRunbookWorkerGroup01 na conta automação chamada Contoso17.

### Exemplo 3: Obter os trabalhadores em um grupo de trabalhadores de um runbook híbrido
```
PS C:\>(Get-AzAutomationHybridWorker -ResourceGroupName ResourceGroupName01 -AutomationAccountName Contoso17 -Name "HybridRunbookWorkerGroup01" ).RunbookWorker
```

Esse comando obtém os trabalhadores de um runbook híbrido no grupo de trabalhadores de runbook híbrido chamado HybridRunbookWorkerGroup01 na conta automação chamada Contoso17.
