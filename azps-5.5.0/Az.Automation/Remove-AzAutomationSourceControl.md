# Remove-AzAutomationSourceControl

## Sinopse
Remove um controle de origem da Automação do Azure.

## Sintaxe

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O Remove-AzAutomationSourceControl cmdlet remove um controle de origem da Automação do Azure.

## Exemplos

### Exemplo 1
Esse comando remove o controle de origem automação chamado VSTSNative na conta chamada devAccount.
Esse comando especifica o parâmetro *Forçar.* Portanto, ele não solicita confirmação.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
