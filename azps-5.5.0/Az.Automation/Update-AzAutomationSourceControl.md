# Update-AzAutomationSourceControl

## Sinopse
Atualiza um controle de origem da Automação do Azure.

## Sintaxe

```
Update-AzAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Update-AzAutomationSourceControl cmdlet modifica o valor de um campo em um controle de origem na Automação do Azure.

## Exemplos

### Exemplo 1
Esse comando define o campo PublishRunbook como false para o controle de origem da Automação do Azure chamado VSTSNative na conta chamada devAccount.


```powershell
Update-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false

```

### Exemplo 2

Atualiza um controle de origem de automação do Azure. (gerado automaticamente)
