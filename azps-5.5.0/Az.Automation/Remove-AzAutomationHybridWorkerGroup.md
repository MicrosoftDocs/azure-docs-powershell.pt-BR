# Remove-AzAutomationHybridWorkerGroup

## Sinopse
Remove o grupo de trabalhadores híbridos da Automação.

## Sintaxe

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O Remove-AzAutomationHybridWorkerGroup cmdlet remove um grupo de trabalhadores híbridos da Automação.

## Exemplos

### Exemplo 1
Esse comando remove um trabalhador híbrido por nome.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
