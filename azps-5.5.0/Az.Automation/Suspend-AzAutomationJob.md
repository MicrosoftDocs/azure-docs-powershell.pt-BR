# Suspend-AzAutomationJob

## Sinopse
Suspende um trabalho de Automação.

## Sintaxe

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Suspend-AzAutomation Job** suspende um trabalho de Automação do Azure.
Especifique um trabalho de automação em execução.
Para retomar um trabalho suspenso, use Resume-AzAutomationJob cmdlet.

## Exemplos

### Exemplo 1: Suspender um trabalho
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Esse comando suspende o trabalho que tem a ID especificada.
