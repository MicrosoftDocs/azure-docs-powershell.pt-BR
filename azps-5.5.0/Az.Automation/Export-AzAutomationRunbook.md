# Export-AzAutomationRunbook

## Sinopse
Exporta um manual de automação.

## Sintaxe

```
Export-AzAutomationRunbook [-Name] <String> [-Slot <String>] [-OutputFolder <String>] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Export-AzAutomationRunbook** exporta uma pasta de trabalho de automação do Azure para um arquivo de script do wps_2 (.ps1), para arquivos de fluxo de trabalho do wps_2 ou wps_2 ou para um arquivo de pasta de trabalho gráfica (.graphrunbook) para runbooks gráficos.
O nome da pasta de executar torna-se o nome do arquivo exportado.

## Exemplos

### Exemplo 1: Exportar um livro de runbook
```
PS C:\>Export-AzAutomationRunbook -ResourceGroupName "ResourceGroup01" -AutomationAccountName "ContosoAutomationAccount" -Name "Runbook03" -Slot "Published" -OutputFolder "C:\Users\PattiFuller\Desktop"
```

Esse comando exporta a versão publicada de um manual de automação para uma área de trabalho do usuário.
