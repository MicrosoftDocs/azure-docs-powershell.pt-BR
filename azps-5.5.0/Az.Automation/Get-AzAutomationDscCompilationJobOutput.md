# Get-AzAutomationDscCompilationJobOutput

## Sinopse
Obtém os fluxos de log de um trabalho de compilação DSC de automação.

## Sintaxe

```
Get-AzAutomationDscCompilationJobOutput [-Id] <Guid> [-Stream <CompilationJobStreamType>]
 [-StartTime <DateTimeOffset>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzAutomationDscCompilation JobOutput** obtém os registros de fluxo de um trabalho de compilação de DSC (Configuração de Estado Desejado) APS na Automação do Azure.

## Exemplos

### Exemplo 1: Obter os logs para um trabalho de compilação de DSC
```
PS C:\>$Jobs = Get-AzAutomationDscCompilationJob -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
PS C:\> $Jobs[0] | Get-AzAutomationDscCompilationJobOutput -Stream "Any"
```

O primeiro comando obtém os trabalhos de compilação na conta automação chamada Contoso17 usando o cmdlet Get-AzAutomationDscCompilationJob dados.
O comando armazena esses objetos na variável $Jobs dados.
O segundo comando obtém a saída do trabalho de compilação de qualquer fluxo para o primeiro membro da matriz $Jobs dados.
