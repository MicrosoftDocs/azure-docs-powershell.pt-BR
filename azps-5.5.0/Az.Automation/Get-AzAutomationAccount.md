# Get-AzAutomationAccount

## Sinopse
Obtém contas de Automação em um grupo de recursos.

## Sintaxe

### ByAll (Padrão)
```
Get-AzAutomationAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByAutomationAccountName
```
Get-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzAutomationAccount** obtém contas de Automação do Azure em um grupo de recursos.
Para obter mais informações sobre contas de automação, consulte New-AzAutomationAccount cmdlet.

## Exemplos

### Exemplo 1: Obter todas as contas
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03"
```

Esse comando obtém todas as contas de Automação no grupo de recursos chamado ResourceGroup03.

### Exemplo 2: Obter uma conta
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03" -Name "ContosoAutomationAccount"
```

Esse comando obtém a conta automação chamada ContosoAutomationAccount no grupo de recursos chamado ContosoResourceGroup.
