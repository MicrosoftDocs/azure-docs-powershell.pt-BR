# New-AzAutomationAccount

## Sinopse
Cria uma conta de Automação.

## Sintaxe

```
New-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-Plan <String>]
 [-Tags <IDictionary>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzAutomationAccount** cria uma conta de Automação do Azure em um grupo de recursos.
Uma conta de Automação é um contêiner para recursos de automação que está isolado dos recursos de outras contas de Automação. Os recursos de automação incluem runbooks, configurações de DSC (Configuração de Estado Desejada), trabalhos e ativos.

## Exemplos

### Exemplo 1: Criar uma conta de automação
```powershell
PS C:\> New-AzAutomationAccount -Name "ContosoAutomationAccount" -Location "East US" -ResourceGroupName "ResourceGroup01"
```

Esse comando cria uma nova conta de automação chamada ContosoAutomationAccount na região leste dos EUA.

### Exemplo 2

Cria uma conta de Automação. (gerado automaticamente)

<!-- Aladdin Generated Example -->
```powershell
New-AzAutomationAccount -Location 'East US' -Name 'ContosoAutomationAccount' -ResourceGroupName 'ResourceGroup01' -Tags <IDictionary>
```
