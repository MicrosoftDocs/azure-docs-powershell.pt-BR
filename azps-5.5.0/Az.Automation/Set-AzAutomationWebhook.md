# Set-AzAutomationWebhook

## Sinopse
Modifica um web browser para um manual de automação.

## Sintaxe

```
Set-AzAutomationWebhook [-Name] <String> [-IsEnabled] <Boolean> [[-Parameters] <IDictionary>] [-RunOn <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzAutomationWebweb modifis** a webwebbook for an Azure Automation runbook.

## Exemplos

### Exemplo 1: Desabilitar um web browser
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -IsEnabled $False
```

Esse comando desabilita um webrecisão chamado Web firewall01 na conta automação chamada AutomationAccount01.

### Exemplo 2
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -RunOn 'Windows'
```

Esse comando define o valor de run on for the web hybrid e força o runbook a ser executado em um grupo de Trabalhadores Híbridos chamado Windows.

### Exemplo 3
```powershell
PS C:\>Set-AzAutomationWebhook -Name "Webhook01" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -RunOn $null
```

Esse comando atualiza o valor de run on do web browser e força o runbook a ser executado em um profissional do Azure runbook.
