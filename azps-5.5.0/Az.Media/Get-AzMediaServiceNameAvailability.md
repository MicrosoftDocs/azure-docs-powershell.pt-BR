# Get-AzMediaServiceNameAvailability

## Sinopse
Verifica se um nome de serviço de mídia está disponível.
Os nomes de serviços de mídia são globalmente exclusivos.

## Sintaxe

```
Get-AzMediaServiceNameAvailability [-DefaultProfile <IAzureContextContainer>] [-AccountName] <String>
 [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzMediaServiceNameAvailability** verifica se um nome de serviço de mídia está disponível.
Os nomes de serviços de mídia são globalmente exclusivos.

## Exemplos

### Exemplo 1: Verificar se um nome de Serviço de Mídia está disponível
```
PS C:\>Get-AzMediaServiceNameAvailability -AccountName "MediaService1"
```

Esse comando verifica se o nome MediaService1 está disponível.
