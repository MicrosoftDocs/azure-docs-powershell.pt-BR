# Get-AzBillingInvoice

## Sinopse
Obter faturas de cobrança da assinatura.
Obter faturas de cobrança de uma conta de cobrança e perfil de cobrança

## Sintaxe

### Lista (Padrão)
```
Get-AzBillingInvoice [-MaxCount <Int32>] [-GenerateDownloadUrl] [-DefaultProfile <IAzureContextContainer>] [-BillingAccountName] [-BillingProfileName]
 [<CommonParameters>]
```

### Últimas
```
Get-AzBillingInvoice [-Latest] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>] [-BillingAccountName] [-BillingProfileName]
```

### Único
```
Get-AzBillingInvoice -Name <System.Collections.Generic.List`1[System.String]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzBillingInvoice** obtém faturas de cobrança da assinatura. 

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzBillingInvoice -Latest
```

Obter a fatura mais recente da assinatura.

### Exemplo 2
```
PS C:\> Get-AzBillingInvoice -Name 2017-02-18-432543543
```

Obter a fatura da assinatura com o nome especificado.

### Exemplo 3
```
PS C:\> Get-AzBillingInvoice
```

Obter todas as faturas disponíveis da assinatura em ordem cronológica inversa, começando com a fatura mais recente sem baixar a URL. 

### Exemplo 4
```
PS C:\> Get-AzBillingInvoice -GenerateDownloadUrl -MaxCount 10
```

Receba as 10 faturas mais recentes da assinatura e inclua a URL de download no resultado.

### Exemplo 5
```
PS C:\> Get-AzBillingInvoice -Name 2017-02-18-432543543 -GenerateDownloadUrl
```

Obter a fatura específica por nome e incluir a URL do download no resultado.

### Exemplo 6
```
PS C:\> Get-AzBillingInvoice -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -GenerateDownloadUrl
```

Obter faturas pelo nome da conta de cobrança e incluir a URL de download de cada fatura no resultado.

### Exemplo 7
```
PS C:\> Get-AzBillingInvoice -Name 0000000000 -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -GenerateDownloadUrl
```

Receba uma fatura específica pelo nome da fatura e pelo nome da conta de cobrança e inclua a URL de download de cada fatura no resultado.

### Exemplo 8
```
PS C:\> Get-AzBillingInvoice -Latest -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -GenerateDownloadUrl
```

Receba a fatura mais recente pelo nome da conta de cobrança e inclua a URL de download da fatura no resultado.

### Exemplo 9
```
PS C:\> Get-AzBillingInvoice -GenerateDownloadUrl -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -BillingProfileName 0000-0000-000-000 -MaxCount 10
```

Receba as 10 faturas mais recentes da conta de cobrança específica e do perfil de cobrança específico e inclua a URL de download no resultado.

### Exemplo 10
```
PS C:\> Get-AzBillingInvoice -Latest -GenerateDownloadUrl -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -BillingProfileName 0000-0000-000-000 
```

Receba a fatura mais recente por meio do nome da conta de cobrança e do nome do perfil de cobrança e inclua a URL de download da fatura no resultado.

### Exemplo 10
```
PS C:\> Get-AzBillingInvoice -BillingAccountName 00000000-0000-0000-0000-000000000000:00000000-0000-0000-0000-000000000000_0000-00-00 -BillingProfileName 0000-0000-000-000 -PeriodStartDate 0000-00-00 -PeriodEndDate 0000-00-00
```

Obter faturas com o nome da conta de cobrança e o nome do perfil de cobrança para um período de cobrança especificado pela data de início e data do períodoEnd.
