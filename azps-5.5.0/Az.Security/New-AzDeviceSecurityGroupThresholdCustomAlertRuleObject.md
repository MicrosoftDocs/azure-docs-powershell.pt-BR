# New-AzDeviceSecurityGroupThresholdCustomAlertRuleObject

## Sinopse
Criar nova regra de alerta personalizado de limite para o grupo de segurança do dispositivo (Segurança IoT)

## Sintaxe

```
New-AzDeviceSecurityGroupThresholdCustomAlertRuleObject -MinThreshold <Int32> -MaxThreshold <Int32>
 -Enabled <Boolean> -Type <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O New-AzDeviceSecurityGroupThresholdCustomAlertRuleObject cmdlet cria uma nova lista de regras de alerta personalizado limite para o grupo de segurança do dispositivo na solução de segurança de IoT.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzDeviceSecurityGroupThresholdCustomAlertRuleObject -MinThreshold 0 -MaxThreshold 10 -Enabled $true -Type "SomeRuleType"

RuleType: "SomeRuleType"
DisplayName: "Display name for some rule type"
Description: "Description for some rule type"
IsEnabled: false
MinThreshold: 0
MaxThreshold: 10
```

Criar nova regra de alerta personalizado de limite do tipo "SomeRuleType" com valores de status habilitados para limite mínimo e máximo
