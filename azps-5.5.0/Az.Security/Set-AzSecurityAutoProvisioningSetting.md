# Set-AzSecurityAutoProvisioningSetting

## Sinopse
Atualiza a configuração de provisionamento automático

## Sintaxe

### SubscriptionLevelResource (Padrão)
```
Set-AzSecurityAutoProvisioningSetting -Name <String> [-EnableAutoProvision]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceid
```
Set-AzSecurityAutoProvisioningSetting [-EnableAutoProvision] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Set-AzSecurityAutoProvisioningSetting -InputObject <PSSecurityAutoProvisioningSetting>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
2010 2010 2010 2010 2010 2010 2010 2010
As Configurações de Provisionamento Automático permitem que você decida se deseja que a Central de Segurança do Azure provisione automaticamente um agente de segurança que será instalado em seus VMs.
O agente de segurança monitorará seu VM para criar alertas de segurança e monitorar a conformidade de segurança do VM.
