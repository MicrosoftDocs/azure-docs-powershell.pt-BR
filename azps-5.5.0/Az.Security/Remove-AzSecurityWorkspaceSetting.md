# Remove-AzSecurityWorkspaceSetting

## Sinopse
Exclui a configuração de espaço de trabalho de segurança desta assinatura.

## Sintaxe

### SubscriptionLevelResource (Padrão)
```
Remove-AzSecurityWorkspaceSetting -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceid
```
Remove-AzSecurityWorkspaceSetting -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Remove-AzSecurityWorkspaceSetting -InputObject <PSSecurityWorkspaceSetting> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Exclui a configuração de espaço de trabalho de segurança desta assinatura.
Esta ação fará com que os agentes de segurança recém-instalados reportem ao espaço de trabalho padrão desta assinatura.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Remove-AzSecurityWorkspaceSetting -Name "default"
```

Exclui a configuração de espaço de trabalho de segurança desta assinatura.
