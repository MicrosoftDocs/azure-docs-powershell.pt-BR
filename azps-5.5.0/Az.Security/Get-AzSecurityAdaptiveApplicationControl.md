# Get-AzSecurityAdaptiveApplicationControl

## Sinopse
Obtém uma lista de grupos de VM/servidor de controle de aplicativos para a assinatura.

## Sintaxe

### SubscriptionScope (Padrão)
```
Get-AzSecurityAdaptiveApplicationControl [-SubscriptionId <String>] [-IncludePathRecommendation <Boolean>] [-Summary <Boolean>] 
[-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Controles de Aplicativo Adaptáveis são calculados automaticamente pela Central de Segurança do Azure, use este cmdlet para obter uma lista de recursos de Controles de Aplicativo Adaptáveis no escopo de uma assinatura.
