# Get-AzCognitiveServicesAccount

## Sinopse
Obtém uma conta.

## Sintaxe

### ResourceGroupParameterSet
```
Get-AzCognitiveServicesAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCognitiveServicesAccount** obtém as contas provisionadas dos Serviços Cognitivas no grupo de recursos especificado pelo parâmetro *ResourceGroupName.*
Se você não especificar o parâmetro *ResourceGroupName,* esse cmdlet obtém todas as contas dos Serviços Cognitivas para a assinatura atual.
