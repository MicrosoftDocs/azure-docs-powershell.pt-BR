# Get-AzCognitiveServicesAccountSku

## Sinopse
Obtém as SKUs disponíveis para uma conta.

## Sintaxe

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCognitiveServicesAccountSku** obtém as SKUs disponíveis para uma conta do Cognitive Services.
A SKU é o plano de camada para uma conta.
Ela define o preço, o limite de chamada e a taxa da conta.
A SKU F0 é um nível gratuito.
As camadas pagas incluem S0, S1, S2 e assim por diante.

## Exemplos

### Exemplo 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```
