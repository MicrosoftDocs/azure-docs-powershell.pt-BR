# Get-AzCognitiveServicesAccountNetworkRuleSet

## Sinopse
Obter a propriedade NetworkRule de uma conta dos Serviços Cognitivas

## Sintaxe

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCognitiveServicesAccountNetworkRuleSet** obtém a propriedade NetworkRule de uma conta do Cognitive Services

## Exemplos

### Exemplo 1: Obter a propriedade NetworkRule de uma conta especificada dos Serviços Cognitivas
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Este comando obtém a propriedade NetworkRule de uma conta especificada dos Serviços Cognitivas
