# Get-AzCognitiveServicesAccountKey

## Sinopse
Obtém as chaves da API de uma conta.

## Sintaxe

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCognitiveServicesAccountKey** obtém as chaves da API de uma conta provisionada dos Serviços Cognitivas.
Uma conta do Cognitive Services tem duas chaves da API: Chave1 e Chave2.
As teclas permitem a interação com o ponto de extremidade da conta dos Serviços Cognitivas.
Use New-AzCognitiveServicesAccountKey para regenerar uma chave.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
