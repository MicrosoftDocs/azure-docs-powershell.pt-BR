# Get-AzBlueprintArtifact

## Sinopse
Recuperar artefatos de uma definição de planta.

## Sintaxe

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Recuperar artefatos de uma definição de planta. Se uma versão de definição de planta não for especificada, a versão de rascunho será recuperada. Caso não haja uma versão de rascunho, a definição de diagrama publicada mais recente será retornada.
