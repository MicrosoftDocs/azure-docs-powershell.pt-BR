# Módulo de Assinatura do Az.Subscription
## Descrição
Os tópicos neste documento de seção são cmdlets do Azure PowerShell para Assinatura do Azure na estrutura do Gerenciador de Recursos do Azure (ARM). Os cmdlets existem no namespace Microsoft.Azure.Commands.Subscription.

## Az.Subscription Cmdlets
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Obtém detalhes do alias da assinatura

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Cria novo alias e assinatura

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Exclui o alias da assinatura

### [Update-AzSubscription](Update-AzSubscription.md)
Atualiza uma assinatura do Azure
