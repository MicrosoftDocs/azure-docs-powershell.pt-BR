# Update-AzSapMonitor

## Sinopse
Patches no campo Marcas de um monitor SAP para a assinatura especificada, o grupo de recursos e o nome do monitor.

## Sintaxe

### UpdateExpanded (Padrão)
```
Update-AzSapMonitor -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>] [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzSapMonitor -InputObject <IHanaOnAzureIdentity> [-Tag <Hashtable>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrição
Patches no campo Marcas de um monitor SAP para a assinatura especificada, o grupo de recursos e o nome do monitor.

## Exemplos

### Exemplo 1: Atualizar um monitor SAP por nome
```powershell
PS C:\> Update-AzSapMonitor -ResourceGroupName nancyc-hn1 -Name ps-spamonitor-t01 -Tag @{'key'=1;'key2'=2; 'key3'=3}

Location Name              Type
-------- ----              ----
westus2  ps-spamonitor-t01 Microsoft.HanaOnAzure/sapMonitors
```

Esse comando atualiza um monitor SAP por nome.

### Exemplo 2: Atualizar um monitor SAP por objeto
```powershell
PS C:\> $sap = Get-AzSapMonitor -ResourceGroupName nancyc-hn1 -Name ps-sapmonitor-t01
PS C:\> Update-AzSapMonitor -InputObject $sap -Tag @{'key'=1;'key2'=2; 'key3'=3}

Location Name              Type
-------- ----              ----
westus2  ps-sapmonitor-t01 Microsoft.HanaOnAzure/sapMonitors
```

Esse comando atualiza um monitor SAP por objeto.
