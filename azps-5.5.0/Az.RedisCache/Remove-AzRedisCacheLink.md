# Remove-AzRedisCacheLink

## Sinopse
Remover um link de replicação geográfica entre dois Caches de Redis.

## Sintaxe

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Remover um link de replicação geográfica entre dois Caches de Redis.

## Exemplos

### Exemplo 1: Remover um link de replicação geográfica
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Esse comando remove links de replicação geográfica em que o Cache Redis chamado mycache1 é principal e o Cache Redis chamado mycache2 é secundário.
