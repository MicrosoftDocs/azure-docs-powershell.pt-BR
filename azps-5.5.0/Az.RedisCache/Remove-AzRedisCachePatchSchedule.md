# Remove-AzRedisCachePatchSchedule

## Sinopse
Remove o cronograma de patch.

## Sintaxe

```
Remove-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzRedisCachePatchSchedule** remove o cronograma de patch de um cache no Cache do Azure Redis.

## Exemplos

### Exemplo 1: Remover o cronograma de patch
```
PS C:\>Remove-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Esse comando remove o cronograma de patch do cache chamado RedisCache06.
