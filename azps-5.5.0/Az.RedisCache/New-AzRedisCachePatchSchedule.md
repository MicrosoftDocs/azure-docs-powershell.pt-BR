# New-AzRedisCachePatchSchedule

## Sinopse
Adiciona um cronograma de patch.

## Sintaxe

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzRedisCachePatchSchedule** adiciona um cronograma de patch a um cache no Cache do Azure Redis.

## Exemplos

### Exemplo 1: Criar e adicionar um cronograma de patch em um cache
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Esse comando adiciona um cronograma de patch ao cache chamado RedisCache06.
O parâmetro Entradas assume como seu valor um comando que usa **New-AzRedisCacheScheduleEntry** para criar um cronograma.
