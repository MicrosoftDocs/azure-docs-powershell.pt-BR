# New-AzRedisCacheScheduleEntry

## Sinopse
Cria uma entrada de cronograma.

## Sintaxe

```
New-AzRedisCacheScheduleEntry -DayOfWeek <String> -StartHourUtc <Int32> [-MaintenanceWindow <TimeSpan>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzRedisCacheScheduleEntry** cria um **objeto PSScheduleEntry.**
Cmdlets do cronograma de patch do Cache do Azure Redis, como o cmdlet New-AzRedisCachePatchSchedule, exigem objetos de entrada de cronograma.

## Exemplos

### Exemplo 1: Criar uma entrada de cronograma para fins de semana
```
PS C:\>New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00"
```

Esse comando cria um **objeto PSScheduleEntry** que representa um cronograma de fim de semana com a hora de início e a janela especificadas.
