# Get-AzSentinelAlertRuleAction

## Sinopse
Obter uma Resposta Automatizada (Ação de Regra de Alerta).

## Sintaxe

### AlertRuleId (Padrão)
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Actionid
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 -ActionId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSentinelAlertRuleAction** obtém uma Resposta Automatizada (Ação de Regra de Alerta) do espaço de trabalho especificado.
Se você especificar os parâmetros *ActionId* e *AlertRuleId,* um único objeto **AlertRuleAction** será retornado.
Se você não especificar o parâmetro *ActionId,* uma matriz contendo todas as Ações para a Regra de Alerta específica no espaço de trabalho especificado será retornada.
Você pode usar o **objeto Ação** para atualizar a Ação, por exemplo, você pode alterar a **Ação de** uma Regra de Alerta.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $AlertRuleActions = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId"
```

Este exemplo obtém  todas as Ações da Regra de Alerta especificada no espaço de trabalho especificado e a armazena na variável $AlertRuleActions dados.

### Exemplo 2
```powershell
PS C:\> $AlertRuleAction = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -ActionId "MyActionId"
```

Este exemplo obtém **uma AlertRuleAction** para a Regra de Alerta especificada no espaço de trabalho especificado e a armazena na variável $AlertRuleAction dados.
