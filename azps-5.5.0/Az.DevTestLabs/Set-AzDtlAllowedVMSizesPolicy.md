# Set-AzDtlAllowedVMSizesPolicy

## Sinopse
Define a política de tamanhos de máquina virtuais permitidos de um laboratório no DevTest Labs.

## Sintaxe

### Habilitar (Padrão)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Desativar
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzDtlAllowedVMIzesPolicy** define a política de tamanhos de máquina virtuais permitidos, que especifica uma lista de tamanhos de máquina virtuais permitidos em um laboratório.
O cmdlet usa o grupo de recursos especificado e o nome do laboratório para definir a política.
