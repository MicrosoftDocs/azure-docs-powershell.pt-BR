# Set-AzCdnEndpoint

## Sinopse
Atualiza um ponto de extremidade de CDN.

## Sintaxe

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzCdnEndpoint** atualiza um ponto de extremidade de REDE de Distribuição de Conteúdo (CDN) do Azure.
