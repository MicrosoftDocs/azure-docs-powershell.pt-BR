# Confirm-AzCdnEndpointProbeURL

## Sinopse
Valida uma URL de negociação.

## Sintaxe

```
Confirm-AzCdnEndpointProbeURL -ProbeUrl <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Confirm-AzCdnEndpointProbeURL** confirma se a URL fornecida pode ser usada para aceleração dinâmica do site.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Confirm-AzCdnEndpointProbeURL -ProbeUrl "http://www.bing.com/images"
IsValid: true
ErrorCode: None
Message:
```

Valida a url da url " http://www.bing.com/images "
