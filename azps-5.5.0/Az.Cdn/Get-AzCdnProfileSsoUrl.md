# Get-AzCdnProfileSsoUrl

## Sinopse
Obtém a URL de entrada única de um perfil de CDN.

## Sintaxe

### ByFieldsParameterSet (Padrão)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCdnProfileSsoUrl** obtém a URL de entrada única do perfil de REDE de Distribuição de Conteúdo (CDN) do Azure.
Essa URL permite que os usuários se conectem a um portal complementar e usem recursos adicionais de CDN.
