# Get-AzCdnEndpointNameAvailability

## Sinopse
Obtém o status de disponibilidade do ponto de extremidade cdn.

## Sintaxe

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCdnEndpointNameAvailability** obtém o status de disponibilidade do ponto de extremidade de Rede de Distribuição de Conteúdo (CDN) do Azure.
