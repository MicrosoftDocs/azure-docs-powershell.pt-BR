# Set-AzCdnProfile

## Sinopse
Atualiza um perfil de CDN.

## Sintaxe

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzCdnProfile** atualiza um perfil de REDE de Distribuição de Conteúdo (CDN) do Azure.
