# Get-AzDataShareSubscription

## Sinopse
Obtém informações sobre o compartilhamento de assinatura na conta de compartilhamento de dados.

## Sintaxe

### ByFieldsParameterSet (Padrão)
```
Get-AzDataShareSubscription -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareSubscription -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzDataShareSubscription** fornece informações sobre o compartilhamento de assinatura em uma conta de compartilhamento de dados. Se o nome da assinatura de compartilhamento for fornecido, o cmdlet fornece informações sobre a assinatura de compartilhamento específica. Caso contrário, o cmdlet fornece uma lista de assinaturas de compartilhamento na conta de compartilhamento de dados.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzDataShareSubscription -ResourceGroupName "ADS" -AccountName "WikiAds" -Name "AdsShareSubscription"

ShareSender             : adsprovider@microsoft.com
ShareSenderCompanyName  : ADS Test

```

Este comando fornece informações sobre o compartilhamento de assinatura AdsShareSubscription em wikiAds da conta de compartilhamento de dados.
