# New-AzSqlServerCommunicationLink

## Sinopse
Cria um link de comunicação para transações de banco de dados elásticas entre dois servidores de banco de dados SQL.

## Sintaxe

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzSqlServerCommunicationLink** cria um link de comunicação para transações de banco de dados elásticas entre dois servidores lógicos no Banco de Dados SQL do Azure.
Transações de banco de dados elásticas podem abranger bancos de dados em um dos servidores emparelhados.
Você pode criar mais de um link em um servidor.
Portanto, transações de banco de dados elásticas podem abranger um número maior de servidores.

## Exemplos

### Exemplo 1: Criar um link de comunicação
```
PS C:\>New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Esse comando cria um link chamado Link01 entre ContosoServer17 e ContosoServer02.
