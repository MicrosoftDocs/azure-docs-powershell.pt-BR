# Get-AzSqlElasticPoolActivity

## Sinopse
Obtém o status das operações em um pool elástica.

## Sintaxe

```
Get-AzSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlElasticPoolActivity** obtém o status das operações em um pool elástica para um banco de dados SQL do Azure.
Você pode ver o status das atualizações de configuração e criação de pool.

## Exemplos

### Exemplo 1: Obter o status das operações de um pool elástica
```
PS C:\>Get-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Esse comando obtém o status das operações do pool elástica chamado ElasticPool01.
