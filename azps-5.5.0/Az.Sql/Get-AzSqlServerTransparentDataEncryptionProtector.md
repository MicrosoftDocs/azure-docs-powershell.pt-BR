# Get-AzSqlServerTransparentDataEncryptionProtector

## Sinopse
Obtém a criptografia de dados transparentes (TDE)

## Sintaxe

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Get-AzSqlServerTransparentDataEncryptionProtector cmdlet obtém informações sobre o TDE.

## Exemplos

### Exemplo 1: Obter a criptografia de dados transparentes (TDE)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Esse comando obtém o TDE para o servidor chamado ContosoServer no grupo de recursos chamado ContosoResourceGroup.
ResourceGroupName ServerName Type ServerKeyVaultKeyName
----------------- ----------                   ---- ---------------------
ContosoResourceGroup ContosoServer ServiceManaged ServiceManaged
