# Get-AzSqlInstanceDatabaseGeoBackup

## Sinopse
Retorna informações sobre o backup redundante do banco de dados da Instância Gerenciada do Azure SQL.

## Sintaxe

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlInstanceDatabaseGeoBackup** obtém um ou mais backups redundantes de bancos de dados SQL do Azure de uma Instância Gerenciada de Banco de Dados SQL do Azure.
