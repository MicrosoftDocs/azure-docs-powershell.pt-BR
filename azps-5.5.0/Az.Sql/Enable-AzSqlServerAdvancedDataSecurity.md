# Enable-AzSqlServerAdvancedDataSecurity

## Sinopse
Habilita a Segurança Avançada de Dados em um servidor.

## Sintaxe

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Enable-AzSqlServerAdvancedDataSecurity** habilita a Segurança de Dados Avançada em um servidor. A Segurança Avançada de Dados é um pacote de segurança unificado que inclui Classificação de Dados, Avaliação de Vulnerabilidade e Proteção Avançada contra Ameaças para seu servidor. (Uma nova conta de armazenamento será criada automaticamente para salvar avaliações de vulnerabilidade. Se uma conta de armazenamento tiver sido criada anteriormente para essa finalidade, ela será usada em vez disso)

## Exemplos

### Exemplo 1: Habilitar a Segurança de Dados Avançada do servidor
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exemplo 2: Habilitar a Segurança de Dados Avançada do servidor a partir do recurso de servidor
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exemplo 3

Habilita a Segurança Avançada de Dados em um servidor. (gerado automaticamente)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlServerAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -ResourceGroupName MyResourceGroup -ServerName s1
```
