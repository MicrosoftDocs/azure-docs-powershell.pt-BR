# Set-AzSqlDatabaseGeoBackupPolicy

## Sinopse
Define uma política de backup geográfica de banco de dados.

## Sintaxe

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzSqlDatabaseGeoBackupPolicy** define a política de backup geo registrada em um banco de dados.
Este é um recurso de Backup do Azure usado para definir a política de armazenamento de backup.
