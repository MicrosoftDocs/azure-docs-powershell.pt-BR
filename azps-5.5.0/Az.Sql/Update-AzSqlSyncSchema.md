# Update-AzSqlSyncSchema

## Sinopse
Atualize o esquema de sincronização de um banco de dados de membro de sincronização ou um banco de dados do hub de sincronização.
Ele obterá o esquema mais recente do banco de dados real e o usará para atualizar o esquema armazenado em cache pelo banco de dados sincronizar metadados.
Se "SyncMemberName" for especificado, ele atualizará o esquema de banco de dados do membro; caso não, ele atualizará o esquema de banco de dados do hub.

## Sintaxe

```
Update-AzSqlSyncSchema [-SyncGroupName] <String> [-SyncMemberName <String>] [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Update-AzSqlSyncSchema** atualiza o esquema de sincronização de um banco de dados de membro de sincronização ou um banco de dados do hub de sincronização.

## Exemplos

### Exemplo 1: Atualizar o esquema de sincronização de um banco de dados do hub
```
PS C:\>Update-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01"
```

Este comando atualiza o esquema de sincronização do banco de dados do hub no grupo de sincronização SyncGroup01

### Exemplo 2: Atualizar o esquema de sincronização de um banco de dados de membro
```
PS C:\>Update-AzSqlSyncSchema -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -SyncGroupName "syncGroup01" -SyncMemberName "syncMember01"
```

Este comando atualiza o esquema de sincronização do banco de dados do membro no membro de sincronização do membro de sincronização01
