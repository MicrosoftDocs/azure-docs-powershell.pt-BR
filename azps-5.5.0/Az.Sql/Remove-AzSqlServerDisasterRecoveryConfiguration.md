# Remove-AzSqlServerDisasterRecoveryConfiguration

## Sinopse
Remove uma configuração de recuperação do sistema do servidor de banco de dados SQL.

## Sintaxe

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzSqlServerDisasterRecoveryConfiguration** remove uma configuração de recuperação do sistema do servidor de banco de dados SQL.
