# Suspend-AzSqlDatabase

## Sinopse
Suspende um banco de dados do SQL Data Warehouse.

## Sintaxe

```
Suspend-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Suspend-AzSqlDatabase** suspende um banco de dados do SQL Data Warehouse do Azure.

## Exemplos

### Exemplo 1: suspende um banco de dados do SQL Data Warehouse do Azure
```
PS C:\>Suspend-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando suspende um banco de dados ativo do SQL Data Warehouse do Azure.
