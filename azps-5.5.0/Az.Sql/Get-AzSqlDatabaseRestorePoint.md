# Get-AzSqlDatabaseRestorePoint

## Sinopse
Recupera os pontos de restauração distintos dos quais um SQL Data Warehouse pode ser restaurado.

## Sintaxe

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlDatabaseRestorePoint** recupera os pontos de restauração distintos de onde um Data Warehouse SQL do Azure pode ser restaurado.
Para um banco de dados SQL do Azure, a janela de restauração é contínua.
Isso significa que qualquer ponto no período de retenção de backup do banco de dados pode ser usado como ponto de restauração.
Esse cmdlet também é suportado pelo serviço Banco de Dados de Extensão do SQL Server no Azure.

## Exemplos

### Exemplo 1: Obter todos os pontos de restauração
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

Esse comando retorna todos os pontos de restauração disponíveis para o banco de dados SQL do Azure chamado Banco de Dados01.
