# Get-AzSqlServer

## Sinopse
Retorna informações sobre servidores de banco de dados SQL.

## Sintaxe

```
Get-AzSqlServer [[-ResourceGroupName] <String>] [[-ServerName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlServer** retorna informações sobre um ou mais servidores de banco de dados SQL do Azure.
Especifique o nome de um servidor para ver informações somente para esse servidor.

## Exemplos

### Exemplo 1: Obter todas as instâncias do SQL Server atribuídas a um grupo de recursos
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01"

ServerName               : server01
Location                 : Central US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server01.database.windows.net

ResourceGroupName        : resourcegroup01
ServerName               : server02
Location                 : West US
SqlAdministratorLogin    : adminLogin
SqlAdministratorPassword :
ServerVersion            : 12.0
Tags                     :
Identity                 :
FullyQualifiedDomainName : server02.database.windows.net
```

Esse comando obtém informações sobre todos os servidores de banco de dados SQL do Azure atribuídos ao grupo de recursos ResourceGroup01.

### Exemplo 2: Obter informações sobre um servidor de banco de dados SQL do Azure
```
PS C:\>Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"

```

Esse comando obtém informações sobre o servidor de banco de dados SQL do Azure chamado Server01.
