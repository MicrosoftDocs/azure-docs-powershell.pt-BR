# Set-AzSqlServerDnsAlias

## Sinopse
Modifica o servidor para o qual o Alias DNS do Azure SQL Server está apontando

## Sintaxe

```
Set-AzSqlServerDnsAlias -Name <String> -TargetServerName <String> [-ResourceGroupName] <String>
 -SourceServerName <String> -SourceServerResourceGroupName <String> -SourceServerSubscriptionId <Guid> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Este comando está atualizando o servidor para o qual o alias está apontando. Esse comando precisa ser emitido enquanto estiver conectado à assinatura onde o novo servidor para o qual o alias apontará está localizado.

## Exemplos

### Exemplo 1
```
PS C:\> Set-AzSqlServerDnsAlias -ResourceGroupName rg -DnsAliasName aliasName -TargetServerName newServer -SourceServerName oldServer -SourceServerResourceGroupName SourceServerRG -SourceServerSubscriptionId 0000-0000-0000-0000
```

Este comando está atualizando o alias que apontava anteriormente para o antigoServer para apontar para o servidor novoServer
