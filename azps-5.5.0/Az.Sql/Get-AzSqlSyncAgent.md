# Get-AzSqlSyncAgent

## Sinopse
Retorna informações sobre os Agentes de Sincronização SQL do Azure.

## Sintaxe

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlSyncAgent** retorna informações sobre um ou mais Agentes de Sincronização SQL do Azure.
Especifique o nome de um agente de sincronização para ver informações somente para esse agente de sincronização.
