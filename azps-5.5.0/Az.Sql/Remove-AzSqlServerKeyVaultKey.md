# Remove-AzSqlServerKeyVaultKey

## Sinopse
Remove uma chave do Cofre de Teclas de um servidor SQL.

## Sintaxe

```
Remove-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Remove-AzSqlServerKeyVaultKey cmdlet remove a tecla Key Vault do servidor SQL especificado.
Observe que as permissões do sql server para o cofre da chave não são alteradas.
Para alterar as permissões, use Set-AzKeyVaultAccessPolicy.
Observe que esse cmdlet não faz alterações no Cofre de Teclas.
Para remover uma chave do Cofre de Teclas, use Remove-AzKeyVaultKey.

## Exemplos

### Exemplo 1: Remover uma tecla do Cofre de Teclas
```
PS C:\> Remove-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Esse comando remove a tecla Key Vault com Id ' https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 do servidor especificado.
