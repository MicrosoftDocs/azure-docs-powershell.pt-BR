# Get-AzSqlInstanceAdvancedDataSecurityPolicy

## Sinopse
Obtém uma política de Segurança de Dados Avançada de uma instância gerenciada.

## Sintaxe

```
Get-AzSqlInstanceAdvancedDataSecurityPolicy [-InputObject <AzureSqlManagedInstanceModel>]
 -InstanceName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlInstanceAdvancedDataSecurityPolicy** recupera a política de Segurança de Dados Avançada de uma instância gerenciada.

## Exemplos

### Exemplo 1: Segurança de Dados Avançada da instância gerenciada
```powershell
PS C:\>  Get-AzSqlInstanceAdvancedDataSecurityPolicy `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exemplo 2: Segurança de Dados Avançada da instância gerenciada do recurso de instância gerenciada
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Get-AzSqlInstanceAdvancedDataSecurityPolicy

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
