# New-AzSqlServerDnsAlias

## Sinopse
Esse comando cria um novo Alias DNS do Azure SQL Server.

## Sintaxe

```
New-AzSqlServerDnsAlias -Name <String> [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Cria um novo Alias DNS do Azure SQL Server que aponta para o servidor especificado.

## Exemplos

### Exemplo 1
```
PS C:\> $serverDNSAlias = New-AzSqlServerDnsAlias -ResourceGroupName rg -ServerName serverName -DnsAliasName aliasName

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Esse comando cria Alias DNS do Azure SQL Server com o nome aliasName que aponta para serverName
