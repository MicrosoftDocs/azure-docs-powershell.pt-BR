# Enable-AzSqlInstanceAdvancedDataSecurity

## Sinopse
Habilita a Segurança Avançada de Dados em uma instância gerenciada.

## Sintaxe

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Enable-AzSqlInstanceAdvancedDataSecurity** habilita a Segurança de Dados Avançada em uma instância gerenciada. A Segurança Avançada de Dados é um pacote de segurança unificado que inclui Classificação de Dados, Avaliação de Vulnerabilidade e Proteção Avançada contra Ameaças para sua instância gerenciada. (Uma nova conta de armazenamento será criada automaticamente para salvar avaliações de vulnerabilidade. Se uma conta de armazenamento tiver sido criada anteriormente para essa finalidade, ela será usada em vez disso)

## Exemplos

### Exemplo 1: Habilitar a Segurança de Dados Avançada da instância gerenciada
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exemplo 2: Habilitar a Segurança de Dados Avançada da instância gerenciada a partir do recurso de servidor
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exemplo 3

Habilita a Segurança Avançada de Dados em uma instância gerenciada. (gerado automaticamente)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlInstanceAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -InstanceName 'ContosoManagedInstanceName' -ResourceGroupName MyResourceGroup
```
