# Get-AzSqlServerDnsAlias

## Sinopse
Obtém ou lista alias DNS do Azure SQL Server.

## Sintaxe

```
Get-AzSqlServerDnsAlias [-Name <String>] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Obter o Alias DNS específico do Azure SQL Server ou listar todos os Aliases DNS do Servidor para o servidor

## Exemplos

### Exemplo 1
```
PS C:\> $serverDNSAliases = Get-AzSqlServerDNSAlias -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
rgname             servername   dnsaliasname2
```

Lista todos os Aliases DNS do Servidor para o servidor específico

### Exemplo 2
```
PS C:\> $serverDNSAliases = Get-AzSqlServerDNSAlias -DnsAliasName dnsaliasname -ServerName servername -ResourceGroupName rgname

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
```

Obtém Alias DNS do Servidor especificado por nome de servidor e alias

### Exemplo 3
```
PS C:\> $serverDNSAliases = Get-AzSqlServerDNSAlias -ServerName servername -ResourceGroupName rgname -DnsAliasName "dnsaliasname*"

ResourceGroupName  ServerName   DnsAliasName
-----------------  ----------   ------------------
rgname             servername   dnsaliasname
rgname             servername   dnsaliasname2
```

Lista todos os Aliases DNS do Servidor para o servidor específico que começam com "dnsaliasname".
