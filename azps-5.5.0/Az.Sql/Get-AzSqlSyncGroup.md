# Get-AzSqlSyncGroup

## Sinopse
Retorna informações sobre os Grupos de Sincronização de Banco de Dados SQL do Azure.

## Sintaxe

```
Get-AzSqlSyncGroup [[-Name] <String>] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlSyncGroup** retorna informações sobre um ou mais Grupos de Sincronização de Banco de Dados SQL do Azure.
Especifique o nome de um grupo de sincronização para ver informações somente para esse grupo de sincronização.
