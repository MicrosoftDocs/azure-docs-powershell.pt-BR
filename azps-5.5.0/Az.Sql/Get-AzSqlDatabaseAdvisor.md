# Get-AzSqlDatabaseAdvisor

## Sinopse
Recebe um ou mais Consultores para um banco de dados SQL do Azure.

## Sintaxe

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlDatabaseAdvisor** obtém um ou mais Supervisores de Banco de Dados SQL do Azure para um banco de dados SQL do Azure.
