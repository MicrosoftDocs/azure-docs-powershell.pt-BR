# Get-AzSqlServerDisasterRecoveryConfiguration

## Sinopse
Obtém uma configuração de recuperação do sistema de servidor de banco de dados.

## Sintaxe

```
Get-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSqlServerDisasterRecoveryConfiguration** obtém uma configuração de recuperação do sistema do servidor de banco de dados SQL.
