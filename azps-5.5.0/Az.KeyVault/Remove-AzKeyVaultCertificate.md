# Remove-AzKeyVaultCertificate

## Sinopse
Remove um certificado de um cofre de chave.

## Sintaxe

### ByVaultNameAndName (Padrão)
```
Remove-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> [-InRemovedState] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Remove-AzKeyVaultCertificate [-InputObject] <PSKeyVaultCertificateIdentityItem> [-InRemovedState] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzKeyVaultCertificate** remove um certificado de um cofre de chave.

## Exemplos

### Exemplo 1: Remover um certificado
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName "ContosoKV01" -Name "SelfSigned01" -PassThru -Force

SecretId           : https://contosokv01.vault.azure.net:443/secrets/selfsigned01/968c3920884a435abf8faea11f565456
Thumbprint         : XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

Esse comando remove o certificado chamado SelfSigned01 do cofre de chaves chamado ContosoKV01.
Esse comando especifica o parâmetro *Forçar.*
Portanto, o cmdlet não solicita confirmação.

### Exemplo 2: limpar permanentemente o certificado excluído do cofre de chave
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName 'Contoso' -Name 'MyCert' -InRemovedState
```

Esse comando remove permanentemente o certificado chamado 'MeuCert' do cofre de chaves chamado 'Contoso'.
Executar este cmdlet requer a permissão de "limpeza", que deve ter sido concedida explicitamente ao usuário neste cofre de chave.
