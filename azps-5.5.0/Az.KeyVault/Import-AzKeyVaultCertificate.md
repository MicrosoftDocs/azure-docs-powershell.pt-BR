# Import-AzKeyVaultCertificate

## Sinopse
Importa um certificado para um cofre de chave.

## Sintaxe

### ImportCertificateFromFile (Padrão)
```
Import-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> -FilePath <String>
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ImportWithPrivateKeyFromString
```
Import-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> -CertificateString <String>
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ImportWithPrivateKeyFromCollection
```
Import-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String>
 [-CertificateCollection] <X509Certificate2Collection> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Import-AzKeyVaultCertificate** importa um certificado para um cofre de chave.
Você pode criar o certificado para importar usando um dos seguintes métodos:
- Use `Add-AzKeyVaultCertificate` para criar uma solicitação de assinatura de certificado e encaminhá-la a uma autoridade de certificação. Ver https://docs.microsoft.com/en-us/azure/key-vault/certificates/create-certificate-signing-request
- Use um arquivo de pacote de certificado existente, como um arquivo .pfx ou .p12, que contém o certificado e a chave privada.

## Exemplos

### Exemplo 1: Importar um certificado de cofre de chave
```powershell
PS C:\> $Password = ConvertTo-SecureString -String "123" -AsPlainText -Force
PS C:\> Import-AzKeyVaultCertificate -VaultName "ContosoKV01" -Name "ImportCert01" -FilePath "C:\Users\contosoUser\Desktop\import.pfx" -Password $Password

              [Not Before]

              [Not After]
```

O primeiro comando usa o cmdlet ConvertTo-SecureString para criar uma senha segura e a armazena na variável $Password dados.
O segundo comando importa o certificado chamado ImportCert01 para o cofre de teclas CosotosoKV01.
