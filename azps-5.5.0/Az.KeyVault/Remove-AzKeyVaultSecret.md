# Remove-AzKeyVaultSecret

## Sinopse
Exclui um segredo em um cofre de chaves.

## Sintaxe

### ByVaultName (Padrão)
```
Remove-AzKeyVaultSecret [-VaultName] <String> [-Name] <String> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKeyVaultSecret [-InputObject] <PSKeyVaultSecretIdentityItem> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Remove-AzKeyVaultSecret cmdlet exclui um segredo em um cofre de chaves.
Se o segredo foi excluído acidentalmente, o segredo pode ser recuperado usando Undo-AzKeyVaultSecretRemoval por um usuário com permissões especiais de "recuperar".
Este cmdlet tem um valor alto para a **propriedade ConfirmImpact.**

O comando especifica os  parâmetros *Forçar* e Confirmar e, portanto, o cmdlet não solicita confirmação.

### Exemplo 3: Limpar o segredo excluído do cofre de chaves permanentemente
```powershell
PS C:\> Remove-AzKeyVaultSecret -VaultName 'Contoso' -Name 'FinanceSecret' -InRemovedState
```

Este comando premoveses o segredo financeSecsecsec from the key vault named Contoso permanently.
Executar este cmdlet requer a permissão de "limpeza", que deve ter sido concedida explicitamente ao usuário para esse cofre de chave.
