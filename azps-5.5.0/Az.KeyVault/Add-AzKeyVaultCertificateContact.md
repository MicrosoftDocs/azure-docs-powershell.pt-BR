# Add-AzKeyVaultCertificateContact

## Sinopse
Adiciona um contato para notificações de certificado.

## Sintaxe

### Interativo (Padrão)
```
Add-AzKeyVaultCertificateContact [-VaultName] <String> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Add-AzKeyVaultCertificateContact [-InputObject] <PSKeyVault> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Add-AzKeyVaultCertificateContact [-ResourceId] <String> [-EmailAddress] <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Add-AzKeyVaultCertificateContact** adiciona um contato para um cofre de chave para notificações de certificado no Cofre de Teclas do Azure.
O contato recebe atualizações sobre eventos como certificado próximo ao vencimento, certificado renovado e assim por diante.
Esses eventos são determinados pela política de certificado.

## Exemplos

### Exemplo 1: Adicionar um contato de certificado do cofre de chave
```powershell
PS C:\> Add-AzKeyVaultCertificateContact -VaultName "ContosoKV01" -EmailAddress "patti.fuller@contoso.com" -PassThru

Email                    VaultName
-----                    ---------
patti.fuller@contoso.com ContosoKV01
```

Este comando adiciona o Batman Ltda como um contato de certificado para o cofre de teclas ContosoKV01 e retorna a lista de contatos do cofre "ContosoKV01".
