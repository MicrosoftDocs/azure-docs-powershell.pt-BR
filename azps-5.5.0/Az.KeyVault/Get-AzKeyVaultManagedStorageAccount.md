# Get-AzKeyVaultManagedStorageAccount

## Sinopse
Obtém contas de armazenamento gerenciadas pelo Azure Key Vault.

## Sintaxe

### ByAccountName (Padrão)
```
Get-AzKeyVaultManagedStorageAccount [-VaultName] <String> [[-AccountName] <String>] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObject
```
Get-AzKeyVaultManagedStorageAccount [-InputObject] <PSKeyVault> [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzKeyVaultManagedStorageAccount [-ResourceId] <String> [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Obtém uma Conta de Armazenamento gerenciada do Azure Do Cofre de Chave se o nome da conta for especificado e as chaves de conta são gerenciadas pelo cofre especificado. Se o nome da conta não for especificado, todas as contas cujas chaves são gerenciadas pelo cofre especificado serão listadas.

## Exemplos

### Exemplo 1: Listar todas as Contas de Armazenamento Gerenciadas do Cofre de Chave
```powershell
PS C:\> Get-AzKeyVaultManagedStorageAccount -VaultName 'myvault'

Vault Name          : myvault
AccountName         : mystorageaccount

                      orage/storageAccounts/mystorageaccount
Enabled             : True
Created             : 4/25/2018 1:50:32 AM
Updated             : 4/25/2018 1:50:32 AM
Tags                :
```

Lista todas as contas cujas chaves são gerenciadas por "myvault" do cofre

### Exemplo 2: Obter uma Conta de Armazenamento Gerenciada do Cofre de Chave
```powershell
PS C:\> Get-AzKeyVaultManagedStorageAccount -VaultName 'myvault' -Name 'mystorageaccount'

```

Obtém os detalhes da Conta de Armazenamento Gerenciada do Cofre-Chave de 'mystorageaccount' se suas chaves são gerenciadas pelo cofre 'myvault'
