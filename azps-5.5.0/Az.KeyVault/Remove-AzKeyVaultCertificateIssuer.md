# Remove-AzKeyVaultCertificateIssuer

## Sinopse
Exclui um emissor de certificado de um cofre de chave.

## Sintaxe

### Padrão (Padrão)
```
Remove-AzKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Remove-AzKeyVaultCertificateIssuer [-InputObject] <PSKeyVaultCertificateIssuerIdentityItem> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzKeyVaultCertificateIssuer** exclui um emissor de certificado de um cofre de chave.

## Exemplos

### Exemplo 1: Remover um emissor de certificado
```powershell
PS C:\> Remove-AzKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force

AccountId           :
ApiKey              :
OrganizationDetails :
Name                : TestIssuer01
IssuerProvider      : test
VaultName           : ContosoKV01
```

Esse comando remove o emissor do certificado chamado TestIssuer01 do cofre de teclas ContosoKV01.
