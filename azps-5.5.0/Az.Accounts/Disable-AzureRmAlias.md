# Disable-AzureRmAlias

## Sinopse
Desabilita aliases de prefixo do AzureRm para módulos do Az.

## Sintaxe

```
Disable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Desabilita aliases de prefixo do AzureRm para módulos do Az. Se -Module for especificado, somente os módulos listados terão aliases desabilitados. Caso contrário, todos os aliases do AzureRm serão desabilitados.

## Exemplos

### Exemplo 1
```
PS C:\> Disable-AzureRmAlias
```

Desabilita todos os prefixos do AzureRm para a sessão atual do PowerShell.

### Exemplo 1
```
PS C:\> Disable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Desabilita aliases do AzureRm para o módulo Az.Accounts para o processo atual e para o usuário atual.
