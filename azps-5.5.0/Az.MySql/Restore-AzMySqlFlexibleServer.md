# Restore-AzMySqlFlexibleServer

## Sinopse
Restaurar um servidor de um backup existente

## Sintaxe

```
Restore-AzMySqlFlexibleServer -Name <String> -ResourceGroupName <String> -InputObject <IServerAutoGenerated>
 -RestorePointInTime <DateTime> [-SubscriptionId <String>] [-Sku <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrição
Restaurar um servidor de um backup existente

## Exemplos

### Exemplo 1: Restaurar o servidor MySql usando a Restauração do PointInTime
```powershell
PS C:\> $restorePointInTime = (Get-Date).AddMinutes(-10)
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | Restore-AzMySqlFlexibleServer -Name mysql-test-restore -ResourceGroupName PowershellMySqlTest -RestorePointInTime $restorePointInTime 

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName   SkuTier       
----               -------- ------------------ ------- ----------------------- -------   -------       
mysql-test-restore eastus   mysql_test         5.7     10240                   Standard_D2ds_v4 GeneralPurpose
```

Esses cmdlets restauram o servidor MySql usando a Restauração do PointInTime.

PROPRIEDADES DE PARÂMETRO COMPLEXOS

Para criar os parâmetros descritos abaixo, construa uma tabela hash contendo as propriedades apropriadas. Para obter informações sobre tabelas hash, execute Get-Help about_Hash_Tables.


INPUTOBJECT: <IServerAutoGenerated> o objeto do servidor de origem a ser restaurado.
  - `Location <String>`: a localização geográfica onde o recurso reside
  - `[Tag <ITrackedResourceTags>]`: Marcas de recurso.
    - `[(Any) <String>]`: indica que qualquer propriedade pode ser adicionada a este objeto.
  - `[AdministratorLogin <String>]`: o nome de logon do administrador de um servidor. Só pode ser especificado quando o servidor está sendo criado (e é necessário para criação).
  - `[AdministratorLoginPassword <String>]`: a senha do logon do administrador (necessário para a criação do servidor).
  - `[AvailabilityZone <String>]`: informações de zona de disponibilidade do servidor.
  - `[CreateMode <CreateMode?>]`: o modo para criar um novo servidor MySQL.
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`: ID de recurso de arm da sub-rede delegada.
  - `[HaEnabled <HaEnabledEnum?>]`: Habilitar HA ou não para um servidor.
  - `[IdentityType <ResourceIdentityType?>]`: o tipo de identidade.
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`: Status mostrando se a criptografia de infraestrutura habilitada pelo servidor.
  - `[MaintenanceWindowCustomWindow <String>]`: indica se a janela personalizada está habilitada ou desabilitada
  - `[MaintenanceWindowDayOfWeek <Int32?>]`: dia da semana para janela de manutenção
  - `[MaintenanceWindowStartHour <Int32?>]`: hora de início da janela de manutenção
  - `[MaintenanceWindowStartMinute <Int32?>]`: minuto de início da janela de manutenção
  - `[PropertiesTag <IServerPropertiesTags>]`: Metadados específicos do aplicativo na forma de pares de valor-chave.
    - `[(Any) <String>]`: indica que qualquer propriedade pode ser adicionada a este objeto.
  - `[ReplicationRole <String>]`: a função de replicação.
  - `[RestorePointInTime <DateTime?>]`: Restaure o tempo de criação de ponto (formato ISO8601), especificando o tempo de restauração.
  - `[SkuName <String>]`: o nome da sKU, por exemplo, Standard_D32s_v3.
  - `[SkuTier <SkuTier?>]`: o nível da SKU específica, por exemplo, GeneralPurpose.
  - `[SourceServerId <String>]`: A ID do servidor MySQL de origem.
  - `[SslEnforcement <SslEnforcementEnum?>]`: Habilitar a imposição ssl ou não ao se conectar ao servidor.
  - `[StorageProfileBackupRetentionDay <Int32?>]`: Backup de dias de retenção para o servidor.
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`: Habilitar o crescimento automático do armazenamento.
  - `[StorageProfileStorageIop <Int32?>]`: IOPS de armazenamento para um servidor.
  - `[StorageProfileStorageMb <Int32?>]`: armazenamento máximo permitido para um servidor.
  - `[Version <ServerVersion?>]`: Versão do servidor.
