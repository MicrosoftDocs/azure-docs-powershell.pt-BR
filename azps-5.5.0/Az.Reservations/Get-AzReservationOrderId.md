# Get-AzReservationOrderId

## Sinopse
Obter lista de `ReservationOrder` IDs aplicáveis.

## Sintaxe

```
Get-AzReservationOrderId [-SubscriptionId <Guid>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Obter IDs dos `ReservationOrder` s aplicáveis que podem ser aplicadas a esta assinatura.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzReservationOrderId
```

Aplicar-se `ReservationOrder` à assinatura padrão

### Exemplo 2
```
PS C:\> Get-AzReservationOrderId -SubscriptionId "1111aaaa-b1b2-c0c2-d0d2-00000fffff"
```

Aplicar-se `ReservationOrder` à assinatura especificada
