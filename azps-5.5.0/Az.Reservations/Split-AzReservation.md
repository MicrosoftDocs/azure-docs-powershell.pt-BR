# Split-AzReservation

## Sinopse
Dividir um `Reservation` .

## Sintaxe

### CommandLine (Padrão)
```
Split-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid> -Quantity <Int32[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Split-AzReservation -Quantity <Int32[]> -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Divida um `Reservation` em dois s com distribuição de quantidade `Reservation` especificada.

## Exemplos

### Exemplo 1
```
PS C:\> Split-AzReservation -ReservationOrderId "00000000-ffff-ffff-0000-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111" -Quantities 2,3
```

Dividir o especificado `Reservation` em dois s com as quantidades `Reservation` correspondentes
