# Set-AzIntegrationAccountGeneratedIcn

## Sinopse
Atualiza o número de controle de troca gerado pela conta de integração (ICN) no grupo de recursos do Azure.

## Sintaxe

```
Set-AzIntegrationAccountGeneratedIcn -ResourceGroupName <String> -Name <String> -AgreementName <String>
 -ControlNumber <String> [-AgreementType <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet Set-AzIntegrationAccountGeneratedIcn atualiza uma conta de integração existente gerada por um número de controle de troca (ICN) e retorna um objeto que representa o número de controle de troca gerado pela conta de integração.
Use este cmdlet para atualizar um número de controle de troca gerado por uma conta de integração.
Você pode atualizar um número de controle de troca gerado por uma conta de integração especificando o nome da conta de integração, o nome do grupo de recursos e o nome do contrato.
Você não pode criar uma nova conta de integração gerada número de controle de troca com esse comando.
Para usar os parâmetros dinâmicos, basta digitá-los no comando ou digitar um sinal de hífen(-) para indicar um nome de parâmetro e pressionar a tecla TAB repetidamente para passar pelos parâmetros disponíveis.
Se você perder um parâmetro de modelo necessário, o cmdlet solicitará o valor.
Os valores de arquivo de parâmetro de modelo especificados na linha de comando têm precedência sobre os valores dos parâmetros de modelo em um objeto de parâmetro de modelo.
Forneça o parâmetro "-AgreementType" para especificar se os números de controle X12 ou Edifact devem ser retornados

## Exemplos

### Exemplo 1
```
PS C:\> $resourceGroup.ResourceGroupName = "ResourceGroup1"
PS C:\> $integrationAccountName = "IntegrationAccount1"
PS C:\> $integrationAccountAgreementName = "X12IntegrationAccountAgreement"
PS C:\> $initialControlNumber = Get-AzIntegrationAccountGeneratedIcn -AgreementType X12 -ResourceGroupName $resourceGroup.ResourceGroupName -Name $integrationAccountName -AgreementName $integrationAccountAgreementName
PS C:\> $incrementedControlNumberValue = [convert]::ToString([convert]::ToInt32($initialControlNumber.ControlNumber, 10) + 100, 10)
PS C:\> Set-AzIntegrationAccountGeneratedIcn -ResourceGroupName $resourceGroup.ResourceGroupName -Name $integrationAccountName -AgreementName $integrationAccountAgreementName -ControlNumber $incrementedControlNumberValue
ControlNumber            : 1100
ControlNumberChangedTime : 2/15/2017 12:36:00 AM
IsMessageProcessingFailed:
```

Esse comando obtém a conta de integração gerada X12 número de controle de troca para um contrato de conta de integração específico, aumenta seu valor em 100 e, em seguida, grava o valor atualizado.

### Exemplo 2
```
PS C:\> $resourceGroup.ResourceGroupName = "ResourceGroup1"
PS C:\> $integrationAccountName = "IntegrationAccount1"
PS C:\> $integrationAccountAgreementName = "EdifactIntegrationAccountAgreement"
PS C:\> $initialControlNumber = Get-AzIntegrationAccountGeneratedIcn -AgreementType EdifactIntegrationAccountAgreement -ResourceGroupName $resourceGroup.ResourceGroupName -Name $integrationAccountName -AgreementName $integrationAccountAgreementName
PS C:\> $incrementedControlNumberValue = [convert]::ToString([convert]::ToInt32($initialControlNumber.ControlNumber, 10) + 100, 10)
PS C:\> Set-AzIntegrationAccountGeneratedIcn -ResourceGroupName $resourceGroup.ResourceGroupName -Name $integrationAccountName -AgreementName $integrationAccountAgreementName -ControlNumber $incrementedControlNumberValue
ControlNumber            : 1100
ControlNumberChangedTime : 2/15/2017 12:36:00 AM
IsMessageProcessingFailed:
```

Esse comando obtém a conta de integração gerada EdifactIntegrationAccountAgreement de um contrato de conta de integração específico, aumenta seu valor em 100 e, em seguida, grava o valor atualizado.
