# Remove-AzIntegrationAccountSchema

## Sinopse
Remove um esquema de conta de integração.

## Sintaxe

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzIntegrationAccountSchema** remove um esquema de conta de integração de um grupo de recursos.
Especificando o nome da conta de integração, o nome do grupo de recursos e o nome do esquema.
Este módulo dá suporte a parâmetros dinâmicos.
Para usar um parâmetro dinâmico, digite-o no comando.
Para descobrir os nomes dos parâmetros dinâmicos, digite um hífen (-) após o nome do cmdlet e pressione a tecla Tab repetidamente para passar pelos parâmetros disponíveis.
Se você omitir um parâmetro de modelo necessário, o cmdlet solicitará o valor.

## Exemplos

### Exemplo 1: Remover um esquema de conta de integração
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

Esse comando remove um esquema de conta de integração chamado IntegrationAccountSchema43.
