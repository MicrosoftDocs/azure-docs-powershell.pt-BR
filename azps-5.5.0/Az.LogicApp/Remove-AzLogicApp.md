# Remove-AzLogicApp

## Sinopse
Remove um aplicativo lógico de um grupo de recursos.

## Sintaxe

```
Remove-AzLogicApp -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-Az LogicApp** remove um aplicativo lógico de um grupo de recursos usando o recurso Aplicativos Lógicos.
Especifique o aplicativo lógico e o grupo de recursos.
Este módulo dá suporte a parâmetros dinâmicos.
Para usar um parâmetro dinâmico, digite-o no comando.
Para descobrir os nomes dos parâmetros dinâmicos, digite um hífen (-) após o nome do cmdlet e pressione a tecla Tab repetidamente para passar pelos parâmetros disponíveis.
Se você omitir um parâmetro de modelo necessário, o cmdlet solicitará o valor.

## Exemplos

### Exemplo 1: Remover um aplicativo lógico
```
PS C:\>Remove-AzLogicApp -ResourceGroupName "ResourceGroup11" -Name "LogicApp03" -Force
```

Esse comando remove um aplicativo lógico de um grupo de recursos.
O comando inclui o parâmetro *Forçar,* o que impede que o comando solicitar confirmação.
