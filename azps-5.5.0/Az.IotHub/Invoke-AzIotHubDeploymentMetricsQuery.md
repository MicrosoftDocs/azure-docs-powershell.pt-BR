# Invoke-AzIotHubDeploymentMetricsQuery

## Sinopse
Invocar uma consulta métrica de implantação de borda IoT.

## Sintaxe

### ResourceSet (Padrão)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
Avalie uma métrica de sistema ou personalizada de destino definida em uma implantação de borda IoT.
Há métricas de sistema pré-definidas que são calculadas pelo Hub Iot e não podem ser personalizadas.
- "Direcionado" mostra os dispositivos de borda de IoT que corresponderem à condição de destino da implantação.
- "Aplicado" mostra os dispositivos de borda IoT direcionados que não são direcionados por outra implantação de prioridade mais alta.
- "Relatório de sucesso" mostra os dispositivos de borda de IoT que relataram que os módulos foram implantados com êxito.
- "Falha de relatório" mostra os dispositivos de borda IoT que relataram que um ou mais módulos não foram implantados com êxito. 
  Para investigar ainda mais o erro, conecte-se remotamente a esses dispositivos e veja os arquivos de log.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Avalie a métrica personalizada definida como "avisoLimit".

### Exemplo 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Avalie a métrica do sistema "Relatório de Sucesso".
