# Get-AzIotHubKey

## Sinopse
Obtém uma tecla IotHub.

## Sintaxe

### ResourceSet (Padrão)
```
Get-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [[-KeyName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubKey [-HubId] <String> [[-KeyName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Obtém uma tecla IotHub.
Você pode listar todas as Teclas ou filtrar a lista por um Nome de Chave específico.

## Exemplos

### Exemplo 1 Obter todas as teclas
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Obtém todas as Teclas do IotHub chamada "myiothub"

### Exemplo 2 Obter informações para uma chave específica
```
PS C:\> Get-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner"
```

Obtém as informações de uma chave chamada "iothu bowlner" para o IotHub chamado "myiothub"
