# Get-AzIotHubJob

## Sinopse
Obtém as informações sobre um trabalho do IotHub.

## Sintaxe

```
Get-AzIotHubJob [-ResourceGroupName] <String> [-Name] <String> [[-JobId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Obtém as informações sobre um Trabalho do IotHub.
Um Trabalho do IotHub é criado quando uma operação de importação ou exportação é inicializada usando os comandos New-AzIotHubExportDevices ou New-AzIotHubImportDevices dados.
Você pode listar todos os trabalhos ou filtrar os trabalhos pelo Identificador de Trabalho.

## Exemplos

### Exemplo 1 Listar todos os Trabalhos
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Obtém todos os trabalhos do IotHub chamado "myiothub"

### Exemplo 2 Obter um trabalho específico
```
PS C:\> Get-AzIotHubJob -ResourceGroupName "myresourcegroup" -Name "myiothub" -JobId 3630fc31-4caa-43e8-a232-ea0577221cb2
```

Obtém informações sobre o trabalho com o identificador "3630fc31-4caa-43e8-a232-ea0577221cb2" para o IotHub chamado "myiothub"
