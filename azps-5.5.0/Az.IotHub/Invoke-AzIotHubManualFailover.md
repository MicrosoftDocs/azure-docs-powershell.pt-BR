# Invoke-AzIotHubManualFailover

## Sinopse
Invoke failover process for the IoT Hub to the geo-paired disaster recovery region.

## Sintaxe

### ResourceSet (Padrão)
```
Invoke-AzIotHubManualFailover [-ResourceGroupName] <String> [-Name] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubManualFailover [-InputObject] <PSIotHub> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubManualFailover [-ResourceId] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Ele disparará o failover do hub de IoT para o local secundário. Essa ação causará perda de tempo e telemetria na sua solução. Esta é uma operação de longa execução e pode levar alguns minutos para ser terminada. Tenha cuidado ao usá-lo.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Iniciando o processo de failover do Hub de IoT do "myiothub".

### Exemplo 2
```powershell
PS C:\> Invoke-AzIotHubManualFailover -ResourceGroupName "myresourcegroup" -Name "myiothub" -AsJob
```

Iniciando o processo de failover do Hub IoT do "myiothub" em segundo plano.
