# Get-AzIotHub

## Sinopse
Obtém informações sobre os IotHubs em uma assinatura.

## Sintaxe

### ListIotHubsByResourceGroup (Padrão)
```
Get-AzIotHub [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetIotHubByName
```
Get-AzIotHub [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Obtém informações sobre os IotHubs em uma assinatura.
Você pode exibir todas as instâncias do IotHub em uma assinatura ou filtrar seus resultados por um grupo de recursos ou um nome específico do IotHub.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzIotHub
```

Obtém todos os IotHubs na assinatura.

### Exemplo 2
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup"
```

Obtém todos os IotHubs da assinatura que pertencem ao grupo de recursos chamado "myresourcegroup".

### Exemplo 3
```
PS C:\> Get-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Obtém informações sobre o IotHub chamado "myiothub".
