# Get-AzIotHubDeployment

## Sinopse
Lista toda ou uma implantação específica de Borda de IoT.

## Sintaxe

### ResourceSet (Padrão)
```
Get-AzIotHubDeployment [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubDeployment [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubDeployment [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Obter os detalhes de uma implantação de Borda IoT ou implantações de Borda de Lista IoT em um Hub de IoT.
Confira https://docs.microsoft.com/azure/iot-edge/module-deployment-monitoring mais informações.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "deploy1"
```

Obter os detalhes de uma implantação de Borda de IoT.

### Exemplo 2
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Listar todas as implantações de borda de IoT em um Hub de IoT.
