# Use-AzHDInsightCluster

## Sinopse
Seleciona um cluster a ser usado com Invoke-RmAzureHDInsightHiveJob cmdlet.

## Sintaxe

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Use-AzHDInsightCluster** seleciona o cluster HDInsight do Azure para o cmdlet Invoke-AzHDInsightHiveJob a ser usado para enviar trabalhos de Colmeia.

## Exemplos

### Exemplo 1: Selecionar um cluster para envio da consulta Desmamartice
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Esse comando seleciona um cluster para um envio de consulta Descarros.
