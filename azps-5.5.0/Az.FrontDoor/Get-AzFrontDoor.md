# Get-AzFrontDoor

## Sinopse
Balanceador de carga da porta da frente

## Sintaxe

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzFrontDoorGet** obtém todas as Portas Front existentes na assinatura atual que corresponde às informações fornecidas.
