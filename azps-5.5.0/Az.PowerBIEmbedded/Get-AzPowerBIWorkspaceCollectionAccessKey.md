# Get-AzPowerBIWorkspaceCollectionAccessKey

## Sinopse
Obtém as chaves de acesso atuais associadas a um conjunto de espaços de trabalho do Power BI.

## Sintaxe

```
Get-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzPowerBIWorkspaceCollectionAccessKey** obtém as chaves de acesso atuais associadas a um conjunto de espaços de trabalho do Power BI.

## Exemplos

### Exemplo 1: Obter teclas de acesso
```
PS C:\>Get-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Esse comando obtém chaves de acesso para o conjunto de espaços de trabalho chamado WCN11 no grupo de recursos especificado.
