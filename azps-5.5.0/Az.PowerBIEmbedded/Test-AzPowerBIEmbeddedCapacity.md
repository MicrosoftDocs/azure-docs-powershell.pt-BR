# Test-AzPowerBIEmbeddedCapacity

## Sinopse
Testa a existência de uma instância de Capacidade Inserida do PowerBI.

## Sintaxe

```
Test-AzPowerBIEmbeddedCapacity [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Test-AzPowerBIEmbeddedCapacity cmdlet testa a existência de uma instância de Capacidade Inserida do PowerBI

## Exemplos

### Exemplo 1
```
PS C:\> Test-AzPowerBIEmbeddedCapacity -Name "testcapacity"
True
```

Esse comando testará se há uma capacidade chamada testcapacity
