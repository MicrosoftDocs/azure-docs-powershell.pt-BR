# Get-AzPowerBIWorkspace

## Sinopse
Obtém os espaços de trabalho em um conjunto de espaços de trabalho do Power BI.

## Sintaxe

```
Get-AzPowerBIWorkspace [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzPowerBIWorkspace** obtém os espaços de trabalho em um conjunto de espaços de trabalho do Power BI.

## Exemplos

### Exemplo 1: Obter espaços de trabalho de um conjunto de espaços de trabalho
```
PS C:\>Get-AzPowerBIWorkspace -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11"
```

Esse comando obtém os espaços de trabalho que pertencem à coleção de espaços de trabalho chamada WCN11 no grupo de recursos especificado.
