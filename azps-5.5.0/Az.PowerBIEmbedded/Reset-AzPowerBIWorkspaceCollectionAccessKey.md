# Reset-AzPowerBIWorkspaceCollectionAccessKey

## Sinopse
Redefine a tecla de acesso especificada.

## Sintaxe

```
Reset-AzPowerBIWorkspaceCollectionAccessKey [-ResourceGroupName] <String> [-WorkspaceCollectionName] <String>
 [-Key1] [-Key2] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Reset-AzPowerBIWorkspaceCollectionAccessKey** redefine a chave de acesso especificada no seu conjunto de espaços de trabalho do Power BI.

## Exemplos

### Exemplo 1: Redefinir a chave de acesso principal
```
PS C:\>Reset-AzPowerBIWorkspaceCollectionAccessKey -ResourceGroupName "ResourceGroup17" -WorkspaceCollectionName "WCN11" -Key1
```

Esse comando redefine a chave de acesso principal do conjunto de espaços de trabalho chamado WCN11 no grupo de recursos especificado.
