# Módulo Az.DevSpaces
## Descrição
Comandos para interagir com o Controlador do Azure DevSpaces.

## Az.DevSpaces Cmdlets
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Obter ou listar o controlador do Azure DevSpaces.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Crie um novo controlador do Azure DevSpaces.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Exclua um controlador DevSpaces.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Atualize o controlador DevSpaces para adicionar marcas.
