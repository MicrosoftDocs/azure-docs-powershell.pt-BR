# Módulo Az.DataFactory
## Descrição
O Azure Data Factory V2 é a plataforma de integração de dados que vai além da organização e processamento em lote de dados de log de séries temporais do Azure Data Factory V1, com um modelo de aplicativo de finalidade geral que oferece suporte a padrões e cenários modernos de data warousousing, SSIS de elevação e deslocamento e aplicativos SaaS orientados por dados. Rediga e gerencie fluxos de trabalho de integração de dados confiáveis e seguros em escala. Use conectores de dados nativos do ADF e o Integration Runtimes para mover e transformar dados locais e na nuvem que podem ser não estruturados, semiestruturados e estruturados com Hadoop, Azure Data Lake, Spark, SQL Server, Db Db e muitas outras plataformas de dados.
