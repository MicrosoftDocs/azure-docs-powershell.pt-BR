# Remove-AzDataFactoryHub

## Sinopse
Remove um hub do Azure Data Factory.

## Sintaxe

### ByFactoryName (Default)
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDataFactoryHub** remove um hub do Azure Data Factory no grupo de recursos do Azure especificado e na fábrica de dados especificada.
Se você remover um hub, todos os serviços e pipelines vinculados no hub também serão removidos.

## Exemplos

### Exemplo 1: Remover um hub
```
PS C:\>Remove-AzDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub"
```

Esse comando remove o hub chamado ContosoDataHub do grupo de recursos do Azure chamado ADFResourceGroup e da fábrica de dados chamada ADFDataFactory.
