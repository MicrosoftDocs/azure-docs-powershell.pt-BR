# Resume-AzDataFactoryPipeline

## Sinopse
Retoma um pipeline suspenso no Data Factory.

## Sintaxe

### ByFactoryName (Default)
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Resume-AzDataFactory Pipelineline** retoma um pipeline suspenso no Azure Data Factory.

## Exemplos

### Exemplo 1: Retomar um pipeline
```powershell
PS C:\>Resume-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to resume pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Esse comando retoma o pipeline chamado DPWikisample na fábrica de dados chamada WikiADF.
Use o cmdlet **Suspend-AzDataFactory Pipelineline** para suspender um pipeline.
O comando retorna um valor de $True.

### Exemplo 2

Retoma um pipeline suspenso no Data Factory. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Resume-AzDataFactoryPipeline -DataFactory $DataFactory -Name 'DPWikisample'
```
