# Remove-AzDataFactoryDataset

## Sinopse
Remove um conjuntos de dados do Azure Data Factory.

## Sintaxe

### ByFactoryName (Default)
```
Remove-AzDataFactoryDataset [-Force] [-DataFactoryName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryDataset [-Force] [-DataFactory] <PSDataFactory> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDataDataoryDataset** remove um conjuntos de dados do Azure Data Factory.

## Exemplos

### Exemplo 1: Remover um conjuntos de dados
```
PS C:\>Remove-AzDataFactoryDataset -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "DAWikiAggregatedData"
Confirm
Are you sure you want to remove dataset 'DAWikiAggregatedData' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
True
```

Esse comando remove o conjuntos de dados chamado DAWikiAggregatedData da fábrica de dados chamada WikiADF.
O comando retorna um valor de $True.
