# Suspend-AzAnalysisServicesServer

## Sinopse
Suspende uma instância do servidor do Analysis Services

## Sintaxe

```
Suspend-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Suspend-AzAnalysisServicesServer cmdlet suspende uma instância do servidor do Analysis Services

## Exemplos

### Exemplo 1
```
PS C:\> Suspend-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Esse comando suspenderá um servidor ativo chamado testserver no grupo de teste de grupo de recursos
