# New-AzRecoveryServicesVault

## Sinopse
Cria um novo cofre dos Serviços de Recuperação.

## Sintaxe

```
New-AzRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzRecoveryServicesVault** cria um novo cofre dos Serviços de Recuperação.

## Exemplos

### Exemplo 1
```
PS C:\> New-AzRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Crie o cofre de serviço de recuperação no grupo de recursos e local determinado.
