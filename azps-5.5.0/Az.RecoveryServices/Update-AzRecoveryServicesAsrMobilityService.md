# Update-AzRecoveryServicesAsrMobilityService

## Sinopse
Atualizações do agente de serviço de mobilidade push para máquinas protegidas.

## Sintaxe

```
Update-AzRecoveryServicesAsrMobilityService [-Account <ASRRunAsAccount>]
 -ReplicationProtectedItem <ASRReplicationProtectedItem> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Update-AzRecoveryServicesAsrMobilityService** tenta pressionar atualizações do agente de serviços de mobilidade para máquinas protegidas (se uma atualização estiver disponível.)

## Exemplos

### Exemplo 1
```
PS C:\> Update-AzRecoveryServicesAsrMobilityService -ReplicationProtectedItem $rpi -Account $fabric.fabricSpecificDetails.RunAsAccounts[0]
```

Trabalho para controlar o Agente de Serviço de Mobilidade do Item Protegido de Replicação de Atualização.
