# Update-AzRecoveryServicesVault

## Sinopse
Atualiza o MSI para o cofre de serviços de recuperação.

## Sintaxe

```
Update-AzRecoveryServicesVault -ResourceGroupName <string> -Name <string> -IdentityType <MSIdentity>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Esse cmdlet é usado para adicionar ou remover o MSI do cofre de serviços de recuperação. Use o param -IdentityType para atribuir uma identidade SystemAssigned ao RSVault. Use IdentityType None para remover o MSI do cofre.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Update-AzRecoveryServicesVault -ResourceGroupName "rgName" -Name "vaultName" -IdentityType SystemAssigned
```

Esse cmdlet é usado para adicionar uma identidade SystemAssigned a um cofre de serviços de recuperação.
