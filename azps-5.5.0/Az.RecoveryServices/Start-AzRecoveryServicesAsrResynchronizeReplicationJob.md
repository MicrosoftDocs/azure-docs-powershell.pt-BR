# Start-AzRecoveryServicesAsrResynchronizeReplicationJob

## Sinopse
Inicia a replicação de sincronização.

## Sintaxe

### Padrão (Padrão)
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem <ASRReplicationProtectedItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Start-AzRecoveryServicesAsrResynchronizeReplicationJob** iniciará a ressincronização da replicação para o item protegido especificado se o protegido estiver em um estado necessário de ressincronização.

## Exemplos

### Exemplo 1
```
PS C:\> Start-AzRecoveryServicesAsrResynchronizeReplicationJob -ReplicationProtectedItem $rpi
```

Inicia o trabalho para ressincronizar a replicação em um item protegido por replicação passada.
