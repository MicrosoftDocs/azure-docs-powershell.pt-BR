# Update-AzRecoveryServicesAsrProtectionDirection

## Sinopse
Atualiza a direção de replicação para o item protegido por replicação especificado ou plano de recuperação. Usado para res proteger/reverter uma falha em relação ao item replicado ou plano de recuperação.

 -Direction <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]

## Descrição
O cmdlet **Update-AzRecoveryServicesAsrProtectionDirection** atualiza a direção de replicação do objeto de Recuperação de Site do Azure especificado após a conclusão de uma operação de failover de confirmação.

## Exemplos

### Exemplo 1
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -RecoveryPlan $RP -Direction PrimaryToRecovery
```

Inicie a operação de direção de atualização para o plano de recuperação especificado e retorna o objeto de trabalho ASR usado para controlar a operação.

### Exemplo 2
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping -LogStorageAccountId $cacheStorageId `
 -ReplicationProtectedItem $rpi
```

Inicie a operação de direção de atualização para o item protegido por replicação especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e pelo uso do armazenamento em cache (na mesma região que o VM).

### Exemplo 3
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping `
 -AzureToAzureDiskReplicationConfiguration $disk1,$disk2 -ReplicationProtectedItem  $rpi
```

Inicie a operação de direção de atualização para o item protegido por replicação especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e configuração de replicação de disco fornecida.

### Exemplo 4
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping `
 -AzureToAzureDiskReplicationConfiguration $disk1,$disk2 -ReplicationProtectedItem  $rpi `
 -DiskEncryptionVaultId  $DiskEncryptionVaultId -DiskEncryptionSecertUrl $DiskEncryptionSecertUrl `
 -KeyEncryptionVaultId $KeyEncryptionVaultId  -KeyEncryptionKeyUrl $KeyEncryptionKeyUrl
```

    
Inicie a operação de direção de atualização para o item protegido por replicação criptografada especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e configuração de replicação de disco fornecida.

### Exemplo 5
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrProtectionDirection -AzureToAzure -ProtectionContainerMapping $B2ApcmMapping -LogStorageAccountId $cacheStorageId `
 -ReplicationProtectedItem $rpi -RecoveryProximityPlacementGroupId $ppg
```

Inicie a operação de direção de atualização para o item protegido por replicação especificado na região do azure de destino definida pelo mapeamento de contêiner de proteção e pelo uso do armazenamento em cache (na mesma região que VM) e do grupo de posicionamento de proximidade.
