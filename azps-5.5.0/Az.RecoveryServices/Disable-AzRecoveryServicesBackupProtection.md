# Disable-AzRecoveryServicesBackupProtection

## Sinopse
Desabilita a proteção para um item protegido por backup.

## Sintaxe

```
Disable-AzRecoveryServicesBackupProtection [-Item] <ItemBase> [-RemoveRecoveryPoints] [-Force]
 [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Disable-AzRecoveryServicesBackupProtection** desabilita a proteção para um item protegido por backup do Azure.
Esse cmdlet interrompe o backup agendado regular de um item.
Esse cmdlet também pode excluir pontos de recuperação existentes para o item de backup, se executado com o parâmetro RemoveRecoveryPoints.
De definir o contexto do cofre usando o Set-AzRecoveryServicesVaultContext cmdlet antes de usar o cmdlet atual.

## Exemplos

### Exemplo 1: Desabilitar a proteção contra backup
```powershell
PS C:\> $Cont = Get-AzRecoveryServicesBackupContainer -ContainerType AzureVM -Status Registered 
PS C:\> $PI = Get-AzRecoveryServicesBackupItem -Container $Cont[0] -WorkloadType AzureVM 
PS C:\> Disable-AzRecoveryServicesBackupProtection -Item $PI[0]
```

O primeiro comando obtém uma matriz de contêineres de backup e, em seguida, o armazena na matriz $Cont dados.
O segundo comando obtém o item de Backup correspondente ao primeiro item de contêiner e o armazena na variável $PI dados.
O último comando desabilita a proteção de backup para o item $PI \[ \] 0, mas mantém os dados.

### Exemplo 2

Desabilita a proteção para um item protegido por backup. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Disable-AzRecoveryServicesBackupProtection -Item $PI[0] -RemoveRecoveryPoints -VaultId $vault.ID
```
