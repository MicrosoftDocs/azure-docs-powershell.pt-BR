# Unregister-AzRecoveryServicesBackupManagementServer

## Sinopse
Desosterra um servidor SCDPM ou um servidor de backup do cofre.

## Sintaxe

```
Unregister-AzRecoveryServicesBackupManagementServer [-AzureRmBackupManagementServer] <BackupEngineBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O **cmdlet Unregister-AzRecoveryServicesBackupManagementServer** não registrou um servidor SCDPM (System Center Data Protection Manager) ou um servidor de backup do Azure a partir do cofre.
Este cmdlet remove referências aos servidores que não foram contados do cofre.
Antes de não poder fazer o registro de um contêiner, você deve excluir todos os dados protegidos associados a esse contêiner.
De definir o contexto do cofre usando o Set-AzRecoveryServicesVaultContext cmdlet antes de usar o cmdlet atual.

## Exemplos

### Exemplo 1: Não registro de um servidor SCDPM do cofre
```powershell
PS C:\>$BMS = Get-AzRecoveryServicesBackupManagementServer -Name "dpmserver01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupManagementServer -AzBackupManagementServer $BMS
```

O primeiro comando obtém o servidor de gerenciamento de backup dpmserver01.contoso.com e o armazena na variável $BMS backup.
O segundo comando não conta com o servidor SCDPM do cofre.

### Exemplo 2

Desosterra um servidor SCDPM ou um servidor de backup do cofre. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Unregister-AzRecoveryServicesBackupManagementServer -AzureRmBackupManagementServer <BackupEngineBase> -VaultId $vault.ID
```
