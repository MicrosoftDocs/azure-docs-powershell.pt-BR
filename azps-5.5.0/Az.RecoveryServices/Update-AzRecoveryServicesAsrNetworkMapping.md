# Update-AzRecoveryServicesAsrNetworkMapping

## Sinopse
Atualiza o mapeamento de rede de recuperação de site do Azure especificado.

## Sintaxe

### ByNetworkObject (Padrão)
```
Update-AzRecoveryServicesAsrNetworkMapping -InputObject <ASRNetworkMapping> -RecoveryNetwork <ASRNetwork>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ById
```
Update-AzRecoveryServicesAsrNetworkMapping -InputObject <ASRNetworkMapping> -RecoveryAzureNetworkId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Update-AzRecoveryServicesAsrNetWorkMapping** atualiza o mapeamento de rede de Recuperação de Site do Azure especificado.

## Exemplos

### Exemplo 1
```
PS C:\> $currentJob = Update-AzRecoveryServicesAsrNetworkMapping -Mapping $NetworkMapping -RecoveryNetwork $RecoveryNetwork
```

Inicia a operação de mapeamento de rede de atualização usando os parâmetros especificados e retorna o trabalho ASR usado para controlar a operação.
