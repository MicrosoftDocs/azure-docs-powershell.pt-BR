# Remove-AzRecoveryServicesAsrServicesProvider

## Sinopse
Exclui/desosterra o provedor especificado de serviços de recuperação de recuperação de sites do Azure do cofre de serviços de recuperação.

## Sintaxe

```
Remove-AzRecoveryServicesAsrServicesProvider -InputObject <ASRRecoveryServicesProvider> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzRecoveryServicesAsrServicesProvider** remove o provedor de serviços de recuperação de site do Azure especificado do cofre.

## Exemplos

### Exemplo 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrServicesProvider -ServicesProvider $ServicesProvider
```

Inicia a exclusão/desatribuição do provedor de serviços de Recuperação de Site do Azure especificado e retorna o trabalho ASR usado para controlar a operação.
