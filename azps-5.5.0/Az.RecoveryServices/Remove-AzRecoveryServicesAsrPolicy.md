# Remove-AzRecoveryServicesAsrPolicy

## Sinopse
Exclui a política de replicação ASR especificada do cofre dos Serviços de Recuperação.

## Sintaxe

```
Remove-AzRecoveryServicesAsrPolicy -InputObject <ASRPolicy> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzRecoveryServicesAsrPolicy** excluiu a política de replicação especificada do cofre dos Serviços de Recuperação.

## Exemplos

### Exemplo 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrPolicy -Policy $Policy
```

Inicia a exclusão da política de replicação especificada e retorna o trabalho ASR usado para controlar a operação.
