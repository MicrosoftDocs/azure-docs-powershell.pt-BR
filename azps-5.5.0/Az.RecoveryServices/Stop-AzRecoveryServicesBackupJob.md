# Stop-AzRecoveryServicesBackupJob

## Sinopse
Cancela um trabalho de execução.

## Sintaxe

### JobFilterSet (Padrão)
```
Stop-AzRecoveryServicesBackupJob [-Job] <JobBase> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdFilterSet
```
Stop-AzRecoveryServicesBackupJob [-JobId] <String> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Stop-AzRecoveryServicesBackup Job** cancela um trabalho de Backup do Azure existente.
Use este cmdlet para interromper um trabalho que demora muito e bloqueia outras atividades.
Você só pode cancelar tipos de trabalho de Backup e Restauração.
De definir o contexto do cofre usando o Set-AzRecoveryServicesVaultContext cmdlet antes de usar o cmdlet atual.

## Exemplos

### Exemplo 1: Interromper um trabalho de backup
```
PS C:\>$Job = Get-AzRecoveryServicesBackupJob -Operation Backup
PS C:\> Stop-AzRecoveryServicesBackupJob -JobID $Job.InstanceId
```

O primeiro comando obtém um trabalho de backup e armazena o trabalho na variável $Job dados.
O último comando interrompe o trabalho especificando a ID de Instância do trabalho de backup no $Job.
