# New-AzAutoscaleWebhook

## Sinopse
Cria uma Webscale webscale automática.

## Sintaxe

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzAutoscaleWebwebscale cria** uma Webscale webscale automática.

## Exemplos

### Exemplo 1

Cria uma Webscale webscale automática. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
