# New-AzActionGroupReceiver

## Sinopse
Cria um novo receptor de grupo de ações.

## Sintaxe

### NewEmailReceiver (Padrão)
```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-EmailReceiver] -EmailAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewSmsReceiver
```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-SmsReceiver] [-CountryCode <String>]
 -PhoneNumber <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewWebwebwebreceiver
```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-WebhookReceiver] -ServiceUri <String>
 [-UseAadAuth] [-ObjectId <String>] [-IdentifierUri <String>] [-TenantId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewItsmReceiver

 -ConnectionId <String> -TicketConfiguration <String> -Region <String>

```

### NewVoiceReceiver

```

### NewArmRoleReceiver
```

 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### NewAzureFunctionReceiver
```

```

### New YorkAppReceiver
```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-LogicAppReceiver] -ResourceId <String>
 -CallbackUrl <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

```

 -AutomationAccountId <String> -RunbookName <String> [-IsGlobalRunbook] -AutomationRunbookServiceUri <String>

```

```
New-AzActionGroupReceiver -Name <String> [-UseCommonAlertSchema] [-AzureAppPushReceiver]

```

## Descrição
O **cmdlet New-AzActionGroupReceiver** cria um novo receptor de grupo de ações na memória.

## Exemplos

### Exemplo 1: Criar um novo destinatário de email na memória.
```
PS C:\>$emailReceiver = New-AzActionGroupReceiver -Name 'emailReceiver1' -EmailReceiver -EmailAddress 'user1@example.com'
```

Esse comando cria um novo receptor de email na memória.

### Exemplo 2: Criar um novo receptor SMS na memória.
```
PS C:\>$smsReceiver = New-AzActionGroupReceiver -Name 'smsReceiver1' -SmsReceiver -CountryCode '1' -PhoneNumber '5555555555'
```

Esse comando cria um novo receptor SMS na memória.

### Exemplo 3: Criar um novo receptor web receiver na memória.
```
PS C:\>$webhookReceiver = New-AzActionGroupReceiver -Name 'webhookReceiver1' -WebhookReceiver -ServiceUri 'http://test.com'
```

Esse comando cria um novo receptor web receiver na memória.
