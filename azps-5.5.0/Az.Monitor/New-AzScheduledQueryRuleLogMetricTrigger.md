# New-AzScheduledQueryRuleLogMetricTrigger

## Sinopse
Cria um objeto do tipo Gatilho Métrico de Log.

## Sintaxe

```
New-AzScheduledQueryRuleLogMetricTrigger -ThresholdOperator <String> -Threshold <Double>
 -MetricTriggerType <String> -MetricColumn <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Cria um objeto do tipo Gatilho Métrico de Log e é opcional.
Essa é a condição de acionamento para a regra de consulta métrica, a ser usada quando você precisar estado tipo de alerta de medida métrica.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $metricTrigger = New-AzScheduledQueryRuleLogMetricTrigger -ThresholdOperator "GreaterThan" -Threshold 5 -MetricTriggerType "Consecutive" -MetricColumn "Computer"
```
