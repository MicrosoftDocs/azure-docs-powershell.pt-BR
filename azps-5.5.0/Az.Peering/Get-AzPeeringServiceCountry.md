# Get-AzPeeringServiceCountry

## Sinopse
Lista os países disponíveis para o serviço de paridade.

## Sintaxe

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Lista os países disponíveis para o serviço de paridade.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista de países disponíveis para o serviço de paridade.
