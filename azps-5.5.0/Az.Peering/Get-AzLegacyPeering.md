# Get-AzLegacyPeering

## Sinopse
Usado para converter recursos de peering herdados em recursos arm (gerenciamento de recursos) do Azure. 

## Sintaxe

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O comando é usado para exibir recursos herdados de Peering, que você pode convertê-los em Recursos de Gerenciamento de Recursos do Azure (ARM).
