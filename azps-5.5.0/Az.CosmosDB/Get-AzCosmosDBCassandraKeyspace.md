# Get-AzCosmosDBCassandraKeyspace

## Sinopse
Obtém um Espaço de Teclas Do CosmosDB.

## Sintaxe

### ByNameParameterSet (Padrão)
```
Get-AzCosmosDBCassandraKeyspace -ResourceGroupName <String> -AccountName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByParentObjectParameterSet
```
Get-AzCosmosDBCassandraKeyspace [-Name <String>] -ParentObject <PSDatabaseAccountGetResults>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzCosmosDBCasskeyspace** cria um novo ou atualiza um Espaço de Keyspace existente do CosmosDB.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzCosmosDBCassandraKeyspace -ResourceGroupName {resourceGroupName} -AccountName {accountName} -Name {name}

Name    Id   Resource
{name}  {id} {resourceObject}
```

Get-AzCosmosDBCassandraKeyspace obtém as propriedades de um Espaço deKeyspace existente. Você pode expandir o Recurso para obter as _etag, _ts, _rid propriedades.
