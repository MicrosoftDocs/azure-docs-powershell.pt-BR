# New-AzCosmosDBCassandraClusterKey

## Sinopse
Cria uma nova Chave Cluster De Cluster do CosmosDB.

## Sintaxe

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O **New-AzCosmosDBCasslusãoClusterKey** cria uma nova Chave de Cluster de Cluster DoSterDB.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
