
## Exemplos

### Exemplo 1
```powershell
PS C:\> Update-AzCosmosDBAccountFailoverPriority -ResourceGroupName rg -Name dbname -FailoverPolicy "region1, region2, region3"
```

FailoverPolicies atualizados com região1 como FailoverPriority 1, região2 como FailoverPriority 2 e região3 como FailoverPriority 3.
