# Get-AzKustoClusterLanguageExtension

## Sinopse
Retorna uma lista de extensões de idioma que podem ser executados em consultas KQL.

## Sintaxe

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrição
Retorna uma lista de extensões de idioma que podem ser executados em consultas KQL.

## Exemplos

### Exemplo 1: Listar todas as extensões de idioma definidas para um cluster
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

O comando acima retorna uma lista de extensões de idioma que podem ser executados em consultas KQL.
