# Get-AzSearchService

## Sinopse
Obtém um serviço de Pesquisa Cognitiva do Azure.

## Sintaxe

### ResourceGroupParameterSet (Padrão)
```
Get-AzSearchService [-ResourceGroupName] <String> [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzSearchService [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzSearchService** obtém o serviço especificado de Pesquisa Cognitiva do Azure.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzSearchService -ResourceGroupName felixwa-01


ResourceGroupName : felixwa-01
Name              : felixwa-basic-search
Location          : West US
Sku               : Basic
ReplicaCount      : 1
PartitionCount    : 1
HostingMode       : Default
Id                : /subscriptions/f9b96b36-1f5e-4021-8959-51527e26e6d3/resourceGroups/felixwa-01/providers/Microsoft.Search/searchServices/felixwa-basic-search
```

Obter um serviço de Pesquisa Cognitiva do Azure com parâmetros especificados.
