# Enable-AzStorageStaticWebsite

## Sinopse
Habilitar o site estático para a conta de Armazenamento do Azure.

## Sintaxe

```
Enable-AzStorageStaticWebsite [[-IndexDocument] <String>] [[-ErrorDocument404Path] <String>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Enable-AzStorageWebWebsite** habilita o site estático para a conta de Armazenamento do Azure.

## Exemplos

### Exemplo 1: Habilitar site estático para a conta de armazenamento do Azure
```
C:\PS>Enable-AzStorageStaticWebsite -IndexDocument $indexdoc -ErrorDocument404Path $errordoc
```

Esse comando habilita o site estático para a conta de Armazenamento do Azure.
