# Remove-AzStorageQueueStoredAccessPolicy

## Sinopse
Remove uma política de acesso armazenada de uma fila de armazenamento do Azure.

## Sintaxe

```
Remove-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzStorageQueueStoredAccessPolicy** remove uma política de acesso armazenada de uma fila de armazenamento do Azure.

## Exemplos

### Exemplo 1: Remover uma política de acesso armazenada de uma fila de armazenamento
```
PS C:\>Remove-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy04"
```

Esse comando remove uma política de acesso chamada Policy04 da fila de armazenamento chamada MyQueue.
