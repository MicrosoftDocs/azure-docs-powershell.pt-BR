## Descrição
O cmdlet **Start-AzStorageFileCopy** começa a copiar um arquivo de origem para um arquivo de destino.

## Exemplos

### Exemplo 1: Iniciar a operação de cópia de arquivo para arquivo usando o nome do compartilhamento e o nome do arquivo
```
PS C:\>Start-AzStorageFileCopy -SrcShareName "ContosoShare01" -SrcFilePath "FilePath01" -DestShareName "ContosoShare02" -DestFilePath "FilePath02"
```

Esse comando inicia uma operação de cópia de arquivo para arquivo.
O comando especifica o nome do compartilhamento e o nome do arquivo

### Exemplo 2: Iniciar a operação de cópia de blob para arquivo usando o nome do contêiner e o nome do blob
```
PS C:\>Start-AzStorageFileCopy -SrcContainerName "ContosoContainer01" -SrcBlobName "ContosoBlob01" -DestShareName "ContosoShare" -DestFilePath "FilePath02"
```

Esse comando inicia uma operação de cópia de blob para arquivo.
O comando especifica o nome do contêiner e o nome do blob
