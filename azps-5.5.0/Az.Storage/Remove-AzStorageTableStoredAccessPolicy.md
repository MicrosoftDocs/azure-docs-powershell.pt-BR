# Remove-AzStorageTableStoredAccessPolicy

## Sinopse
Remove uma política de acesso armazenada de uma tabela de armazenamento do Azure.

## Sintaxe

```
Remove-AzStorageTableStoredAccessPolicy [-Table] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzStorageTableStoredAccessPolicy** remove uma política de acesso armazenada de uma tabela de armazenamento do Azure.

## Exemplos

### Exemplo 1: Remover uma política de acesso armazenada de uma tabela de armazenamento
```
PS C:\>Remove-AzStorageTableStoredAccessPolicy -Table "MyTable" -Policy "Policy05"
```

Este comando remove a política chamada Policy05 da tabela de armazenamento chamada MyTable.
