# Disable-AzStorageStaticWebsite

## Sinopse
Desabilitar o site estático para a conta de Armazenamento do Azure.

## Sintaxe

```
Disable-AzStorageStaticWebsite [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Disable-AzStorageWebWebsite** desabilita o site estático para a conta de Armazenamento do Azure.

## Exemplos

### Exemplo 1: Desabilitar o site estático para uma conta de armazenamento do Azure
```
C:\PS>Disable-AzStorageStaticWebsite
```

Esse comando desabilita o site estático para uma conta de Armazenamento do Azure.
