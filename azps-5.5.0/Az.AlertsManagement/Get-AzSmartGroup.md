# Get-AzSmartGroup

## Sinopse
Obtém informações sobre Grupos Inteligentes

## Sintaxe

### SmartGroupsListByFilter (Padrão)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
**O cmdlet Get-AzSmartGroup** obtém informações sobre grupos inteligentes.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Listar todos os grupos inteligentes formados na última 1 hora. Use Format-List para obter os detalhes completos de cada grupo inteligente na lista.

### Exemplo 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Obter detalhes do Grupo Inteligente por ID (GUID) ou ID do Recurso (ID de ARM completa)
