# Módulo Az.ManagementPartner
## Descrição
Cmdlets do Parceiro de Gerenciamento para o Gerenciador de Recursos do Azure.

## Az.ManagementPartner Cmdlets
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Obtém a ID do MPN (Microsoft Partner Network) do usuário ou entidade de serviço autenticado atual. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Associa uma ID do Microsoft Partner Network (MPN) ao usuário autenticado ou entidade de serviço atual.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Exclua a ID do MPN (Microsoft Partner Network) do usuário autenticado ou entidade de serviço atual.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Atualiza a ID do MPN (Microsoft Partner Network) do usuário ou entidade de serviço autenticado atual.
