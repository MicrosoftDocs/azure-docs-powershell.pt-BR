# Get-AzRelayOperation

## Sinopse
Listar operações de retransmissão com suporte

## Sintaxe

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Get-AzRelayOperation lista** as Operações com suporte de Retransmissão.
