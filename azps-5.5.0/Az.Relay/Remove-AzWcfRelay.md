# Remove-AzWcfRelay

## Sinopse
Remove o WcfRelay do namespace de Retransmissão especificado.

## Sintaxe

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Remove-AzWcfRelay** remove o WcfRelay do namespace de Relay especificado.

## Exemplos

### Exemplo 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Remove o WcfRelay `TestWCFRelay1` do `TestNameSpace-Relay1` namespace.
