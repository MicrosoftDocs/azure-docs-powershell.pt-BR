# Stop-AzDataMigrationService

## Sinopse
Interrompe uma instância do Serviço de Migração de Banco de Dados do Azure que está em execução.

## Sintaxe

### ComponentNameParameterSet (Padrão)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Stop-AzDataMigrationService cmdlet interrompe uma instância do Serviço de Migração de Banco de Dados do Azure que está em execução.

## Exemplos

### Exemplo 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

O exemplo acima interrompe uma instância do Serviço de Migração de Banco de Dados do Azure chamada TestService com base no nome do serviço passado como parâmetro de entrada

### Exemplo 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

O exemplo acima interrompe uma instância do Serviço de Migração de Banco de Dados do Azure com base no objeto PSDataMigrationService passado como parâmetro de entrada.
