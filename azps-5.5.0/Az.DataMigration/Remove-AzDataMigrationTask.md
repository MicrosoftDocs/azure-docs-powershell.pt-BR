# Remove-AzDataMigrationTask

## Sinopse
Remove uma tarefa do Azure Database Migration Service do Azure.

## Sintaxe

### ComponentNameParameterSet (Padrão)
```
Remove-AzDataMigrationTask -ResourceGroupName <String> -ServiceName <String> -ProjectName <String>
 -Name <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Remove-AzDataMigrationTask [-InputObject] <PSProjectTask> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzDataMigrationTask [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Remove-AzDataMigrationTask cmdlet remove uma tarefa do Azure Database Migration Service do Azure.

## Exemplos

### Exemplo 1
```
PS C:\> Remove-AzDataMigrationTask -TaskName TestTask -ProjectName myTestProject -ServiceName MyTestService
 -ResourceGroupName MyResourceGroup
```

O exemplo anterior remove uma tarefa do Serviço de Migração de Banco de Dados do Azure chamada TestTask do Azure com base no parâmetro de nome da tarefa.

### Exemplo 2
```
PS C:\> Remove-AzDataMigrationTask -InputObject $TestTask
```

O exemplo anterior remove uma tarefa do Serviço de Migração de Banco de Dados do Azure com base no objeto PSProjectTask passado.
