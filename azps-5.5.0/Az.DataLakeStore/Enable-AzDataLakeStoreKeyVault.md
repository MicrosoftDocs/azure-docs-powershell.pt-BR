# Enable-AzDataLakeStoreKeyVault

## Sinopse
Tenta habilitar um Cofre de Chave gerenciado pelo usuário para criptografia da conta especificada do Data Lake Store.

## Sintaxe

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Enable-AzDataLakeStoreKeyVault** tenta habilitar um Cofre de Chave gerenciado pelo usuário para criptografia da conta especificada do Data Lake Store.

## Exemplos

### Exemplo 1: Habilitar o Cofre de Teclas para a conta contosoADLS
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Esse comando tenta habilitar o Cofre de Teclas gerenciado pelo usuário para a conta do Data Lake Store chamada ContosoADLS.
