# Set-AzDataLakeStoreItemPermission

## Sinopse
Modifica a octal de permissão de um arquivo ou pasta no Data Lake Store.

## Sintaxe

```
Set-AzDataLakeStoreItemPermission [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Permission] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzDataLakeStoreItemPermission** modifica o octal de permissão de um arquivo ou pasta no Data Lake Store.

## Exemplos

### Exemplo 1: Definir a octal de permissão para um item
```
PS C:\>Set-AzDataLakeStoreItemPermission -AccountName "ContosoADL" -Path "/file.txt" -Permission 0770
```

Esse comando define a octal de permissão para um arquivo como 0770, o que significa limpar o bit auto sticky, definir permissões de leitura/gravação/execução para o proprietário do arquivo, definir permissões de leitura/gravação/execução para o grupo proprietário do arquivo e limpar permissões de leitura/gravação/execução para outros.
