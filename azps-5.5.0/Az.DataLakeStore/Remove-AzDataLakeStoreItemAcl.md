# Remove-AzDataLakeStoreItemAcl

## Sinopse
Limpa o ACL de um arquivo ou pasta no Data Lake Store.

## Sintaxe

```
Remove-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Default] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDataLakeStoreItemAcl** limpa a lista de controles de acesso (ACL) de um arquivo ou pasta na Data Lake Store.

## Exemplos

### Exemplo 1: Remover o ACL de uma pasta
```
PS C:\>Remove-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/"
```

Esse comando remove o ACL do diretório raiz da conta contosoADL.
