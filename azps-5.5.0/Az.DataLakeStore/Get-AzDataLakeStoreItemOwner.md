# Get-AzDataLakeStoreItemOwner

## Sinopse
Obtém o proprietário de um arquivo ou pasta no Data Lake Store.

## Sintaxe

```
Get-AzDataLakeStoreItemOwner [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Type] <Owner>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzDataLakeStoreItemOwner** obtém o proprietário de um arquivo ou pasta na Data Lake Store.

## Exemplos

### Exemplo 1: Obter o proprietário de um diretório
```
PS C:\>Get-AzDataLakeStoreItemOwner -AccountName "ContosoADL" -Path / -Type User
```

Esse comando obtém o proprietário do usuário para o diretório raiz da conta contosoADL.
