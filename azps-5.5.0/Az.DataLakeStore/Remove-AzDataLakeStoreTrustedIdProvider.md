# Remove-AzDataLakeStoreTrustedIdProvider

## Sinopse
Remove o provedor de identidade confiável especificado no Data Lake Store especificado.

## Sintaxe

```
Remove-AzDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDataLakeStoreTrustedIdProvider** remove o provedor de identidade confiável especificado na Data Lake Store especificada.

## Exemplos

### Exemplo 1: Remover um provedor de identidade confiável.
```
PS C:\> Remove-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

Remove o provedor "MeuProvider" da conta "ContosoADL"
