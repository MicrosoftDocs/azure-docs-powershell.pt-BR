# Get-AzDataLakeStoreChildItemSummary

## Sinopse
Obtém o resumo do tamanho total, arquivos e diretórios contidos no caminho especificado

## Sintaxe

```
Get-AzDataLakeStoreChildItemSummary [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Concurrency <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **Get-AzDataLakeStoreChildItemSummary** recupera o resumo de conteúdo de um determinado caminho. Ele calcula recursivamente o número total de arquivos, diretórios e tamanho total de todos os arquivos sob o caminho determinado.

## Exemplos

### Exemplo 1: Obter o resumo de conteúdo de uma pasta
```
PS C:\> Get-AzDataLakeStoreChildItemSummary -Account ContosoADL -Path /a -Concurrency 128
```

Ele lista o número total de diretórios, arquivos e seu tamanho contidos em /a.
