# Set-AzDataLakeStoreItemAcl

## Sinopse
Modifica o ACL de um arquivo ou pasta no Data Lake Store.

## Sintaxe

```
Set-AzDataLakeStoreItemAcl [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Acl] <DataLakeStoreItemAce[]> [-PassThru] [-Recurse] [-Concurrency <Int32>] [-ShowProgress]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzDataLakeStoreItemAcl** modifica a lista de controles de acesso (ACL) de um arquivo ou pasta na Data Lake Store.

## Exemplos

### Exemplo 1: Definir o ACL para um arquivo e uma pasta
```
PS C:\>$ACL = Get-AzDataLakeStoreItemAclEntry -AccountName "ContosoADL" -Path /
PS C:\> Set-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/MyFiles/Test.txt" -Acl $ACL
```

O primeiro comando obtém o ACL para o diretório raiz da conta contosoADL e o armazena na variável $ACL raiz.
O segundo comando define o ACL para o Test.txt para o arquivo no $ACL.

### Exemplo 2: Definir a ACL para pasta recursivamente
```
PS C:\>$ACL = Get-AzDataLakeStoreItemAclEntry -AccountName "ContosoADL" -Path /Folder1
PS C:\> Set-AzDataLakeStoreItemAcl -AccountName "ContosoADL" -Path "/Folder2" -Acl $ACL -Recurse -Concurrency 128
```

O primeiro comando obtém o ACL para a Pasta1 do diretório da conta ContosoADL e, em seguida, armazena-o na variável $ACL dados.
O segundo comando define a ACL recursivamente para a Pasta2 e seus sub-diretórios e arquivos para aquele no $ACL.
