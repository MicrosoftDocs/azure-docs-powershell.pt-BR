# Get-AzDataLakeAnalyticsJobPipeline

## Sinopse
Obtém pipelines ou pipelines de trabalho do Data Lake Analytics.

## Sintaxe

### GetAllInAccount (Padrão)
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetBySpecificJobJobline
```
Get-AzDataLakeAnalyticsJobPipeline [-Account] <String> [-PipelineId] <Guid> [-SubmittedAfter <DateTimeOffset>]
 [-SubmittedBefore <DateTimeOffset>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **Get-AzDataLakeAnalytics Job Pipeline do** Azure Data Lake Analytics obtém um pipeline especificado do Azure Data Lake Analytics ou uma lista de pipelines.

## Exemplos

### Exemplo 1: Obter um pipeline especificado
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -Account "contosoadla" -PipelineId 83cb7ad2-3523-4b82-b909-d478b0d8aea3
```

Este comando obtém o pipeline especificado com a id '83cb7ad2-3523-4b82-b909-d478b0d8aea3' na conta "contosoadla".

### Exemplo 2: Obter uma lista de todos os pipelines na conta
```
PS C:\>Get-AzDataLakeAnalyticsJobPipeline -AccountName "contosoadla"
```

Este comando obtém uma lista de todos os pipelines na conta "contosoadla"
