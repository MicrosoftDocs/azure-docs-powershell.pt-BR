# Get-AzDataLakeAnalyticsComputePolicy

## Sinopse
Obtém uma política de computação do Data Lake Analytics ou uma lista de políticas de computação.

## Sintaxe

```
Get-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **Get-AzDataLakeAnalyticsComputePolicy** obtém uma política de computação especificada do Azure Data Lake Analytics ou uma lista de políticas.

## Exemplos

### Exemplo 1: Obter uma política de computação especificada
```
PS C:\>Get-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Esse comando obtém a política de computação especificada com o nome 'myPolicy' na conta 'contosoadla'.

### Exemplo 2: Obter uma lista de todas as políticas de computação na conta
```
PS C:\>Get-AzDataLakeAnalyticsComputePolicy -AccountName "contosoadla"
```

Esse comando obtém uma lista de todas as políticas de computação na conta "contosoadla"
