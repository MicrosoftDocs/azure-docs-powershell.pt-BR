# Get-AzHpcCacheUsageModel

## Sinopse
Obtém todos os usageModels para Destino de Armazenamento NFS.

## Sintaxe

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzHpcCacheUsageModel** retorna uma lista de modelos de uso para o Destino de Armazenamento NFS.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
