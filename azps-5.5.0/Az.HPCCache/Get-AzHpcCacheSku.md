# Get-AzHpcCacheSku

## Sinopse
Obtém todas as SKUs disponíveis na assinatura.

## Sintaxe

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzHpcCacheSku** retorna uma lista de SKUs disponíveis na assinatura.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
