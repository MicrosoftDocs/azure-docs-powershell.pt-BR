# Stop-AzSynapseTrigger

## Sinopse
Interrompe um gatilho em um espaço de trabalho.

## Sintaxe

### StopByName (Padrão)
```
Stop-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByObject
```
Stop-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Stop-AzSynapseTrishot** interrompe um gatilho em um espaço de trabalho. Se o gatilho estiver no estado "Iniciado", o cmdlet parará o gatilho e não invocará mais os pipelines. Se o gatilho já estiver no estado "Parado", esse cmdlet não terá efeito.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Stop-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Interrompe um gatilho chamado ContosoTrição no espaço de trabalho ContosoWorkspace.

### Exemplo 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Stop-AzSynapseTrigger -Name ContosoTrigger
```

Interrompe um gatilho chamado ContosoTri pipeline no espaço de trabalho ContosoWorkspace.

### Exemplo 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Stop-AzSynapseTrigger
```

Parar um gatilho chamado ContosoTri pipeline no espaço de trabalho ContosoWorkspace.
