# Test-AzSynapseSparkPool

## Sinopse
Verifica a existência de um pool de Miniapsis de Análise Synapse.

## Sintaxe

### TestByNameParameterSet (Padrão)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Test-AzSynapseSparkPool** verifica a existência de um pool de miniaptas de análise synapse.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

Esse comando verifica a existência do pool de Spark especificado.
