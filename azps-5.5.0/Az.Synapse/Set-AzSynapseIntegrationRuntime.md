## Descrição
O cmdlet **Set-AzSynapseIntegrationRuntime** atualiza um tempo de execução de integração com parâmetros específicos.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Description 'New description'
```

O cmdlet atualiza a descrição do tempo de execução de integração chamado "test-selfhost-ir".

### Exemplo 2
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' `
                                        -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"
```

O cmdlet adiciona o espaço de trabalho para usar o tempo de execução de integração compartilhada. Ao usar `-SharedIntegrationRuntimeResourceId` o parâmetro, `-Type` o parâmetro também deve ser incluído. Observe que o espaço de trabalho precisa ter permissão para usar o tempo de execução de integração antes de executar o cmdlet.
