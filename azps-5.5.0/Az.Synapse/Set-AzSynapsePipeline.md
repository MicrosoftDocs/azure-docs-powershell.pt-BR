# Set-AzSynapsePipeline

## Sinopse
Cria um pipeline no espaço de trabalho.

## Sintaxe

### SetByName (Padrão)
```
Set-AzSynapsePipeline -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapsePipeline -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzSynapse Pipelineline** cria um pipeline no espaço de trabalho.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Set-AzSynapsePipeline -WorkspaceName ContosoWorkspace -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Esse comando cria um pipeline chamado ContosoSpaceline no espaço de trabalho chamado ContosoWorkspace.
O comando baseia o pipeline em informações no pipeline.jsno arquivo.
Este arquivo inclui informações sobre atividades.

### Exemplo 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Set-AzSynapsePipeline -Name ContosoPipeline -DefinitionFile "C:\pipeline.json"
```

Esse comando cria um pipeline chamado Contoso Pipeline no espaço de trabalho chamado ContosoWorkspace por meio de pipeline.
O comando baseia o pipeline em informações no pipeline.jsno arquivo.
Este arquivo inclui informações sobre atividades.
