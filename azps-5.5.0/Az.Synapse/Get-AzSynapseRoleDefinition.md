# Get-AzSynapseRoleDefinition

## Sinopse
Obtém uma definição de função do Synapse Analytics.

## Sintaxe

### GetByWorkspaceNameAndIdParameterSet (Padrão)
```
Get-AzSynapseRoleDefinition -WorkspaceName <String> [-Id <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByWorkspaceNameAndNameParameterSet
```
Get-AzSynapseRoleDefinition -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByWorkspaceObjectAndIdParameterSet
```
Get-AzSynapseRoleDefinition -WorkspaceObject <PSSynapseWorkspace> -Id <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByWorkspaceObjectAndNameParameterSet
```
Get-AzSynapseRoleDefinition -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzSynapseRoleDefinition obtém** uma Definição de Função de Análise Synapse do Azure.
Se você não especificar um nome de função ou uma ID de função, este cmdlet obtém todas as definições de função.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzSynapseRoleDefinition -WorkspaceName ContosoWorkspace
```

Esse comando obtém todas as definições de função em um espaço de trabalho.

### Exemplo 2
```powershell
PS C:\> Get-AzSynapseRoleDefinition -WorkspaceName ContosoWorkspace -Name ContosoRole
```

Esse comando obtém a definição de função no espaço de trabalho ContosoWorkspace com o nome ContosoRole.

### Exemplo 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseRoleDefinition
```

Esse comando obtém todas as definições de função em um espaço de trabalho por meio do pipeline.
