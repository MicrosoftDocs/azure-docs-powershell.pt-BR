# Start-AzSynapseTrigger

## Sinopse
Inicia um gatilho em um espaço de trabalho.

## Sintaxe

### StartByName (Padrão)
```
Start-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByObject
```
Start-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByInputObject
```
Start-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Start-AzSynapseTrishot** inicia um gatilho em um espaço de trabalho. Se o gatilho estiver no estado "Parado", o cmdlet iniciará o gatilho e eventualmente invocará pipelines com base em sua definição. Se o gatilho já estiver no estado "Iniciado", esse cmdlet não terá efeito.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Start-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Inicia um gatilho chamado ContosoTri fazer isso no espaço de trabalho ContosoWorkspace.

### Exemplo 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Start-AzSynapseTrigger -Name ContosoTrigger
```

Inicia um gatilho chamado ContosoTri pipeline no espaço de trabalho ContosoWorkspace.

### Exemplo 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Start-AzSynapseTrigger
```

Inicia um gatilho chamado ContosoTri pipeline no espaço de trabalho ContosoWorkspace.
