# New-AzStreamAnalyticsOutput

## Sinopse
Cria ou atualiza saídas para um trabalho do Stream Analytics.

## Sintaxe

```
New-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzStreamAnalyticsOutput** cria uma saída dentro de um trabalho do Stream Analytics ou atualiza uma saída existente.
O nome da saída pode ser especificado na . Arquivo JSON ou na linha de comando.
Se ambos são especificados, o nome na linha de comando deve corresponder ao nome no arquivo.
Se você especificar uma saída que já  existe e não especificar o parâmetro Forçar, o cmdlet perguntará se deve ou não substituir a saída existente.
Se você especificar o *parâmetro Forçar* e especificar um nome de saída existente, a saída será substituída sem confirmação.

## Exemplos

### Exemplo 1: Adicionar uma saída a um trabalho
```powershell
PS C:\>New-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Output.json" -JobName "StreamingJob" -Name "Output"
```

Esse comando cria uma nova saída chamada Saída no trabalho chamado Streaming Job.
Se uma saída existente com esse nome já estiver definida, o cmdlet perguntará se deve ou não substituí-la.

### Exemplo 2: Substituir uma definição de saída de trabalho
```powershell
PS C:\>New-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Output.json" -JobName "StreamingJob" -Name "Output" -Force
```

Esse comando substitui a definição de Saída no trabalho chamado Streaming Job sem confirmação.
