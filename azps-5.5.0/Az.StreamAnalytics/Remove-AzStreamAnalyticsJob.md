# Remove-AzStreamAnalyticsJob

## Sinopse
Remove um trabalho do Stream Analytics.

## Sintaxe

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzStreamAnalytics Job** assíncrona exclui de forma assíncrona um trabalho específico do Stream Analytics no Azure.

## Exemplos

### EXEMPLO 1: Remover um trabalho
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Esse comando remove o trabalho Streaming Job.
