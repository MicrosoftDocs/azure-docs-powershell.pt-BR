# Get-AzStreamAnalyticsOutput

## Sinopse
Obtém as saídas definidas em um trabalho ou saída especificado do Stream Analytics.

## Sintaxe

```
Get-AzStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzStreamAnalyticsOutput** lista todas as saídas definidas em um trabalho especificado do Stream Analytics ou obtém informações sobre uma saída específica.

## Exemplos

### Exemplo 1: obter informações sobre saídas de trabalho
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob"
```

Esse comando retorna informações sobre as saídas definidas no trabalho Streaming Job.

### Exemplo 2: obter informações sobre uma saída de trabalho específica
```powershell
PS C:\>Get-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Esse comando retorna informações sobre a saída chamada Saída definida no trabalho Streaming Job.
