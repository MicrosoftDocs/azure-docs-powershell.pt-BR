# Start-AzStreamAnalyticsJob

## Sinopse
Inicia um trabalho do Stream Analytics.

## Sintaxe

```
Start-AzStreamAnalyticsJob [-Name] <String> [[-OutputStartMode] <String>] [[-OutputStartTime] <DateTime>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Start-AzStreamAnalytics Job** implanta de forma assíncrona e inicia um trabalho de Análise de Fluxo no Azure.

## Exemplos

### Exemplo 1: Iniciar um trabalho do Stream Analytics
```powershell
PS C:\> Start-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob" -OutputStartMode "CustomTime" -OutputStartTime "2014-07-03T01:00Z"
```

Este comando inicia o trabalho Streaming Job Streaming Job e especifica que o fluxo de evento de saída deve começar no timestamp 2014-07-03T01:00Z.

### -OutputStartMode
Especifica o modo de início do trabalho.
Os valores válidos são: 
- JobStartTime – Esse valor indica que o ponto de partida do fluxo de eventos de saída deve começar quando o trabalho for iniciado.
- CustomTime - Esse valor indica que o ponto inicial do fluxo de eventos de saída deve começar em um horário personalizado especificado no parâmetro *OutputStartTime.* 
 - LastOutputEventTime - Esse valor indica que o ponto de partida do stream de evento de saída deve começar a partir do último tempo de saída do evento.
Se a propriedade estiver ausente, o padrão será JobStartTime.

Especifica a hora de início da saída.
Esse valor é um carimbo de hora formatado iso-8601 que indica o ponto de partida do fluxo de eventos de saída ou $Null para indicar que o fluxo de eventos de saída será iniciado sempre que o trabalho de streaming for iniciado.
Essa propriedade deve ter um valor se *o OutputStartMode* estiver definido como CustomTime.
