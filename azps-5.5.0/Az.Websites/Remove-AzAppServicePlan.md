# Remove-AzAppServicePlan

## Sinopse
Remove um plano de Serviço do Aplicativo Azure.

## Sintaxe

### S1
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-AppServicePlan] <PSAppServicePlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Remove-AzAppServicePlan** remove um plano de Serviço do Aplicativo Azure.

## Exemplos

### Exemplo 1: Remover um plano de Serviço de Aplicativo
```
PS C:\>Remove-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Esse comando remove o plano de Serviço do Aplicativo Azure chamado ContosoASP que pertence ao grupo de recursos chamado Default-Web-WestUS.
