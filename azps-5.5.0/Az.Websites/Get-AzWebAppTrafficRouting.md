# Get-AzWebAppTrafficRouting

## Sinopse
Obter uma Regra de roteamento para o nome de Slot determinado.

## Sintaxe

```
Get-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RuleName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzWebAppTrafficRouting** obtém uma configuração de regra de roteamento de um Slot do Azure Web App.

## Exemplos

### Exemplo 1 Obtém a regra de roteamento específica do slot webapp
```powershell
PS C:\> Get-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite"  -RuleName 'Stg'
```

Esse comando obtém uma regra de roteamento para um determinado slot de webapp.
