# Remove-AzWebAppCertificate

## Sinopse
Cria um certificado gerenciado pelo serviço de aplicativos para um Azure Web App. 

## Sintaxe

```
Remove-AzWebAppCertificate [-ResourceGroupName] <String> [-ThumbPrint] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzWebAppCertificate** cria um Certificado Gerenciado pelo Serviço do Azure App

## Exemplos

### Exemplo 1
```powershell
PS C:\>Remove-AzWebAppCertificate -ResourceGroupName Default-Web-WestUS -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3" 
```

Este comando remove o certificado gerenciado pelo serviço de aplicativo para o aplicativo Web determinado.
