# Reset-AzWebAppSlotPublishingProfile

## Sinopse

## Sintaxe

### S1
```
Reset-AzWebAppSlotPublishingProfile [-ResourceGroupName] <String> [-Name] <String> [-Slot] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Reset-AzWebAppSlotPublishingProfile [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Reset-AzWebAppSlotPublishingProfile** redefine o perfil de publicação para o Slot de Aplicativo Web especificado.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Reset-AzWebAppSlotPublishingProfile -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -Slot "slot001"
```

Esse comando redefine o perfil de publicação do Slot nomeado slot001 para o Web App ContosoWebApp associado ao grupo de recursos Default-Web-WestUS.
