# Get-AzWebAppCertificate

## Sinopse
Recebe um certificado do Azure Web App.

## Sintaxe

```
Get-AzWebAppCertificate [[-ResourceGroupName] <String>] [[-Thumbprint] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzWebAppCertificate** obtém informações sobre certificados do Azure Web App associados a um grupo de recursos especificado.
Se você sabe a miniatura do certificado, também pode usar esse cmdlet para obter informações sobre um certificado especificado.

## Exemplos

### Exemplo 1: Obter certificados do Web App em um grupo de recursos
```
PS C:\>Get-AzWebAppCertificate -ResourceGroupName "ContosoResourceGroup"
```

Esse comando retorna informações sobre os certificados do Web App carregados associados ao grupo de recursos ContosoResourceGroup.

### Exemplo 2: Obter um certificado de aplicativo Web especificado
```
PS C:\>Get-AzWebAppCertificate -ResourceGroupName "ContosoResourceGroup" -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3"
```

Esse comando obtém o certificado ContosoResourceGroup Web App com a impressão digital E3A38EBA60CAA1C162785A2E1C44A15AD450199C3.
