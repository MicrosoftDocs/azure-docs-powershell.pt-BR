# Get-AzWebAppAccessRestrictionConfig

## Sinopse
Obtém a configuração de Restiction do Access para um Azure Web App.

## Sintaxe

```
Get-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String> [[-SlotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzWebAppAccessRestrictionConfig obtém** a configuração de Restrição do Access sobre um Azure Web App.

## Exemplos

### Exemplo 1: Obter uma Configuração de Restrição de Acesso do Web App de um grupo de recursos
```
PS C:\>Get-AzWebAppAccessRestrictionConfig -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite"
```

Esse comando obtém a configuração de restrição de acesso de um Web App chamado ContosoSite que pertence ao grupo de recursos Default-Web-WestUS.
