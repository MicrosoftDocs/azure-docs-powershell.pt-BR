# Start-AzWebApp

## Sinopse
Inicia um Azure Web App.

## Sintaxe

### S1
```
Start-AzWebApp [-ResourceGroupName] <String> [-Name] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### S2
```
Start-AzWebApp [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Start-AzWebApp** inicia um Azure Web App.

## Exemplos

### Exemplo 1: Iniciar um Aplicativo Web
```
PS C:\>Start-AzWebApp -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp"
```

Esse comando inicia o Web App chamado ContosoWebApp que pertence ao grupo de recursos chamado Default-Web-WestUS.
