# Reset-AzWebAppPublishingProfile

## Sinopse

## Sintaxe

### S1
```
Reset-AzWebAppPublishingProfile [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Reset-AzWebAppPublishingProfile [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Reset-AzWebAppPublishingProfile** redefine o perfil de publicação do Web App especificado.

## Exemplos

### Exemplo 1

O exemplo a seguir redefine o perfil de publicação do IpRule do Web App associado ao grupo de recursos MyResourceGroup.

```powershell <!-- Aladdin Generated Example --> 
Reset-AzWebAppPublishingProfile -Name IpRule -ResourceGroupName MyResourceGroup
```
