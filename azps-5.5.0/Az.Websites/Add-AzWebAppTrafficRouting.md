# Add-AzWebAppTrafficRouting

## Sinopse
Adicione uma Regra de Roteamento ao Slot.

## Sintaxe

```
Add-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Add-AzWebAppTrafficRouting** adiciona uma regra de Roteamento a um Slot do Azure Web App.

## Exemplos

### Exemplo 1 Adicionar uma regra de roteamento para transferir 15% do tráfego de produção para o slot Stg
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

Este comando adiciona uma regra de roteamento para transferir 15% do tráfego de produção para o slot Stg

### Exemplo 2 Adicionar uma regra de roteamento para transferir o tráfego de produção para intervalos stg de 50% a 90% de maneira incremental.
```powershell
PS C:\>Add-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

Esse comando adiciona uma regra de roteamento para transferir o tráfego de produção para intervalos stg de 50% a 90% de maneira incremental.
