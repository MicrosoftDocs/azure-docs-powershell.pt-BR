# Get-AzSupportService

## Sinopse
Obter serviços para os quais o suporte está disponível. 

## Sintaxe

### ListParameterSet (Padrão)
```
Get-AzSupportService [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzSupportService -Id <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Obtém a lista atual de serviços do Azure para os quais o suporte está disponível. Cada serviço pode conter uma ou mais informações de tipo de recurso do Gerenciador de Recursos do Azure (ARM). Tipos de recursos (exemplo: 'microsoft.compute/virtualmachines') podem ser úteis para mapear o produto e o recurso ARM certos ao criar um tíquete de suporte técnico. Cada serviço do Azure tem seu próprio conjunto de categorias chamada classificação de problemas. Obter a lista atual de classificação de problemas para um serviço usando Get-AzSupportProblemClassification. Você pode usar o GUID de classificação de problemas e serviço para criar um novo tíquete de suporte usando o New-AzSupportTicket.

Sempre use os GUIDs de classificação de problemas e serviço obtidos programaticamente. Essa prática garante que você tenha os GUIDs de classificação de problemas e de serviço mais recentes para criação de tíquetes de suporte.
