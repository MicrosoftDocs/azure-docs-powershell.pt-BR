# Remove-AzMlCommitmentPlan

## Sinopse
Exclui um plano de compromisso.

## Sintaxe

### RemoveByNameAndResourceGroup
```
Remove-AzMlCommitmentPlan -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByObject
```
Remove-AzMlCommitmentPlan -MlCommitmentPlan <CommitmentPlan> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Exclui um plano de compromisso do Azure Machine Learning. Observe que os planos de compromisso que têm associações de compromisso não podem ser excluídos. As associações de compromisso só podem ser excluídas pelo recurso de destino. Por exemplo, se você excluir um serviço Web do Azure Machine Learning, a associação de compromissos que associa o serviço Web a um plano de compromisso também será excluída.

## Exemplos

### Exemplo 1: Excluir um plano de compromisso
```
Remove-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName"
```
