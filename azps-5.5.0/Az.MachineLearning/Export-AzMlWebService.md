# Export-AzMlWebService

## Sinopse
Exporta o objeto de definição de serviço web como uma cadeia de caracteres formatada JSON.

## Sintaxe

### ExportToFile
```
Export-AzMlWebService -WebService <WebService> -OutputFile <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ExportToJSON
```
Export-AzMlWebService -WebService <WebService> [-ToJsonString] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Exporta o objeto de definição para o serviço Web especificado como uma cadeia de caracteres formatada JSON.
Você pode retornar a cadeia de caracteres imediatamente ou salvá-la em um arquivo.

## Exemplos

### Exemplo 1: Exportar como cadeia de caracteres
```
Export-AzMlWebService -WebService $svc -ToJsonString
```

### Exemplo 2: Exportar para arquivo
```
Export-AzMlWebService -WebService $svc -OutputFile "C:\mlservice.json"
```
