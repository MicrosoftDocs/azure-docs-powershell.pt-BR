# Add-AzMlWebServiceRegionalProperty

## Sinopse
Cria propriedades regionais do serviço Web.

## Sintaxe

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName <String> -Name <String> -Region <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Cria propriedades regionais do Azure Machine Learning para um serviço Web existente.

## Exemplos

### Exemplo 1: Adicionar novas propriedades regionais para a West Central US

```
Add-AzMlWebServiceRegionalProperty -ResourceGroupName "myresourcegroup" -Name "mywebservicename" -Region westcentralus
```

Este comando de exemplo cria uma propriedade regional para um serviço Web na região "Oeste Central dos EUA".
