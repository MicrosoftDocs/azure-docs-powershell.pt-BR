# Remove-AzStorageSyncService

## Sinopse
Esse comando excluirá o serviço de sincronização de armazenamento especificado.

## Sintaxe

### StringParameterSet (Padrão)
```
Remove-AzStorageSyncService [-ResourceGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncService [-InputObject] <PSStorageSyncService> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncService [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Esse comando excluirá o serviço de sincronização de armazenamento especificado. Um serviço de sincronização de armazenamento só pode ser removido quando todos os grupos de sincronização contidos e servidores registrados são excluídos primeiro.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Remove-AzStorageSyncService -Force -ResourceGroupName "myResourceGroup" -Name "myStorageSyncServiceName"
```

Esse comando removerá o serviço de sincronização de armazenamento.
