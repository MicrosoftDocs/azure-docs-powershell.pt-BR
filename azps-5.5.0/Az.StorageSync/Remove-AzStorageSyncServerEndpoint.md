# Remove-AzStorageSyncServerEndpoint

## Sinopse
Esse comando excluirá o ponto de extremidade do servidor especificado. A sincronização com esse local será parada imediatamente.

## Sintaxe

### StringParameterSet (Padrão)
```
Remove-AzStorageSyncServerEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzStorageSyncServerEndpoint [-InputObject] <PSServerEndpoint> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzStorageSyncServerEndpoint [-ResourceId] <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Remover um ponto de extremidade de servidor é uma operação prejudicial. Esse local do servidor interromperá a sincronização. Essa operação não deve ser executada para resolver problemas de sincronização. Se esse local do servidor (incl. são arquivos) for adicionado novamente ao mesmo grupo de sincronização que um ponto de extremidade de servidor, ele poderá levar a arquivos de conflito e outras consequências não intencionais. Este comando destina-se apenas à desativação.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Remove-AzStorageSyncServerEndpoint -Force -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName" -Name "myServerEndpointName"
```

Esse comando removerá o ponto de extremidade do servidor.
