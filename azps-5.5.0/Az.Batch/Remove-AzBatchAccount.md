# Remove-AzBatchAccount

## Sinopse
Remove uma conta em lotes.

## Sintaxe

```
Remove-AzBatchAccount [-AccountName] <String> [[-ResourceGroupName] <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Remove-AzBatchAccount** remove uma conta do Azure Batch.
Este cmdlet solicita antes de remover uma conta, a menos que você especifique o *parâmetro Force.*

## Exemplos

### Exemplo 1: Remover uma conta em lotes
```
PS C:\>Remove-AzBatchAccount -AccountName "pfuller"
```

Esse comando remove a conta Em lotes chamada pfuller.
Esse comando solicita a confirmação antes de excluir a conta.
