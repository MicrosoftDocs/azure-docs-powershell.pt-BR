# New-AzBatchAccountKey

## Sinopse
Regenera uma chave de uma conta em lotes.

## Sintaxe

```
New-AzBatchAccountKey [-AccountName] <String> [-ResourceGroupName <String>] -KeyType <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzBatchAccountKey** regenera a chave primária ou secundária de uma conta do Azure Batch.
O cmdlet retorna um objeto **BatchAccountContext que** tem as propriedades **PrimaryAccountKey** e **SecondaryAccountKey** atuais.

## Exemplos

### Exemplo 1: Regenerar a chave da conta primária em uma conta em lote
```
PS C:\>New-AzBatchAccountKey -AccountName "pfuller" -KeyType "Primary"
AccountName                  : pfuller
Location                     : westus
ResourceGroupName            : CmdletExampleRG
DedicatedCoreQuota           : 20
LowPriorityCoreQuota         : 20
PoolQuota                    : 20
ActiveJobAndJobScheduleQuota : 20
Tags                         :
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Esse comando regenera a chave da conta primária na conta em lote chamada pfuller.
