# Módulo Az.ConnectedKubernetes
## Descrição
Microsoft Azure PowerShell: cmdlets ConnectedKubernetes

## Az.ConnectedKubernetes Cmdlets
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Retorna as propriedades do cluster conectado especificado, incluindo nome, identidade, propriedades e detalhes adicionais do cluster.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API para registrar um novo cluster K8s e, assim, criar um recurso rastreado no ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Exclua um cluster conectado, removendo o recurso rastreado no Gerenciador de Recursos do Azure (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API para atualizar determinadas propriedades do recurso de cluster conectado
