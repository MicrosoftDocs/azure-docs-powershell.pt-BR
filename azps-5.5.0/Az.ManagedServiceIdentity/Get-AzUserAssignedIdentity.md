# Get-AzUserAssignedIdentity

## Sinopse
Obtém identidades/identidades atribuídas pelo usuário.

## Sintaxe

### SuscriptionParameterSet (Padrão)
```
Get-AzUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **Get-AzUserAssignedIdentity** recebe identidades de usuário existentes atribuídas.
