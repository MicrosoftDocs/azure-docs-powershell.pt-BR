# New-AzUserAssignedIdentity

## Sinopse
Cria uma nova Identidade atribuída pelo Usuário ou atualiza uma identidade atribuída pelo usuário existente.

## Sintaxe

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzUserAssignedIdentity** cria uma nova Identidade atribuída pelo Usuário. Quando usado com uma identidade já existente, atualiza a identidade.
Para adicionar marcas do Gerenciador de Recursos do Azure à identidade, use o cmdlet Set-AzResource dados.
