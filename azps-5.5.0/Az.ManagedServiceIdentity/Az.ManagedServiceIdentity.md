# Módulo Az.ManagedServiceIdentity
## Descrição
Este tópico exibe os tópicos de ajuda para os Cmdlets de Identidade de Serviço Gerenciado do Azure.

## Az.ManagedServiceIdentity Cmdlets
### [Get-AzUserAssignedIdentity](Get-AzUserAssignedIdentity.md)
Obtém identidades/identidades atribuídas pelo usuário.

### [New-AzUserAssignedIdentity](New-AzUserAssignedIdentity.md)
Cria uma nova Identidade atribuída pelo Usuário ou atualiza uma identidade atribuída pelo usuário existente.

### [Remove-AzUserAssignedIdentity](Remove-AzUserAssignedIdentity.md)
Remove uma identidade atribuída pelo usuário.
