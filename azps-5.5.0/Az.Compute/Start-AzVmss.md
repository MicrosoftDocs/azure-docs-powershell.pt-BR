# Start-AzVmss

## Sinopse
Inicia o VMSS ou um conjunto de máquinas virtuais dentro do VMSS.

## Sintaxe

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Start-AzVmss** inicia todas as máquinas virtuais no VMSS (Virtual Machine Scale Set) ou em um conjunto de máquinas virtuais.
Você pode usar o parâmetro *InstanceId* para selecionar um conjunto de máquinas virtuais.

## Exemplos

### Exemplo 1: Iniciar um conjunto específico de máquinas virtuais no VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

Esse comando inicia um conjunto específico de máquinas virtuais especificada pela matriz de cadeia de caracteres de ID de instância que pertencem ao VMSS chamado ContosoVMSS.

### Exemplo 2: Iniciar todas as máquinas virtuais no VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

Esse comando inicia todas as máquinas virtuais que pertencem ao VMSS chamado ContosoVMSS.
