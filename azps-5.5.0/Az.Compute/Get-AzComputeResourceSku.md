# Get-AzComputeResourceSku

## Sinopse
Listar todas as SKIs de recursos de computação

## Sintaxe

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Listar todas as SKIs de recursos de computação

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzComputeResourceSku "westus";
```

Listar todas as SKIs de recursos de computação na região oeste dos EUA
