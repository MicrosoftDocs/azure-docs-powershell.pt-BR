# Disable-AzVmssDiskEncryption

## Sinopse
Desabilita a criptografia de disco em um conjunto de escala de VM.

## Sintaxe

```
Disable-AzVmssDiskEncryption [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [[-ExtensionName] <String>] [-VolumeType <String>] [-ForceUpdate] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Desabilita a criptografia de disco em um conjunto de escala de VM.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Disable-AzVmssDiskEncryption -ResourceGroupName "Group001" -VMScaleSetName "VMSS001"
```

Desabilita a criptografia de disco no conjunto de escala VM chamado VMSS001 que pertence ao grupo de recursos chamado Group001.

### Exemplo 2

Desabilita a criptografia de disco em um conjunto de escala de VM. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Disable-AzVmssDiskEncryption -ResourceGroupName 'Group001' -VMScaleSetName 'VMSS001' -VolumeType OS
```
