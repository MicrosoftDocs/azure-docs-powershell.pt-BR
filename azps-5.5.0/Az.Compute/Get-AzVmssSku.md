# Get-AzVmssSku

## Sinopse
Obtém as SKUs disponíveis para o VMSS.

## Sintaxe

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzVmssSku** obtém as SKUs disponíveis para o VMSS (Virtual Machine Scale Set).

## Exemplos

### Exemplo 1: Obter todas as SKUs disponíveis no VMSS
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Esse comando obtém todas as SKUs disponíveis do VMSS chamado ContosoVMSS que pertence ao grupo de recursos chamado ContosoGroup.
