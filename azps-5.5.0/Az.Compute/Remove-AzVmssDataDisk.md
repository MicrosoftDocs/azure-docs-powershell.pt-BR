# Remove-AzVmssDataDisk

## Sinopse
Remove um disco de dados do VMSS.

## Sintaxe

### NameParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InstoTomParameterSet
```
Remove-AzVmssDataDisk [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Lun] <Int32>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzVmssDataDisk** remove um disco de dados da instância VMSS (Virtual Machine Scale Set).

## Exemplos

### Exemplo 1
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Name 'DataDisk1'
```

Esse comando remove o disco de dados chamado 'DataDisk1' do objeto VMSS.

### Exemplo 2
```
PS C:\> Remove-AzVmssDataDisk -VirtualMachineScaleSet $vmss -Lun 0
```

Esse comando remove o disco de dados de OBJECT 0 do objeto VMSS.
