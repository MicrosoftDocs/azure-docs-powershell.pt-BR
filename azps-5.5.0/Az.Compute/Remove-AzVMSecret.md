# Remove-AzVMSecret

## Sinopse
Remove (a) segredo(s) de um objeto de máquina virtual

## Sintaxe

```
Remove-AzVMSecret [-VM] <PSVirtualMachine> [[-SourceVaultId] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Remove-AzVMSecret cmdlet remove (a) segredo(s) de um objeto virtual de máquina.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzVM -ResourceGroupName "rg1" -Name "vm1" | Remove-AzVMSecret | Update-AzVM
```

Remove todos os segredos de uma máquina virtual "vm1" no grupo de recursos "rg1" e atualiza o VM
