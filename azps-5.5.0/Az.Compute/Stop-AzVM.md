# Stop-AzVM

## Sinopse
Interrompe uma máquina virtual do Azure.

## Sintaxe

### ResourceGroupNameParameterSetName (Padrão)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Stop-AzVM** interrompe uma máquina virtual do Azure.

## Exemplos

### Exemplo 1: Parar uma máquina virtual
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Esse comando interrompe a máquina virtual chamada VirtualMachine07 no ResourceGroup11.
