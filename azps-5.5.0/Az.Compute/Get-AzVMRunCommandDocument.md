# Get-AzVMRunCommandDocument

## Sinopse
Obter um documento de comando executar.

## Sintaxe

```
Get-AzVMRunCommandDocument [-Location] <String> [[-CommandId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Obter um documento de comando executar.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus' -CommandId 'RunPowerShellScript'
```

Obtém um documento de comando de executar específico para 'RunPowerShellScript' em 'westus'.

### Exemplo 2
```
PS C:\> Get-AzVMRunCommandDocument -Location 'westus'
```

Lista todos os comandos de executar disponíveis em 'westus'.
