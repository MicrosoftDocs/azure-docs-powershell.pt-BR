# Get-AzVMAEMExtension

## Sinopse
Obtém informações sobre a extensão AEM.

## Sintaxe

```
Get-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzVMAEMExtension obtém** informações sobre a extensão Azure Enhanced Monitoring (AEM).

## Exemplos

### Exemplo 1: Obter a extensão AEM
```
PS C:\> Get-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Esse comando obtém informações sobre a extensão AEM da máquina virtual chamada contoso-server.
