# Get-AzVMDscExtension

## Sinopse
Obtém as configurações da extensão DSC em uma determinada máquina virtual.

## Sintaxe

### GetDscExtension (Padrão)
```
Get-AzVMDscExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### VMParameterSet
```
Get-AzVMDscExtension [-Status] [-VM <PSVirtualMachine>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzVMDscExtension** obtém as configurações da extensão de Estado Desejado (DSC) em uma determinada máquina virtual.

## Exemplos

### Exemplo 1: Obter as configurações de uma extensão DSC
```
PS C:\> Get-AzVMDscExtension -ResourceGroupName "ResourceGroup002" -VMName "VM07" -Name "DSC"
```

Esse comando obtém as configurações da extensão chamada DSC na máquina virtual chamada VM07.
