# Test-AzVMAEMExtension

## Sinopse
Verifica a configuração da extensão AEM.

## Sintaxe

```
Test-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-OSType] <String>]
 [[-WaitTimeInMinutes] <Int32>] [-SkipStorageCheck] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Test-AzVMAEMExtension** verifica a configuração da extensão Azure Enhanced Monitoring (AEM).
A extensão AEM coleta os dados de desempenho.
Este cmdlet verifica se os dados de desempenho estão disponíveis.

## Exemplos

### Exemplo 1: Verificar a configuração da extensão AEM
```
PS C:\> Test-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Esse comando verifica a configuração da extensão AEM para a máquina virtual chamada contoso-server.
