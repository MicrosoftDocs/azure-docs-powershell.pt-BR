# Get-AzVMImagePublisher

## Sinopse
Obtém os editores do VMImage.

## Sintaxe

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzVMImagePublisher** obtém os editores do VMImage.

## Exemplos

### Exemplo 1: Obter os editores do VMImage para uma região
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

Esse comando obtém os editores de instâncias do VMImage para a região central dos EUA em seu perfil do Azure.
