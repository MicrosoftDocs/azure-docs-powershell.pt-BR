# New-AzADDomainServiceReplicaSet

## Sinopse
Criar um objeto na memória para o ReplicaSet

## Sintaxe

```
New-AzADDomainServiceReplicaSet [-Location <String>] [-SubnetId <String>] [<CommonParameters>]
```

## Descrição
Criar um objeto na memória para o ReplicaSet
