## Descrição
O cmdlet **Get-AzManagementGroupDeploymentWhatIfResult** obtém o modelo ARM What-If resultado para uma implantação de modelo no escopo do grupo de gerenciamento especificado. Ela retorna uma lista de alterações indicando quais recursos serão atualizados se a implantação for aplicada sem fazer alterações nos recursos reais. Para especificar o formato do resultado de retorno, use o parâmetro *Formatação* De resultado.

## Exemplos

### Exemplo 1: Obter um What-If no escopo do grupo de gerenciamento
```powershell
PS C:\> Get-AzManagementGroupDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -ManagementGroupId "myManagementGroup" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "FullResourcePayloads"
```

Esse comando obtém um What-If no escopo do grupo de gerenciamento usando um arquivo de modelo personalizado e um arquivo de parâmetro no disco.
O comando usa o *parâmetro Local* para especificar onde armazenar os dados de implantação.
O comando usa o parâmetro *ManagementGroupId* para especificar o grupo de gerenciamento no qual o modelo será implantado.
O comando usa o parâmetro *TemplateFile* para especificar um arquivo de modelo.
O comando usa o parâmetro *TemplateParameterFile* para especificar um arquivo de parâmetro de modelo.
O comando usa o parâmetro *ResultFormat* para definir o What-If para incluir cargas completas de recursos.

### Exemplo 2: Obter um What-If no escopo do grupo de gerenciamento com ResourceIdOnly
```powershell
PS C:\> Get-AzManagementGroupDeploymentWhatIfResult `
    -DeploymentName "deploy-01" `
    -Location "West US" `
    -ManagementGroupId "myManagementGroup" `
    -TemplateFile "D:\Azure\Templates\ServiceTemplate.json" `
    -TemplateParameterFile "D:\Azure\Templates\ServiceParameters.json" `
    -ResultFormat "ResourceIdOnly"
```

Esse comando obtém um What-If no escopo do grupo de gerenciamento usando um arquivo de modelo personalizado e um arquivo de parâmetro no disco.
O comando usa o *parâmetro Local* para especificar onde armazenar os dados de implantação.
O comando usa o parâmetro *ManagementGroupId* para especificar o grupo de gerenciamento no qual o modelo será implantado.
O comando usa o parâmetro *TemplateFile* para especificar um arquivo de modelo.
O comando usa o parâmetro *TemplateParameterFile* para especificar um arquivo de parâmetro de modelo.
O comando usa o parâmetro *ResultFormat* para definir o What-If para conter somente IDs de recurso.
