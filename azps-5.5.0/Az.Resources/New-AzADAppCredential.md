# New-AzADAppCredential

## Sinopse
Adiciona uma credencial a um aplicativo existente.

## Sintaxe

### ApplicationObjectIdWithPasswordParameterSet (Padrão)
```
New-AzADAppCredential -ObjectId <String> -Password <SecureString> [-StartDate <DateTime>] [-EndDate <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]

## Descrição
O New-AzADAppCredential cmdlet pode ser usado para adicionar uma nova credencial ou para rolar credenciais para um aplicativo.
O aplicativo é identificado fornecendo a ID do objeto do aplicativo ou a ID do aplicativo.

## Exemplos

### Exemplo 1 - Criar uma nova credencial de aplicativo usando uma senha

```
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> New-AzADAppCredential -ObjectId 1f89cf81-0146-4f4e-beae-2007d0668416 -Password $SecureStringPassword
```

Uma nova credencial de senha é adicionada ao aplicativo existente com a ID do objeto '1f89cf81-0146-4f4e-beae-2007d0668416'.

### Exemplo 2 - Criar uma nova credencial de aplicativo usando um certificado

```
PS C:\> $cer = New-Object System.Security.Cryptography.X509Certificates.X509Certificate2
PS C:\> $cer.Import("C:\myapp.cer")
PS C:\> $binCert = $cer.GetRawCertData()
PS C:\> $credValue = [System.Convert]::ToBase64String($binCert)
PS C:\> New-AzADAppCredential -ApplicationId 4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58 -CertValue $credValue -StartDate $cer.NotBefore -EndDate $cer.NotAfter
```

O certificado X509 público da base64 fornecido ("myapp.cer") é adicionado ao aplicativo existente com a ID do aplicativo '4589cd6b-3d79-4bb4-93b8-a0b99f3bfc58'.

### Exemplo 3 - Criar uma nova credencial de aplicativo usando a canalização

```
PS C:\> $SecureStringPassword = ConvertTo-SecureString -String "password" -AsPlainText -Force
PS C:\> Get-AzADApplication -ObjectId 1f89cf81-0146-4f4e-beae-2007d0668416 | New-AzADAppCredential -Password $SecureStringPassword
```

Obtém o aplicativo com a ID do objeto '1f89cf81-0146-4f4e-beae-2007d0668416' e os canos que para o New-AzADAppCredential para criar uma nova credencial de aplicativo para esse aplicativo com a senha determinada.
