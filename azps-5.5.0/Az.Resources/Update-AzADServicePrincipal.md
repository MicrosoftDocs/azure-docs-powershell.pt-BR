# Update-AzADServicePrincipal

## Sinopse
Atualiza uma entidade de serviço existente do Azure Active Directory.

## Sintaxe

### SpObjectIdWithDisplayNameParameterSet (Padrão)
```
Update-AzADServicePrincipal -ObjectId <String> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SpApplicationIdWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -ApplicationId <Guid> [-Homepage <String>] [-IdentifierUri <String[]>]
 [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SPNWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -ServicePrincipalName <String> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectWithDisplayNameParameterSet
```
Update-AzADServicePrincipal -InputObject <PSADServicePrincipal> [-DisplayName <String>] [-Homepage <String>]
 [-IdentifierUri <String[]>] [-KeyCredential <KeyCredential[]>] [-PasswordCredential <PasswordCredential[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Atualiza uma entidade de serviço existente do Azure Active Directory. Para atualizar as credenciais associadas a esta entidade de serviço, use New-AzADSpCredential cmdlet. Para atualizar as propriedades associadas ao aplicativo subjacente, use Update-AzADApplication cmdlet.

## Exemplos

### Exemplo 1: Atualizar o nome de exibição de uma entidade de serviço

```powershell
PS C:\> Update-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 -DisplayName MyNewDisplayName
```

Atualiza o nome de exibição da entidade de serviço com a ID do objeto '784136ca-3ae2-4fdd-a388-89d793e7c780' como 'MyNewDisplayName'.

### Exemplo 2: Atualizar o nome de exibição de uma entidade de serviço usando a piping

```powershell
PS C:\> Get-AzADServicePrincipal -ObjectId 784136ca-3ae2-4fdd-a388-89d793e7c780 | Update-AzADServicePrincipal -DisplayName MyNewDisplayName
```

Obtém a entidade de serviço com a ID do objeto '784136ca-3ae2-4fdd-a388-89d793e7c780' e passa para o cmdlet Update-AzADServicePrincipal para atualizar o nome de exibição da entidade de serviço para "MyNewDisplayName".

### Exemplo 3

Atualiza uma entidade de serviço existente do Azure Active Directory. (gerado automaticamente)
