# Remove-AzDeploymentManagerArtifactSource

## Sinopse
Exclui a fonte de artefato especificada.

## Sintaxe

### Interativo (Padrão)
```
Remove-AzDeploymentManagerArtifactSource [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceid
```
Remove-AzDeploymentManagerArtifactSource [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Remove-AzDeploymentManagerArtifactSource [-InputObject] <PSArtifactSource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDeploymentManagerArtifactSource** exclui uma fonte de artefato Especifique a fonte de artefato pelo nome e nome do grupo de recursos. Como alternativa, você pode fornecer o objeto ArtifactSource ou o ResourceId.

## Exemplos

### Exemplo 1: Excluir uma fonte de artefato
### Exemplo 1
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -ResourceGroupName "ContosoResourceGroup" -Name "ContosoArtifactSource"
```

Esse comando exclui uma fonte de artefato chamada ContosoArtifactSource no ContosoResourceGroup.

### Exemplo 2: Excluir uma fonte de artefato usando o identificador de recurso
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/artifactSources/ContosoArtifactSource"
```

Esse comando exclui uma fonte de artefato chamada ContosoArtifactSource no ContosoResourceGroup.

### Exemplo 3: Excluir uma fonte de artefato usando um objeto
```powershell
PS C:\> Remove-AzDeploymentManagerArtifactSource -InputObject $artifactSourceObject
```

Esse comando exclui uma fonte de artefato cujo nome e Grupo de Recursos corresponderem às propriedades Nome e NomeDoGrupo de Recursos da $artifactSourceObject, respectivamente.
