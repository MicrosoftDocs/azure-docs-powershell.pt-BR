# Remove-AzDeploymentManagerRollout

## Sinopse
Exclui a adoção.

## Sintaxe

### Interativo (Padrão)
```
Remove-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceid
```
Remove-AzDeploymentManagerRollout [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Remove-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDeploymentManagerRollout** exclui uma rollout em um estado de terminal.
Especifique a distribuição pelo nome e nome do grupo de recursos. Como alternativa, você pode fornecer o objeto Distribuição ou o ResourceId.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Esse comando exclui uma apostila chamada ContosoRollout no Grupo ContosoResource.

### Exemplo 2: Excluir uma distribuição usando o identificador de recurso
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Esse comando exclui uma apostila chamada ContosoRollout no Grupo ContosoResource.

### Exemplo 3: Excluir uma apostila usando o objeto de adoção.
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Esse comando exclui uma distribuição cujo nome e Grupo de Recursos corresponderem às propriedades Nome e NomeDoGrupo de Recursos da $rolloutObject, respectivamente.
