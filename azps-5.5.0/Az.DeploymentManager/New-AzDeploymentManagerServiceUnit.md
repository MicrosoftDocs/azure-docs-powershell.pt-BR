# New-AzDeploymentManagerServiceUnit

## Sinopse
Cria uma unidade de serviço sob a topologia de serviço e serviço especificada.

## Descrição
O cmdlet **New-AzDeploymentManagerServiceUnit** cria um serviço sob um serviço em uma topologia de serviço e retorna um objeto que representa essa unidade de serviço.
Especifique a unidade de serviço pelo nome, nome do serviço, topologia de serviço em que ela está e o nome do grupo de recursos. 

O cmdlet retorna um objeto ServiceUnit. Você pode modificar esse objeto localmente e aplicar alterações ao serviço usando o cmdlet Set-AzDeploymentManagerService usuário.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzDeploymentManagerServiceUnit -ResourceGroupName ContosoResourceGroup -ServiceTopologyName ContosoServiceTopology -ServiceName ContosoService2 -Name ContosoService2Storage -Location "Central US" -TargetResourceGroup service2ResourceGroup -DeploymentMode Incremental -TemplateArtifactSourceRelativePath "Templates/Service2.Storage.json" -ParametersArtifactSourceRelativePath "Parameters/Service2Storage.Parameters.json"
```

Este cmdlet cria uma nova unidade de serviço com o nome ContosoService2Storage no Grupo ContosoResource sob o serviço ContosoService2 em topologia ContosoServiceTopology, no local Central dos EUA. Os arquivos Modelo e parâmetros são definidos como caminhos relativos para o local de origem do artefato referenciado na Topologia de Serviço ContosoServiceTopology. Os recursos definidos neste modelo devem ser implantados no serviço de grupo de recursos de destino2ResourceGroup com o modo de implantação definido como Incremental.

### Exemplo 2
```powershell
PS C:\> New-AzDeploymentManagerServiceUnit -ResourceGroupName ContosoResourceGroup -ServiceTopologyName ContosoServiceTopology1 -ServiceName ContosoService2 -Name ContosoService2Storage -Location "Central US" -TargetResourceGroup service2ResourceGroup -DeploymentMode Complete -TemplateUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts/Templates/Service2.Storage.json?sasParameters" -ParametersUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts/Parameters/Service2Storage.Parameters.json?sasParameters"
```

Este cmdlet cria uma nova unidade de serviço com o nome ContosoService2Storage no Grupo ContosoResource sob o serviço ContosoService2 em topologia ContosoServiceTopology, no local Central dos EUA. As referências modelo e parâmetros são fornecidas como SAS Uri como fonte de artefato ResourceId não foi fornecida na Topologia de Serviço ContosoServiceTopology1. Os recursos definidos neste modelo devem ser implantados no serviço de grupo de recursos de destino2ResourceGroup com o modo de implantação definido como Concluído.
