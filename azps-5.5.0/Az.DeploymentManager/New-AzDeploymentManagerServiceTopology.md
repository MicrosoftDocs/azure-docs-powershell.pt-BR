# New-AzDeploymentManagerServiceTopology

## Sinopse
Cria uma topologia de serviço.

## Sintaxe

```
New-AzDeploymentManagerServiceTopology -ResourceGroupName <String> -Name <String> -Location <String>
 [-ArtifactSourceId <String>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzDeploymentManagerServiceTopology** cria uma topologia de serviço.

Você pode modificar o objeto ServiceTopology retornado localmente e aplicar alterações à topologia usando o cmdlet Set-AzDeploymentManagerServiceTopology.
O objeto retornado 

O objeto retornado tem um campo ResourceId que pode ser referenciado em um recurso de distribuição para indicar que os serviços declarados nesta topologia de serviço seriam implantados na distribuição.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzDeploymentManagerServiceTopology -ResourceGroupName ContosoResourceGroup -Name ContosoServiceTopology -Location "Central US" -ArtifactSourceId "/subscriptions/XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/artifactSources/ContosoArtifactSource"
```

Este cmdlet cria uma nova topologia de serviço no grupo de recursos ContosoResourceGroup com o nome ContosoServiceTopology e no local Central dos EUA. A fonte de artefato ResourceId indica que os artefatos necessários para as definições da unidade de serviço nesta topologia precisam ser lidos a partir da fonte de artefato especificada.

### Exemplo 2
```powershell
PS C:\> New-AzDeploymentManagerServiceTopology -ResourceGroupName ContosoResourceGroup -Name ContosoServiceTopology -Location "Central US"
```

Este cmdlet cria uma nova topologia de serviço no grupo de recursos ContosoResourceGroup com o nome ContosoServiceTopology e no local Central dos EUA. A ausência de uma referência de origem de artefato indica que os artefatos necessários para as definições da unidade de serviço nesta topologia seriam fornecidos como URIs SAS absolutas na unidade de serviço.
