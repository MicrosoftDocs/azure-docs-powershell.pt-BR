# New-AzDeploymentManagerStep

## Sinopse
Cria uma etapa.

## Sintaxe

### Aguarde (Padrão)
```
New-AzDeploymentManagerStep -ResourceGroupName <String> -Name <String> -Location <String> -Duration <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### HealthCheckFile
```
New-AzDeploymentManagerStep -ResourceGroupName <String> -Name <String> -Location <String>
 -HealthCheckPropertiesFile <String> [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### HealthCheckObject
```
New-AzDeploymentManagerStep -ResourceGroupName <String> -Name <String> -Location <String>
 -HealthCheckProperties <PSHealthCheckStepProperties> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzDeploymentManagerStep** cria uma etapa de implantação que pode ser referenciada em implantações.
*Especifique o Nome,* *ResourceGroupName* e as propriedades necessárias.

Você pode modificar o objeto retornado localmente e aplicar as alterações à etapa usando o cmdlet Set-AzDeploymentManagerStep texto.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep -Location "Central US" -Duration PT20M
```

Cria uma etapa no Grupo ContosoResource com o nome ContosoService1WaitStep com Central US como o local do recurso. A propriedade Duração fornece a duração que a adoção aguardará antes de executar a próxima etapa.
