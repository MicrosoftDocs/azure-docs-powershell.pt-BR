# Remove-AzDeploymentManagerStep

## Sinopse
Exclui a etapa.

## Sintaxe

### Interativo (Padrão)
```
Remove-AzDeploymentManagerStep [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Resourceid
```
Remove-AzDeploymentManagerStep [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Inputobject
```
Remove-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzDeploymentManagerStep** exclui uma etapa.
Especifique a etapa pelo nome e pelo nome do grupo de recursos. Como alternativa, você pode fornecer o objeto Step ou o ResourceId.

## Exemplos

### Exemplo 1: Remover uma etapa
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceGroupName ContosoResourceGroup -Name ContosoService1WaitStep
```

Esse comando exclui uma etapa chamada ContosoService1WaitStep em ContosoResourceGroup.

### Exemplo 2: Remover uma etapa usando o identificador de recurso
```powershell
PS C:\> Remove-AzDeploymentManagerStep -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/steps/ContosoService1WaitStep"
```

Esse comando exclui uma etapa chamada ContosoService1WaitStep em ContosoResourceGroup.

### Exemplo 3: Remover uma etapa usando um objeto retornado por New-AzDeploymentManagerStep
### Exemplo 1
```powershell
PS C:\> Remove-AzDeploymentManagerStep -InputObject $stepObject
```

 Esse comando exclui uma etapa cujo nome e Grupo de Recursos corresponderem às propriedades Nome e ResourceGroupName da $stepObject, respectivamente.
