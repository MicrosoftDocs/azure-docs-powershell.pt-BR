# Set-AzDeploymentManagerArtifactSource

## Sinopse
Atualiza a fonte de artefatos.

## Sintaxe

```
Set-AzDeploymentManagerArtifactSource [-InputObject] <PSArtifactSource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzDeploymentManagerArtifactSource** atualiza uma fonte de artefatos com o objeto de origem de artefato especificado.
O cmdlet retorna o objeto ArtifactSource atualizado.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzDeploymentManagerArtifactSource -InputObject $artifactSourceObject
```

Esse comando atualiza uma fonte de artefato cujo nome e Grupo de Recursos corresponderem às propriedades Nome e NomeDoGrupo de Recursos do $artifactSourceObject, respectivamente.
A fonte de artefatos seria atualizada para as propriedades definidas no $artifactSourceObject.
