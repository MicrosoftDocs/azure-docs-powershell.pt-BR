# New-AzApplicationGatewayRewriteRuleCondition

## Sinopse
Adiciona uma condição à Regra Reescrita para um gateway de aplicativo.

## Sintaxe

```
New-AzApplicationGatewayRewriteRuleCondition -Variable <String> [-Pattern <String>] [-IgnoreCase] [-Negate]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
**O cmdlet AzApplicationGatewayRewriteRuleCondition** cria uma condição de regra de reescrita para um gateway de aplicativo do Azure.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayRewriteRuleCondition -Variable "var_request_uri" -Pattern "http" -IgnoreCase
PS C:\> $condition

Variable   : var_request_uri
Pattern    : http
IgnoreCase : True
Negate     : False

PS C:\> $condition | Format-Table

Variable        Pattern IgnoreCase Negate
--------        ------- ---------- ------
var_request_uri http          True  False
```
Esse comando cria uma condição em uma regra de reescrita e armazena o resultado na variável chamada $condition.
