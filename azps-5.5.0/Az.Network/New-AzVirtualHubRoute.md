# New-AzVirtualHubRoute

## Sinopse
Cria um objeto de Rota do Hub Virtual do Azure.

## Sintaxe

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Cria um objeto de Rota do Hub Virtual do Azure.

## Exemplos

### Exemplo 1

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

O acima criará um objeto de rota do hub virtual que pode ser incluído na tabela de rota do hub virtual.

A rota do hub virtual é um objeto na memória que pode ser usado para criar um objeto VirtualHubRouteTable.
