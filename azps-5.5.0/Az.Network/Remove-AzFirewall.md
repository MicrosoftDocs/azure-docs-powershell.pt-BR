# Remove-AzFirewall

## Sinopse
Remover um Firewall.

## Sintaxe

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzFirewall** remove um Firewall do Azure.

## Exemplos

### Exemplo 1: Criar e excluir um Firewall
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Este exemplo cria um Firewall e o exclui. Para suprimir o prompt ao excluir o Firewall, use o sinalizador -Force.
