# Get-AzApplicationGatewaySslPredefinedPolicy

## Sinopse
Obtém políticas SSL predefinidas fornecidas pelo Gateway de Aplicativo.

## Sintaxe

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApplicationGatewaySslPredefinedPolicy** obtém Políticas SSL predefinidas fornecidas pelo Gateway de Aplicativo.
