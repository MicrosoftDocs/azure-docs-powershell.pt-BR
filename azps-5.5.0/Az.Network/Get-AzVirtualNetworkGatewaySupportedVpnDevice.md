# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## Sinopse
Este commandlet retorna uma lista de marcas de dispositivo VPN com suporte, modelos e versões de firmware.

## Sintaxe

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
Este commandlet retorna uma lista de marcas de dispositivo VPN com suporte, modelos e versões de firmware.

## Exemplos

### Exemplo 1
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

Retorna uma lista de marcas de dispositivo VPN com suporte, modelos e versões de firmware:
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
