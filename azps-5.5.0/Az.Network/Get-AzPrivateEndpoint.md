# Get-AzPrivateEndpoint

## Sinopse
Obter um ponto de extremidade particular

## Sintaxe

### NoExpand (Padrão)
```
Get-AzPrivateEndpoint [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Expandir
```
Get-AzPrivateEndpoint -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzPrivateEndpoint** obtém um ou mais pontos de extremidade particulares.
