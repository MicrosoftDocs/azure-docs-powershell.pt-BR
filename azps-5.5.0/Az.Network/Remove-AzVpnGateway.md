# Remove-AzVpnGateway

## Sinopse
O Remove-AzVpnGateway cmdlet remove um gateway VPN do Azure. Este é um gateway específico para a conectividade definida pelo software WAN virtual do Azure.

## Sintaxe

### ByVpnGatewayName (Padrão)
```
Remove-AzVpnGateway -ResourceGroupName <String> -Name <String> [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayObject
```
Remove-AzVpnGateway -InputObject <PSVpnGateway> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVpnGatewayResourceId
```
Remove-AzVpnGateway -ResourceId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Remove-AzVpnGateway cmdlet remove um gateway VPN do Azure. Este é um gateway específico para a conectividade definida pelo software WAN virtual do Azure.

## Exemplos

### Exemplo 1

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Remove-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -Passthru
```

Este exemplo cria um grupo De recursos, WAN Virtual, Hub Virtual, gateway VPN escalável na Central dos EUA e o exclui imediatamente. Para suprimir o prompt ao excluir o Gateway Virtual, use o sinalizador -Force.
Isso excluirá o VpnGateway e todas as VpnConnections anexadas a ele.

### Exemplo 2

```powershell
PS C:\> New-AzResourceGroup -Location "West US" -Name "testRG"
PS C:\> $virtualWan = New-AzVirtualWan -ResourceGroupName testRG -Name myVirtualWAN -Location "West US"
PS C:\> $virtualHub = New-AzVirtualHub -VirtualWan $virtualWan -ResourceGroupName "testRG" -Name "westushub" -AddressPrefix "10.0.0.1/24"
PS C:\> New-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" -VirtualHubId $virtualHub.Id -BGPPeeringWeight 10 -VpnGatewayScaleUnit 2
PS C:\> Get-AzVpnGateway -ResourceGroupName "testRG" -Name "testvpngw" | Remove-AzVpnGateway-Passthru
```

Este exemplo cria um grupo De recursos, WAN Virtual, Hub Virtual, gateway VPN escalável na Central dos EUA e o exclui imediatamente. Essa exclusão acontece usando a piping do PowerShell, que usa o objeto VpnGateway retornado pelo comando Get-AzVpnGateway usuário.
Para suprimir o prompt ao excluir o Gateway Virtual, use o sinalizador -Force.
Isso excluirá o VpnGateway e todas as VpnConnections anexadas a ele.
