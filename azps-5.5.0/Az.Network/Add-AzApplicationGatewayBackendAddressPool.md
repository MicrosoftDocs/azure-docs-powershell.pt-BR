# Add-AzApplicationGatewayBackendAddressPool

## Sinopse
Adiciona um pool de endereços back-end a um gateway de aplicativo.

## Sintaxe

```
Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-BackendIPAddresses <String[]>] [-BackendFqdns <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Add-AzApplicationGatewayBackendAddressPool** adiciona um pool de endereços back-end a um gateway de aplicativo.
Um endereço back-end pode ser especificado usando um endereço IP, um nome de domínio totalmente qualificado (FQDN) ou IDs de configuração IP.

## Exemplos

### Exemplo 1: Adicionar um pool de endereços back-end usando um FQDN de servidor back-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway $AppGw -Name "Pool02" -BackendFqdns "contoso1.com", " contoso1.com"
```

O primeiro comando obtém o gateway de aplicativo chamado ApplicationGateway01 no grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw usuário. O segundo comando adiciona o pool de endereços back-end do gateway de aplicativo armazenado $AppGw usando FQDNs.

### Exemplo 2: Adicionar um pool de endereços back-end usando endereços IP do servidor back-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayBackendAddressPool -ApplicationGateway $ AppGw -Name "Pool02" -BackendIPAddresses "10.10.10.10", "10.10.10.11"
```

O primeiro comando obtém o gateway de aplicativo chamado ApplicationGateway01 no grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw usuário. O segundo comando adiciona o pool de endereços back-end do gateway de aplicativo armazenado no $AppGw usando endereços IP.
