# Remove-AzApplicationGatewayBackendHttpSetting

## Sinopse
Remove as configurações HTTP de back-end de um gateway de aplicativo.

## Sintaxe

```
Remove-AzApplicationGatewayBackendHttpSetting -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Remove-AzApplicationGatewayBackendHttpSetting cmdlet remove as configurações http de protocolo HTTP de back-end de um gateway de aplicativo do Azure.

## Exemplos

### Exemplo 1: Remover configurações HTTP de back-end de um gateway de aplicativo
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayBackendHttpSetting -ApplicationGateway $AppGw -Name "BackEndSetting02"
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando remove a configuração HTTP de back-end chamada BackEndSetting02 do gateway de aplicativo armazenado no $AppGw.
