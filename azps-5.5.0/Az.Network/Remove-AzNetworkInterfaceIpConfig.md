# Remove-AzNetworkInterfaceIpConfig

## Sinopse
Remove uma configuração IP de interface de rede de uma interface de rede.

## Sintaxe

```
Remove-AzNetworkInterfaceIpConfig -Name <String> -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzNetworkInterfaceIpConfig** remove uma configuração IP da interface de rede de uma interface de rede do Azure.

## Exemplos

### Exemplo 1: Excluir uma configuração IP de uma interface de rede
```powershell
$nic = Get-AzNetworkInterface -Name mynic -ResourceGroupName myrg

Remove-AzNetworkInterfaceIpConfig -Name IPConfig-1 -NetworkInterface $nic

Set-AzNetworkInterface -NetworkInterface $nic
```

O primeiro comando obtém uma interface de rede chamada mynic e a armazena na variável $nic. O segundo comando remove a configuração IP chamada IPConfig-1 associada a esta interface de rede. O terceiro comando define as alterações feitas na interface de rede.
