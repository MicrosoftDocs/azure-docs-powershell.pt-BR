# Remove-AzLocalNetworkGateway

## Sinopse
Exclui um Gateway de Rede Local

## Sintaxe

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O Gateway de Rede Local é o objeto que representa seu dispositivo VPN local.
O cmdlet **Remove-AzLocalNetworkGateway** exclui o objeto que representa seu gateway local com base no Nome e Nome do Grupo de Recursos.

## Exemplos

### Exemplo 1: Excluir um Gateway de Rede Local
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Exclui o objeto do Gateway de Rede Local com o nome "myLocalGW" no grupo de recursos "myRG" Observação: Primeiro, você deve excluir todas as conexões com o Gateway de Rede Local usando o cmdlet **Remove-AzVirtualNetworkGatewayConnection.**
