# Set-AzApplicationGatewayTrustedClientCertificate

## Sinopse
Modifica a cadeia de certificados ca de cliente confiável de um gateway de aplicativo.

## Sintaxe

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzApplicationGatewayTrustedClientCertificate** modifica a cadeia de certificados ca de cliente confiável de um gateway de aplicativo.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Cenários de exemplo acima mostram como atualizar um objeto de cadeia de certificados de cliente confiável existente. O primeiro comando obtém um gateway de aplicativo e o armazena na variável $gw aplicativo. O segundo comando modifica o objeto de cadeia de certificados da CA de cliente confiável existente com um novo arquivo de cadeia de certificados da CA. O terceiro comando atualiza o gateway do aplicativo no Azure.
