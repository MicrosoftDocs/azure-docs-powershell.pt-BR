# Test-AzPrivateIPAddressAvailability

## Sinopse
Teste a disponibilidade de um endereço IP particular em uma rede virtual.

## Sintaxe

### TestByResource
```
Test-AzPrivateIPAddressAvailability -VirtualNetwork <PSVirtualNetwork> -IPAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByResourceId
```
Test-AzPrivateIPAddressAvailability -ResourceGroupName <String> -VirtualNetworkName <String>
 -IPAddress <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Test-AzPrivateIPAddressAvailability** testa se um endereço IP particular especificado está disponível em uma rede virtual.
Esse cmdlet retornará uma lista de endereços IP particulares disponíveis se o endereço IP particular solicitado for retirado.

## Exemplos

### Exemplo 1: Testar se um endereço IP está disponível usando o pipeline
```
PS C:\>Get-AzVirtualNetwork -Name $vnetName -ResourceGroupName $rgname | Test-AzPrivateIPAddressAvailability -IPAddress "10.0.1.10"
```

Esse comando obtém uma rede virtual e usa o operador de pipeline para **passá-lo para Test-AzPrivateIPAddressAvailability,** que testa se o endereço IP particular especificado está disponível.
