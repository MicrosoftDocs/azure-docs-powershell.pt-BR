# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## Sinopse
Cria a entrada RuleGroupOverride em ManagedRuleSets para a política de firewall.

## Sintaxe

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O **New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** cria uma entrada ruleGroupOverride em um managedRuleSet para uma política de firewall.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Cria uma entrada RuleGroupOverride com nome de grupo como $ruleName e Regras como $rule 1, $rule 2. Atribui o mesmo ao $overrideEntry
