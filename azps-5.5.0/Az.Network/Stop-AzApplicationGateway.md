# Stop-AzApplicationGateway

## Sinopse
Interrompe um gateway de aplicativo

## Sintaxe

```
Stop-AzApplicationGateway -ApplicationGateway <PSApplicationGateway> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **cmdlet Stop-AzApplicationGateway** interrompe um gateway de aplicativo.

## Exemplos

### Exemplo 1: Parar um gateway de aplicativo
```
PS C:\>Stop-AzApplicationGateway -ApplicationGateway $AppGw
```

Esse comando interrompe o gateway de aplicativo armazenado na variável $AppGw dados.
