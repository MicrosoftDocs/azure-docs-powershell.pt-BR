# Remove-AzPrivateEndpoint

## Sinopse
Remove um ponto de extremidade particular.

## Sintaxe

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Remove-AzPrivateEndpoint** remove um ponto de extremidade particular. 

## Exemplos

### Exemplo 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

Este exemplo remove um ponto de extremidade particular chamado MyPrivateEndpoint1.
