# Remove-AzApplicationGatewayFrontendIPConfig

## Sinopse
Remove uma configuração IP front-end de um gateway de aplicativo.

## Sintaxe

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzApplicationGatewayFrontendIPConfig** remove o IP frontend de um gateway de aplicativo do Azure.

## Exemplos

### Exemplo 1: Remover uma configuração ip front-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 e o armazena na variável $AppGw aplicativo.
O segundo comando remove a configuração IP de front-end chamada FrontEndIP02 do gateway de aplicativo armazenado no $AppGw.
