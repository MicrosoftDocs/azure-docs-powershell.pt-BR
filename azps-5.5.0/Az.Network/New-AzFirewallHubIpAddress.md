# New-AzFirewallHubIpAddress

## Sinopse
Endereços ip associados ao firewall no hub virtual

## Sintaxe

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Endereços IP associados ao firewall no hub virtual. Eles podem ser endereços públicos e privados

## Exemplos

### Exemplo 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

Este exemplo cria um objeto de endereço Ip do Hub com uma contagem de 2 IPs públicos. O objeto HubIPAddress éssociado ao firewall no hub virtual.
