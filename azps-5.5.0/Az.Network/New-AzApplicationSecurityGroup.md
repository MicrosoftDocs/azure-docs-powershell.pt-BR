# New-AzApplicationSecurityGroup

## Sinopse
Cria um grupo de segurança de aplicativos.

## Sintaxe

```
New-AzApplicationSecurityGroup -ResourceGroupName <String> -Name <String> -Location <String> [-Tag <Hashtable>]
 [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzApplicationSecurityGroup** cria um grupo de segurança de aplicativos.

## Exemplos

### Exemplo 1
```
PS C:\> New-AzApplicationSecurityGroup -ResourceGroupName "MyResourceGroup" -Name "MyApplicationSecurityGroup" -Location "West US"
```

Este exemplo cria um grupo de segurança de aplicativos sem associações. Depois de criada, as configurações de IP na interface de rede podem ser incluídas no grupo. As regras de segurança também podem se referir ao grupo como suas fontes ou destinos.
