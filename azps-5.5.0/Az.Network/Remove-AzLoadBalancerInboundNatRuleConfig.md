# Remove-AzLoadBalancerInboundNatRuleConfig

## Sinopse
Remove uma configuração de regra NAT de entrada de um balanceador de carga.

## Sintaxe

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzLoadBalancerInboundNatRuleConfig** remove uma configuração de regra nat (conversão de endereço de rede de entrada) de um balanceador de carga do Azure.

## Exemplos

### 1: Excluir uma regra NAT de entrada de um balanceador de carga do Azure
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

O primeiro comando carrega um balanceador de carga já existente chamado "mylb" e o armazena na variável $load balanceador. O segundo comando remove a regra NAT de entrada associada a esse balanceador de carga.
