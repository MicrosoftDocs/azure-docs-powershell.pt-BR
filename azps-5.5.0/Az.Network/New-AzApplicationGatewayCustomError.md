# New-AzApplicationGatewayCustomError

## Sinopse
Cria um erro personalizado com código de status http e url de página de erro personalizada 

## Sintaxe

```
New-AzApplicationGatewayCustomError -StatusCode <String> -CustomErrorPageUrl <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzApplicationGatewayCustomError** cria um erro personalizado.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $customError403Url = "https://mycustomerrorpages.blob.core.windows.net/errorpages/403-another.htm"
PS C:\> $ce = New-AzApplicationGatewayCustomError -StatusCode HttpStatus403 -CustomErrorPageUrl $customError403Url
```

Esse comando cria o erro personalizado do código de status http 403.
