# New-AzApplicationGatewayTrustedRootCertificate

## Sinopse
Cria um Certificado Raiz Confiável para um gateway de aplicativo.

## Sintaxe

```
New-AzApplicationGatewayTrustedRootCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzApplicationGatewayTrustedRootCertificate** cria um Certificado Raiz Confiável para um gateway de aplicativo do Azure.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $certFilePath = ".\rootCA.cer"
PS C:\> $trc = New-AzApplicationGatewayTrustedRootCertificate -Name "trc1" -CertificateFile $certFilePath
```

Esse comando cria um Certificado Raiz Confiável chamado Lista "trc1" e armazena o resultado na variável chamada $trc.
