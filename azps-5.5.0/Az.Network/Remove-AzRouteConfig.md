# Remove-AzRouteConfig

## Sinopse
Remove uma rota de uma tabela de rota.

## Sintaxe

```
Remove-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzRouteConfig** remove uma rota de uma tabela de rota do Azure.

## Exemplos

### Exemplo 1: Remover uma rota
```
PS C:\>Get-AzRouteTable -ResourceGroupName "ResourceGroup11" -Name "RouteTable01" | Remove-AzRouteConfig -Name "Route02" | Set-AzRouteTable

ResourceGroupName : ResourceGroup11

                    k/routeTables/RouteTable01
Etag              : W/"47099b62-60ec-4bc1-b87b-fad56cb8bed1"
ProvisioningState : Succeeded
Tags              : 
Routes            : [
                      {
                        "Name": "Route07",
                        "Etag": "W/\"47099b62-60ec-4bc1-b87b-fad56cb8bed1\"",
                        "Id": "/subscriptions/xxxx-xxxx-xxxx-xxxx/resourceGroups/ResourceGroup11/providers/Micro
                    soft.Network/routeTables/RouteTable01/routes/Route07",
                        "AddressPrefix": "10.1.0.0/16",
                        "NextHopType": "VnetLocal",
                        "NextHopIpAddress": null, 
                        "ProvisioningState": "Succeeded"
                      }
                    ]

```

Esse comando obtém a tabela de rota chamada Roteável01 usando o cmdlet **Get-AzRouteTable.**
O comando passa essa tabela para o cmdlet atual usando o operador de pipeline.
O cmdlet atual remove a rota chamada Rota02 e o passa o resultado para o cmdlet **Set-AzRouteTable,** que atualiza a tabela para refletir suas alterações.
A tabela não contém mais a rota chamada Rota02.
