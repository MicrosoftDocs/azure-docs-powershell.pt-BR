# Get-AzApplicationGatewayProbeConfig

## Sinopse
Obtém uma configuração de configuração de configuração de saúde existente de um Gateway de Aplicativo.

## Sintaxe

```
Get-AzApplicationGatewayProbeConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Get-AzApplicationGatewayProbeConfig cmdlet obtém uma configuração de configuração de problemas de saúde existente de um Gateway de Aplicativo.

## Exemplos

### Exemplo 1: Obter uma confirmação existente de um gateway de aplicativo
```
PS C:\>Get-AzApplicationGatewayProbeConfig -ApplicationGateway Gateway -Name "Probe02"
```

Esse comando obtém a confirmação de saúde chamada " Sor02 " do gateway de aplicativo chamado Gateway.
