# Get-AzApplicationGatewayAvailableSslOption

## Sinopse
Obtém todas as opções de SSL disponíveis para a política SSL do Gateway de Aplicativo.

## Sintaxe

```
Get-AzApplicationGatewayAvailableSslOption [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApplicationGatewayAvailableSslOption** obtém todas as opções de ssl disponíveis para a política ssl

## Exemplos

### Exemplo 1
```
PS C:\>$sslOptions = Get-AzApplicationGatewayAvailableSslOption
```

Esses comandos retornam todas as opções de SSL disponíveis para a política SSL.
