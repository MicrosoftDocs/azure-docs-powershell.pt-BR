# Get-AzApplicationGatewayTrustedClientCertificate

## Sinopse
Obtém a cadeia de certificados da CA de cliente confiável com um nome específico do Gateway de Aplicativos.

## Sintaxe

```
Get-AzApplicationGatewayTrustedClientCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Get-AzApplicationGatewayTrustedClientCertificate cmdlet obtém a cadeia de certificados de CA do cliente confiável com um nome específico do Gateway de Aplicativos.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClientCert = Get-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName
```

O primeiro comando obtém o Gateway de Aplicativos e o armazena $gw variável. O segundo comando obtém a cadeia de certificados de CA do cliente confiável com um nome especificado do Gateway de Aplicativo.
