# New-AzPrivateDnsZoneConfig

## Sinopse
Cria a configuração de zona DNS do grupo de zona dns particular.

## Sintaxe

```
New-AzPrivateDnsZoneConfig -Name <String> [-PrivateDnsZoneId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzPrivateDnsZoneConfig** permite que você crie um novo objeto de configuração de zona DNS que será definido no grupo de zona DNS.

## Exemplos

### Cria configuração de zona DNS
```powershell
PS C:\> $dnsZone = New-AzPrivateDnsZone -ResourceGroupName "rg" -Name "test.vault.azure.com"
PS C:\> $config = New-AzPrivateDnsZoneConfig -Name "test-vault-azure-com" -PrivateDnsZoneId $dnsZone.ResourceId
```

O exemplo acima cria zona DNS e, em seguida, cria a configuração de zona DNS. `New-AzPrivateDnsZone` cmdlet é comprovado pelo módulo Az.PrivateDns.
