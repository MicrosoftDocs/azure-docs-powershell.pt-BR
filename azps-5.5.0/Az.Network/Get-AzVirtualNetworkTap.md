# Get-AzVirtualNetworkTap

## Sinopse
Recebe um toque de rede virtual

## Sintaxe

### ListParameterSet (Padrão)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzVirtualNetworkTap** recebe um toque de rede virtual do Azure ou uma lista de toques de rede virtual do Azure em um grupo de recursos.

## Exemplos

### Exemplo 1: Toque em uma rede virtual
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Esse comando obtém uma referência de toque virtualNetwork para determinado "VirtualTap1" em "ResourceGroup1".

### Exemplo 2: Obter todos os toques de rede virtual usando filtragem
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Esse comando obtém todas as referências de toque do VirtualNetwork que começam com "VirtualTap".
