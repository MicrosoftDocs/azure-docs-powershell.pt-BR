# Remove-AzExpressRoutePortIdentity

## Sinopse
Remove uma identidade de um ExpressRoutePort.

## Sintaxe

```
Remove-AzExpressRoutePortIdentity -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzExpressRoutePortIdentity** remove a identidade de um objeto local do Azure ExpressRoutePort. Use **Remove-AzExpressRoutePort** para removê-lo do ExpressRoutePort.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $expressroutePort = Remove-AzExpressRoutePortIdentity -ExpressRoutePort $expressroutePort
```
