# Get-AzLoadBalancerFrontendIpConfig

## Sinopse
Obtém uma configuração IP front-end em um balanceador de carga.

## Sintaxe

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzLoadBalancerFrontendIpConfig obtém** uma configuração IP front-end ou uma lista de configurações de IP front-end em um balanceador de carga.

## Exemplos

### Exemplo 1: Obter uma configuração IP front-end em um balanceador de carga
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

O primeiro comando obtém o balanceador de carga chamado MyLoadBalancer e o armazena na variável $slb.
O segundo comando obtém a configuração ip front-end associada a esse balanceador de carga.
