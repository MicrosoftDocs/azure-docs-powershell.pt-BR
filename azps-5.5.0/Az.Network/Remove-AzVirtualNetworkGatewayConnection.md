# Remove-AzVirtualNetworkGatewayConnection

## Sinopse
Exclui uma conexão virtual de gateway de rede

## Sintaxe

```
Remove-AzVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
A Conexão virtual do Gateway de Rede é o objeto que representa o túnel IPsec (Site para Site ou Vnet-para-Vnet) conectado ao seu Gateway de Rede Virtual no Azure.
O cmdlet **Remove-AzVirtualNetworkGatewayConnection** exclui o objeto da sua conexão com base no Nome e no Nome do Grupo de Recursos.

## Exemplos

### Exemplo 1: Excluir uma conexão virtual de gateway de rede
```powershell
Remove-AzVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

Exclui o objeto da Conexão virtual de Gateway de Rede com o nome "meuTunnel" dentro do grupo de recursos "myRG"
