# New-AzFirewallPolicyNetworkRule

## Sinopse
Criar uma nova Regra de Rede de Política de Firewall do Azure

## Sintaxe

```
New-AzFirewallPolicyNetworkRule -Name <String> [-Description <String>] -SourceAddress <String[]>
 [-SourceIpGroup <String[]>] -DestinationAddress <String[]> [-DestinationIpGroup <String[]>]
 -DestinationPort <String[]> [-DestinationFqdn <String[]>] -Protocols <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzFirewallPolicyNetworkRule** cria uma regra de Rede para uma Política de Firewall do Azure.

## Exemplos

### Exemplo 1
```powershell
PS C:\> New-AzFirewallPolicyNetworkRule -Name NRC1 -Protocol "TCP" -SourceAddress "192.168.0.0/16" -DestinationAddress * -DestinationPort *
```

Este exemplo cria uma regra de rede com o endereço de origem, protocolo, endereço de destino e porta de destino
