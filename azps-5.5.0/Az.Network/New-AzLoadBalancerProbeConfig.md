# New-AzLoadBalancerProbeConfig

## Sinopse
Cria uma configuração de configuração de configuração para um balanceador de carga.

## Sintaxe

```
New-AzLoadBalancerProbeConfig -Name <String> [-Protocol <String>] -Port <Int32> -IntervalInSeconds <Int32>
 -ProbeCount <Int32> [-RequestPath <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzLoadBalancerProbeConfig** cria uma configuração de configuração de configuração para um balanceador de carga do Azure.

## Exemplos

### Exemplo 1: Criar uma configuração de configuração de configuração de configuração de
```powershell
PS C:\>New-AzLoadBalancerProbeConfig -Name "MyProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 15
```

Esse comando cria uma configuração de configuração de configuração chamada MyProbe usando o protocolo HTTP.
A nova carga será conectada a um serviço balanceado de carga na porta 80.

### Exemplo 2

Cria uma configuração de configuração de configuração para um balanceador de carga. (gerado automaticamente)

<!-- Aladdin Generated Example -->
```powershell
New-AzLoadBalancerProbeConfig -IntervalInSeconds 15 -Name 'MyProbe' -Port 80 -ProbeCount 15 -Protocol 'http' -RequestPath 'healthcheck.aspx'
```
