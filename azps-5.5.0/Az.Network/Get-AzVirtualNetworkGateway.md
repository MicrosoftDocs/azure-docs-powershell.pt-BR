# Get-AzVirtualNetworkGateway

## Sinopse
Obtém um Gateway de Rede Virtual

## Sintaxe

```
Get-AzVirtualNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Gateway de Rede Virtual é o objeto que representa seu gateway no Azure.
O cmdlet **Get-AzVirtualNetworkGateway** retorna o objeto do gateway no Azure com base no Nome e Nome do Grupo de Recursos.

## Exemplos

### Exemplo 1: Obter um Gateway de Rede Virtual
```powershell
Get-AzVirtualNetworkGateway -Name myGateway1 -ResourceGroupName myRG
```

Retorna o objeto do Gateway de Rede Virtual com o nome "myGateway1" dentro do grupo de recursos "myRG"

### Exemplo 2: Obter um Gateway de Rede Virtual
```powershell
Get-AzVirtualNetworkGateway -Name myGateway* -ResourceGroupName myRG
```

Retorna todos os Gateways de Rede Virtual que começam com "myGateway" no grupo de recursos "myRG"
