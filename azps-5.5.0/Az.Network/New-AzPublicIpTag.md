# New-AzPublicIpTag

## Sinopse
Cria uma marca IP.

## Sintaxe

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzPublicIpTag** cria uma Marca IP.

## Exemplos

### Exemplo 1: Criar uma nova marca IP
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

Esse comando cria uma nova Marca IP com o Tagtype, como "FirstPartyUsage" e uma marca como "/Sql". Isso é usado na criação de publicIpAddress com essas marcas específicas para alocação.
