# Test-AzDnsAvailability

## Sinopse
Verifica se um nome de domínio na cloudapp.azure.com está disponível para uso.

## Sintaxe

```
Test-AzDnsAvailability -DomainNameLabel <String> -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
Verifica se um nome de domínio na cloudapp.azure.com está disponível para uso.

## Exemplos

### Exemplo 1: verifique se contoso.westus.cloudapp.azure.com está disponível para uso.
```
Test-AzDnsAvailability -DomainNameLabel contoso -Location westus
```
