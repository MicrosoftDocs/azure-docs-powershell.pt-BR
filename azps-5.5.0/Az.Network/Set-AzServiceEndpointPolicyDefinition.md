# Set-AzServiceEndpointPolicyDefinition

## Sinopse
Atualiza uma definição de política de ponto de extremidade de serviço.

## Sintaxe

```
Set-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Set-AzServiceEndpointPolicyDefinition** cria uma definição de política de ponto de extremidade de serviço.

## Exemplos

### Exemplo 1: atualiza uma definição de política de ponto de extremidade de serviço em uma política de ponto de extremidade de serviço
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicyDefinition -Name "Policydef1" -ServiceEndpointPolicy $serviceEndpointPolicy
```

Esse comando atualiza uma definição de política de ponto de extremidade de serviço chamada Policydef1 na política de ponto de extremidade de serviço definida pelo objeto $ServiceEndpointPolicy.
