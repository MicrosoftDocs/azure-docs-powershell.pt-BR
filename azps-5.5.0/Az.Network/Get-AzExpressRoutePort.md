# Get-AzExpressRoutePort

## Sinopse
Obtém um recurso do Azure ExpressRoutePort.

## Sintaxe

### ResourceNameParameterSet (Padrão)
```
Get-AzExpressRoutePort [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePort -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzExpressRoutePort** é usado para recuperar um objeto do ExpressRoutePort da sua assinatura. O objeto do expressrouteport retornado pode ser usado como entrada para outros cmdlets que operam no ExpressRoutePort.

## Exemplos

### Exemplo 1
```powershell
PS C:\> Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Obtém o objeto ExpressRoutePort com nome $PortName no grupo de recursos $rg em sua assinatura.

### Exemplo 2
```powershell
PS C:\> Get-AzExpressRoutePort -Name test*
```

Obtém todos os objetos do ExpressRoutePort cujo nome começa com "teste".

### Exemplo 3
```powershell
PS C:\> Get-AzExpressRoutePort -ResourceId $id
```

Obtém o objeto ExpressRoutePort com a $id ResourceId.
