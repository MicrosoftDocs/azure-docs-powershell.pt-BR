# Get-AzApplicationGatewayFrontendIPConfig

## Sinopse
Obtém a configuração de IP front-end de um gateway de aplicativo.

## Sintaxe

```
Get-AzApplicationGatewayFrontendIPConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApplicationGatewayFrontendIPConfig obtém** a configuração ip front-end de um gateway de aplicativo.

## Exemplos

### Exemplo 1: obter uma configuração IP front-end especificada
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIP= Get-AzApplicationGatewayFrontendIPConfig -Name "FrontEndIP01" -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando obtém a configuração IP de front-end chamada FrontEndIP01 do $AppGw e a armazena na variável $FrontEndIP.

### Exemplo 2: Obter uma lista de configurações de IP front-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $FrontEndIPs= Get-AzApplicationGatewayFrontendIPConfig  -ApplicationGateway $AppGw
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 do grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando obtém uma lista das configurações de IP front-end do $AppGw e as armazena na variável $FrontEndIPs dados.
