# New-AzPublicIpPrefix

## Sinopse
Cria um Prefixo IP Público

## Sintaxe

```
New-AzPublicIpPrefix -Name <String> -ResourceGroupName <String> [-Location <String>] [-Sku <String>]
 [-Tier <String>] -PrefixLength <UInt16> [-IpAddressVersion <String>] [-IpTag <PSPublicIpPrefixTag[]>]
 [-Zone <String[]>] [-CustomIpPrefix <PSCustomIpPrefix>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet New-AzPublicIpPrefix cria** um prefixo IP público.

## Exemplos

### Exemplo 1: Criar um novo prefixo Ip público
```powershell
PS C:\> $publicIpPrefix = New-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName -PrefixLength 30
```

Esse comando cria um novo recurso de prefixo IP público. 

### Exemplo 2: Criar um novo prefixo Ip público global
```powershell
PS C:\> $publicIpPrefix = New-AzPublicIpPrefix -ResourceGroupName $rgname -name $rname -location $location -Tier Global -PrefixLength 30
```

Esse comando cria um novo recurso de prefixo IP público global.
