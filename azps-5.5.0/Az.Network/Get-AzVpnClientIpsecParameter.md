# Get-AzVpnClientIpsecParameter

## Sinopse
Obtém os parâmetros Ipsec vpn definidos no Gateway de Rede Virtual para conexões ponto a site.

## Sintaxe

```
Get-AzVpnClientIpsecParameter [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O Gateway de Rede Virtual é o objeto que representa seu gateway no Azure.
O cmdlet **Get-AzVpnClientIpsecParameter** retorna o objeto dos parâmetros ipsec de vpn definidos no gateway no Azure com base no Nome do Gateway e no Nome do Grupo de Recursos.

## Exemplos

### Exemplo 1: obtém os parâmetros Ipsec vpn definidos no Gateway de Rede Virtual para conexões ponto a site.
```powershell
PS C:\> $VpnClientIPsecParameters = Get-AzVpnClientIpsecParameter -Name myGateway -ResourceGroupName myRG
```

Retorna o objeto dos parâmetros ipsec vpn definidos no Gateway de Rede Virtual com o nome "myGateway" no grupo de recursos "myRG"
