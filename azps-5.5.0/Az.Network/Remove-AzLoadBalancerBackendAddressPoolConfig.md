# Remove-AzLoadBalancerBackendAddressPoolConfig

## Sinopse
Remove uma configuração de pool de endereços de back-end de um balanceador de carga.

## Sintaxe

```
Remove-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzLoadBalancerBackendAddressPoolConfig** remove um pool de endereços back-end de um balanceador de carga.

## Exemplos

### Exemplo 1: Remover uma configuração de pool de endereços back-end de um balanceador de carga
```
PS C:\>Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup" | Remove-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" | Set-AzLoadBalancer
```

Esse comando obtém o balanceador de carga chamado MyLoadBalancer e **passa-o para Remove-AzLoadBalancerBackendAddressPoolConfig,** que remove a configuração BackendAddressPool02 do MyLoadBalancer.
Por fim, o Set-AzLoadBalancer cmdlet atualiza o MyLoadBalancer.
Observe que uma configuração de pool de endereços back-end deve existir antes que você possa excluí-la.
