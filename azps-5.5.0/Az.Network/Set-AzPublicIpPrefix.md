# Set-AzPublicIpPrefix

## Sinopse
Define as Marcas de um PublicIpPrefix existente

## Sintaxe

```
Set-AzPublicIpPrefix -PublicIpPrefix <PSPublicIpPrefix> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O **cmdlet Set-AzPublicIpPrefix define** as Marcas de um prefixo IP público.

## Exemplos

### Definir as marcas para prefixo ip público
```powershell
PS C:\> $publicIpPrefix = Get-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName

PS C:\> $publicIpPrefix.Tags = "TestTag"

PS C:\> Set-AzPublicIpPrefix -PublicIpPrefix $publicIpPrefix
```

O primeiro comando obtém um Prefixo IP público existente, o segundo comando define a Propriedade marcas e o terceiro comando atualiza o objeto existente.
