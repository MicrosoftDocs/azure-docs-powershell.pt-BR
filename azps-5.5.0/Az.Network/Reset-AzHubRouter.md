# Reset-AzHubRouter

## Sinopse
Redefine o RoutingState de um recurso VirtualHub.

## Sintaxe

### ByVirtualHubName (Padrão)
```
Reset-AzHubRouter -ResourceGroupName <String> -Name <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualHubResourceId
```
Reset-AzHubRouter -ResourceId <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualHubObject
```
Reset-AzHubRouter -InputObject <PSVirtualHub> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
Redefine o Estado de Roteamento de um recurso VirtualHub existente somente se o Estado de Roteamento do hub virtual não estiver Provisionado.

## Exemplos

### Exemplo 1

```powershell
PS C:\> Reset-AzHubRouter -ResourceGroupName "testRG" -Name "westushub"
```

Redefina o estado de roteamento do hub virtual usando o ResourceGroupName e o ResourceName.

### Exemplo 2

```powershell
PS C:\> Reset-AzHubRouter -ResourceId "/subscriptions/testSub/resourceGroups/testRG/providers/Microsoft.Network/virtualHubs/westushub"
```

Redefina o estado de roteamento do hub virtual usando seu ResourceId.

### Exemplo 3

```powershell
PS C:\> Reset-AzHubRouter -InputObject $virtualHub
```

Redefina o estado de roteamento do hub virtual usando um objeto de entrada. O objeto de entrada é do tipo PSVirtualHub.

### Exemplo 4

```powershell
PS C:\> Get-AzVirtualHub -ResourceGroupName "testRG" -Name "westushub" | Reset-AzHubRouter
```

Um objeto do hub virtual existente pode ser recuperado e passado como objeto de entrada para o Reset-AzHubRouter.
