# New-AzApplicationGatewayClientAuthConfiguration

## Sinopse
Cria uma nova configuração de autenticação de cliente para perfil SSL.

## Sintaxe

```
New-AzApplicationGatewayClientAuthConfiguration [-VerifyClientCertIssuerDN]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **New-AzApplicationGatewayClientAuthConfiguration** cria uma nova configuração de autenticação de cliente para o perfil SSL.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $clientAuthConfig = New-AzApplicationGatewayClientAuthConfiguration -VerifyClientCertIssuerDN
```

O comando cria uma nova configuração de auth do cliente e a armazena $clientAuthConfig variável a ser usada em um perfil SSL.
