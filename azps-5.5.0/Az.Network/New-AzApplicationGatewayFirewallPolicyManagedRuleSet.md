# New-AzApplicationGatewayFirewallPolicyManagedRuleSet

## Sinopse
Cria um ManagedRuleSet para firewallPolicy

## Sintaxe

```
New-AzApplicationGatewayFirewallPolicyManagedRuleSet -RuleSetType <String> -RuleSetVersion <String>
 [-RuleGroupOverride <PSApplicationGatewayFirewallPolicyManagedRuleGroupOverride[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O **New-AzApplicationGatewayFirewallPolicyManagedRuleSet** cria regras gerenciadas para uma política de firewall.

## Exemplos

### Exemplo 1
```powershell
PS C:\> $managedRuleSet = New-AzApplicationGatewayFirewallPolicyManagedRuleSet -RuleSetType $ruleSetType 
-RuleSetVersion $ruleSetVersion -RuleGroupOverrides $ruleGroupOverride1, $ruleGroupOverride2
```

Cria um ManagedRuleSet com ruleSetType como $ruleSetType, ruleSetVersion como $ruleSetVersion e RuleGroupOverrides como uma lista com inteiros como $ruleGroupOverride 1, $ruleGroupOverride 2 O novo ManagedRuleSet é atribuído ao $managedRuleSet
