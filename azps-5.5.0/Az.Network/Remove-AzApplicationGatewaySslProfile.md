# Remove-AzApplicationGatewaySslProfile

## Sinopse
Remove o perfil SSL de um gateway de aplicativo.

## Sintaxe

```
Remove-AzApplicationGatewaySslProfile -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzApplicationGatewaySslProfile** remove o perfil SSL de um gateway de aplicativo.

## Exemplos

### Exemplo 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "SslProfile01"
```

O primeiro comando obtém um gateway de aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando remove o perfil SSL chamado SslProfile01 do gateway de aplicativo armazenado no $AppGw.
