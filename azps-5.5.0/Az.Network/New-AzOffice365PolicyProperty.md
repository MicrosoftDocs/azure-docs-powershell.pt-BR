# New-AzOffice365PolicyProperty

## Sinopse
Defina uma nova política de quebra de tráfego do Office 365 a ser usada com um site de Dispositivo Virtual.

## Sintaxe

```
New-AzOffice365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O New-AzOffice365PolicyProperties define uma política de saída do Office 365 que deve ser usada com um site de Dispositivo Virtual. 

## Exemplos

### Exemplo 1
```powershell
PS C:\> $o365Policy = New-AzOffice365PolicyProperty -Allow -Optimize 
```

Crie um objeto de política de política de quebra de tráfego do Office 365 para ser usado com comandos de site de Dispositivo Virtual.
