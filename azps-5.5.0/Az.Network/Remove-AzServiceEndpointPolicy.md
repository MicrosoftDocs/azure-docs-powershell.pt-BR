# Remove-AzServiceEndpointPolicy

## Sinopse
Remove uma política de ponto de extremidade de serviço.

## Sintaxe

### RemoveByNameParameterSet (Padrão)
```
Remove-AzServiceEndpointPolicy -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzServiceEndpointPolicy -ResourceId <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzServiceEndpointPolicy -InputObject <PSServiceEndpointPolicy> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzServiceEndpointPolicy** remove uma política de ponto de extremidade de serviço.

## Exemplos

### Exemplo 1: remove uma política de ponto de extremidade de serviço usando o nome
```
Remove-AzServiceEndpointPolicy -Name "Policy1" -ResourceGroup "resourcegroup1"
```

Esse comando remove uma política de ponto de extremidade de serviço com o nome Política1 que pertence ao grupo de recursos com o nome "resourcegroup1"

### Exemplo 2: Remover uma política de ponto de extremidade de serviço usando o objeto de entrada
```
Remove-AzServiceEndpointPolicy -InputObject $Policy1 -ResourceGroup "resourcegroup1"
```

Esse comando remove uma Política de objeto de política de ponto de extremidade de serviço1 que pertence ao grupo de recursos com o nome "resourcegroup1"
