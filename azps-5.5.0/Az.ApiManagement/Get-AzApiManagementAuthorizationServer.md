# Get-AzApiManagementAuthorizationServer

## Sinopse
Obtém um servidor de autorização de Gerenciamento de API.

## Sintaxe

### ContextParameterSet (Padrão)
```
Get-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServer [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApiManagementAuthorizationServer** obtém todos os servidores de autorização do Azure API Management ou um servidor de autorização especificado.
ClientSec ltd não será incluído nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementAuthorizationServerClientSec limited.**

## Exemplos

### Exemplo 1: Obter todos os servidores de autorização
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext
```

Esse comando obtém todos os servidores de autorização do Gerenciamento de API.

### Exemplo 2: Obter um servidor de autorização especificado
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "0123456789"
```

Esse comando obtém o servidor de autorização especificado.
