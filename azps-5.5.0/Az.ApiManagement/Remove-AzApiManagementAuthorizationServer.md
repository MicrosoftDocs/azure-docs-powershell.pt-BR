# Remove-AzApiManagementAuthorizationServer

## Sinopse
Remove um servidor de autorização.

## Sintaxe

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzApiManagementAuthorizationServer** remove um servidor de autorização do Azure API Management.

## Exemplos

### Exemplo 1: Remover um servidor de autorização
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Esse comando remove o Servidor de Autorização de Gerenciamento de API especificado.
Como o *parâmetro Forçar* é especificado, nenhuma confirmação é necessária.
