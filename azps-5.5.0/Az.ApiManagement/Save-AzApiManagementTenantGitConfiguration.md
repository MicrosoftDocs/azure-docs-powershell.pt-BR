# Save-AzApiManagementTenantGitConfiguration

## Sinopse
Salva as alterações criando uma confirmação para a configuração atual.

## Sintaxe

```
Save-AzApiManagementTenantGitConfiguration -Context <PsApiManagementContext> -Branch <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Save-AzApiManagementTenantGitConfiguration** salva as alterações criando um confirmação que contém o instantâneo de configuração atual em uma ramificação no repositório.

## Exemplos

### Exemplo 1: Salvar alterações na configuração
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Save-AzApiManagementTenantGitConfiguration -Context $apimContext -Branch 'master' -PassThru
```

Esse comando salva as alterações criando uma confirmação com o instantâneo de configuração atual para a ramificação especificada no repositório.
