# Get-AzApiManagementLogger

## Sinopse
Obtém objetos lo também.

## Sintaxe

### GetAllLoggers (Padrão)
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByLoggerId
```
Get-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApiManagementLogger** obtém uma **Loção** de Gerenciamento de API do Azure ou todos os loção.

## Exemplos

### Exemplo 1: Obter todos os lagos
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext
```

Esse comando obtém todas as loções para o contexto especificado.

### Exemplo 2: Obter um loção específico
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123"
```

Esse comando remove um lago que tem a ID Loção123.
