# Get-AzApiManagementAuthorizationServerClientSecret

## Sinopse
Obtém um segredo do cliente de servidor de autorização de gerenciamento de API.

## Sintaxe

### ContextParameterSet (Padrão)
```
Get-AzApiManagementAuthorizationServerClientSecret -Context <PsApiManagementContext> [-ServerId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzApiManagementAuthorizationServerClientSecret [-ServerId <String>] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApiManagementAuthorizationServerClientSec ltd** obtém o segredo do cliente do servidor de autorização do Gerenciamento de API do Azure.

## Exemplos

### Exemplo 1: Obter um segredo de cliente de servidor de autorização especificado por id
```
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementAuthorizationServerClientSecret -Context $ApiMgmtContext -ServerId "0123456789"
```

Esse comando obtém o segredo de servidor de autorização especificado.
