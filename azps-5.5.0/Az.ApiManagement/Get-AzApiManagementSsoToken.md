# Get-AzApiManagementSsoToken

## Sinopse
Obtém um link com um token SSO para um portal de gerenciamento implantado de um serviço de Gerenciamento de API.

## Sintaxe

### ExpandedParameter (Padrão)
```
Get-AzApiManagementSsoToken -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObject
```
Get-AzApiManagementSsoToken -InputObject <PsApiManagement> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApiManagementSsoToken** retorna um link (URL) contendo um único token de SSO (sign-on) para um portal de gerenciamento implantado de um serviço de Gerenciamento de API.

## Exemplos

### Exemplo 1: Obter o token SSO de um serviço de Gerenciamento de API
```
PS C:\>Get-AzApiManagementSsoToken -ResourceGroupName "Contoso" -Name "ContosoApi"
```

Esse comando obtém o token SSO de um serviço de Gerenciamento de API.
