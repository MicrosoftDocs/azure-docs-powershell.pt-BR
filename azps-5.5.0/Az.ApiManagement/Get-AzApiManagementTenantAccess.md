# Get-AzApiManagementTenantAccess

## Sinopse
Obtém a configuração de acesso de um locatário.

## Sintaxe

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrição
O cmdlet **Get-AzApiManagementTenantAccess** obtém a configuração de acesso de locatário para um locatário.
As teclas não serão incluídas nos detalhes do resultado. Para obter o segredo do cliente, use **Get-AzApiManagementTenantAccessSec limited.**

## Exemplos

### Exemplo 1: Obter configuração de acesso de locatário
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Esse comando obtém a configuração de acesso do locatário para o contexto especificado.
