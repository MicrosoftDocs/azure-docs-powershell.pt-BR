# Remove-AzApiManagementApiRevision

## Sinopse
Remoção de uma revisão de API específica

## Sintaxe

### ByApiId (Padrão)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrição
O cmdlet **Remove-AzApiManagementApiRevision** remove uma revisão de API específica.

## Exemplos

### Exemplo 1: Remover uma Revisão da API
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Esse comando remove a revisão `2` da API do serviço de Gerenciamento de `echo-api` API.

### Exemplo 2

Esse comando remove a 2 revisão da api de eco da API do serviço de Gerenciamento de API. (gerado automaticamente)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
