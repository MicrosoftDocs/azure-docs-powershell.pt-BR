# New-AzBatchAccountKey

## Sinopse
Regenera uma chave de uma conta em lotes.

## SYNTAX

```
New-AzBatchAccountKey [-AccountName] <String> [-ResourceGroupName <String>] -KeyType <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzBatchAccountKey** regenera a chave primária ou secundária de uma conta em lotes do Azure.
O cmdlet retorna um objeto **BatchAccountContext** que tem suas propriedades **PrimaryAccountKey** e **SecondaryAccountKey** atuais.

## EXEMPLOS

### Exemplo 1: regenerar a chave da conta primária em uma conta em lotes
```
PS C:\>New-AzBatchAccountKey -AccountName "pfuller" -KeyType "Primary"
AccountName                  : pfuller

Location                     : westus

ResourceGroupName            : CmdletExampleRG

CoreQuota                    : 20

PoolQuota                    : 20

ActiveJobAndJobScheduleQuota : 20

Tags                         : 
TaskTenantUrl                : https://cmdletexample.westus.batch.azure.com
```

Esse comando regenera a chave da conta primária na conta em lotes chamada pfuller.
