# Remove-AzHDInsightPersistedScriptAction

## Sinopse
Remove uma ação de script persistente de um cluster HDInsight.

## SYNTAX

```
Remove-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzHDInsightPersistedScriptAction** remove uma ação de script persistente da lista de ações de script persistentes do cluster do Azure HDInsight especificado.
O script removido não será mais executado quando o cluster estiver ampliado.

## EXEMPLOS

### Exemplo 1: remover uma ação de script da lista de ações de script persistentes em um cluster
```
PS C:\>Remove-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Esse comando Remove a ação de script chamada ScriptAction da lista de ações persistentes de script no cluster especificado.
