# Add-AzDataLakeStoreFirewallRule

## Sinopse
Adiciona uma regra de firewall à conta especificada do data Lake Store.

## SYNTAX

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzDataLakeStoreFirewallRule** adiciona uma regra de firewall à conta especificada do data Lake Store.

## EXEMPLOS

### Exemplo 1: adicionar uma nova regra de firewall a uma conta do repositório data Lake
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Isso cria uma nova regra de firewall chamada "myrule" na conta "ContosoADL" com um intervalo IP de 127.0.0.1-127.0.0.2
