# Start-AzRecoveryServicesAsrSwitchProcessServerJob

## Sinopse
Alterne a replicação de um servidor de processo para outro para o balanceamento de carga.

## SYNTAX

```
Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric <ASRFabric> -SourceProcessServer <ASRProcessServer>
 -TargetProcessServer <ASRProcessServer> [-ReplicationProtectedItem <ASRReplicationProtectedItem[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O **Start-AzRecoveryServicesAsrSwitchProcessServerJob** alterna a movimentação de dados de replicação das máquinas virtuais especificadas ou um servidor de processo especificado para o servidor de processo de destino especificado. Usado para balancear a carga ou alternar a replicação entre servidores de processo.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer
```

Trabalho para acompanhar a mudança do servidor de processo para todos os itens protegidos de origem do servidor de processo de destino.

### Exemplo 2
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer -ReplicatedItem $rpList
```

Trabalho para acompanhar o processo de comutação do servidor para o item protegido de replicação do servidor de processo de origem para destino.
