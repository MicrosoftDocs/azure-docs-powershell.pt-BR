# Módulo AZ. DataBox
## Descritivo
Módulo para criar, cancelar, excluir e buscar trabalhos do Databox

## Cmdlets AZ. DataBox
### [Get-AzDataBoxCredential](Get-AzDataBoxCredential.md)
Obtém as credenciais de databox de um trabalho específico

### [Get-AzDataBoxJob](Get-AzDataBoxJob.md)
Obtém informações sobre trabalhos do Databox

### [New-AzDataBoxJob](New-AzDataBoxJob.md)
Cria um novo trabalho do databox usando os parâmetros especificados

### [Remove-AzDataBoxJob](Remove-AzDataBoxJob.md)
Exclui o trabalho databox

### [Parar-AzDataBoxJob](Stop-AzDataBoxJob.md)
Cancela um trabalho em andamento do databox se o trabalho estiver em estado cancelável.
