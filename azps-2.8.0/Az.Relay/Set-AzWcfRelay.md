# Set-AzWcfRelay

## Sinopse
Atualiza a descrição de um WcfRelay no namespace de retransmissão especificado.

## SYNTAX

### WcfRelayInputObjectSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-InputObject <PSWcfRelayAttributes>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### WcfRelayPropertiesSet
```
Set-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String> [-UserMetadata <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Set-AzWcfRelay atualiza a descrição do WcfRelay no namespace de retransmissão especificado.
