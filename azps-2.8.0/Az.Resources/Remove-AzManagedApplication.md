# Remove-AzManagedApplication

## Sinopse
Remove um aplicativo gerenciado

## SYNTAX

### RemoveByNameAndResourceGroup (padrão)
```
Remove-AzManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzManagedApplication** remove um aplicativo gerenciado

## EXEMPLOS

### Exemplo 1: remover o aplicativo gerenciado por ID do recurso
```
PS C:\>$Application = Get-AzManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplication -Id $Application.ResourceId -Force
```

O primeiro comando obtém um aplicativo gerenciado chamado myApp usando o cmdlet Get-AzManagedApplication.
O comando o armazena na variável $Application.
O segundo comando Remove o aplicativo gerenciado identificado pela propriedade **ResourceId** de $Application.
