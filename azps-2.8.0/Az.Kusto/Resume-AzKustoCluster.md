# Resume-AzKustoCluster

## Sinopse
Retomar um cluster Kusto suspenso.

## SYNTAX

### ByNameAndResourceGroup (padrão)
```
Resume-AzKustoCluster [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Resume-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Resume-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Retomar um cluster Kusto suspenso.

## EXEMPLOS

### Exemplo 1-retomar um cluster Kusto suspenso por nome

```
PS C:\> Resume-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

O comando acima reinicia o cluster Kusto suspenso chamado "mykustocluster" localizado no grupo de recursos "testrg".

### Exemplo 2-retomar um cluster de Kusto suspenso por tubulação

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Resume-AzKustoCluster
```

O comando acima Obtém o cluster Kusto chamado "mykustocluster" localizado no grupo de recursos "testrg" usando o `Get-AzKustoCluster` cmdlet e, em seguida, canaliza o resultado para `Resume-AzKustoCluster` retomá-lo.
