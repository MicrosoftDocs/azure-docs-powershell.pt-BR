# Get-AzPrivateDnsVirtualNetworkLink

## Sinopse
Obtém um link de rede virtual associado à zona DNS privada especificada.

## SYNTAX

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzPrivateDnsVirtualNetworkLink** Obtém links de rede virtual associados a uma determinada zona de DNS particular do grupo de recursos especificado.
Se você especificar o parâmetro *Name* , um único objeto **PSPrivateDnsVirtualNetworkLink** será retornado.
Se você não especificar o parâmetro *Name* , uma matriz que contenha todos os links associados à zona no grupo de recursos especificado será retornada.
Você pode usar o objeto **PSPrivateDnsVirtualNetworkLink** para atualizar o link.
