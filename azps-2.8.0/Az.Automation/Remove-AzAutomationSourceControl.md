# Remove-AzAutomationSourceControl

## Sinopse
Remove um controle de origem da automação do Azure.

## SYNTAX

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Remove-AzAutomationSourceControl remove um controle de origem da automação do Azure.

## EXEMPLOS

### Exemplo 1
Esse comando Remove o controle de origem de automação chamado VSTSNative na conta chamada devAccount.
Esse comando especifica o parâmetro *Force* . Portanto, ele não solicita a confirmação.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
