# Publish-AzAutomationRunbook

## Sinopse
Publica um runbook.

## SYNTAX

```
Publish-AzAutomationRunbook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Publish-AzAutomationRunbook** publica um runbook para uso no ambiente de produção da automação do Azure.

## EXEMPLOS

### Exemplo 1: publicar um runbook
```
PS C:\>Publish-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01"
```

Esse comando publica o runbook chamado Runbk01 na conta de automação do Azure chamado Contoso17.
