# Remove-AzLoadBalancerProbeConfig

## Sinopse
Remove uma configuração de teste de um balanceador de carga.

## SYNTAX

```
Remove-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzLoadBalancerProbeConfig** remove uma configuração de teste de um balanceador de carga.

## EXEMPLOS

### Exemplo 1: remover uma configuração de teste de um balanceador de carga
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $loadbalancer
```

O primeiro comando obtém o balanceador de carga chamado MyLoadBalancer e, em seguida, armazena-o na variável $loadbalancer.
O segundo comando exclui a configuração nomeada myprobe do balanceador de carga em $loadbalancer.
