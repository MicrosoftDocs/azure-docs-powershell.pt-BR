# Get-AzSqlElasticPoolAdvisor

## Sinopse
Obtém um ou mais conselheiros para um pool elástico do Azure SQL.

## SYNTAX

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlElasticPoolAdvisor** Obtém um ou mais conselheiros de pool elástico do SQL do Azure para um pool elástico do Azure SQL.
