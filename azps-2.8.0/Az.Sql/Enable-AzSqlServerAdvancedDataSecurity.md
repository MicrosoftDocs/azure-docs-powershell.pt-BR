# Enable-AzSqlServerAdvancedDataSecurity

## Sinopse
Habilita a segurança avançada de dados em um servidor.

## SYNTAX

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Enable-AzSqlServerAdvancedDataSecurity** permite a segurança avançada de dados em um servidor. A segurança avançada de dados é um pacote de segurança unificado que inclui classificação de dados, avaliação de vulnerabilidade e proteção avançada contra ameaças para o seu servidor. (Uma nova conta de armazenamento será criada automaticamente para salvar avaliações de vulnerabilidade. Se uma conta de armazenamento foi criada anteriormente para essa finalidade, ela será usada em vez disso.

## EXEMPLOS

### Exemplo 1-habilitar a segurança de dados avançados do servidor
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### Exemplo 2-habilitar a segurança de dados avançados do servidor do recurso do servidor
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```
