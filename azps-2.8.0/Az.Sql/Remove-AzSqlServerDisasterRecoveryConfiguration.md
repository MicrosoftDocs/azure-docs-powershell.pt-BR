# Remove-AzSqlServerDisasterRecoveryConfiguration

## Sinopse
Remove uma configuração de recuperação do sistema de servidor de banco de dados SQL.

## SYNTAX

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzSqlServerDisasterRecoveryConfiguration** remove uma configuração de recuperação do sistema de servidor de banco de dados SQL.
