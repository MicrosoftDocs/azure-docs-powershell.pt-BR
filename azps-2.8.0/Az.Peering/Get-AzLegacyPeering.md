# Get-AzLegacyPeering

## Sinopse
Usado para converter recursos de emparelhamento herdados para recursos de gerenciamento de recursos do Azure (ARM). 

## SYNTAX

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O comando é usado para exibir recursos de emparelhamento herdados que você deseja converter em recursos de gerenciamento de recursos do Azure (ARM).
