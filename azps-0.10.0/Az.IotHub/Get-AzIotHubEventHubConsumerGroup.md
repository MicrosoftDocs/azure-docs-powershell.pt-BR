# Get-AzIotHubEventHubConsumerGroup

## Sinopse
Obtém todas as consumergroups do eventhub.

## SYNTAX

```
Get-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obtém todos os consumergroups do eventhub para os diferentes EventHubs usados pelo IotHub.

## EXEMPLOS

### O exemplo 1 Obtém todos os consumergroups do eventhub para o eventhub de telemetria
```
PS C:\> Get-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Obtém todos os consumergroups do eventhub para o eventhub do iothub chamado myiothub
