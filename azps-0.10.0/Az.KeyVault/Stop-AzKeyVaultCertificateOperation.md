# Stop-AzKeyVaultCertificateOperation

## Sinopse
Cancela uma operação de certificado no cofre de chaves.

## SYNTAX

```
Stop-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzKeyVaultCertificateOperation** cancela uma operação de certificado no serviço de compartimento de chave do Azure.
