# New-AzADGroup

## Sinopse
Cria um novo grupo do Active Directory.

## SYNTAX

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Cria um novo grupo do Active Directory.

## EXEMPLOS

### Exemplo 1-criar um novo grupo de anúncios

```
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Cria um novo grupo de anúncios com o nome "MyGroupDisplayName" e o apelido do email " myemail@domain.com ".
