# Stop-AzVM

## Sinopse
Interrompe uma máquina virtual do Azure.

## SYNTAX

### ResourceGroupNameParameterSetName (padrão)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzVM** interrompe uma máquina virtual do Azure.

## EXEMPLOS

### Exemplo 1: parar uma máquina virtual
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Esse comando para a máquina virtual nomeada VirtualMachine07 em ResourceGroup11.
