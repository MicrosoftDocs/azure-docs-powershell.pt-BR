# Remove-AzApplicationGatewayUrlPathMapConfig

## Sinopse
Remove os mapeamentos de caminho de URL para um pool de servidores back-end.

## SYNTAX

```
Remove-AzApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayUrlPathMapConfig** remove mapeamentos de caminho de URL para um pool de servidores back-end.
