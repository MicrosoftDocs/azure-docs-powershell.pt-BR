# Remove-AzVirtualNetworkGateway

## Sinopse
Exclui um gateway de rede virtual

## SYNTAX

```
Remove-AzVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O gateway de rede virtual é o objeto que representa o gateway no Azure.

O cmdlet **Get-AzVirtualNetworkGateway** retorna o objeto do seu gateway no Azure com base no nome e no nome do grupo de recursos.

## EXEMPLOS

### 1: excluir um gateway de rede virtual
```
Remove-AzVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Exclui o objeto do gateway de rede virtual com o nome "mygateway" no grupo de recursos "myRG"

Observação: você deve primeiro excluir todas as conexões com o gateway de rede virtual usando o cmdlet **Remove-AzVirtualNetworkGatewayConnection** .
