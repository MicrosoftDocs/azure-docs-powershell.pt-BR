# Set-AzApplicationGatewaySslCertificate

## Sinopse
Define o estado da meta de um certificado SSL.

## SYNTAX

```
Set-AzApplicationGatewaySslCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> -Password <SecureString> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzApplicationGatewaySslCertificate** define o estado da meta de um certificado SSL.

## EXEMPLOS

### Exemplo 1: definir o estado da meta de um certificado SSL
```
PS C:\> $appGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $password = ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force
PS C:\> $cert = Set-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert01" -CertificateFile "D:\cert01.pfx" -Password $password
```

Esse comando define o estado da meta para um certificado SSL do gateway do aplicativo chamado ApplicationGateway01.
