# Remove-AzApplicationGatewayAuthenticationCertificate

## Sinopse
Remove um certificado de autenticação de um gateway de aplicativo.

## SYNTAX

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String>
 -ApplicationGateway <PSApplicationGateway> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayAuthenticationCertificate** remove um certificado de autenticação de um Application Gateway do Azure.
