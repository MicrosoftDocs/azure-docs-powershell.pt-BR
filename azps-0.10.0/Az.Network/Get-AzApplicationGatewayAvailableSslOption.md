# Get-AzApplicationGatewayAvailableSslOption

## Sinopse
Obtém todas as opções de SSL disponíveis para a política SSL do Application Gateway.

## SYNTAX

```
Get-AzApplicationGatewayAvailableSslOption [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzApplicationGatewayAvailableSslOption** obtém todas as opções SSL disponíveis para a política SSL

## EXEMPLOS

### Exemplo 1
```
PS C:\>$sslOptions = Get-AzApplicationGatewayAvailableSslOption
```

Esses comandos retornam todas as opções SSL disponíveis para a política SSL.
