# Get-AzStorageUsage

## Sinopse
Obtém o uso do recurso de armazenamento da assinatura atual.

## SYNTAX

```
Get-AzStorageUsage [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzStorageUsage** Obtém o uso do recurso do armazenamento do Azure para a assinatura atual.

## EXEMPLOS

### Exemplo 1: obter o uso dos recursos de armazenamento
```
PS C:\>Get-AzStorageUsage
```

Este comando obtém o uso dos recursos de armazenamento da assinatura atual.
