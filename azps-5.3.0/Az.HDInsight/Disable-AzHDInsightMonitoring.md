# Disable-AzHDInsightMonitoring

## Sinopse
Desabilita o monitoramento em um cluster HDInsight e os logs relevantes deixarão de fluir para o espaço de trabalho de monitoramento especificado durante a ativação.

## SYNTAX

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Disable-AzHDInsightMonitoring** desabilita o monitoramento em um cluster do Azure HDInsight.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

O monitoramento será desabilitado no cluster HDInsight e os logs relevantes deixarão de fluir para o espaço de trabalho monitoramento.

### Exemplo 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

O monitoramento será desabilitado no cluster HDInsight e os logs relevantes deixarão de fluir para o espaço de trabalho monitoramento.
