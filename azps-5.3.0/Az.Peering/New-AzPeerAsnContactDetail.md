# New-AzPeerAsnContactDetail

## Sinopse
Cria um detalhe de contato na memória para PeerAsn. 

## SYNTAX

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Crie um detalhe de contato do PeerAsn de memória.

## EXEMPLOS

### Criar um detalhe de contato e adicionar ao PeerAsn
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

A função e o e-mail são obrigatórios. O telefone é opcional. Telefone compatível com +-() ou espaços. Um PeerAsn deve incluir pelo menos um detalhe de contato do tipo "NOC"
