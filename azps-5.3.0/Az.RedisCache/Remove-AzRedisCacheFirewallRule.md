# Remove-AzRedisCacheFirewallRule

## Sinopse
Remover uma regra de firewall de um cache do Redis.

## SYNTAX

### NormalParameterSet (padrão)
```
Remove-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> -RuleName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PSRedisFirewallRuleObject
```
Remove-AzRedisCacheFirewallRule -InputObject <PSRedisFirewallRule> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Remover uma regra de firewall de um cache do Redis.

## EXEMPLOS

### Exemplo 1: remover uma única regra de firewall
```
PS C:\>Remove-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone" -PassThru
True
```

Esse comando Remove uma regra de firewall chamada ruleone do cache Redis chamado myCache.
