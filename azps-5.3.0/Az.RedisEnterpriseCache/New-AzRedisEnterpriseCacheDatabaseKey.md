# New-AzRedisEnterpriseCacheDatabaseKey

## Sinopse
Gera novamente as teclas de acesso do banco de dados RedisEnterprise.

## SYNTAX

```
New-AzRedisEnterpriseCacheDatabaseKey -ClusterName <String> -ResourceGroupName <String>
 -KeyType <AccessKeyType> [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Gera novamente as teclas de acesso do banco de dados RedisEnterprise.

## EXEMPLOS

### Exemplo 1: regenerar a chave de acesso primária
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Primary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
new-primary-key                              secondary-key

```

Esse comando regenera a chave de acesso de segredo primário usada para autenticar conexões para o banco de dados do cache corporativo do Redis chamado myCache.

### Exemplo 2: regenerar a chave de acesso secundário
```powershell
PS C:\> New-AzRedisEnterpriseCacheDatabaseKey -Name "MyCache" -ResourceGroupName "MyGroup" -KeyType "Secondary"

PrimaryKey                                   SecondaryKey
----------                                   ------------
primary-key                                  new-secondary-key

```

Esse comando regenera a chave de acesso de segredo secundário usada para autenticar conexões para o banco de dados do cache corporativo do Redis chamado myCache.
