# Módulo de AZ. Advisor
## Descritivo
Comandos para interagir com o Azure Advisor.

## Cmdlets do AZ. Advisor
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Obtém uma lista de recomendações do Azure Advisor.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Desabilitar uma recomendação do Azure Advisor.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Habilita a recomendação (s) do Azure Advisor.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Obter as configurações do Azure Advisor para a assinatura ou o grupo de recursos específico.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Atualiza ou cria a configuração do Azure Advisor.
