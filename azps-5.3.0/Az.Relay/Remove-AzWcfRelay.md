# Remove-AzWcfRelay

## Sinopse
Remove o WcfRelay do namespace especificado de retransmissão.

## SYNTAX

```
Remove-AzWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzWcfRelay** remove o WcfRelay do namespace de retransmissão especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Remove o WcfRelay `TestWCFRelay1` do namespace `TestNameSpace-Relay1` .
