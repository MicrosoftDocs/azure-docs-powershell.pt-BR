# Update-AzKeyVaultManagedHsm

## Sinopse
Atualize o estado de um HSM gerenciado do Azure.

## SYNTAX

### UpdateByNameParameterSet (padrão)
```
Update-AzKeyVaultManagedHsm -Name <String> -ResourceGroupName <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UpdateByInputObjectParameterSet
```
Update-AzKeyVaultManagedHsm -InputObject <PSManagedHsm> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UpdateByResourceIdParameterSet
```
Update-AzKeyVaultManagedHsm -ResourceId <String> [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Esse cmdlet atualiza o estado de um HSM gerenciado do Azure.

## EXEMPLOS

### Exemplo 1: atualizar um HSM gerenciado diretamente

Atualiza as marcas do HSM gerenciado nomeado `$hsmName` no grupo de recursos `$resourceGroupName` .

### Exemplo 2: atualizar um HSM gerenciado usando o encanamento
```powershell
PS C:\> Get-AzKeyVaultManagedHsm -Name $hsmName -ResourceGroupName $resourceGroupName | Update-AzKeyVaultManagedHsm -Tag @{testKey="testValue"}
```

Atualiza as marcas do HSM gerenciado usando a sintaxe de tubulação.
