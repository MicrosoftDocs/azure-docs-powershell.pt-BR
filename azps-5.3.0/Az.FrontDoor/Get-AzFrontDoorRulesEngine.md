# Get-AzFrontDoorRulesEngine

## Sinopse
Obter configurações do mecanismo de regras.

## SYNTAX

```
Get-AzFrontDoorRulesEngine -ResourceGroupName <String> -FrontDoorName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzFrontDoorRulesEngine** Obtém uma configuração de mecanismo de regras específica ou obtém todas as configurações de mecanismo de regras associadas a uma porta frontal. 

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name rulesengine3

Name         RulesEngineRules
----         ----------------
rulesEngine3 {rules1}
```

Obter configuração de mecanismo de regras específicas.

### Exemplo 2

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName

Name         RulesEngineRules
----         ----------------
rulesEngine1 {Rule1}
rulesEngine2 {Rule1}
rulesEngine3 {rules1}
```

Obter todas as configurações do mecanismo de regras em uma porta frontal.

### Exemplo 3

PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name nonexistent
