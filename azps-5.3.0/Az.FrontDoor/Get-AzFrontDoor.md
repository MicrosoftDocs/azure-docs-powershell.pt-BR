# Get-AzFrontDoor

## Sinopse
Obter balanceador de carga de porta frontal

## SYNTAX

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdletGet **Get-AzFrontDoor** obtém todas as portas frontais existentes na assinatura atual que corresponde às informações fornecidas.
