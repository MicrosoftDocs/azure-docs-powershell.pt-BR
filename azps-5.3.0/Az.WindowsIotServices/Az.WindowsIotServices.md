# Módulo AZ. WindowsIotServices
## Descritivo
Microsoft Azure PowerShell: cmdlets do WindowsIotServices

## Cmdlets AZ. WindowsIotServices
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Obter os metadados relacionados a não segurança de um serviço de dispositivo IoT do Windows.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Criar ou atualizar os metadados de um serviço de dispositivo IoT do Windows.
O padrão usual para modificar uma propriedade é recuperar os metadados do serviço de dispositivo Windows IoT e os metadados de segurança e, em seguida, combiná-los com os valores modificados em um novo corpo para atualizar o serviço de dispositivo IoT do Windows.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Excluir um serviço de dispositivo IoT do Windows.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Atualiza os metadados de um serviço de dispositivo IoT do Windows.
O padrão usual para modificar uma propriedade é recuperar os metadados do serviço de dispositivo Windows IoT e os metadados de segurança e, em seguida, combiná-los com os valores modificados em um novo corpo para atualizar o serviço de dispositivo IoT do Windows.
