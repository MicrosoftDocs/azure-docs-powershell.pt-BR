# Módulo AZ. ResourceGraph
## Descritivo
Microsoft Azure PowerShell: cmdlets do ResourceGraph

## Cmdlets AZ. ResourceGraph
### [Get-AzResourceGraphQuery](Get-AzResourceGraphQuery.md)
Obtenha uma única consulta de gráfico por meio de seu Resource resourcer.

### [New-AzResourceGraphQuery](New-AzResourceGraphQuery.md)
Criar uma nova consulta de gráfico.

### [Remove-AzResourceGraphQuery](Remove-AzResourceGraphQuery.md)
Excluir uma consulta de gráfico.

### [Pesquisa-AzGraph](Search-AzGraph.md)
Consulta os recursos gerenciados pelo Azure Resource Manager.

### [Update-AzResourceGraphQuery](Update-AzResourceGraphQuery.md)
Atualiza uma consulta de gráfico que já foi adicionada.
