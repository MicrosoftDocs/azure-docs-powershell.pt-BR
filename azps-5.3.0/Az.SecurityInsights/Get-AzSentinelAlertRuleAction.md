# Get-AzSentinelAlertRuleAction

## Sinopse
Obter uma resposta automática (ação de regra de alerta).

## SYNTAX

### AlertRuleId (padrão)
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ActionID
```
Get-AzSentinelAlertRuleAction -ResourceGroupName <String> -WorkspaceName <String> -AlertRuleId <String>
 -ActionId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSentinelAlertRuleAction** Obtém uma resposta automática (ação de regra de alerta) do espaço de trabalho especificado.
Se você especificar os parâmetros *ActionID* e *AlertRuleId* , um único objeto **AlertRuleAction** será retornado.
Se você não especificar o parâmetro *ActionID* , uma matriz que contenha todas as ações para a regra de alerta específica no espaço de trabalho especificado será retornada.
Você pode usar o objeto **Action** para atualizar a ação, por exemplo, você pode alterar a **ação** para uma regra de alerta.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $AlertRuleActions = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId"
```

Este exemplo obtém todas as **ações** da regra de alerta especificada no espaço de trabalho especificado e a armazena na variável $AlertRuleActions.

### Exemplo 2
```powershell
PS C:\> $AlertRuleAction = Get-AzSentinelAlertRuleAction -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleId "MyAlertRuleId" -ActionId "MyActionId"
```

Este exemplo obtém um **AlertRuleAction** para a regra de alerta especificada no espaço de trabalho especificado e, em seguida, armazena-o na variável $AlertRuleAction.
