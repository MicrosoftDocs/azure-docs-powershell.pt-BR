# Get-AzMarketplacePrivateStoreOffer

## Sinopse
Obtenha uma ou mais ofertas de repositório particular.

## SYNTAX

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obtenha uma ou mais ofertas de repositório particular com planos públicos + privados que foram adicionados em escopo do locatário. Se a ID da assinatura for apresentada, obtenha uma ou mais ofertas da loja privada com planos particulares somente sob o escopo da assinatura
