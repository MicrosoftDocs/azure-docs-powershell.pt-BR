# Módulo AZ. Marketplace
## Descritivo
Comandos para interagir com o Azure Marketplace.

## Cmdlets do AZ. Marketplace
### [Get-AzMarketplacePrivateStoreOffer](Get-AzMarketplacePrivateStoreOffer.md)
Obtenha uma ou mais ofertas de repositório particular.

### [Get-AzMarketplacePrivateStore](Get-AzMarketplacePrivateStore.md)
Obter lista da loja particular

### [Remove-AzMarketplacePrivateStoreOffer](Remove-AzMarketplacePrivateStoreOffer.md)
Remover uma oferta do repositório particular.

### [Set-AzMarketplacePrivateStoreOffer](Set-AzMarketplacePrivateStoreOffer.md)
Atualiza ou cria a oferta para armazenamento particular.
