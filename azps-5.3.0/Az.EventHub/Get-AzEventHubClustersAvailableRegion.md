# Get-AzEventHubClustersAvailableRegion

## Sinopse
Obtém os detalhes do cluster do Eventhub único ou da lista de clusters no grupo de recursos especificado

## SYNTAX

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
A lista cmdlet Get-AzEventHubClustersAvailableRegion de regiões onde o Dedicated está disponível para criação.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzEventHubClustersAvailableRegion
```
