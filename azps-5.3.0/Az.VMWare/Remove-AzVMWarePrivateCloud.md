# Remove-AzVMWarePrivateCloud

## Sinopse
Excluir uma nuvem privada

## SYNTAX

### Excluir (padrão)
```
Remove-AzVMWarePrivateCloud -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzVMWarePrivateCloud -InputObject <IVMWareIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Excluir uma nuvem privada

## EXEMPLOS

### Exemplo 1: excluir nuvem privada
```powershell
PS C:\> Remove-AzVMWarePrivateCloud -ResourceGroupName azps-test-group -Name azps-test-cloud

```

Excluir nuvem privada

  - `[Location <String>]`: Região do Azure
  - `[PrivateCloudName <String>]`: Nome da nuvem privada
  - `[ResourceGroupName <String>]`: O nome do grupo de recursos. O nome não diferencia maiúsculas de minúsculas.
