# Get-AzIotHubDeployment

## Sinopse
Lista todas ou uma implantação de borda IoT específica.

## SYNTAX

### ResourceSet (padrão)
```
Get-AzIotHubDeployment [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubDeployment [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceSet
```
Get-AzIotHubDeployment [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
Obter os detalhes de uma implantação de borda IoT ou lista de implantações de borda IoT em um hub IoT.
Consulte https://docs.microsoft.com/azure/iot-edge/module-deployment-monitoring para obter mais informações.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "deploy1"
```

Obtenha os detalhes de uma implantação de borda IoT.

### Exemplo 2
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Listar todas as implantações de borda IoT em um hub IoT.
