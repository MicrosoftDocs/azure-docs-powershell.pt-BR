# Get-AzSqlVirtualCluster

## Sinopse
Retorna informações sobre o cluster virtual do Azure SQL.

## SYNTAX

### GetVirtualClusterByResourceGroup (padrão)
```
Get-AzSqlVirtualCluster [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetVirtualClusterByNameAndResourceGroup
```
Get-AzSqlVirtualCluster [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetVirtualClusterByResourceId
```
Get-AzSqlVirtualCluster [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlVirtualCluster** retorna informações sobre um ou mais clusters virtuais do Azure SQL.
Especifique o nome de um cluster virtual para ver informações somente para esse cluster.
