# Remove-AzSqlServer

## Sinopse
Remove um servidor de banco de dados SQL do Azure.

## SYNTAX

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzSqlServer** remove um servidor de banco de dados SQL do Azure.
A operação de exclusão é assíncrona e pode demorar algum tempo, portanto verifique se a operação foi concluída antes de executar qualquer operação adicional que dependa do servidor ser completamente excluído.
Por exemplo, você precisa criar um novo servidor que use o mesmo nome.

## EXEMPLOS

### Exemplo 1: remover um servidor
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Esse comando Remove o servidor do banco de dados SQL do Azure chamado Server01.
