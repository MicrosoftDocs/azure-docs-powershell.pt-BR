# Remove-AzSqlSyncAgent

## Sinopse
Remove um agente de sincronização do SQL do Azure.

## SYNTAX

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzSqlSyncAgent** remove um agente de sincronização do SQL do Azure.

## EXEMPLOS

### Exemplo 1: remover um agente de sincronização
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Esse comando Remove o agente SQL Sync do Azure chamado syncAgent01.
