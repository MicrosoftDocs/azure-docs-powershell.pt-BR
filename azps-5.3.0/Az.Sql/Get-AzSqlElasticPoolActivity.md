# Get-AzSqlElasticPoolActivity

## Sinopse
Obtém o status das operações em um pool elástico.

## SYNTAX

```
Get-AzSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String> [-OperationId <Guid>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlElasticPoolActivity** Obtém o status das operações em um pool elástico para um banco de dados SQL do Azure.
Você pode ver o status da criação e das atualizações de configuração do pool.

## EXEMPLOS

### Exemplo 1: obter o status das operações para um pool elástico
```
PS C:\>Get-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Esse comando obtém o status das operações do pool elástico chamado ElasticPool01.
