# Módulo AZ. AppConfiguration
## Descritivo
Microsoft Azure PowerShell: cmdlets do AppConfiguration

## Cmdlets AZ. AppConfiguration
### [Get-AzAppConfigurationStore](Get-AzAppConfigurationStore.md)
Obter ou listar repositórios de configuração de aplicativo.

### [Get-AzAppConfigurationStoreKey](Get-AzAppConfigurationStoreKey.md)
Lista a tecla de acesso para o repositório de configuração especificado.

### [New-AzAppConfigurationStore](New-AzAppConfigurationStore.md)
Cria um repositório de configuração com os parâmetros especificados.

### [New-AzAppConfigurationStoreKey](New-AzAppConfigurationStoreKey.md)
Gera novamente uma tecla de acesso para o repositório de configuração especificado.

### [Remove-AzAppConfigurationStore](Remove-AzAppConfigurationStore.md)
Exclui um repositório de configuração.

### [Test-AzAppConfigurationStoreNameAvailability](Test-AzAppConfigurationStoreNameAvailability.md)
Verifica se o nome do repositório de configuração está disponível para uso.

### [Update-AzAppConfigurationStore](Update-AzAppConfigurationStore.md)
Atualiza um repositório de configuração com os parâmetros especificados.
