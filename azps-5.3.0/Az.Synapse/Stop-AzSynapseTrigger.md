# Stop-AzSynapseTrigger

## Sinopse
Interrompe um gatilho em um espaço de trabalho.

## SYNTAX

### StopByName (padrão)
```
Stop-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByObject
```
Stop-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzSynapseTrigger** interrompe um gatilho em um espaço de trabalho. Se o gatilho estiver no estado ' iniciado ', o cmdlet para o gatilho e não invoca mais os pipelines. Se o gatilho já estiver no estado ' interrompido ', esse cmdlet não terá efeito.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Stop-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Interrompe um gatilho chamado ContosoTrigger na ContosoWorkspace do espaço de trabalho.

### Exemplo 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Stop-AzSynapseTrigger -Name ContosoTrigger
```

Interrompe um gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace pelo pipeline.

### Exemplo 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Stop-AzSynapseTrigger
```

Pare um gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace por meio do pipeline.
