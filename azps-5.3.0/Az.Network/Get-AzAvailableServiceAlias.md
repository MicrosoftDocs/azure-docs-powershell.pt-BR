# Get-AzAvailableServiceAlias

## Sinopse
Obter aliases de serviço disponíveis na região.

## SYNTAX

```
Get-AzAvailableServiceAlias -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzAvailableServiceAlias** permite que você recupere todos os aliases de serviço disponíveis para uma sub-rede no local fornecido.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzAvailableServiceAlias -Location "westus"

Name                         Id                                                                                                                                   Type                                      ResourceName
----                         --                                                                                                                                   ----                                      ------------

```
