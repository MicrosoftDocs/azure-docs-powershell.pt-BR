# Set-AzApplicationGatewayAuthenticationCertificate

## Sinopse
Atualiza um certificado de autenticação para um gateway de aplicativo.

## SYNTAX

```
Set-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzApplicationGatewayAuthenticationCertificate** atualiza um certificado de autenticação para um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: atualizar um certificado de autenticação
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Set-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01" -CertificateFile "C:\cert2.cer"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

O primeiro comando obtém o gateway do aplicativo chamado appGwName e armazena o resultado na variável $appgw.
O segundo comando atualiza o certificado de autenticação chamado cert01 no gateway do aplicativo.
O terceiro comando atualiza o Application Gateway.
