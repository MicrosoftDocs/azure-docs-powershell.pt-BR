# Remove-AzApplicationGatewayClientAuthConfiguration

## Sinopse
Remove a configuração de autenticação de cliente de um objeto de perfil SSL.

## SYNTAX

```
Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayClientAuthConfiguration** remove a configuração de autenticação de cliente de um objeto de perfil SSL.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "Profile01" -ApplicationGateway $AppGw
PS C:\> Remove-AzApplicationGatewayClientAuthConfiguration -SslProfile $profile
```

O primeiro comando obtém o gateway do aplicativo chamado ApplicationGateway01 no grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw. O segundo comando obtém o perfil SSL chamado Profile01 para $AppGw e o armazena na variável $profile. O último comando Remove a configuração de autenticação de cliente do perfil SSL armazenado em $profile.
