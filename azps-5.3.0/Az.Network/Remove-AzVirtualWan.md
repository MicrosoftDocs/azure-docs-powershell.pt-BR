# Remove-AzVirtualWan

## Sinopse
Remove uma WAN virtual do Azure.

## SYNTAX

### ByVirtualWanName (padrão)
```
Remove-AzVirtualWan -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanObject
```
Remove-AzVirtualWan -InputObject <PSVirtualWan> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanResourceId
```
Remove-AzVirtualWan -ResourceId <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Remove uma WAN virtual do Azure.

## EXEMPLOS

### Exemplo 1

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Passthru
```

Este exemplo cria uma WAN virtual em um grupo de recursos e, em seguida, a exclui imediatamente. Para suprimir o prompt ao excluir a rede de longa distância virtual, use o sinalizador-Force.

### Exemplo 2

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -InputObject $virtualWan -Passthru
```

Este exemplo cria uma WAN virtual em um grupo de recursos e, em seguida, a exclui imediatamente. Essa exclusão ocorre usando o objeto de WAN virtual retornado por New-AzVirtualWan.
Para suprimir o prompt ao excluir a rede de longa distância virtual, use o sinalizador-Force.

### Exemplo 3

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -ResourceId $virtualWan.Id -Passthru
```

Este exemplo cria uma WAN virtual em um grupo de recursos e, em seguida, a exclui imediatamente. Essa exclusão ocorre usando a ID do recurso de WAN virtual retornada por New-AzVirtualWan.
Para suprimir o prompt ao excluir a rede de longa distância virtual, use o sinalizador-Force.
