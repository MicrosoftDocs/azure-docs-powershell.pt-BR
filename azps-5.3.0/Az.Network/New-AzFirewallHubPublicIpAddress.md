# New-AzFirewallHubPublicIpAddress

## Sinopse
Assoicated de IP público para o firewall no virtual Hub

## SYNTAX

```
New-AzFirewallHubPublicIpAddress [-Count <Int32>] [-Addresses <PSAzureFirewallPublicIpAddress[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Assoicated de IP público para o firewall no virtual Hub

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzFirewallHubPublicIpAddress -Count 2
```

Isso criará dois IPS públicos no firewall anexado ao Hub virtual. Isso criará o endereço IP no back-end. Não podemos fornecer IPAddresses explicitamente para um novo firewall.

### Exemplo 2
```powershell
PS C:\> $publicIp1 = New-AzFirewallPublicIpAddress -Address 10.2.3.4
PS C:\> $publicIp2 = New-AzFirewallPublicIpAddress -Address 20.56.37.46
PS C:\> New-AzFirewallHubPublicIpAddress -Count 3 -Addresses $publicIp1, $publicIp2
```

Isso criará um 1 novo IP público no firewall, mantendo $publicIp 1 $publicIp 2 que já existem no firewall.
