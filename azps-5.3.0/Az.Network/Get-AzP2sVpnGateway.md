# Get-AzP2sVpnGateway

## Sinopse
Obtém um P2SVpnGateway existente em VirtualHub.

## SYNTAX

### ListBySubscriptionId (padrão)
```
Get-AzP2sVpnGateway [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ListByResourceGroupName
```
Get-AzP2sVpnGateway [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzP2sVpnGateway** permite que você obtenha um P2SVpnGateway existente em VirtualHub.
