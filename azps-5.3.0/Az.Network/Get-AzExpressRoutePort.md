# Get-AzExpressRoutePort

## Sinopse
Obtém um recurso do Azure ExpressRoutePort.

## SYNTAX

### ResourceNameParameterSet (padrão)
```
Get-AzExpressRoutePort [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePort -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzExpressRoutePort** é usado para recuperar um objeto ExpressRoutePort da sua assinatura. O objeto expressrouteport retornado pode ser usado como entrada para outros cmdlets que operam em ExpressRoutePort.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Obtém o objeto ExpressRoutePort com o nome $PortName no grupo de recursos $rg na sua assinatura.

### Exemplo 2
```powershell
PS C:\> Get-AzExpressRoutePort -Name test*
```

Obtém todos os objetos ExpressRoutePort cujo nome começa com "Test".

### Exemplo 3
```powershell
PS C:\> Get-AzExpressRoutePort -ResourceId $id
```

Obtém o objeto ExpressRoutePort com $id ResourceId.
