# Get-AzNetworkVirtualAppliance

## Sinopse
Obter ou listar dispositivos virtuais de rede.

## SYNTAX

### ResourceNameParameterSet (padrão)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
Os comandos Get-AzNetworkVirtualAppliance Obtém ou lista recursos de dispositivos virtuais da rede.
