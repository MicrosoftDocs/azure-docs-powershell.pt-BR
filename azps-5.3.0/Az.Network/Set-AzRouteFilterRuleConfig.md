# Set-AzRouteFilterRuleConfig

## Sinopse
Modifica a regra de filtro de rota de um filtro de rota.

## SYNTAX

```
Set-AzRouteFilterRuleConfig -RouteFilter <PSRouteFilter> [-Force] -Name <String> -Access <String>
 -RouteFilterRuleType <String> -CommunityList <String[]> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzRouteFilterRuleConfig** modifica a regra de filtro de rota de um filtro de rota.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
PS C:\> $rf = Set-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01" -Access Deny -RouteFilterRuleType Community -CommunityList "12076:5010","12076:5040"
PS C:\> Set-AzRouteFilter -RouteFilter $rf
```

O primeiro comando obtém o filtro de rota chamado RouteFilter01 e o armazena na variável $rf.
O segundo comando modifica a regra de filtro de rota chamada Rule01 e armazena o filtro de rota atualizado na variável $rf.
O terceiro comando salva o filtro de rota atualizado.
