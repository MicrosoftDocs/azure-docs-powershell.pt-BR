# Remove-AzApplicationGatewayAuthenticationCertificate

## Sinopse
Remove um certificado de autenticação de um gateway de aplicativo.

## SYNTAX

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayAuthenticationCertificate** remove um certificado de autenticação de um Application Gateway do Azure.

## EXEMPLOS

### Exemplo 1: remover um certificado de autenticação de um aplicativo gateway
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

O primeiro comando obtém o gateway do aplicativo chamado appGwName e armazena o resultado na variável $appgw.
O segundo comando Remove o certificado de autenticação chamado cert01 do gateway do aplicativo.
O terceiro comando atualiza o Application Gateway.
