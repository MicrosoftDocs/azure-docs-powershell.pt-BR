# Remove-AzApplicationGatewayPrivateLinkConfiguration

## Sinopse
Remove uma configuração de privateLink de um gateway de aplicativo.

## SYNTAX

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayPrivateLinkConfiguration** remove uma configuração privateLink de um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: remover uma configuração de PrivateLink do gateway do aplicativo
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na variável $AppGw.
O segundo comando Remove a configuração privateLink chamada privateLinkConfig01 do gateway do aplicativo armazenado em $AppGw.
