# Save-AzTenantDeploymentTemplate

## Sinopse
Salva um modelo de implantação em um arquivo.

## SYNTAX

### SaveByDeploymentName (padrão)
```
Save-AzTenantDeploymentTemplate -DeploymentName <String> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SaveByDeploymentObject
```
Save-AzTenantDeploymentTemplate -DeploymentObject <PSDeployment> [-Path <String>] [-Force] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Save-AzTenantDeploymentTemplate**  salva um modelo de implantação em um arquivo JSON para uma implantação no escopo do locatário.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Save-AzTenantDeploymentTemplate -DeploymentName "TestDeployment"
```

Esse comando obtém o modelo de implantação do TestDeployment e o salva como um arquivo JSON no diretório atual.

### Exemplo 2: obter uma implantação e salvar seu modelo
```
PS C:\>Get-AzTenantDeploymentTemplate -Name "RolesDeployment" | Save-AzTenantDeploymentTemplate
```

Esse comando obtém a implantação "RolesDeployment" no escopo do locatário atual e salva seu modelo.
