# Remove-AzWvdRegistrationInfo

## Sinopse
Remova as informações de registro da área de trabalho virtual do Windows.

## SYNTAX

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Remova as informações de registro da área de trabalho virtual do Windows.

## EXEMPLOS

### Exemplo 1: excluir um token de registro da área de trabalho virtual do Windows
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Este comando exclui um token de registro da área de trabalho virtual do Windows em um pool de hosts.
