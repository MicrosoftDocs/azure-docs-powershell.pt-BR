# Módulo AZS. inscrições
## Descritivo
Versão prévia do módulo de assinaturas do azurestack para os usuários do locatário. Isso permite que o functianality exiba e se inscreva em ofertas públicas e gerenciem assinaturas.

## Cmdlets AZS. inscrições
### [Get-AzsDelegatedProviderOffer](Get-AzsDelegatedProviderOffer.md)
Obter a lista de ofertas para o provedor delegado especificado.

### [Get-AzsOffer](Get-AzsOffer.md)
Obtenha a lista de ofertas públicas.

### [Get-AzsSubscription](Get-AzsSubscription.md)
Obter a lista de assinaturas.

### [New-AzsSubscription](New-AzsSubscription.md)
Criar uma assinatura.

### [Remove-AzsSubscription](Remove-AzsSubscription.md)
Exclua a assinatura especificada.

### [Set-AzsSubscription](Set-AzsSubscription.md)
Criar ou atualizar uma assinatura.
