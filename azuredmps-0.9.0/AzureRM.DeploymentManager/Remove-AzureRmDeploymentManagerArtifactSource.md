# Remove-AzureRmDeploymentManagerArtifactSource

## Sinopse
Exclui uma fonte de artefato.

## SYNTAX

### Interativo (padrão)
```
Remove-AzureRmDeploymentManagerArtifactSource [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Identificação
```
Remove-AzureRmDeploymentManagerArtifactSource [-ResourceId] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzureRmDeploymentManagerArtifactSource [-ArtifactSource] <PSArtifactSource> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmDeploymentManagerArtifactSource** exclui uma fonte de artefato especifica a origem do artefato por seu nome e nome do grupo de recursos. Como alternativa, você pode fornecer o objeto artefato ou a ResourceId.

## EXEMPLOS

### Exemplo 1: excluir uma fonte de artefato
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ResourceGroupName "ContosoResourceGroup" -Name "ContosoArtifactSource"
```

Esse comando exclui uma fonte de artefato nomeada ContosoArtifactSource em ContosoResourceGroup.

### Exemplo 2: excluir uma fonte de artefato usando o identificador de recursos
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/artifactSources/ContosoArtifactSource"
```

Esse comando exclui uma fonte de artefato nomeada ContosoArtifactSource em ContosoResourceGroup.

### Exemplo 3: excluir uma fonte de artefato usando um objeto
```powershell
PS C:\> Remove-AzureRmDeploymentManagerArtifactSource -ArtifactSource $artifactSourceObject
```

Esse comando exclui uma fonte de artefato cujo nome e o meu nome do participante correspondem às propriedades Name e ResourceGroupName do $artifactSourceObject, respectivamente.
