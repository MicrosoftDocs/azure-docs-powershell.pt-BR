# Set-AzureRmDeploymentManagerStep

## Sinopse
Atualiza uma etapa.

## SYNTAX

```
Set-AzureRmDeploymentManagerStep [-Step] <PSStepResource> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmDeploymentManagerStep** atualiza uma etapa com o objeto Step especificado.
O cmdlet retorna o objeto Step atualizado.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzureRmDeploymentManagerStep -Step $stepObject
```

Esse comando atualiza uma etapa cujo nome e o meu nome do elemento de Resource correspondem às propriedades Name e ResourceGroupName do $stepObject, respectivamente.
A etapa seria atualizada para as propriedades definidas no $stepObject.
