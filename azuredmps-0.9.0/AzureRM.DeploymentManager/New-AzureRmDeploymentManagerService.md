# New-AzureRmDeploymentManagerService

## Sinopse
Cria um serviço em uma topologia de serviço.

## SYNTAX

### Interativo (padrão)
```
New-AzureRmDeploymentManagerService [-ResourceGroupName] <String> [-ServiceTopologyName] <String>
 -Name <String> -Location <String> -TargetLocation <String> -TargetSubscriptionId <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByServiceTopologyObject
```
New-AzureRmDeploymentManagerService [-ResourceGroupName] <String> -Name <String> -Location <String>
 -TargetLocation <String> -TargetSubscriptionId <String> [-ServiceTopology] <PSServiceTopologyResource>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByServiceTopologyResourceId
```
New-AzureRmDeploymentManagerService [-ResourceGroupName] <String> -Name <String> -Location <String>
 -TargetLocation <String> -TargetSubscriptionId <String> [-ServiceTopologyId] <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmDeploymentManagerService** cria um serviço em uma topologia de serviço e retorna um objeto que representa esse serviço.
Especifique o serviço pelo nome, topologia de serviço em que ele se encontra e o nome do grupo de recursos. 

O cmdlet retorna um objeto de serviço. Você pode modificar esse objeto localmente e, em seguida, aplicar as alterações ao serviço usando o cmdlet Set-AzureRmDeploymentManagerService.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzureRmDeploymentManagerService -ResourceGroupName ContosoResourceGroup -ServiceTopologyName ContosoServiceTopology -Name ContosoService1 -Location "Central US" -TargetLocation "East US" -TargetSubscriptionId XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX
```

Cria um novo serviço com o nome ContosoService1 em ContosoServiceTopology de topologia de serviço no grupo de recursos ContosoResourceGroup, no centro de localização dos EUA. A propriedade TargetLocation indica que o serviço ContosoService1 deve ser implantado na região leste dos EUA na assinatura especificada.
