# New-AzureRmDataFactoryV2LinkedServiceEncryptedCredential

## Sinopse
Criptografar credencial no serviço vinculado com tempo de execução de integração especificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### ByFactoryName (padrão)
```
New-AzureRmDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-WhatIf] [-Confirm]
```

### ByFactoryObject
```
New-AzureRmDataFactoryV2LinkedServiceEncryptedCredential [-IntegrationRuntimeName] <String>
 [-DefinitionFile] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-WhatIf] [-Confirm]
```

## DESCRITIVO
O cmdlet New-AzureRmDataFactoryV2LinkedServiceEncryptCredential criptografar credenciais no serviço vinculado com tempo de execução de integração especificado.

## EXEMPLOS

### Exemplo 1: criptografar o creadentials em um serviço vinculado
```
PS C:\> New-AzureRmDataFactoryV2LinkedServiceEncryptCredential -ResourceGroupName resourceGroup -DataFactoryName myDataFactory -IntegrationRuntimeName myIR -File D:\sql.json
```

Esse comando criptografa a credencial no arquivo D:\sql.jscom o tempo de execução de integração chamado myIR.
