# Stop-AzureRmDataFactoryV2IntegrationRuntime

## Sinopse
Interrompe um tempo de execução de integração dedicada gerenciado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### ByIntegrationRuntimeName (padrão)
```
Stop-AzureRmDataFactoryV2IntegrationRuntime [-Force] [-Name] <String> [-ResourceGroupName] <String>
 [-DataFactoryName] <String> [-WhatIf] [-Confirm]
```

### ByResourceId
```
Stop-AzureRmDataFactoryV2IntegrationRuntime [-Force] [-ResourceId] <String> [-WhatIf] [-Confirm]
```

### ByIntegrationRuntimeObject
```
Stop-AzureRmDataFactoryV2IntegrationRuntime [-Force] [-InputObject] <PSIntegrationRuntime> [-WhatIf]
 [-Confirm]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmDataFactoryV2IntegrationRuntime** interrompe um tempo de execução de integração dedicada gerenciado no estado ' iniciado ', que foi iniciado pelo cmdlet Start-AzureRmDataFactoryV2IntegrationRuntime. Os recursos são liberados e o estado é transferido para ' parado '.

## EXEMPLOS

### Exemplo 1: parar um tempo de execução de integração gerenciada que está no estado ' iniciado '.
```
PS C:\> Stop-AzureRmDataFactoryV2IntegrationRuntime -ResourceGroupName 'rg-test-dfv2' -DataFactoryName 'test-df' -Name 'test-reserlved-ir'
```

O tempo de execução de integração gerenciada ' test-reserlved-IV ' está no estado ' iniciado '. Após a execução do cmdlet Stop-AzureRmDataFactoryV2IntegrationRuntime, os recursos são liberados e o estado é transferido para ' parado '.
