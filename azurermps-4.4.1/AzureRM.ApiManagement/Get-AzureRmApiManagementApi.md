# Get-AzureRmApiManagementApi

## Sinopse
Obtém uma API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### Todas as APIs (padrão)
```
Get-AzureRmApiManagementApi -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Localizar por ID
```
Get-AzureRmApiManagementApi -Context <PsApiManagementContext> -ApiId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Localizar por nome
```
Get-AzureRmApiManagementApi -Context <PsApiManagementContext> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Localizar por ID do produto
```
Get-AzureRmApiManagementApi -Context <PsApiManagementContext> -ProductId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmApiManagementApi** Obtém uma ou mais APIs de gerenciamento de API do Azure.

## EXEMPLOS

### Exemplo 1: obter todas as APIs de gerenciamento
```
PS C:\>Get-AzureRmApiManagementApi -Context $ApiMgmtContext
```

Esse comando obtém todas as APIs do contexto especificado.

### Exemplo 2: obter uma API de gerenciamento por ID
```
PS C:\>Get-AzureRmApiManagementApi -Context $ApiMgmtContext -ApiId $ApiId
```

Este comando obtém a API com a ID especificada.

### Exemplo 3: obter uma API de gerenciamento por nome
```
PS C:\>Get-AzureRmApiManagementApi -Context $ApiMgmtContext -Name "EchoApi"
```

Este comando obtém a API com o nome especificado.
