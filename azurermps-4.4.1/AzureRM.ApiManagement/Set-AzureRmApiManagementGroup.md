# Set-AzureRmApiManagementGroup

## Sinopse
Configura um grupo de gerenciamento de API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-Name <String>]
 [-Description <String>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmApiManagementGroup** configura um grupo de gerenciamento de API.

## EXEMPLOS

### Exemplo 1: configurar um grupo de gerenciamento
```
PS C:\>Set-AzureRmApiManagementGroup -Context $APImContext -Description "Updated Management Group" -Name "Group0001"
```

Esse comando configura um grupo de gerenciamento chamado Group0001.
