# Get-AzureRmBatchAccountKeys

## Sinopse
Obtém as chaves de uma conta em lotes.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmBatchAccountKeys [-AccountName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmBatchAccountKeys** Obtém as chaves de uma conta em lotes do Azure na assinatura atual.
