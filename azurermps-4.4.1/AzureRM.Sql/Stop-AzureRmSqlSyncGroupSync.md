# Stop-AzureRmSqlSyncGroupSync

## Sinopse
Interrompe a sincronização de um grupo de sincronização.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String>
 [-DatabaseName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmSqlSyncGroupSync** interrompe uma sincronização de grupo de sincronização.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Stop-AzureRmSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Esse comando interrompe a sincronização que está em andamento para o grupo de sincronização mysg.
