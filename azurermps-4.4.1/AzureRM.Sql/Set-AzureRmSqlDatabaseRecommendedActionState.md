# Set-AzureRmSqlDatabaseRecommendedActionState

## Sinopse
Atualiza o estado de uma ação recomendada do banco de dados SQL do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Set-AzureRmSqlDatabaseRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -DatabaseName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureRmSqlDatabaseRecommendedActionState** atualiza o estado de uma ação recomendada do banco de dados SQL do Azure.
Isso permite que uma ação recomendada seja aplicada, revertida ou descartada com base no novo estado.
