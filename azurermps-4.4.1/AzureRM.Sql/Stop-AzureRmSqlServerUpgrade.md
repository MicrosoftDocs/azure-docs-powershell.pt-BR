# Stop-AzureRmSqlServerUpgrade

## Sinopse
Interrompe a atualização de um servidor de banco de dados SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmSqlServerUpgrade** interrompe a atualização de um servidor de banco de dados SQL do Azure.

## EXEMPLOS

### Exemplo 1: parar uma atualização do servidor
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

Esse comando interrompe a solicitação de atualização do servidor chamado Server02 atribuído ao grupo de recursos chamado ResourceGroup01.
