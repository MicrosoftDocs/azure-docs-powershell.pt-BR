# Get-AzureRmSqlServerBackupLongTermRetentionVault

## Sinopse
Obtém um cofre de retenção de longo prazo do servidor.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmSqlServerBackupLongTermRetentionVault [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSqlServerBackupLongTermRetentionVault** Obtém o cofre de retenção de longo prazo registrado neste servidor.
O cofre é um recurso de backup do Azure usado para armazenar dados de backup.
