# Suspend-AzureRmSqlDatabase

## Sinopse
Suspende um banco de dados de data warehouse do SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Suspend-AzureRmSqlDatabase [-ServerName] <String> -DatabaseName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Suspend-AzureRmSqlDatabase** suspende um banco de dados de SQL data warehouse do Azure.

## EXEMPLOS

### Exemplo 1: suspende um banco de dados de SQL data warehouse do Azure
```
PS C:\>Suspend-AzureRmSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando suspende um banco de dados ativo do Azure SQL data warehouse.
