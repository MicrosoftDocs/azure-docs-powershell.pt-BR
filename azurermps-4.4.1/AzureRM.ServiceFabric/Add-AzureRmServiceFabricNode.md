# Add-AzureRmServiceFabricNode

## Sinopse
Adicione nós ao tipo de nó específico no cluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Add-AzureRmServiceFabricNode [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 -NumberOfNodesToAdd <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
Use **Add-AzureRmServiceFabricNode** para adicionar nós ao tipo de nó específico. Você só precisa especificar o número de nós que deseja adicionar a um tipo de nó.

## EXEMPLOS

### Exemplo 1
```
PS c:> Add-AzureRmServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NumberOfNodesToAdd 2 -NodeTypeName 'nt1'
```

Esse comando adicionará 2 nós ao tipo de nó ' N1 '.
