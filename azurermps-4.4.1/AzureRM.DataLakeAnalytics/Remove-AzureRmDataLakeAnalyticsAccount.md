# Remove-AzureRmDataLakeAnalyticsAccount

## Sinopse
Exclui uma conta do data Lake Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmDataLakeAnalyticsAccount** exclui permanentemente uma conta do Azure data Lake Analytics.

## EXEMPLOS

### Exemplo 1: remover uma conta
```
PS C:\>Remove-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Este comando Remove a conta do data Lake Analytics especificada.
