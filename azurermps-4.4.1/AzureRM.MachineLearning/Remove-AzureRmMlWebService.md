# Remove-AzureRmMlWebService

## Sinopse
Exclui um serviço Web.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### Remover um recurso do serviço Web do Azure ML por nome e grupo de recursos.
```
Remove-AzureRmMlWebService -ResourceGroupName <String> -Name <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Remover um serviço Web do Azure ML especificado como um objeto.
```
Remove-AzureRmMlWebService -MlWebService <WebService> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Exclui um serviço Web do Azure Machine Learning referenciado pelo nome e grupo de recursos.
