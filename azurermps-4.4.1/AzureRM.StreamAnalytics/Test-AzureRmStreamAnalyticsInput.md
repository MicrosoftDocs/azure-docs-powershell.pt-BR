# Test-AzureRmStreamAnalyticsInput

## Sinopse
Testa o status de conexão de uma entrada.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Test-AzureRmStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Test-AzureRmStreamAnalyticsInput** testa a capacidade do Stream Analytics de se conectar a uma entrada.

## EXEMPLOS

### EXEMPLO 1: testar o status de conexão de um fluxo de entrada
```
PS C:\>Test-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Isso testa o status de conexão da entrada EntryStream em StreamingJob.
