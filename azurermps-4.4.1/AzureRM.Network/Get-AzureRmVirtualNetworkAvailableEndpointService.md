# Get-AzureRmVirtualNetworkAvailableEndpointService

## Sinopse
Lista os serviços de ponto de extremidade disponíveis para localização.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmVirtualNetworkAvailableEndpointService -Location <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
Get-AzureRmVirtualNetworkAvailableEndpointService lista os serviços de ponto de extremidade disponíveis no local especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzureRmVirtualNetworkAvailableEndpointService -Location westus

-Name              Id                                                                                             Type
-----              --                                                                                             ----
-Microsoft.Storage /subscriptions/id/providers/Microsoft.Network/virtualNetworkEndpointServices/Microsoft.Storage Microsoft.Network/virtualNetworkEndpointServices
```

Obtém serviços de ponto de extremidade disponíveis na região oeste.
