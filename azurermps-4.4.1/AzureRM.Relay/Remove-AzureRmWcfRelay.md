# Remove-AzureRmWcfRelay

## Sinopse
Remove o WcfRelay do namespace especificado de retransmissão.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmWcfRelay [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmWcfRelay** remove o WcfRelay do namespace de retransmissão especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzureRmWcfRelay -ResourceGroupName Default-ServiceBus-WestUS -NamespaceName TestNameSpace-Relay1 -Name TestWCFRelay1
```

Remove o WcfRelay `TestWCFRelay1` do namespace `TestNameSpace-Relay1` .
