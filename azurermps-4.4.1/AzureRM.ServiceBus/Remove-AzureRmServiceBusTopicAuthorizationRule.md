# Remove-AzureRmServiceBusTopicAuthorizationRule

## Sinopse
Remove a regra de autorização de um tópico do namespace especificado de barramento de serviço.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceBusTopicAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Topic <String>
 -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmServiceBusTopicAuthorizationRule** remove a regra de autorização de um tópico do namespace de barramento de serviço especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzureRmServiceBusTopicAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1 -AuthorizationRuleName SBTopicAuthoRule1
```

Remove a regra `SBTopicAuthoRule1` de autorização do tópico `SB-Topic_exampl1` do namespace `SB-Example1` .
