# Get-AzureRmServiceBusOperation

## Sinopse
Lista de operações do ServiceBus com suporte

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmServiceBusOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmServiceBusOperation** lista as operações com suporte do ServiceBus.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzureRmServiceBusOperation
```

Lista as operações com suporte do ServiceBus
