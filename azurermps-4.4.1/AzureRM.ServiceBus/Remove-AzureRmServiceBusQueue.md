# Remove-AzureRmServiceBusQueue

## Sinopse
Remove a fila do namespace especificado de barramento de serviço.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceBusQueue -ResourceGroupName <String> -Namespace <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmServiceBusQueue** remove a fila do namespace de barramento de serviço especificado.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Remove-AzureRmServiceBusQueue -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1
```

Remove a fila `SB-Queue_exampl1` do namespace `SB-Example1` .
