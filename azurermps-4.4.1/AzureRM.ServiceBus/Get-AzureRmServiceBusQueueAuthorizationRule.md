# Get-AzureRmServiceBusQueueAuthorizationRule

## Sinopse
Obtém a descrição de uma regra de autorização especificada para uma determinada fila de barramento de serviço. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmServiceBusQueueAuthorizationRule [-ResourceGroup] <String> -Namespace <String> -Queue <String>
 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmServiceBusQueueAuthorizationRule** Obtém a descrição de uma regra de autorização especificada em uma determinada fila de barramento do serviço.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Get-AzureRmServiceBusQueueAuthorizationRule -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -QueueName SB-Queue_exampl1 -AuthorizationRuleName SBAuthoRule1
```

Retorna a descrição da regra de autorização especificada para uma determinada fila de barramento do serviço.
