# Remove-AzureKeyVaultCertificateIssuer

## Sinopse
Exclui um emissor de certificado de um cofre de chaves.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureKeyVaultCertificateIssuer [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureKeyVaultCertificateIssuer** exclui um emissor de certificado de um cofre de chaves.

## EXEMPLOS

### Exemplo 1: remover um emissor de certificado
```
PS C:\>Remove-AzureKeyVaultCertificateIssuer -VaultName "ContosoKV01" -Name "TestIssuer01" -Force
```

Esse comando Remove o emissor do certificado chamado TestIssuer01 do cofre da chave ContosoKV01.
