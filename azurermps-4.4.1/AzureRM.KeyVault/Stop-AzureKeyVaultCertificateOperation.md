# Stop-AzureKeyVaultCertificateOperation

## Sinopse
Cancela uma operação de certificado no cofre de chaves.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Stop-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureKeyVaultCertificateOperation** cancela uma operação de certificado no serviço de compartimento de chave do Azure.
