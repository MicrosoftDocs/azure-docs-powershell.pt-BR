# Remove-AzureRmRedisCache

## Sinopse
Remove um cache Redis.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmRedisCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmRedisCache** remove um cache do Azure Redis.

## EXEMPLOS

### Exemplo 1: remover um cache Redis e retornar o resultado
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Esse comando Remove um cache Redis e exibe se a operação é bem-sucedida.

### Exemplo 2: remover um cache do Redis e não exibir o resultado
```
PS C:\>Remove-AzureRmRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Esse comando Remove um cache Redis.
Como o parâmetro *PassThru* não é especificado, o resultado da operação não é exibido.
