# Remove-AzureRmStorageAccount

## Sinopse
Remove uma conta de armazenamento do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmStorageAccount** remove uma conta de armazenamento do Azure.

## EXEMPLOS

### Exemplo 1: remover uma conta de armazenamento
```
PS C:\>Remove-AzureRmStorageAccount -ResourceGroupName "RG01" -AccountName "MyStorageAccount"
```

Este comando Remove a conta de armazenamento especificada.
