# Get-AzsRPHealth

## Sinopse
Retorna uma lista da integridade de cada serviço.

## SYNTAX

### Lista (padrão)
```
Get-AzsRPHealth [-Location <String>] [-ResourceGroupName <String>] [-Filter <String>] [-Skip <Int32>]
 [-Top <Int32>] [<CommonParameters>]
```

### Obter
```
Get-AzsRPHealth [-Name] <String> [-Location <String>] [-ResourceGroupName <String>] [<CommonParameters>]
```

### Identificação
```
Get-AzsRPHealth -ResourceId <String> [<CommonParameters>]
```

## DESCRITIVO
Retorna uma lista da integridade de cada serviço. A propriedade AlertSummary inclui detalhes sobre contagens de aviso/erro.

## EXEMPLOS

### EXEMPLO 1
```
Get-AzsRPHealth
```

Retorna uma lista da integridade de cada serviço.

### EXEMPLO 2
```
Get-AzsRPHealth -Name "e56bc7b8-c8b5-4e25-b00c-4f951effb22c"
```

Retorna a integridade de um serviço.
