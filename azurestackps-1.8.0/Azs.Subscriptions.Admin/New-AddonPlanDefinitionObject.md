# New-AddonPlanDefinitionObject

## Sinopse
Contém o nome do plano desejado para vincular ou desvinculado de uma oferta.

## SYNTAX

```

```

## DESCRITIVO
Contém o nome do plano desejado para vincular ou desvinculado de uma oferta.
