# Módulo AZS. Gallery. admin
## Descritivo
Versão prévia do módulo do administrador da Galeria AzureStack, que fornece funcionalidade para gerenciar itens da galeria no Marketplace.

## AZS. Gallery. cmdlets de administração
### [Add-AzsGalleryItem](Add-AzsGalleryItem.md)
Carrega um item de galeria de provedores para o armazenamento.

### [Get-AzsGalleryItem](Get-AzsGalleryItem.md)
Lista itens da galeria.

### [Remove-AzsGalleryItem](Remove-AzsGalleryItem.md)
Excluir um item de galeria específico.
