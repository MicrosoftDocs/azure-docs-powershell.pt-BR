# Módulo AZS. Update. admin
## Descritivo
Versão prévia do módulo do administrador de atualização de pilha do Azure.  Neste módulo, os administradores podem:
- Listar e instalar as atualizações disponíveis
- Retomar atualizações interrompidas
- Exibir atualizações instaladas

## Cmdlets do AZS. Update. admin
### [Get-AzsUpdate](Get-AzsUpdate.md)
Obter a lista de atualizações disponíveis.

### [Get-AzsUpdateLocation](Get-AzsUpdateLocation.md)
Obter a lista de locais de atualização.

### [Get-AzsUpdateRun](Get-AzsUpdateRun.md)
Obter a lista de execuções de atualização.

### [Install-AzsUpdate](Install-AzsUpdate.md)
Aplicar uma atualização específica em um local de atualização.

### [Currículo-AzsUpdateRun](Resume-AzsUpdateRun.md)
Retoma uma execução de atualização iniciada anteriormente que falhou.
