# Grant-AzureRmHDInsightHttpServicesAccess

## Sinopse
Concede acesso HTTP ao cluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Grant-AzureRmHDInsightHttpServicesAccess [-ClusterName] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Grant-AzureRmHDInsightHttpServicesAccess** concede acesso http a um cluster do Azure HDINSIGHT usando ODBC, Ambari, Oozie e Web Services.

## EXEMPLOS

### Exemplo 1: conceder acesso HTTP a um cluster do Azure HDInsight
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzureRmHDInsightHttpServicesAccess `
            -ClusterName $clusterName `
            -HttpCredential $newClusterCreds
```

Esse comando concede acesso HTTP ao cluster chamado seu-Hadoop-001.
