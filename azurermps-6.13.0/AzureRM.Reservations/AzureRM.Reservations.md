# Módulo AzureRM. reservas
## Descritivo
Os tópicos desta seção documentam os cmdlets do Azure PowerShell para reservas do Azure na estrutura ARM (Azure Resource Manager). Os cmdlets existem no namespace Microsoft. Azure. Commands. reservas.

## Cmdlets AzureRM. reservas
### [Get-AzureRmReservation](Get-AzureRmReservation.md)
Obter `Reservation` s em uma determinada ordem de reserva

### [Get-AzureRmReservationCatalog](Get-AzureRmReservationCatalog.md)
Obter o catálogo de reservas disponíveis

### [Get-AzureRmReservationHistory](Get-AzureRmReservationHistory.md)
Obter `Reservation` histórico de revisão.

### [Get-AzureRmReservationOrder](Get-AzureRmReservationOrder.md)
Obter `ReservationOrder`

### [Get-AzureRmReservationOrderId](Get-AzureRmReservationOrderId.md)
Obtenha a lista de `ReservationOrder` IDs aplicáveis.

### [Mesclar AzureRmReservation](Merge-AzureRmReservation.md)
Combina dois `Reservation` s.

### [Split-AzureRmReservation](Split-AzureRmReservation.md)
Dividir a `Reservation` .

### [Update-AzureRmReservation](Update-AzureRmReservation.md)
Atualizar um `Reservation` .
