# Get-AzureRmFrontDoor

## Sinopse
Obter balanceador de carga de porta frontal

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdletGet **Get-AzureRmFrontDoor** obtém todas as portas frontais existentes na assinatura atual que corresponde às informações fornecidas.
