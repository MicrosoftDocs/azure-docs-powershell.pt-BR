# New-AzureRmRecoveryServicesVault

## Sinopse
Cria um novo cofre de serviços de recuperação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmRecoveryServicesVault** cria um novo cofre de serviços de recuperação.

## EXEMPLOS

### Exemplo 1
```
PS C:\> New-AzureRmRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Criar cofre de serviços de recuperação no grupo de recursos e determinada localização.
