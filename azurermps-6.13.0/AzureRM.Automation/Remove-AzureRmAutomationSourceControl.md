# Remove-AzureRmAutomationSourceControl

## Sinopse
Remove um controle de origem da automação do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Remove-AzureRmAutomationSourceControl remove um controle de origem da automação do Azure.

## EXEMPLOS

### Exemplo 1
Esse comando Remove o controle de origem de automação chamado VSTSNative na conta chamada devAccount.
Esse comando especifica o parâmetro *Force* . Portanto, ele não solicita a confirmação.

```powershell
PS C:\> Remove-AzureRmAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
