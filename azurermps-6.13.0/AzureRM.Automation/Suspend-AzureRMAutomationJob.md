# Suspend-AzureRmAutomationJob

## Sinopse
Suspende um trabalho de automação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Suspend-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Suspend-AzureRmAutomationJob** suspende um trabalho de automação do Azure.
Especifique um trabalho de automação em execução.
Para retomar um trabalho suspenso, use o cmdlet Resume-AzureRmAutomationJob.

## EXEMPLOS

### Exemplo 1: suspender um trabalho
```
PS C:\>Suspend-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Esse comando suspende o trabalho que tem a ID especificada.
