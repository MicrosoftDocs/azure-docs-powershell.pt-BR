# Remove-AzureRmAutomationHybridWorkerGroup

## Sinopse
Remove o grupo do Hybrid Worker da automação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Remove-AzureRmAutomationHybridWorkerGroup remove um grupo do Hybrid Worker da automação.

## EXEMPLOS

### Exemplo 1
Esse comando Remove um trabalhador híbrido por nome.

```powershell
PS C:\> Remove-AzureRmAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
