# New-AzureRmSqlSyncAgentKey

## Sinopse
Cria uma chave do agente do SQL Sync do Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmSqlSyncAgentKey** cria uma chave do agente do SQL Sync do Azure.

## EXEMPLOS

### Exemplo 1: criar uma chave do agente de sincronização para um agente de sincronização do SQL do Azure.
```
PS C:\> New-AzureRmSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Esse comando cria uma chave do agente de sincronização para um agente do Azure SQL Sync.
