# Remove-AzureRmSqlServerAuditing

## Sinopse
Remove a auditoria de um SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmSqlServerAuditing [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmSqlServerAuditing** remove a auditoria de um SQL Server do Azure.
Para usar esse cmdlet, especifique os parâmetros *ResourceGroupName* e *ServerName* para identificar o servidor.
Após executar esse cmdlet, a auditoria dos bancos de dados no Azure SQL Server não é realizada.
Se o comando for bem-sucedido e você especificar o parâmetro *PassThru* , o cmdlet retornará um objeto que descreve a política de auditoria atual e os identificadores do SQL Server do Azure.
Os identificadores de servidor incluem **ResourceGroupName** e **nomedoservidor**.

## EXEMPLOS

### Exemplo 1: remover a auditoria de um SQL Server do Azure
```
PS C:\>Remove-AzureRmSqlServerAuditing -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Esse comando Remove a auditoria de todos os bancos de dados localizados em Server01 no grupo de recursos.
