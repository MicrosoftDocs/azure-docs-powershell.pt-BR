# Get-AzureRmSqlInstance

## Sinopse
Retorna informações sobre a instância de banco de dados gerenciado do Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### GetInstanceByResourceGroup (padrão)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmSqlInstance** retorna informações sobre uma ou mais instâncias gerenciadas do SQL do Azure.
Especifique o nome de uma instância para ver informações somente para essa instância.
