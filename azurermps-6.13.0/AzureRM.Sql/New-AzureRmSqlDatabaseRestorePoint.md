# New-AzureRmSqlDatabaseRestorePoint

## Sinopse
Cria um novo ponto de restauração a partir do qual um banco de dados SQL pode ser restaurado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmSqlDatabaseRestorePoint -RestorePointLabel <String> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmSqlDatabaseRestorePoint** cria um novo ponto de restauração no qual um SQL data warehouse do Azure pode ser restaurado.
Este cmdlet tem suporte no momento para SQL data warehouse do Azure.

## EXEMPLOS

### Exemplo 1: criar um ponto de restauração
```
PS C:\>New-AzureRmSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -RestorePointLabel "RestorePoint01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : DISCRETE
RestorePointCreationDate : 8/12/2015 12:00:00 AM
EarliestRestoreDate      : 
RestorePointLabel        : RestorePoint01
```

Esse comando cria um ponto de restauração para o SQL data warehouse do Azure e retorna os detalhes do ponto de restauração.
