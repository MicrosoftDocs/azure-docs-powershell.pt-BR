# Remove-AzureRmIotHubKey

## Sinopse
Remove uma chave IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Remove uma chave IotHub.
Se houver várias chaves com o mesmo nome que a primeira da lista será removida.

## EXEMPLOS

### Exemplo 1 excluir um IotHub
```
PS C:\> Remove-AzureRmIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Remove a chave chamada iothubowner1 da IotHub chamada "myiothub"
