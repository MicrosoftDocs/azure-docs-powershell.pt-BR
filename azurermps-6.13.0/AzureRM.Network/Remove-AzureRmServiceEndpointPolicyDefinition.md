# Remove-AzureRmServiceEndpointPolicyDefinition

## Sinopse
{{Preencher o resumo}}

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmServiceEndpointPolicy** remove uma política de ponto de extremidade de serviço.

## EXEMPLOS

### Exemplo 1: Remove uma política de ponto de extremidade do serviço usando Name
```
Remove-AzureRmServiceEndpointPolicyDefinition -Name "PolicyDef1"
```

Esse comando Remove uma política de ponto de extremidade de serviço com o nome PolicyDef1
