# Add-AzureRmApplicationGatewayIPConfiguration

## Sinopse
Adiciona uma configuração de IP a um gateway de aplicativo.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### SetByResourceId
```
Add-AzureRmApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SubnetId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Add-AzureRmApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-Subnet <PSSubnet>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzureRmApplicationGatewayIPConfiguration** adiciona uma configuração de IP a um gateway de aplicativo.
As configurações de IP contêm a sub-rede na qual o Application Gateway é implantado.

## EXEMPLOS

### Exemplo 1: adicionar uma configuração de rede virtual a um gateway de aplicativo
```
PS C:\>$Vnet = Get-AzureRmVirtualNetwork -Name "Vnet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzureRmVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $Vnet 
PS C:\> $AppGw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzureRmApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Appgwsubnet01" -Subnet $Subnet
```

O primeiro comando cria uma rede virtual.
O segundo comando cria uma sub-rede usando a rede virtual previamente criada.
O terceiro comando obtém o gateway do aplicativo e o armazena na variável $AppGw.
O comando Fouth adiciona a configuração de IP ao gateway do aplicativo armazenado no $AppGw.
