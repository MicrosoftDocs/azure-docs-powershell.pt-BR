# Get-AzureRmDdosProtectionPlan

## Sinopse
Obtém um plano de proteção contra DDoS.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### GetByNameAndGroup
```
Get-AzureRmDdosProtectionPlan -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Programação
```
Get-AzureRmDdosProtectionPlan [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Get-AzureRmDdosProtectionPlan Obtém um plano de proteção contra DDoS.

## EXEMPLOS

```
                      {
                        "Id": "/subscriptions/d1dbd366-9871-45ac-84b7-fb318152a9e0/resourceGroups/ResourceGroupName/providers/Microsoft.Network/virtualNetworks/VnetName"

```

Nesse caso, precisamos especificar ambos os atributos **ResourceGroupName** e **Name** , que correspondem ao grupo de recursos e ao nome do plano de proteção contra DDoS, respectivamente.

Nesse cenário, especificamos somente o parâmetro **ResourceGroupName** para obter uma lista de planos de proteção contra DDoS.
