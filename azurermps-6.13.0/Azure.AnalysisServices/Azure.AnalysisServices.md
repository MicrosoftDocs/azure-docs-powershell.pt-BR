# Módulo Azure. AnalysisServices
## Descritivo
Este módulo contém comandos para executar operações em um determinado ambiente do Azure Analysis Services

## Cmdlets. AnalysisServices do Azure
### [Add-AzureAnalysisServicesAccount](Add-AzureAnalysisServicesAccount.md)
Adiciona uma conta autenticada para usar nas solicitações do cmdlet do servidor do Azure Analysis Services.

### [Export-AzureAnalysisServicesInstanceLog](Export-AzureAnalysisServicesInstanceLog.md)
Exporta um log de uma instância do servidor do Analysis Services no ambiente atualmente conectado, conforme especificado no comando Add-AzureAnalysisServicesAccount

### [Restart-AzureAnalysisServicesInstance](Restart-AzureAnalysisServicesInstance.md)
Reinicia uma instância do servidor do Analysis Services no ambiente atualmente conectado, conforme especificado no comando Add-AzureAnalysisServicesAccount

### [Sync-AzureAnalysisServicesInstance](Sync-AzureAnalysisServicesInstance.md)
Sincroniza um banco de dados especificado na instância especificada do servidor do Analysis Services para todas as instâncias de dimensionamento de consulta no ambiente atualmente conectado, conforme especificado no comando Add-AzureAnalysisServicesAccount
