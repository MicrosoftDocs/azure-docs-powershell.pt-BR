# Remove-AzureRmServiceFabricNodeType

## Sinopse
Remover um tipo de nó completo de um cluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Use **Remove-AzureRmServiceFabricNodeType** para remover todos os nós de um tipo de nó específico e o tipo de nó de um cluster. Este comando não pode ser usado para excluir o tipo de nó primário.

## EXEMPLOS

### Exemplo 1
```
PS c:> Remove-AzureRmServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

Esse comando removerá o NodeType ' NT1 ' do cluster.
