# Get-AzureRmDataLakeStoreItem

## Sinopse
Obtém os detalhes de um arquivo ou pasta no data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmDataLakeStoreItem** Obtém os detalhes de um arquivo ou pasta no repositório data Lake.

## EXEMPLOS

### Exemplo 1: obter detalhes de um arquivo do repositório data Lake
```
PS C:\>Get-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Este comando obtém os detalhes do arquivo Test.csv do repositório data Lake.
