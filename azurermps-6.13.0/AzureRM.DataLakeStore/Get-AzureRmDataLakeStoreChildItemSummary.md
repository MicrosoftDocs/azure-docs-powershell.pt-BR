# Get-AzureRmDataLakeStoreChildItemSummary

## Sinopse
Obtém o resumo do tamanho total, dos arquivos e das pastas contidos no caminho especificado

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmDataLakeStoreChildItemSummary [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Concurrency <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
**Get-AzureRmDataLakeStoreChildItemSummary** recupera o resumo de conteúdo para um determinado caminho. Calcula recursivamente o número total de arquivos, diretórios e tamanho total de todos os arquivos sob o caminho fornecido.

## EXEMPLOS

### Exemplo 1: obter o resumo de conteúdo de uma pasta
```
PS C:\> Get-AzureRmDataLakeStoreChildItemSummary -Account ContosoADL -Path /a -Concurrency 128
```

Ele lista o número de diretórios, arquivos e o tamanho total contidos em/a.
