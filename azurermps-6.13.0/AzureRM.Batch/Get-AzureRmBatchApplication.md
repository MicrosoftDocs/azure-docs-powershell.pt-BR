# Get-AzureRmBatchApplication

## Sinopse
Obtém informações sobre o aplicativo especificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [[-ApplicationId] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRmBatchApplication** Obtém informações sobre um aplicativo em uma conta em lotes do Azure.

## EXEMPLOS

### Exemplo 1: exibir os aplicativos em uma conta em lotes
```
PS C:\>Get-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup"
ApplicationId AllowUpdates DisplayName

------------- ------------ ----------------------------

litware       False        Litware Advanced Reticulator
```

Esse comando exibe todos os aplicativos na conta do ContosoBatch.
