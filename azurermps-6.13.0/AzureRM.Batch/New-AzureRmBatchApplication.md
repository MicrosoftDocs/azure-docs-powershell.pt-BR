# New-AzureRmBatchApplication

## Sinopse
Adiciona um aplicativo à conta em lotes especificada.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
New-AzureRmBatchApplication [-AccountName] <String> [-ResourceGroupName] <String> [-ApplicationId] <String>
 [[-AllowUpdates] <Boolean>] [[-DisplayName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureRmBatchApplication** adiciona um aplicativo à conta em lotes do Azure especificada.

## EXEMPLOS

### Exemplo 1: adicionar um aplicativo vazio a uma conta em lotes
```
PS C:\>New-AzureRmBatchApplication -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationId "Litware" -AllowUpdates $True -DisplayName "Litware Advanced Reticulator"
```

Esse comando cria o aplicativo Litware na conta ContosoBatch.
O aplicativo inicialmente não contém pacotes.
