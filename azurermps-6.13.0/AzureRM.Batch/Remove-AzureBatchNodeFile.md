# Remove-AzureBatchNodeFile

## Sinopse
Exclui um arquivo de nó para uma tarefa ou um nó de computação.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### Ela
```
Remove-AzureBatchNodeFile -JobId <String> -TaskId <String> -Path <String> [-Force] [-Recursive]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ComputeNode
```
Remove-AzureBatchNodeFile [-PoolId] <String> [-ComputeNodeId] <String> -Path <String> [-Force] [-Recursive]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### InputObject
```
Remove-AzureBatchNodeFile [[-InputObject] <PSNodeFile>] [-Force] [-Recursive]
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureBatchNodeFile** exclui um arquivo de nó de lote do Azure para uma tarefa ou um nó de computação.

## EXEMPLOS

### Exemplo 1: excluir um arquivo assocated com uma tarefa
```
PS C:\>Remove-AzureBatchNodeFile -JobId "Job-000001" -TaskId "Task26" -Path "wd\testFile.txt" -BatchContext $Context
```

Esse comando exclui o arquivo de nó chamado wd\testFile.txt.
Esse arquivo está associado à tarefa que tem a ID Task26 sob o job job-000001.

### Exemplo 2: excluir um arquivo de um nó de computação
```
PS C:\>Remove-AzureBatchNodeFile -PoolId "Pool07" -ComputeNodeId "tvm-2316545714_1-20150725t213220z" -Path "startup\testFile.txt" -BatchContext $Context
```

Esse comando exclui o arquivo de nó que é chamado startup\testFile.txt do nó de computação especificado no pool que tem a ID Pool07.

### Exemplo 3: remover um arquivo usando o pipeline
```
PS C:\>Get-AzureBatchNodeFile -JobId "Job-000001" -TaskId "Task26" -Path "wd\testFile2.txt" -BatchContext $Context | Remove-AzureBatchNodeFile -Force -BatchContext $Context
```

Esse comando obtém o arquivo de nó usando **Get-AzureBatchNodeFile**.
Esse arquivo está associado à tarefa que tem a ID Task26 sob o job job-000001.
O comando passa esse arquivo para o cmdlet atual usando o pipeline.
O cmdlet atual remove o arquivo de nó.
O comando especifica o parâmetro *Force* .
Portanto, o comando não solicita confirmação.
