# Módulo AzureRM. Maps
## Descritivo
Este tópico exibe ajuda para os cmdlets do Azure Maps.

## Cmdlets AzureRM. Maps
### [Get-AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
Obtém a conta.

### [Get-AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
Obtém as chaves de API para uma conta.
Essas chaves são o mecanismo de autenticação usado em chamadas subsequentes para mapas do Azure.

### [New-AzureRmMapsAccount](New-AzureRmMapsAccount.md)
Cria uma conta do Azure Maps.

### [New-AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
Regenera a chave de uma conta.

### [Remove-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
Exclui uma conta do Azure Maps.
