# Módulo AzureRM. PolicyInsights
## Descritivo
Este tópico exibe ajuda para os cmdlets do Azure Policy insights.

## Cmdlets AzureRM. PolicyInsights
### [Get-AzureRmPolicyEvent](Get-AzureRmPolicyEvent.md)
Obtém eventos de avaliação de política gerados à medida que os recursos são criados ou atualizados.

### [Get-AzureRmPolicyRemediation](Get-AzureRmPolicyRemediation.md)
Obtém correções de política.

### [Get-AzureRmPolicyState](Get-AzureRmPolicyState.md)
Obtém os Estados de conformidade de política para recursos.

### [Get-AzureRmPolicyStateSummary](Get-AzureRmPolicyStateSummary.md)
Obtém o resumo dos Estados mais recentes de conformidade com a política para recursos.

### [Remove-AzureRmPolicyRemediation](Remove-AzureRmPolicyRemediation.md)
Exclui uma correção de política.

### [Start-AzureRmPolicyRemediation](Start-AzureRmPolicyRemediation.md)
Cria e inicia uma correção de política para uma atribuição de política.

### [Parar-AzureRmPolicyRemediation](Stop-AzureRmPolicyRemediation.md)
Cancela uma remediação de política em andamento.
