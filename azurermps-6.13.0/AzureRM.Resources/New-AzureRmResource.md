# New-AzureRmResource

## Sinopse
Cria um recurso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## DESCRITIVO
O cmdlet **New-AzureRmResource** cria um recurso do Azure, como um site, um servidor de banco de dados SQL do Azure ou um banco de dados SQL do Azure em um grupo de recursos.

## EXEMPLOS

### Exemplo 1: criar um recurso
```
PS> New-AzureRmResource -Location "West US" -Properties @{test="test"} -ResourceName TestSite06 -ResourceType microsoft.web/sites -ResourceGroupName ResourceGroup11 -Force
```

Esse comando cria um recurso que é um site do ResourceGroup11.

### Exemplo 2: criar um recurso usando splatting
```
PS> $prop = @{
    Location          = "West US" 
    Properties        = @{test = "test"} 
    ResourceName      = "TestSite06" 
    ResourceType      = "microsoft.web/sites" 
    ResourceGroupName = "ResourceGroup11" 
    Force             = $true
}

PS> New-AzureRmResource @prop
```

Esse comando cria um recurso que é um site do ResourceGroup11.
