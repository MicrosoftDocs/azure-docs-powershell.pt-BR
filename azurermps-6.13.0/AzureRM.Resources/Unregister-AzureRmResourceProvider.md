# Unregister-AzureRmResourceProvider

## Sinopse
Cancela o registro de um provedor de recursos.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Unregister-AzureRmResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Unregister-AzureRmResourceProvider** cancela o registro de um provedor de recursos do Azure.
