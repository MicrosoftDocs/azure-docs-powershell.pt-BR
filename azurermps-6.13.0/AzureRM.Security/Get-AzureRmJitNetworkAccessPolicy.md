# Get-AzureRmJitNetworkAccessPolicy

## Sinopse
Obtém as políticas de acesso à rede JIT

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### SubscriptionScope (padrão)
```
Get-AzureRmJitNetworkAccessPolicy [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupScope
```
Get-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceGroupLevelResource
```
Get-AzureRmJitNetworkAccessPolicy -ResourceGroupName <String> -Location <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Identificação
```
Get-AzureRmJitNetworkAccessPolicy -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
As políticas de acesso à rede just in time (JIT) permitem definir uma política, permitindo que os usuários simples criem uma conexão de rede temporária para uma VM.
A política define quais portas, protocolo e endereços IP de origem podem solicitar uma conexão com uma VM e a duração máxima antes que a conexão seja fechada automaticamente.
Na política, você também pode ver a solicitação de conexão que foi feita com essa política.
