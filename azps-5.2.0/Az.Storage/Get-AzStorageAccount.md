# Get-AzStorageAccount

## Sinopse
Obtém uma conta de armazenamento.

## SYNTAX

### ResourceGroupParameterSet
```
Get-AzStorageAccount [[-ResourceGroupName] <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-IncludeGeoReplicationStats] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### BlobRestoreStatusParameterSet
```
Get-AzStorageAccount [-ResourceGroupName] <String> [-Name] <String> [-IncludeBlobRestoreStatus] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzStorageAccount** Obtém uma conta de armazenamento especificada ou todas as contas de armazenamento em um grupo de recursos ou na assinatura.

## EXEMPLOS

### Exemplo 1: obter uma conta de armazenamento especificada
```
PS C:\>Get-AzStorageAccount -ResourceGroupName "RG01" -Name "mystorageaccount"
```

Este comando obtém a conta de armazenamento especificada.

### Exemplo 2: obter todas as contas de armazenamento em um grupo de recursos
```
PS C:\>Get-AzStorageAccount -ResourceGroupName "RG01"
```

Este comando obtém todas as contas de armazenamento em um grupo de recursos.

### Exemplo 3: obter todas as contas de armazenamento na assinatura
```
PS C:\>Get-AzStorageAccount
```

Este comando obtém todas as contas de armazenamento na assinatura.

### Exemplo 4: obter uma conta de armazenamento com seu status de restauração de BLOB
```
PS C:\> $account = Get-AzStorageAccount -ResourceGroupName "myresourcegoup" -StorageAccountName "mystorageaccount" -IncludeBlobRestoreStatus

PS C:\> $account.BlobRestoreStatus

Status     RestoreId                            FailureReason Parameters.TimeToRestore     Parameters.BlobRanges                 
------     ---------                            ------------- ------------------------     ---------------------                 
InProgress a70cd4a1-f223-4c86-959f-cc13eb4795a8               2020-02-10T13:45:04.7155962Z [container1/blob1 -> container2/blob2]
```

Esse comando obtém uma conta de armazenamento com seu status de restauração de BLOB e mostra o status da restauração de BLOB.
