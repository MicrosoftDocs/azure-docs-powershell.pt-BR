# Stop-AzHDInsightJob

## Sinopse
Interrompe um trabalho em execução especificado em um cluster.

## SYNTAX

```
Stop-AzHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzHDInsightJob** interrompe um trabalho em execução especificado em um cluster do Azure HDInsight.

## EXEMPLOS

### Exemplo 1: parar um trabalho no cluster especificado
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Esse comando interrompe um trabalho no cluster chamado seu-Hadoop-001.
