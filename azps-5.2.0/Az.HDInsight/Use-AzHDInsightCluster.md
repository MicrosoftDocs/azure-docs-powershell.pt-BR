# Use-AzHDInsightCluster

## Sinopse
Seleciona um cluster para ser usado com o cmdlet Invoke-RmAzureHDInsightHiveJob.

## SYNTAX

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **use-AzHDInsightCluster** seleciona o cluster do Azure HDInsight para o cmdlet Invoke-AzHDInsightHiveJob usar para enviar trabalhos de Hive.

## EXEMPLOS

### Exemplo 1: selecionar um cluster para envio de consulta de Hive
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Esse comando seleciona um cluster para um envio de consulta de Hive.
