# Set-AzDtlAllowedVMSizesPolicy

## Sinopse
Define a política de tamanhos de máquinas virtuais permitidos de um laboratório no DevTest Labs.

## SYNTAX

### Habilitar (padrão)
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Enable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Desabilita
```
Set-AzDtlAllowedVMSizesPolicy [[-VmSizes] <String[]>] [-Disable] [-LabName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzDtlAllowedVMSizesPolicy** define a política de tamanhos de máquinas virtuais permitidos, que especifica uma lista de tamanhos de máquinas virtuais permitidos em um laboratório.
O cmdlet usa o grupo de recursos e o nome do laboratório especificado para definir a política.
