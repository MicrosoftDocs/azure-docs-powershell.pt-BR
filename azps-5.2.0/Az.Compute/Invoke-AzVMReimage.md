# Invoke-AzVMReimage

## Sinopse
Reimagem de uma máquina virtual do Azure.

## SYNTAX

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Invoke-AzVMReimage** reimagemia uma máquina virtual do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Este comando reimagemia a máquina virtual nomeada VirtualMachine07 em ResourceGroup11.
