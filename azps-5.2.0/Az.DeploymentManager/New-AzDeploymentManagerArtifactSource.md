# New-AzDeploymentManagerArtifactSource

## Sinopse
Cria uma fonte de artefato.

## SYNTAX

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzDeploymentManagerArtifactSource** cria uma fonte de artefato.
Especifique o *nome*, *ResourceGroupName* e propriedades necessárias.

Você pode modificar o objeto retornado localmente e, em seguida, aplicar as alterações à fonte de artefatos usando o cmdlet Set-AzDeploymentManagerArtifactSource.

O cmdlet retorna um objeto artefato que tem um ResourceId que pode ser referenciado no cmdlet New-AzDeploymentManagerServiceTopology para que os artefatos necessários para um recurso de recurso, o modelo e os arquivos de parâmetros, possam ser referenciados neste local.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

Cria uma fonte de artefatos no ContosoResourceGroup com o nome ContosoArtifactSource com EUA como o local do recurso. A propriedade SasUri fornece um URI de SAS de armazenamento do Azure para o contêiner de armazenamento em que os artefatos são armazenados.
