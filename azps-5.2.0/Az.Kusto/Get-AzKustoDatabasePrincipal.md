# Get-AzKustoDatabasePrincipal

## Sinopse
Retorna uma lista de objetos de banco de dados do cluster e do banco de dados Kusto fornecidos.

## SYNTAX

```
Get-AzKustoDatabasePrincipal -ClusterName <String> -DatabaseName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Retorna uma lista de objetos de banco de dados do cluster e do banco de dados Kusto fornecidos.

## EXEMPLOS

### Exemplo 1: lista de entidades de banco de dados do cluster e do banco de dados Kusto fornecidos
