# Get-AzSqlElasticPoolDatabase

## Sinopse
Obtém bancos de dados elásticos em um pool elástico e seus valores de propriedade.

## SYNTAX

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlElasticPoolDatabase** Obtém bancos de dados elásticos em um pool elástico e seus valores de propriedade.
Você pode especificar o nome de um banco de dados elástico no banco de dados SQL do Azure para ver os valores de propriedade somente para esse banco de dados.

## EXEMPLOS

### Exemplo 1: obter todos os bancos de dados em um pool elástico
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Esse comando obtém todos os bancos de dados em um pool elástico chamado ElasticPool01.

### Exemplo 2: obter todos os bancos de dados em um pool elástico usando a filtragem
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Esse comando obtém todos os bancos de dados em um pool elástico denominado ElasticPool01 que começam com "banco de dados".
