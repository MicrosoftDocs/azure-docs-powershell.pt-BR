# Remove-AzSqlSyncGroup

## Sinopse
Remove um grupo de sincronização do banco de dados SQL do Azure.

## SYNTAX

```
Remove-AzSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzSqlSyncGroup** remove um grupo de sincronização do banco de dados SQL do Azure.

## EXEMPLOS

### Exemplo 1: remover um grupo de sincronização
```
PS C:\>Remove-AzSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Esse comando Remove o grupo de sincronização do banco de dados SQL do Azure chamado syncGroup01.
