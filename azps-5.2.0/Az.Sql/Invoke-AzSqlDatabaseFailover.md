# Invoke-AzSqlDatabaseFailover

## Sinopse
Failover de um banco de dados.

## SYNTAX

```
Invoke-AzSqlDatabaseFailover [-DatabaseName] <String> [-ReadableSecondary] [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Invoke-AzSqlDatabaseFailover failover em um banco de dados SQL do Azure. Se o banco de dados estiver em um pool elástico, esse comando fará o failover do banco de dados fornecido sem afetar os outros bancos de dados no mesmo pool elástico.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando fará o failover da réplica primária do banco de dados chamada "Database01" no servidor chamado "Server01"

### Exemplo 2
```powershell
PS C:\> Invoke-AzSqlDatabaseFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -ReadableSecondary
```

Esse comando fará o failover da réplica secundária legível do banco de dados chamada "Database01" no servidor chamado "Server01"
