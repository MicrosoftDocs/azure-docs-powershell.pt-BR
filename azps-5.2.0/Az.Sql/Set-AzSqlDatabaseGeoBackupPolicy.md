# Set-AzSqlDatabaseGeoBackupPolicy

## Sinopse
Define uma política de backup geográfico do banco de dados.

## SYNTAX

```
Set-AzSqlDatabaseGeoBackupPolicy -State <GeoBackupPolicyState> [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzSqlDatabaseGeoBackupPolicy** define a política de backup geográfico registrada em um banco de dados.
Este é um recurso de backup do Azure usado para definir a política de armazenamento de backup.
