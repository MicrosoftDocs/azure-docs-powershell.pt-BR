# New-AzRecoveryServicesAsrProtectableItem

## Sinopse
Adicione (descubra) um servidor físico à lista de itens que protegem.

## SYNTAX

```
New-AzRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer> -FriendlyName <String>
 -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O **New-AzRecoveryServicesAsrProtectableItem** adiciona um novo item de proteção à lista de itens que protegem os itens que protegem em um contêiner de proteção em uma malha ASR (aplicável somente para o tipo de malha do VMware).

## EXEMPLOS

### Exemplo 1

Adicione (descubra) um servidor físico à lista de itens que protegem. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
New-AzRecoveryServicesAsrProtectableItem -FriendlyName XXXXXXXXXX -IPAddress $ipaddresss -OSType Windows -ProtectionContainer $pc
```
