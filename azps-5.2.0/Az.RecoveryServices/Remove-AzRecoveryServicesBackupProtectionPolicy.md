# Remove-AzRecoveryServicesBackupProtectionPolicy

## Sinopse
Exclui uma política de proteção de backup de um cofre.

## SYNTAX

### PolicyName (padrão)
```
Remove-AzRecoveryServicesBackupProtectionPolicy [-Name] <String> [-PassThru] [-Force] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Políticaobject
```
Remove-AzRecoveryServicesBackupProtectionPolicy [-Policy] <PolicyBase> [-PassThru] [-Force] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzRecoveryServicesBackupProtectionPolicy** exclui políticas de backup para um cofre.
Antes de poder excluir uma política de proteção de backup, a política não deve ter itens de backup associados.
Antes de excluir a política, verifique se cada item associado está associado a alguma outra política.
Para associar outra política a um item de backup, use o cmdlet Enable-AzRecoveryServicesBackupProtection.
Defina o contexto do cofre usando o cmdlet Set-AzRecoveryServicesVaultContext antes de usar o cmdlet atual.

## EXEMPLOS

### Exemplo 1: remover uma política
```powershell
PS C:\>$Pol= Get-AzRecoveryServicesBackupProtectionPolicy -Name "NewPolicy"
PS C:\> Remove-AzRecoveryServicesBackupProtectionPolicy -Policy $Pol
```

O primeiro comando obtém a política de proteção de backup chamada NewPolicy e a armazena na variável $Pol.
O segundo comando Remove o objeto de política em $Pol.

### Exemplo 2

Exclui uma política de proteção de backup de um cofre. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzRecoveryServicesBackupProtectionPolicy -Name 'V2VM' -VaultId $vault.ID
```
