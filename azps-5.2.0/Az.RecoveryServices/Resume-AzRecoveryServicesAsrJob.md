# Resume-AzRecoveryServicesAsrJob

## Sinopse
Retoma um trabalho suspenso do Azure site Recovery.

## SYNTAX

### ByObject (padrão)
```
Resume-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-Comment <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Resume-AzRecoveryServicesAsrJob -Name <String> [-Comment <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **resume-AzRecoveryServicesAsrJob** retoma um trabalho suspenso do Azure site Recovery.

## EXEMPLOS

### Exemplo 1
```
PS C:\> $currentJob = Resume-AzRecoveryServicesAsrJob -Job $Job
```

Retome o trabalho especificado se ele estiver em um estado de espera ou suspenso e retornar o objeto de trabalho ASR atualizado correspondente ao trabalho ASR.
