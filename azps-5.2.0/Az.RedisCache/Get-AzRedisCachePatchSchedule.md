# Get-AzRedisCachePatchSchedule

## Sinopse
Obtém um cronograma de correção.

## SYNTAX

```
Get-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzRedisCachePatchSchedule** Obtém um cronograma de patch para um cache no Azure Redis cache.

## EXEMPLOS

### Exemplo 1: obter o cronograma do patch
```
PS C:\>Get-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06"
```

Esse comando obtém o cronograma do patch do cache chamado RedisCache06.
