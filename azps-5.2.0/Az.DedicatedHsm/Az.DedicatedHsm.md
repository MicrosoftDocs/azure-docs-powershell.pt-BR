# Módulo AZ. DedicatedHsm
## Descritivo
Microsoft Azure PowerShell: cmdlets do DedicatedHsm

## Cmdlets AZ. DedicatedHsm

Obtém o HSM dedicado do Azure especificado.

Exclui o HSM dedicado do Azure especificado.

### [Update-AzDedicatedHsm](Update-AzDedicatedHsm.md)
Atualize um HSM exclusivo na assinatura especificada.
