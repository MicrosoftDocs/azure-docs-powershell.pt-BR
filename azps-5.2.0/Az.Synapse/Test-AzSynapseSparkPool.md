# Test-AzSynapseSparkPool

## Sinopse
Verifica a existência de um pool Spark do Analytics Synapse.

## SYNTAX

### TestByNameParameterSet (padrão)
```
Test-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByParentObjectParameterSet
```
Test-AzSynapseSparkPool -Name <String> -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Test-AzSynapseSparkPool** verifica a existência de um pool Spark do Analytics do Synapse.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Test-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

Esse comando verifica a existência do pool do Spark especificado.
