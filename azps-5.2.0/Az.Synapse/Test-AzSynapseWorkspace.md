# Test-AzSynapseWorkspace

## Sinopse
Verifica a existência de um espaço de trabalho de análise do Synapse.

## SYNTAX

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Test-AzSynapseWorkspace** verifica a existência de um espaço de trabalho analítico do Synapse.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

Esse comando verifica a existência de um espaço de trabalho de análise do Synapse.
