# Start-AzSynapseTrigger

## Sinopse
Inicia um gatilho em um espaço de trabalho.

## SYNTAX

### StartByName (padrão)
```
Start-AzSynapseTrigger -WorkspaceName <String> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByObject
```
Start-AzSynapseTrigger -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StartByInputObject
```
Start-AzSynapseTrigger -InputObject <PSTriggerResource> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Start-AzSynapseTrigger** inicia um gatilho em um espaço de trabalho. Se o gatilho estiver no estado ' parado ', o cmdlet iniciará o gatilho e eventualmente invocará os pipelines com base em sua definição. Se o gatilho já estiver no estado ' iniciado ', esse cmdlet não terá efeito.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Start-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Inicia um gatilho chamado ContosoTrigger na ContosoWorkspace do espaço de trabalho.

### Exemplo 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Start-AzSynapseTrigger -Name ContosoTrigger
```

Inicia um gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace pelo pipeline.

### Exemplo 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Start-AzSynapseTrigger
```

Inicia um gatilho chamado ContosoTrigger no espaço de trabalho ContosoWorkspace pelo pipeline.
