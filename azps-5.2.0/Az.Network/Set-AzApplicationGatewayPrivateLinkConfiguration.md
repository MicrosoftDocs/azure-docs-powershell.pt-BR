# Set-AzApplicationGatewayPrivateLinkConfiguration

## Sinopse
Modifica uma configuração de PrivateLink para um gateway de aplicativo.

## SYNTAX

```
Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzApplicationGatewayPrivateLinkConfiguration** modifica uma configuração privateLink para um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: definir uma configuração de PrivateLink
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $privateLinkIpConfiguration01
```

O primeiro comando obtém o gateway do aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando define a configuração privateLink com o nome privateLinkConfig01 para usar a configuração de IP armazenada em $privateLinkIpConfiguration 01
