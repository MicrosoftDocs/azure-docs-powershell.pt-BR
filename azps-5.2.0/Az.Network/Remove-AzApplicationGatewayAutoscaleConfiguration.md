# Remove-AzApplicationGatewayAutoscaleConfiguration

## Sinopse
Remove a configuração de autoescala de um aplicativo gateway.

## SYNTAX

```
Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayAutoscaleConfiguration** remove a configuração de autoescala de um gateway de aplicativo existente.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

O primeiro comando obtém o gateway do aplicativo e o armazena em $gw variável.
O segundo comando Remove a configuração de autoescala do gateway do aplicativo.
O terceiro comando atualiza o Application Gateway no Azure.
