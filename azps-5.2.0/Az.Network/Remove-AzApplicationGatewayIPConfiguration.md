# Remove-AzApplicationGatewayIPConfiguration

## Sinopse
Remove uma configuração de IP de um aplicativo gateway.

## SYNTAX

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayIPConfiguration** remove uma configuração de IP de um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: remover uma configuração de IP de um gateway do aplicativo do Azure
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na variável $AppGw.
O segundo comando Remove a configuração de IP denominada Subnet02 do gateway do aplicativo armazenado em $AppGw.
