# Invoke-AzIotHubDeploymentMetricsQuery

## Sinopse
Invocar uma consulta métrica de implantação de borda IoT.

## SYNTAX

### ResourceSet (padrão)
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceGroupName] <String> [-IotHubName] <String> -Name <String>
 -MetricName <String> [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-InputObject] <PSIotHub> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ResourceSet
```
Invoke-AzIotHubDeploymentMetricsQuery [-ResourceId] <String> -Name <String> -MetricName <String>
 [-MetricType <PSConfigurationMetricType>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
Avaliar um destino personalizado ou um sistema métrico definido em uma implantação de borda IoT.
Há métricas de sistema predefinidas que são calculadas por Hub IOT e não podem ser personalizadas.
- "Direcionado" mostra os dispositivos de borda IoT que correspondem à condição de direcionamento de implantação.
- "Aplicado" mostra os dispositivos de borda IoT direcionados que não são direcionados a outra implantação de prioridade mais alta.
- "Relatório de sucesso" mostra os dispositivos de borda IoT que relataram que os módulos foram implantados com êxito.
- "Falha no relatório" mostra os dispositivos de borda IoT que relataram que um ou mais módulos não foram implantados com êxito. 
  Para investigar ainda mais o erro, conecte-se remotamente a esses dispositivos e veja os arquivos de log.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzIotHubDeploymentMetricsQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "warningLimit"
```

Avaliar a métrica ' warningLimit ' definida como personalizada.

### Exemplo 2
```powershell
PS C:\> Invoke-AzIotHubDeployMetric -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "myDeploy1" -MetricName "Reporting Success" -MetricType "system"
```

Avaliar a métrica do sistema ' êxito no relatório '.
