# Update-AzServiceFabricReliability

## Sinopse
Atualize a camada de confiabilidade do tipo de nó primário em um cluster.

## SYNTAX

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Use **Update-AzServiceFabricReliability** para atualizar a confiabilidade do tipo de nó primário em um cluster.

## EXEMPLOS

### Exemplo 1
```powershell
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Esse comando altera a camada de confiabilidade do tipo de nó principal para prata.

### Exemplo 2

Atualize a camada de confiabilidade do tipo de nó primário em um cluster. (gerado automaticamente)

<!-- Aladdin Generated Example -->
```powershell
Update-AzServiceFabricReliability -AutoAddNode -Name 'Contoso01SFCluster' -ReliabilityLevel None -ResourceGroupName 'Group1'
```
