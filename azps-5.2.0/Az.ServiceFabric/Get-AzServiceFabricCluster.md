# Get-AzServiceFabricCluster

## Sinopse
Obter os detalhes do recurso de cluster.

## SYNTAX

### BySubscription (padrão)
```
Get-AzServiceFabricCluster [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroup
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByName
```
Get-AzServiceFabricCluster [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O **Get-AzServiceFabricCluster** receberá os detalhes do recurso de cluster.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzServiceFabricCluster -ResourceGroupName 'Group1' -ClusterName 'Contoso01SFCluster'
```

Esse comando receberá os detalhes do recurso de cluster do cluster ' mycluster '.
