# Get-AzOperationalInsightsCluster

## Sinopse
Obter ou listar clusters

## SYNTAX

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obter ou listar clusters, listar clusters em grupo de recursos quando "-ClusterName" não foi fornecido, clusters de lista em assinatura quando "-ClusterName" e "ResourceGroupName" não foram fornecidos.

## EXEMPLOS

### Exemplo 1
```powershell
Get-AzOperationalInsightsCluster -ResourceGroupName {rg-name} -ClusterName {cluster-name}

```

Obter cluster
