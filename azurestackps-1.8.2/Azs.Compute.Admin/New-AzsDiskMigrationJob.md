# New-AzsDiskMigrationJob

## Sinopse
Inicia um trabalho de migração de disco gerenciado para migrar discos gerenciados para o compartilhamento de destino especificado.

## SYNTAX

```
New-AzsDiskMigrationJob [-Disks] <Disk[]> [-TargetShare] <String> [[-Location] <String>] [-Name] <String>
 [<CommonParameters>]
```

## DESCRITIVO
Crie um trabalho de migração de disco.

## EXEMPLOS

### --------------------------EXEMPLO 1--------------------------
```
New-AzsDiskMigrationJob -Name "MyMigrationJob" -Disks $disks -location local -TargetShare "\\SU1FileServer.azurestack.local\SU1_ObjStore"
```

Iniciar um trabalho de migração de disco gerenciado para os primeiros 20 discos
