# Get-AzsSubscription

## Sinopse
Obter a lista de assinaturas.

## SYNTAX

### Lista (padrão)
```
Get-AzsSubscription [<CommonParameters>]
```

### Obter
```
Get-AzsSubscription [-SubscriptionId] <String> [<CommonParameters>]
```

## DESCRITIVO
Obter a lista de assinaturas.

## EXEMPLOS

### EXEMPLO 1
```
Get-AzsSubscription
```

Obter a lista de assinaturas.
