# Get-AzsKeyVaultQuota

## Sinopse
Obter uma lista de todos os objetos de cota para o keyvault em um local.

## SYNTAX

```
Get-AzsKeyVaultQuota [[-Location] <String>] [<CommonParameters>]
```

## DESCRITIVO
Obter uma lista de todos os objetos de cota para o keyvault em um local.

## EXEMPLOS

### EXEMPLO 1
```
Get-AzsKeyVaultQuota
```

Obter uma lista de todos os objetos de cota para o keyvault em um local.
