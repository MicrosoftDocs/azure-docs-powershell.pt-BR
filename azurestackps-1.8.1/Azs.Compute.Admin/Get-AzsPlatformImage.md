# Get-AzsPlatformImage

## Sinopse
Retorna as imagens da máquina virtual carregadas no repositório de imagens da plataforma.

## SYNTAX

### Lista (padrão)
```
Get-AzsPlatformImage [-Publisher <String>] [-Offer <String>] [-Sku <String>] [-Location <String>]
 [<CommonParameters>]
```

### Obter
```
Get-AzsPlatformImage -Publisher <String> -Offer <String> -Sku <String> -Version <String> [-Location <String>]
 [<CommonParameters>]
```

### Identificação
```
Get-AzsPlatformImage -ResourceId <String> [<CommonParameters>]
```

## DESCRITIVO
Retorna imagens da plataforma.

## EXEMPLOS

### --------------------------EXEMPLO 1--------------------------
```
Get-AzsPlatformImage
```

Retorna as imagens da máquina virtual carregadas no repositório de imagens da plataforma no local local.

### --------------------------EXEMPLO 2--------------------------
```
Get-AzsPlatformImage -Publisher Canonical -Offer UbuntuServer -Sku 16.04-LTS -Version 0.1.0
```

Obter uma imagem de plataforma específica.
