# Get-AzsLocation

## Sinopse
Obtenha uma lista de todos os locais do AzureStack.

## SYNTAX

### Lista (padrão)
```
Get-AzsLocation [<CommonParameters>]
```

### Obter
```
Get-AzsLocation [-Name] <String> [<CommonParameters>]
```

## DESCRITIVO
Obtenha uma lista de todos os locais do AzureStack.

## EXEMPLOS

### --------------------------EXEMPLO 1--------------------------
```
Get-AzsLocation
```

Obtenha uma lista de todos os locais do AzureStack.
