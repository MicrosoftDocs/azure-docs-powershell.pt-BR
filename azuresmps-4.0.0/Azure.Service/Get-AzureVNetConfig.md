# Get-AzureVNetConfig

## Sinopse
Obtém a configuração de rede virtual do Azure da assinatura atual.

## SYNTAX

```
Get-AzureVNetConfig [-ExportToFile <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureVNetConfig** recupera a configuração de rede virtual da assinatura atual do Azure.
Se o parâmetro *ExportToFile* for especificado, um arquivo de configuração de rede será criado.

## EXEMPLOS

### Exemplo 1: obter a configuração de rede virtual de uma assinatura atual do Azure
```
PS C:\> Get-AzureVNetConfig
```

Esse comando obtém a configuração de rede virtual da assinatura atual do Azure e a exibe.

### Exemplo 2: obter a configuração de rede virtual da assinatura atual do Azure e salvá-la em um arquivo local
```
PS C:\> Get-AzureVNetConfig -ExportToFile "c:\temp\MyAzNets.netcfg"
```

Esse comando obtém a configuração de rede virtual da assinatura atual do Azure e salva-a em um arquivo local.
