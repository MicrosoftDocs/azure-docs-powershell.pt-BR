# Add-AzureVirtualIP

## Sinopse
Adiciona um IP virtual a um serviço na nuvem.

## SYNTAX

```
Add-AzureVirtualIP [-ServiceName] <String> [-VirtualIPName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzureVirtualIP** adiciona um novo IP virtual (VIP) ao seu serviço do Azure.
O novo IP virtual tem um nome, mas não foi atribuído um endereço IP.

O endereço IP é alocado somente quando você associa um ponto de extremidade ao VIP.
Consulte Add-AzureEndpoint para obter mais detalhes.

Sua assinatura será cobrada por VIPs adicionais somente depois que elas estiverem associadas a um ponto de extremidade.

## EXEMPLOS

### Exemplo 1: adicionar um IP virtual a um serviço
```
PS C:\> Add-AzureVirtualIP -VirtualIPName "Vip01" -ServiceName "ContosoService03"
OperationDescription OperationId                          OperationStatus
-------------------- -----------                          ---------------
Add-AzureVirtualIP   4bd7b638-d2e7-216f-ba38-5221233d70ce Succeeded
```

Esse comando adiciona um endereço IP virtual a um serviço.
