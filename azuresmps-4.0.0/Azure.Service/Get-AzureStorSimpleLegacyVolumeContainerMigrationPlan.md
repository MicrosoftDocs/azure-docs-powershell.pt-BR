# Get-AzureStorSimpleLegacyVolumeContainerMigrationPlan

## Sinopse
Obtém planos de migração para contêineres herdados.

## SYNTAX

```
Get-AzureStorSimpleLegacyVolumeContainerMigrationPlan [-LegacyConfigId <String>]
 [-LegacyContainerNames <String[]>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureStorSimpleLEgacyVolumeContainerMigrationPlan** Obtém planos de migração para contêineres herdados.
Especifique um plano de migração por sua ID de configuração herdada.
Se a criação do plano de migração ainda estiver em andamento, esse cmdlet obtém o status do plano de migração.
Se o plano de migração estiver concluído, esse cmdlet retornará o plano de migração real para o conjunto de contêineres herdados.
Se você não especificar o parâmetro *LegacyConfigId* , esse cmdlet retornará uma lista de IDs de configuração.
