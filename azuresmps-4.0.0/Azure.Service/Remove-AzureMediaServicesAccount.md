# Remove-AzureMediaServicesAccount

## Sinopse
Remove a conta especificada do Azure Media Services.

**Observação:**  Esta versão foi preterida, confira a [versão mais recente](https://docs.microsoft.com/powershell/module/azurerm.media/?view=azurermps-5.4.0#media_services).

## SYNTAX

```
Remove-AzureMediaServicesAccount -Name <String> [-Force] [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
Este tópico descreve o cmdlet na versão 0.8.10 do módulo do Microsoft Azure PowerShell.
Para obter a versão do módulo que você está usando, no console do PowerShell do Azure, digite `(Get-Module -Name Azure).Version` .

O cmdlet **Remove-AzureMediaServicesAccount** remove uma conta de serviços de mídia.

## EXEMPLOS

### Exemplo 1: excluir uma conta de serviços de mídia
```
PS C:\> Remove-AzureMediaServicesAccount -Name "mediaservicesaccount" -Force
```
