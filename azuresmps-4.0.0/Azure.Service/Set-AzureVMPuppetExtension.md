# Set-AzureVMPuppetExtension

## Sinopse
Define a extensão Puppet para uma máquina virtual.

## SYNTAX

```
Set-AzureVMPuppetExtension [-PuppetMasterServer] <String> [[-Version] <String>] [-Disable]
 [[-ReferenceName] <String>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureVMPuppetExtension** define a extensão Puppet para uma máquina virtual.

## EXEMPLOS

### Exemplo 1: definir a extensão Puppet para uma máquina virtual
```
PS C:\> Set-AzureVMPuppetExtension -VM $VM
```

Este exemplo define a extensão Puppet para a máquina virtual especificada, conforme armazenada na variável $VM.
