# Remove-AzureLocalNetworkGateway

## Sinopse
Remove um gateway de rede local do Azure.

## SYNTAX

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureLocalNetworkGateway** remove um gateway de rede local do Azure.
