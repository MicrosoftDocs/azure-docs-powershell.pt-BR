# Get-AzureVMChefExtension

## Sinopse
Obtém a extensão chefe aplicada a uma máquina virtual.

## SYNTAX

```
Get-AzureVMChefExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureVMChefExtension** Obtém a extensão chefe aplicada a uma máquina virtual.

## EXEMPLOS

### Exemplo 1: obter a extensão chefe aplicada na máquina virtual especificada
```
PS C:\> Get-AzureVMChefExtension -VM $VM;
```

Este comando obtém a extensão chefe aplicada na máquina virtual especificada.
