# Get-AzureRemoteAppPlan

## Sinopse
Lista os planos do RemoteApp do Azure.

## SYNTAX

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRemoteAppPlan** lista os planos do Azure RemoteApp.

## EXEMPLOS

## OS
