# Copy-AzureRemoteAppUserDisk

## Sinopse
Copia o disco de usuário de um usuário de uma coleção do Azure RemoteApp para outra.

## SYNTAX

```
Copy-AzureRemoteAppUserDisk [-SourceCollectionName] <String> [-DestinationCollectionName] <String>
 [-UserUpn] <String> [-OverwriteExistingUserDisk] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Copy-AzureRemoteAppUserDisk** copia o disco de usuário de um usuário de uma coleção do Azure RemoteApp para outra.

## EXEMPLOS

### Exemplo 1: copiar um disco de usuário
```
PS C:\> Copy-AzureRemoteAppUserDisk -DestinationCollectionName "Contoso02" -SourceCollectionName "Contoso01" -UserUpn "PattiFuller@contoso.com" -OverwriteExistingUserDisk
```

Esse comando copia o disco do usuário de um usuário do Azure Active Directory que tem o UPN PattiFuller@contoso.com da coleção Contoso01 para a coleção Contoso02.
Se um disco do usuário PattiFuller@contoso.com já existir no Contoso02, esse comando o substituirá.
