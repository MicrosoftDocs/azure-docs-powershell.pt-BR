# Remove-AzurePublicIP

## Sinopse
Remove a configuração de IP público de uma máquina virtual do Azure.

## SYNTAX

```
Remove-AzurePublicIP [[-PublicIPName] <String>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzurePublicIP** remove a configuração de IP público de uma máquina virtual do Azure.

## EXEMPLOS

### Exemplo 1: remover a configuração de IP público
```
PS C:\> Get-AzureVM -ServiceName "FTPInAzure" -Name "FTPInstance" | Remove-AzurePublicIP | Update-AzureVM
```

Esse comando obtém a máquina virtual chamada FTPInstance no serviço chamado FTPInAzure usando o cmdlet **Get-AzureVM** .
O comando transmite a máquina virtual para o cmdlet atual usando o operador pipeline.
O cmdlet atual remove a configuração de IP público da máquina virtual.
O comando passa a máquina virtual para o cmdlet **Update-AzureVM** , que implementa suas alterações.
