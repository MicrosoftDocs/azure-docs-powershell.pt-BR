# Stop-AzureVirtualNetworkGatewayDiagnostics

## Sinopse
Interrompe uma sessão de diagnóstico do gateway de rede virtual em execução.

## SYNTAX

```
Stop-AzureVirtualNetworkGatewayDiagnostics -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureVirtualNetworkGatewayDiagnostics** interrompe uma sessão de diagnóstico do gateway de rede virtual em execução.
Esse comando salva os resultados da sessão de diagnóstico na conta de armazenamento que o cmdlet Start-AzureVirtualNetworkGatewayDiagnostics especifica.
