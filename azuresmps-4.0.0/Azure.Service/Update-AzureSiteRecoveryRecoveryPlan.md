# Update-AzureSiteRecoveryRecoveryPlan

## Sinopse
Atualiza um plano de recuperação na recuperação do site.

## SYNTAX

```
Update-AzureSiteRecoveryRecoveryPlan -File <String> [-WaitForCompletion] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Update-AzureSiteRecoveryRecoveryPlan** atualiza um plano de recuperação no Azure site Recovery e, em seguida, o publica.

## EXEMPLOS

### Exemplo 1: atualizar um plano de recuperação
```
PS C:\> Update-AzureSiteRecoveryRecoveryPlan -File "C:\Users\Contoso\Desktop\RecoveryPlan.xml"
```

Esse comando atualiza o plano de recuperação especificado e o publica.
