# New-AzureVNetGateway

## Sinopse
Cria um gateway VPN em uma rede virtual.

## SYNTAX

```
New-AzureVNetGateway -VNetName <String> [-GatewayType <String>] [-GatewaySKU <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzureVNetGateway** cria um gateway de rede virtual privada (VPN) em uma rede virtual.
Você também pode especificar a SKU do gateway, seja padrão, padrão ou HighPerformance.
Você pode especificar o tipo, ou StaticRouting ou DynamicRouting.

## EXEMPLOS

### Exemplo 1: criar um gateway de rede virtual
```
PS C:\> New-AzureVNetGateway -VNetName "ContosoVN07" -GatewayType "DynamicRouting" -GatewaySKU "Default"
```

Esse comando cria um gateway de rede virtual para a rede virtual chamada ContosoVN07.
O gateway é o SKU padrão e usa o roteamento dinâmico.
