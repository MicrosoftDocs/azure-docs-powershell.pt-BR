# Set-AzureStorSimpleAccessControlRecord

## Sinopse
Atualiza o IQN de um registro de controle de acesso.

## SYNTAX

```
Set-AzureStorSimpleAccessControlRecord -ACRName <String> -IQNInitiatorName <String> [-WaitForComplete]
 [-NewName <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureStorSimpleAccessControlRecord** atualiza o nome qualificado iSCSI (iqn) de um registro de controle de acesso existente.

## EXEMPLOS
