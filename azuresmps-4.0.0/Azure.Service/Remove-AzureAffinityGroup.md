# Remove-AzureAffinityGroup

## Sinopse
Exclui um grupo de afinidade em uma assinatura.

## SYNTAX

```
Remove-AzureAffinityGroup [-Name] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureAffinityGroup** exclui um grupo de afinidade do Azure na assinatura atual.
Você não pode excluir um grupo de afinidade que tenha membros.
Você deve primeiro excluir todos os membros de um grupo de afinidade.

## EXEMPLOS

### Exemplo 1: remover um grupo de afinidade
```
PS C:\> Remove-AzureAffinityGroup -Name "South01"
```

Esse comando exclui o grupo de afinidade South01 na assinatura atual.
