# Set-AzureDns

## Sinopse
Modifica o endereço IP de um servidor DNS.

## SYNTAX

```
Set-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureDns** modifica o endereço IP de um servidor DNS para um serviço do Azure.

## EXEMPLOS

### Exemplo 1: modificar o endereço IP de um servidor DNS
```
PS C:\> Set-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.5 -Name "Dns07"
```

Esse comando modifica o endereço IP do servidor DNS chamado Dns07 para o serviço chamado ContosoService.
