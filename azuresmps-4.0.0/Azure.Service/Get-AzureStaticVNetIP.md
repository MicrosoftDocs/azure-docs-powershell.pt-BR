# Get-AzureStaticVNetIP

## Sinopse
Obtém as informações de endereço IP da VNet estático de um objeto de máquina virtual, se houver.

## SYNTAX

```
Get-AzureStaticVNetIP -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureStaticVNetIP** Obtém as informações de endereço IP de rede virtual estática (VNet) de um objeto de máquina virtual, se houver.
