# Remove-AzureServiceRemoteDesktopExtension

## Sinopse
Remove a extensão da área de trabalho remota do serviço de nuvem aplicada a todas as funções ou funções nomeadas em um slot de implantação especificado.

## SYNTAX

### RemoveByRoles (padrão)
```
Remove-AzureServiceRemoteDesktopExtension [[-ServiceName] <String>] [[-Slot] <String>] [[-Role] <String[]>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### RemoveAllRoles
```
Remove-AzureServiceRemoteDesktopExtension [[-ServiceName] <String>] [[-Slot] <String>]
 [-UninstallConfiguration] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureServiceRemoteDesktopExtension** remove a extensão da área de trabalho remota do serviço de nuvem aplicada a todas as funções ou funções nomeadas em um determinado slot de implantação.

## EXEMPLOS

### Exemplo 1: remover a extensão da área de trabalho remota
```
PS C:\> Remove-AzureServiceRemoteDesktopExtension -ServiceName $svc
```

Esse comando Remove a extensão da área de trabalho remota.

### Exemplo 2: remover a extensão da área de trabalho remota de uma função especificada
```
PS C:\> Remove-AzureServiceRemoteDesktopExtension -ServiceName $svc -Role "WebRole1"
```

Este comando Remove a extensão da área de trabalho remota de uma função especificada.
