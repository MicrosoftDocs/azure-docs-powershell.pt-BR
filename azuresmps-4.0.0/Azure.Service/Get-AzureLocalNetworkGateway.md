# Get-AzureLocalNetworkGateway

## Sinopse
Obtém um gateway de rede local.

## SYNTAX

```
Get-AzureLocalNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureLocalNetworkGateway** Obtém um gateway de rede local.
