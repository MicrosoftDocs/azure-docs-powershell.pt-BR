# Remove-AzureStorageAccount

## Sinopse
Exclui a conta de armazenamento especificada de uma assinatura.

## SYNTAX

```
Remove-AzureStorageAccount [-StorageAccountName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureStorageAccount** remove uma conta de uma assinatura do Azure.

## EXEMPLOS

### Exemplo 1: remover uma conta de armazenamento
```
PS C:\> Remove-AzureStorageAccount -StorageAccountName "ContosoStore01"
```

Este comando Remove a conta de armazenamento ContosoStore01 da assinatura especificada.
