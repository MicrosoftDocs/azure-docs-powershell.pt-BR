# Start-AzureWebsiteJob

## Sinopse
Inicia um trabalho da Web para um site.

## SYNTAX

```
Start-AzureWebsiteJob -JobName <String> -JobType <WebJobType> [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Start-AzureWebsiteJob** inicia um trabalho da Web para um site.

## EXEMPLOS

### Exemplo 1: iniciar um trabalho da Web para um site
```
PS C:\> Start-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob -JobType Continuous
```

Inicia um trabalho da Web chamado MyWebJob para mywebsite.
