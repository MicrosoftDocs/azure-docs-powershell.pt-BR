# Remove-AzureVMDiagnosticsExtension

## Sinopse
Remove a extensão de diagnóstico do Azure de uma máquina virtual.

## SYNTAX

```
Remove-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureVMDiagnosticsExtension** remove uma extensão de diagnóstico do Microsoft Azure de uma máquina virtual.
Você deve passar a saída deste cmdlet para o cmdlet **Update-AzureVM** .

## EXEMPLOS

### Exemplo 1: remover a extensão de diagnóstico do Azure de uma máquina virtual
```
PS C:\> Remove-AzureVMDiagnosticsExtension -VM $VM | Update-AzureVM
```

Esse comando Remove a extensão de diagnóstico do Azure de uma máquina virtual.
