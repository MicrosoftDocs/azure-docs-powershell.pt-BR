# Rename-AzureRemoteAppTemplateImage

## Sinopse
Renomeia uma imagem de modelo do Azure RemoteApp.

## SYNTAX

```
Rename-AzureRemoteAppTemplateImage [-ImageName] <String> [-NewName] <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Rename-AzureRemoteAppTemplateImage** renomeia uma imagem de modelo do Azure RemoteApp.

## EXEMPLOS

### Exemplo 1: renomear uma imagem de modelo
```
PS C:\> Rename-AzureRemoteAppTemplateImage -ImageName "ContosoApps" -NewName "ContosoFinanceApps"
```

Esse comando renomeia a imagem de modelo do Azure RemoteApp chamada ContosoApps para ContosoFinanceApps.
