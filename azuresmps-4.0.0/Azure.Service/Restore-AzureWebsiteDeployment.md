# Restore-AzureWebsiteDeployment

## Sinopse
Reimplanta uma implantação anterior de um site no Azure.

## SYNTAX

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
Este tópico descreve o cmdlet na versão 0.8.10 do módulo do Microsoft Azure PowerShell.
Para obter a versão do módulo que você está usando, no console do PowerShell do Azure, digite `(Get-Module -Name Azure).Version` .

O cmdlet **Restore-AzureWebsiteDeployment** reimplanta uma implantação anterior de um site no Azure.
Esse processo substitui a implantação atual pela implantação selecionada.

## EXEMPLOS

### Exemplo 1: reimplantar um site
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

Esse comando reimplanta a implantação que tem a ID f876543210 para o site chamado ContosoSite.
