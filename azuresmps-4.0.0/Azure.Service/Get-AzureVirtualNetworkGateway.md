# Get-AzureVirtualNetworkGateway

## Sinopse
Obtém um gateway de rede virtual.

## SYNTAX

```
Get-AzureVirtualNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureVirtualNetworkGateway** Obtém um gateway de rede virtual do Azure.
