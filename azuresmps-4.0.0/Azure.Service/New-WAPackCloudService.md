# New-WAPackCloudService

## Sinopse
Cria um serviço na nuvem.

## SYNTAX

```
New-WAPackCloudService -Name <String> -Label <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
Esses tópicos são preteridos e serão removidos no futuro.
Este tópico descreve o cmdlet na versão 0.8.1 do módulo do Microsoft Azure PowerShell.
Para descobrir a versão do módulo que você está usando, no console do PowerShell do Azure, digite `(Get-Module -Name Azure).Version` .

O cmdlet **New-WAPackCloudService** cria um serviço na nuvem.

## EXEMPLOS

### Exemplo 1: criar um serviço de nuvem
```
PS C:\> New-WAPackCloudService -Name "ContosoCloudService01" -Label "A label"
```

O comando cria um serviço em nuvem chamado ContosoCloudService01 com um rótulo.
