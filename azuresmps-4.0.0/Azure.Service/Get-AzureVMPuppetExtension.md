# Get-AzureVMPuppetExtension

## Sinopse
Obtém a extensão Puppet aplicada a uma máquina virtual.

## SYNTAX

```
Get-AzureVMPuppetExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureVMPuppetExtension** Obtém a extensão Puppet aplicada a uma máquina virtual.

## EXEMPLOS

### Exemplo 1: obter a extensão Puppet aplicada em uma máquina virtual
```
PS C:\> Get-AzureVMPuppetExtension -VM $VM
```

Esse comando obtém a extensão Puppet aplicada a uma máquina virtual.
