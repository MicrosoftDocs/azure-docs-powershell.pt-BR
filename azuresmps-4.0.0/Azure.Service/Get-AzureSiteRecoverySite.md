# Get-AzureSiteRecoverySite

## Sinopse
Obtém os sites do Hyper-V de um cofre de recuperação do site.

## SYNTAX

```
Get-AzureSiteRecoverySite [-Vault <ASRVault>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureSiteRecoverySite** Obtém os sites do Hyper-V em um cofre do Azure site Recovery.

## EXEMPLOS

### Exemplo 1: obter sites de recuperação
```
PS C:\> Get-AzureSiteRecoverySite
Type                                    Name                                    ID
----                                    ----                                    --
HyperVSite                              RecoverySite07                          f16829b4-5b01-4209-a6cf-8e0aff1fe328
```

Este comando obtém os sites de recuperação do cofre atual.
