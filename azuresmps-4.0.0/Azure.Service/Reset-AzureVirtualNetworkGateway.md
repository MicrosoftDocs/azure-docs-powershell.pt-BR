# Reset-AzureVirtualNetworkGateway

## Sinopse
Redefine um gateway de rede virtual.

## SYNTAX

```
Reset-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Reset-AzureVirtualNetworkGateway** redefine um gateway de rede virtual.
