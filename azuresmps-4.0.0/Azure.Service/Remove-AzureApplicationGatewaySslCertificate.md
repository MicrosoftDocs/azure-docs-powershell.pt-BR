# Remove-AzureApplicationGatewaySslCertificate

## Sinopse
Remove um certificado SSL de um aplicativo gateway.

## SYNTAX

```
Remove-AzureApplicationGatewaySslCertificate -Name <String> -CertificateName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureApplicationGatewaySslCertificate** remove um certificado Secure Sockets Layer (SSL) de um Application Gateway.

## EXEMPLOS

### Exemplo 1: remover um certificado SSL
```
PS C:\> Remove-AzureApplicationGatewaySslCertificate -Name "ApplicationGateway08" -CertificateName "SslCertificate13"
```

Esse comando Remove um certificado SSL chamado SslCertificate13 do gateway do aplicativo chamado ApplicationGateway08.
