# Get-AzureVirtualNetworkGatewayDiagnostics

## Sinopse
Obtém os resultados do diagnóstico do gateway de rede virtual do Azure.

## SYNTAX

```
Get-AzureVirtualNetworkGatewayDiagnostics -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureVirtualNetworkGatewayDiagnostics** Obtém os resultados do diagnóstico do gateway de rede virtual do Azure.
