# Remove-AzureRemoteAppUserDisk

## Sinopse
Remove o disco de usuário de um usuário de uma coleção do Azure RemoteApp.

## SYNTAX

```
Remove-AzureRemoteAppUserDisk [-CollectionName] <String> [-UserUpn] <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRemoteAppUserDisk** remove o disco de usuário de um usuário de uma coleção do Azure RemoteApp.

## EXEMPLOS

### Exemplo 1: remover um disco de usuário
```
PS C:\> Remove-AzureRemoteAppUserDisk -CollectionName "Contoso01" -UserUpn "PattiFuller@contoso.com"
```

Esse comando Remove o disco de usuário de um usuário do Azure Active Directory que tem o UPN PattiFuller@contoso.com da coleção Contoso01.
