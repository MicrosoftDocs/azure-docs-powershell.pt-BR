# Get-AzureRemoteAppLocation

## Sinopse
Lista os locais aos quais o Azure RemoteApp dá suporte.

## SYNTAX

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureRemoteAppLocation** lista os locais aos quais o Azure RemoteApp dá suporte.

## EXEMPLOS

## OS
