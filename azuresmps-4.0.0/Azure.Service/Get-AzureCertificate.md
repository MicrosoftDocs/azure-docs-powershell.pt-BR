# Get-AzureCertificate

## Sinopse
Obtém um objeto de certificado de um serviço do Azure.

## SYNTAX

```
Get-AzureCertificate [-ServiceName] <String> [-ThumbprintAlgorithm <String>] [-Thumbprint <String>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureCertificate** Obtém um objeto de certificado de um serviço do Azure.

## EXEMPLOS

### Exemplo 1: obter certificados de um serviço
```
PS C:\> $AzureCert = Get-AzureCertificate -ServiceName "ContosoService"
```

Esse comando obtém objetos de certificado do serviço denominado ContosoService e, em seguida, os armazena na variável $AzureCert.

### Exemplo 2: obter um certificado de um serviço
```
PS C:\> $AzureCert = Get-AzureCertificate -ServiceName "ContosoService" -Thumbprint '5383CE0343CB6563281CA97C1D4D712209CFFA97'
```

Esse comando obtém o objeto de certificado identificado pela impressão digital especificada do serviço denominado ContosoService e, em seguida, armazena-o na variável $AzureCert.
