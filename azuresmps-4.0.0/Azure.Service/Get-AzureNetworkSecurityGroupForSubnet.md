# Get-AzureNetworkSecurityGroupForSubnet

## Sinopse
Obtém o grupo de segurança de rede para uma sub-rede.

## SYNTAX

```
Get-AzureNetworkSecurityGroupForSubnet -VirtualNetworkName <String> -SubnetName <String> [-Detailed]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureNetworkSecurityGroupForSubnet** Obtém o grupo de segurança de rede associado a uma sub-rede.
