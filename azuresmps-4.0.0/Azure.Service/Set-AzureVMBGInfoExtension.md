# Set-AzureVMBGInfoExtension

## Sinopse
Define a extensão BGInfo para uma máquina virtual.

## SYNTAX

### SetBGInfoExtension (padrão)
```
Set-AzureVMBGInfoExtension [-Disable] [[-ReferenceName] <String>] [[-Version] <String>] -VM <IPersistentVM>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### UninstallBGInfoExtension
```
Set-AzureVMBGInfoExtension [-Uninstall] [[-ReferenceName] <String>] [[-Version] <String>] -VM <IPersistentVM>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzureVMBGInfoExtension** define a extensão BGInfo para uma máquina virtual.

## EXEMPLOS

### Exemplo 1: definir a extensão do BGInfo para uma máquina virtual
```
PS C:\> Set-AzureVMBGInfoExtension -VM $VM
```

Esse comando define a extensão BGInfo para a máquina virtual especificada como armazenada na variável $VM.
