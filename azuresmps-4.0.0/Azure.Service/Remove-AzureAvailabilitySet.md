# Remove-AzureAvailabilitySet

## Sinopse
Remove um conjunto de disponibilidade de uma máquina virtual do Azure.

## SYNTAX

```
Remove-AzureAvailabilitySet -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureAvailabilitySet** remove um conjunto de disponibilidade de uma máquina virtual do Azure.
