# Remove-AzureReservedIPAssociation

## Sinopse
Remove a associação do endereço IP reservado para a VM ou o serviço de nuvem.

## SYNTAX

```
Remove-AzureReservedIPAssociation [-ReservedIPName] <String> [-ServiceName] <String>
 [[-VirtualIPName] <String>] [[-Slot] <String>] [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureReservedIPAssociation** dissocia um endereço IP reservado de uma máquina virtual (VM) ou de um serviço na nuvem.
Quando a operação é concluída, o endereço IP reservado é gratuito e a VM/VIP Obtém um endereço IP público dinâmico do inventário do Azure.

## EXEMPLOS

### Exemplo 1: remover uma associação de IP reservado
```
PS C:\> Remove-AzureReservedIPAssociation -ReservedIPName "ResIp14" -ServiceName "PipTestWestEurope"
```

Esse comando desassocia o endereço IP reservado chamado ResIp14 do serviço chamado PipTestWestEurope.
PipTestWestEurope será atribuído um novo VIP dinâmico.
