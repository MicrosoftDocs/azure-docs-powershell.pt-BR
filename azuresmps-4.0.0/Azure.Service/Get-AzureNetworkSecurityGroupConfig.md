# Get-AzureNetworkSecurityGroupConfig

## Sinopse
Obtém detalhes de um grupo de segurança de rede.

## SYNTAX

```
Get-AzureNetworkSecurityGroupConfig [-Detailed] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureNetworkSecurityGroupConfig** retorna detalhes de um grupo de segurança de rede.
Especifique o parâmetro *detalhado* para exibir as regras de segurança de rede.
