# New-AzureServiceADDomainExtensionConfig

## Sinopse
Gera a configuração da extensão de domínio do AD para serviços de nuvem.

## DESCRITIVO
O cmdlet **New-AzureServiceADDomainExtensionConfig** gera a configuração da extensão de domínio do Active Directory (AD) para serviços de nuvem.

## EXEMPLOS

### Exemplo 1: especificar uma configuração de domínio do AD
```
PS C:\> $ExtensionCfg = New-AzureServiceADDomainExtensionConfig -Role WorkerRole1 -DomainName $Domain -Credential $Cred -JoinOption 35;

PS C:\> New-AzureDeployment -ServiceName $CloudSvc -Slot "Production" -Package $Pkg -Configuration $Config -ExtensionConfiguration $ExtensionCfg;
```

Esse comando gera uma configuração para a extensão de domínio do anúncio.
