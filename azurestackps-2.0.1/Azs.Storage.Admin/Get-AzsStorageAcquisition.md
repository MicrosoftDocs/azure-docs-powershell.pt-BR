# Get-AzsStorageAcquisition

## Sinopse
Retorna uma lista de aquisições de BLOB.

## SYNTAX

```
Get-AzsStorageAcquisition [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRITIVO
Retorna uma lista de aquisições de BLOB.

## EXEMPLOS

### Exemplo 1:
```powershell
PS C:\> Get-AzsStorageAcquisition
```

Obtenha a lista de Acquistions de BLOB.
