# Get-AzsStorageDestinationShare

## Sinopse
Retorna uma lista de compartilhamentos de destino que o sistema considera os melhores candidatos para a migração.

## SYNTAX

```
Get-AzsStorageDestinationShare [-SourceShareName] <String> [-FarmName] <String> [[-ResourceGroupName] <String>]
 [<CommonParameters>]
```

## DESCRITIVO
Retorna uma lista de compartilhamentos de destino que o sistema considera os melhores candidatos para a migração.

## EXEMPLOS

### --------------------------EXEMPLO 1--------------------------
```
Get-AzsStorageDestinationShare -FarmName f9b8e2e2-e4b4-44e0-9d92-6a848b1a5376 -SourceShareName "||SU1FileServer.azurestack.local|SU1_ObjStore"
```

Obtenha uma lista de compartilhamentos de destino que o sistema considera melhores candidatos para a migração.
