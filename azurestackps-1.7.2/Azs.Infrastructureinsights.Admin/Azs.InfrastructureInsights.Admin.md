# Módulo AZS. InfrastructureInsights. admin
## Descritivo
Versão prévia do módulo do operador InfrastructureInsights, que permite que o oeprators exiba a integridade da infraestrutura.

## Cmdlets do AZS. InfrastructureInsights. admin
### [Fechar-AzsAlert](Close-AzsAlert.md)
Fecha o alerta fornecido.

### [Get-AzsAlert](Get-AzsAlert.md)
Retorna a lista de todos os alertas em um determinado local.

### [Get-AzsRegionHealth](Get-AzsRegionHealth.md)
Retorna uma lista de status de integridade da região.

### [Get-AzsRegistrationHealth](Get-AzsRegistrationHealth.md)
Retorna uma lista da integridade de cada recurso em um serviço.

### [Get-AzsRPHealth](Get-AzsRPHealth.md)
Retorna uma lista da integridade de cada serviço.

### [Reparar-AzsAlert](Repair-AzsAlert.md)
Repara o alerta fornecido.
