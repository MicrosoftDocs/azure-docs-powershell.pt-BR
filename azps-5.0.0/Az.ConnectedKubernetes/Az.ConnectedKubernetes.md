# Módulo AZ. ConnectedKubernetes
## Descritivo
Microsoft Azure PowerShell: cmdlets do ConnectedKubernetes

## Cmdlets AZ. ConnectedKubernetes
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Retorna as propriedades do cluster conectado especificado, incluindo o nome, a identidade, as propriedades e detalhes adicionais do cluster.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API para registrar um novo cluster K8s e, assim, criar um recurso controlado no ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Exclua um cluster conectado, removendo o recurso controlado no braço do Azure Resource Manager (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API para atualizar determinadas propriedades do recurso de cluster conectado
