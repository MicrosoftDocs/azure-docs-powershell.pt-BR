# Enable-AzHDInsightMonitoring

## Sinopse
Habilita o monitoramento em um cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho monitoramento especificado durante a ativação.

## SYNTAX

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Enable-AzHDInsightMonitoring** habilita o monitoramento em um cluster do Azure HDInsight.

## EXEMPLOS

### Exemplo 1
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

O monitoramento será habilitado no cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho monitoramento com ID 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Exemplo 2
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

O monitoramento será habilitado no cluster HDInsight e os logs relevantes serão enviados para o espaço de trabalho monitoramento com ID 1d364e89-bb71-4503-aa3d-a23535aea7bd
