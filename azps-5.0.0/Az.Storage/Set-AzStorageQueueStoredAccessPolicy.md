# Set-AzStorageQueueStoredAccessPolicy

## Sinopse
Define uma política de acesso armazenado para uma fila de armazenamento do Azure.

## SYNTAX

```
Set-AzStorageQueueStoredAccessPolicy [-Queue] <String> [-Policy] <String> [-Permission <String>]
 [-StartTime <DateTime>] [-ExpiryTime <DateTime>] [-NoStartTime] [-NoExpiryTime] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzStorageQueueStoredAccessPolicy** define uma política de acesso armazenado para uma fila de armazenamento do Azure.

## EXEMPLOS

### Exemplo 1: definir uma política de acesso armazenado na fila com permissão completa
```
PS C:\> Set-AzStorageQueueStoredAccessPolicy -Queue "MyQueue" -Policy "Policy07" -Permission arup
```

Esse comando define uma política de acesso chamada Policy07 para fila de armazenamento chamada MyQueue.
