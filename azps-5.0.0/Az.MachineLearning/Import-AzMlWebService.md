# Import-AzMlWebService

## Sinopse
Importa um objeto JSON para uma definição de serviço Web.

## SYNTAX

### ImportFromJSONFile
```
Import-AzMlWebService -InputFile <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ImportFromJSONString.
```
Import-AzMlWebService -JsonString <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Import-AzMlWebService importa, especificado diretamente ou em um arquivo referenciado, e cria um objeto de definição de serviço Web que pode ser passado para o cmdlet New-AzMlWebService.

## EXEMPLOS

### Exemplo 1: importar da cadeia de caracteres
```
Import-AzMlWebService -JsonString $jsonDefinition
```

### Exemplo 2: importar do caminho do arquivo
```
Import-AzMlWebService -InputFile "C:\mlservice.json"
```
