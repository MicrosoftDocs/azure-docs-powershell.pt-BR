# Remove-AzResourceMoverMoveResource

## Sinopse
Exclui um recurso de movimentação da coleção move.

## SYNTAX

```
Remove-AzResourceMoverMoveResource -MoveCollectionName <String> -Name <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRITIVO
Exclui um recurso de movimentação da coleção move.

## EXEMPLOS

### Exemplo 1: remover o recurso da coleção mover
