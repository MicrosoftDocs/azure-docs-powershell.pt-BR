# Get-AzDataFactoryGateway

## Sinopse
Obtém informações sobre gateways lógicos no Azure data Factory.

## SYNTAX

### ByFactoryName (padrão)
```
Get-AzDataFactoryGateway [-DataFactoryName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzDataFactoryGateway** Obtém informações sobre gateways lógicos no Azure data Factory.
Se você especificar o nome de um gateway, esse cmdlet obterá informações sobre esse gateway.
Se você não especificar um nome, esse cmdlet obterá informações sobre todos os gateways de uma fábrica de dados.
Se quiser adicionar um Microsoft SQL Server local como um serviço vinculado a uma fábrica de dados, você deve instalar um gateway em seu computador local.

Esse comando obtém informações sobre todos os gateways lógicos para o data Factory chamado WikiADF no grupo de recursos chamado ADF.
