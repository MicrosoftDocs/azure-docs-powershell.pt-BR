# Get-AzApiManagementSsoToken

## Sinopse
Obtém um link com um token de SSO para um portal de gerenciamento implantado de um serviço de gerenciamento de API.

## SYNTAX

### ExpandedParameter (padrão)
```
Get-AzApiManagementSsoToken -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByInputObject
```
Get-AzApiManagementSsoToken -InputObject <PsApiManagement> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzApiManagementSsoToken** retorna um link (URL) que contém um token de logon único (SSO) para um portal de gerenciamento implantado de um serviço de gerenciamento de API.

## EXEMPLOS

### Exemplo 1: obter o token de SSO de um serviço de gerenciamento de API
```
PS C:\>Get-AzApiManagementSsoToken -ResourceGroupName "Contoso" -Name "ContosoApi"
```

Esse comando obtém o token de SSO de um serviço de gerenciamento de API.
