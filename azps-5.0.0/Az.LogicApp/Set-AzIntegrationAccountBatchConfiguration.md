# Set-AzIntegrationAccountBatchConfiguration

## Sinopse
Modifica uma configuração em lotes da conta de integração.

## SYNTAX

### ByIntegrationAccountAndParameters (padrão)
```
Set-AzIntegrationAccountBatchConfiguration -ResourceGroupName <String> -ParentName <String> -Name <String>
 [-BatchGroupName <String>] [-MessageCount <Int32>] [-BatchSize <Int32>] [-ScheduleInterval <Int32>]
 [-ScheduleFrequency <String>] [-ScheduleTimeZone <String>] [-ScheduleStartTime <DateTime>]
 [-Metadata <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

```

 -BatchConfigurationDefinition <String> [-Metadata <Hashtable>] [-DefaultProfile <IAzureContextContainer>]
```

## DESCRITIVO
O cmdlet **set-AzIntegrationAccountBatchConfiguration** modifica uma configuração em lotes da conta de integração.

## EXEMPLOS

### Exemplo 1: modificar uma configuração em lotes usando um arquivo local
```powershell
PS C:\> Set-AzIntegrationAccountBatchConfiguration -ResourceGroupName "sampleResourceGroup" -IntegrationAccountName "sampleIntegrationAccount" -BatchConfigurationName "sampleBatchConfig" -BatchConfigurationFilePath $batchConfigurationFilePath

```

Modifique uma configuração em lotes chamada "sampleBatchConfig" usando o arquivo local localizado no caminho do arquivo contido em "$batchConfigurationFilePath".

Modifique uma configuração em lotes chamada "sampleBatchConfig" usando a cadeia de caracteres JSON contida em "$batchConfigurationContent".

### Exemplo 3: modificar uma configuração em lotes usando parâmetros
