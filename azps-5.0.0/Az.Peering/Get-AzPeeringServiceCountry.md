# Get-AzPeeringServiceCountry

## Sinopse
Lista os países disponíveis para o serviço de emparelhamento.

## SYNTAX

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Lista os países disponíveis para o serviço de emparelhamento.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Lista de países disponíveis para o serviço de emparelhamento.
