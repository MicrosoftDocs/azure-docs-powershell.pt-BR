# New-AzRedisCachePatchSchedule

## Sinopse
Adiciona um cronograma de correção.

## SYNTAX

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzRedisCachePatchSchedule** adiciona um cronograma de patches a um cache no Azure Redis cache.

## EXEMPLOS

### Exemplo 1: criar e adicionar um cronograma de patch em um cache
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Esse comando adiciona um cronograma de patches ao cache chamado RedisCache06.
O parâmetro Entries usa como seu valor um comando que usa **New-AzRedisCacheScheduleEntry** para criar um cronograma.
