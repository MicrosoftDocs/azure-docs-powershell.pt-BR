# Remove-AzOperationalInsightsWorkspace

## Sinopse
Remove um espaço de trabalho.

## SYNTAX

```
Remove-AzOperationalInsightsWorkspace [-ResourceGroupName] <String> [-Name] <String> [-ForceDelete] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzOperationalInsightsWorkspace** exclui um espaço de trabalho existente.
Se esse espaço de trabalho tiver sido vinculado a uma conta existente por meio do parâmetro *CódigoDoCliente* no momento da criação, a conta original não será excluída no portal dos insights operacionais.

## EXEMPLOS

### Exemplo 1: remover um espaço de trabalho por nome
```
PS C:\>Remove-AzOperationalInsightsWorkspace -ResourceGroupName "ContosResourceGroup" -Name "MyWorkspace"
```

Esse comando Remove o espaço de trabalho chamado MyWorkspace do grupo de recursos chamado ContosoResourceGroup.

### Exemplo 2: remover um espaço de trabalho usando o pipeline e sem confirmação
```
PS C:\>Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosResourceGroup" -Name "MyWorkspace" | Remove-AzOperationalInsightsWorkspace -Force
```

Esse comando usa o cmdlet Get-AzOperationalInsightsWorkspace para obter o espaço de trabalho chamado MyWorkspace e, em seguida, o passa para o cmdlet **Remove-AzOperationalInsightsWorkspace** usando o operador pipeline para removê-lo.
Como o parâmetro *Force* é especificado, o comando não o avisa antes de remover o espaço de trabalho.

### Exemplo 3: forçar exclusão do espaço de trabalho (não pode ser recuperada)
```
PS C:\> $workspace = New-AzOperationalInsightsWorkspace -ResourceGroupName $rgname -Name $wsname -Location $wslocation
PS C:\> $workspace | Remove-AzOperationalInsightsWorkspace -ForceDelete
```

Force excluir um espaço de trabalho.
