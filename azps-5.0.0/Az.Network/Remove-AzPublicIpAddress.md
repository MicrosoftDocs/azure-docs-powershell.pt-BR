# Remove-AzPublicIpAddress

## Sinopse
Remove um endereço IP público.

## SYNTAX

```
Remove-AzPublicIpAddress -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzPublicIpAddress** remove um endereço IP público do Azure.

## EXEMPLOS

### 1: remover um recurso de endereço IP público
```
Remove-AzPublicIpAddress -Name $publicIpName -ResourceGroupName $rgName
```

Este comando Remove o recurso de endereço IP público chamado $publicIpName no $rgName do grupo de recursos.
