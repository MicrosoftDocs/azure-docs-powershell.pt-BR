# New-AzPublicIpPrefix

## Sinopse
Cria um prefixo de IP público

## SYNTAX

```
New-AzPublicIpPrefix -Name <String> -ResourceGroupName <String> [-Location <String>] [-Sku <String>]
 -PrefixLength <UInt16> [-IpAddressVersion <String>] [-IpTag <PSPublicIpPrefixTag[]>] [-Zone <String[]>]
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzPublicIpPrefix** cria um prefixo de IP público.

## EXEMPLOS

### Exemplo 1: criar um novo prefixo de IP público
```powershell
PS C:\> $publicIpPrefix = New-AzPublicIpPrefix -Name $prefixName -ResourceGroupName $rgName -PrefixLength 30
```

Esse comando cria um novo recurso de prefixo de IP público.
