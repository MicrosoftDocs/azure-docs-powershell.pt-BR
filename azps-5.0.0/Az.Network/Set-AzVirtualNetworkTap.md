# Set-AzVirtualNetworkTap

## Sinopse
Atualiza uma rede virtual, toque em.

## SYNTAX

```
Set-AzVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzVirtualNetworkTap** atualiza um toque de rede virtual.

## EXEMPLOS

### Exemplo 1: configurar um toque de rede virtual
```
PS C:\>$vTap = Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzVirtualNetworkTap -VirtualNetworkTap $vTap
```

O comando atualiza a IpConfiguration de destino e atualiza o toque da rede virtual.
Se houver qualquer toque de configuração que faz referência a ele, todo o tráfego de origem não será iniciado para a nova postagem de configuração de IP de destino.
