# Get-AzApplicationGatewayIdentity

## Sinopse
Obter identidade atribuída ao gateway do aplicativo.

## SYNTAX

```
Get-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzApplicationGatewayIdentity** Obtém a identidade atribuída ao gateway do aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzApplicationGatewayIdentity -ApplicationGateway $gw
```

Este exemplo mostra como obter identidade do Application Gateway do Application Gateway.
