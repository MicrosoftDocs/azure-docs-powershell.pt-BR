# Get-AzNetworkUsage

## Sinopse
Lista os usos da rede para uma assinatura

## SYNTAX

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Get-AzNetworkUsage Obtém limites e uso atual para recursos de rede.

## EXEMPLOS

### Exemplo 1
