# Get-AzLoadBalancerInboundNatPoolConfig

## Sinopse
Obtém uma ou mais configurações de pool de NAT de entrada de um balanceador de carga.

## SYNTAX

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzLoadBalancerInboundNatPoolConfig** Obtém uma ou mais configurações de pool de NAT de entrada de um balanceador de carga.

## EXEMPLOS

### 1: obter
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
