# New-AzP2sVpnGateway

## Sinopse

## DESCRITIVO
O cmdlet **New-AzP2sVpnGateway** permite que você crie um novo P2SVpnGateway em VirtualHub para obter a conectividade do ponto para o site do Point para os clientes do site para o Azure VirtualWan.
