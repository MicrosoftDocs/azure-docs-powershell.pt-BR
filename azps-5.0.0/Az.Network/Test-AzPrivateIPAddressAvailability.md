# Test-AzPrivateIPAddressAvailability

## Sinopse
Teste a disponibilidade de um endereço IP privado em uma rede virtual.

## SYNTAX

### TestByResource
```
Test-AzPrivateIPAddressAvailability -VirtualNetwork <PSVirtualNetwork> -IPAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### TestByResourceId
```
Test-AzPrivateIPAddressAvailability -ResourceGroupName <String> -VirtualNetworkName <String>
 -IPAddress <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Test-AzPrivateIPAddressAvailability** testa se um endereço IP privado especificado está disponível em uma rede virtual.
Esse cmdlet retorna uma lista de endereços IP particulares disponíveis se o endereço IP particular solicitado for tirado.

## EXEMPLOS

### Exemplo 1: testar se um endereço IP está disponível usando o pipeline
```
PS C:\>Get-AzVirtualNetwork -Name $vnetName -ResourceGroupName $rgname | Test-AzPrivateIPAddressAvailability -IPAddress "10.0.1.10"
```

Este comando obtém uma rede virtual e usa o operador pipeline para passá-la para **Test-AzPrivateIPAddressAvailability** , o que testa se o endereço IP privado especificado está disponível.
