# Get-AzVirtualNetworkTap

## Sinopse
Obtém uma rede virtual toque

## SYNTAX

### ListParameterSet (padrão)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzVirtualNetworkTap** Obtém um toque de rede virtual do Azure ou uma lista da rede virtual do Azure toca em um grupo de recursos.

## EXEMPLOS

### Exemplo 1: obter uma rede virtual, toque em
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Este comando obtém uma referência de toque do VirtualNetwork para determinado "VirtualTap1" em "ResourceGroup1".

### Exemplo 2: obter todos os toques de redes virtuais usando a filtragem
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Esse comando obtém todas as referências ao VirtualNetwork em que começam com "VirtualTap".
