# Unregister-AzRecoveryServicesBackupManagementServer

## Sinopse
Cancela o registro de um servidor SCDPM ou de um servidor de backup do cofre.

## SYNTAX

```
Unregister-AzRecoveryServicesBackupManagementServer [-AzureRmBackupManagementServer] <BackupEngineBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Unregister-AzRecoveryServicesBackupManagementServer** cancela o registro de um servidor de sistema de proteção de dados do System Center (SCDPM) ou um servidor do Azure backup do cofre.
Esse cmdlet Remove referências aos servidores que não estão registrados no cofre.
Antes de cancelar o registro de um contêiner, você deve excluir todos os dados protegidos associados a esse contêiner.
Defina o contexto do cofre usando o cmdlet Set-AzRecoveryServicesVaultContext antes de usar o cmdlet atual.

## EXEMPLOS

### Exemplo 1: cancelar o registro de um servidor SCDPM do cofre
```powershell
PS C:\>$BMS = Get-AzRecoveryServicesBackupManagementServer -Name "dpmserver01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupManagementServer -AzBackupManagementServer $BMS
```

O primeiro comando obtém o servidor de gerenciamento de backup chamado dpmserver01.contoso.com e, em seguida, armazena-o na variável $BMS.
O segundo comando cancela o registro do servidor SCDPM do cofre.

### Exemplo 2

Cancela o registro de um servidor SCDPM ou de um servidor de backup do cofre. (gerado automaticamente)

```powershell <!-- Aladdin Generated Example --> 
Unregister-AzRecoveryServicesBackupManagementServer -AzureRmBackupManagementServer <BackupEngineBase> -VaultId $vault.ID
```
