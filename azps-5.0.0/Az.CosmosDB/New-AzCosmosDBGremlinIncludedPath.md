# New-AzCosmosDBGremlinIncludedPath

## Sinopse
Cria um novo objeto do tipo PSIncludedPath. Ele pode ser passado como um valor de parâmetro para Set-AzCosmosDBGremlinGraph.

## SYNTAX

```
New-AzCosmosDBGremlinIncludedPath [-Path <String>] [-Index <PSIndexes[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Objeto correspondente ao IncludedPath da API Gremlin.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $index1 = New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash
New-AzCosmosDBGremlinIncludedPath -Path "/*" -Index $index1
Path Indexes
---- -------
/*   {Microsoft.Azure.Commands.CosmosDB.Models.PSIndexes}
```
