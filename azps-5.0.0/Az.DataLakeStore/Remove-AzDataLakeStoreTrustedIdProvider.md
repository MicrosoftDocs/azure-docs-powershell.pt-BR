# Remove-AzDataLakeStoreTrustedIdProvider

## Sinopse
Remove o provedor de identidade confiável especificado no repositório data Lake especificado.

## SYNTAX

```
Remove-AzDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzDataLakeStoreTrustedIdProvider** remove o provedor de identidade confiável especificado no repositório data Lake especificado.

## EXEMPLOS

### Exemplo 1: remover um provedor de identidade confiável.
```
PS C:\> Remove-AzDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

Remove o provedor "MyProvider" da conta "ContosoADL"
