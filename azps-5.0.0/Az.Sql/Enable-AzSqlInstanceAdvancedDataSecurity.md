# Enable-AzSqlInstanceAdvancedDataSecurity

## Sinopse
Habilita a segurança avançada de dados em uma instância gerenciada.

## SYNTAX

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Enable-AzSqlInstanceAdvancedDataSecurity** habilita a segurança avançada de dados em uma instância gerenciada. A segurança avançada de dados é um pacote de segurança unificado que inclui classificação de dados, avaliação de vulnerabilidade e proteção avançada contra ameaças para sua instância gerenciada. (Uma nova conta de armazenamento será criada automaticamente para salvar avaliações de vulnerabilidade. Se uma conta de armazenamento foi criada anteriormente para essa finalidade, ela será usada em vez disso.

## EXEMPLOS

### Exemplo 1: habilitar a instância gerenciada segurança de dados avançada
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exemplo 2: habilitar a instância gerenciada segurança de dados avançada do recurso do servidor
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Exemplo 3

Habilita a segurança avançada de dados em uma instância gerenciada. (gerado automaticamente)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlInstanceAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -InstanceName 'ContosoManagedInstanceName' -ResourceGroupName MyResourceGroup
```
