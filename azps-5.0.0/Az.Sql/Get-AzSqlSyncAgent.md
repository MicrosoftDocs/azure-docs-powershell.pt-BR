# Get-AzSqlSyncAgent

## Sinopse
Retorna informações sobre os agentes do Azure SQL Sync.

## SYNTAX

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzSqlSyncAgent** retorna informações sobre um ou mais agentes de sincronização do SQL do Azure.
Especifique o nome de um agente de sincronização para ver informações somente para esse agente de sincronização.
