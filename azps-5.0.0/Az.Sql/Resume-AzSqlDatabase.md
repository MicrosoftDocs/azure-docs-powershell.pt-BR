# Resume-AzSqlDatabase

## Sinopse
Retoma um banco de dados de data warehouse do SQL.

## SYNTAX

```
Resume-AzSqlDatabase [-ServerName] <String> -DatabaseName <String> [-AsJob] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **resume-AzSqlDatabase** retoma um banco de dados de SQL data warehouse do Azure.

## EXEMPLOS

### Exemplo 1: retoma um banco de dados de SQL data warehouse do Azure
```
PS C:\>Resume-AzSqlDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Esse comando retoma um banco de dados suspenso do Azure SQL data warehouse.
