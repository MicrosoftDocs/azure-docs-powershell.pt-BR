# Set-AzSynapseDataset

## Sinopse
Cria ou atualiza um DataSet no espaço de trabalho.

## SYNTAX

### SetByName (padrão)
```
Set-AzSynapseDataset -WorkspaceName <String> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByObject
```
Set-AzSynapseDataset -WorkspaceObject <PSSynapseWorkspace> -Name <String> -DefinitionFile <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzSynapseDataset** cria um DataSet ou atualiza um conjunto de um conjunto de um existente no espaço de trabalho.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Set-AzSynapseDataset -WorkspaceName ContosoWorkspace -Name ContosoDataset -DefinitionFile "C:\\samples\\Dataset.json"
```

Esse comando cria um conjunto de um conjunto de um conjunto de ContosoDataset no espaço de trabalho chamado ContosoWorkspace.
O comando baseia o DataSet nas informações do Dataset.jsem arquivo.
