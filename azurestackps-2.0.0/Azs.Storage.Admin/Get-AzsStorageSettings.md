# Get-AzsStorageSettings

## Sinopse
Retorna as configurações do provedor de recursos de armazenamento.

## SYNTAX

```
Get-AzsStorageSettings [-Location <String>] [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## DESCRITIVO
Retorna as configurações do provedor de recursos de armazenamento.

## EXEMPLOS

### Exemplo 1:
```powershell
PS C:\> Get-AzsStorageSettings
```

Obter as configurações de armazenamento.
