# Add-AzsPlatformImage

## Sinopse
Cria uma nova plataforma de plataforma com o Publisher, a oferta, os SKUs e a versão fornecidos.

## SYNTAX

### Createexpanded (padrão)
```
Add-AzsPlatformImage -Offer <String> -Publisher <String> -Sku <String> -Version <String> [-Location <String>]
 [-SubscriptionId <String>] [-BillingPartNumber <String>] [-DataDisks <IDataDisk[]>] [-OsType <OSType>]
 [-OsUri <String>] [-ProvisioningState <ProvisioningState>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Criados
```
Add-AzsPlatformImage -Offer <String> -Publisher <String> -Sku <String> -Version <String>
 -NewImage <IPlatformImageParameters> [-Location <String>] [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentity
```
Add-AzsPlatformImage -InputObject <IComputeAdminIdentity> -NewImage <IPlatformImageParameters>
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityExpanded
```
Add-AzsPlatformImage -InputObject <IComputeAdminIdentity> [-BillingPartNumber <String>]
 [-DataDisks <IDataDisk[]>] [-OsType <OSType>] [-OsUri <String>] [-ProvisioningState <ProvisioningState>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Cria uma nova plataforma de plataforma com o Publisher, a oferta, os SKUs e a versão fornecidos.

## INFORMA

Propriedades de parâmetros complexas para criar os parâmetros descritos abaixo, construa uma tabela de hash contendo as propriedades adequadas. Para obter informações sobre tabelas de hash, execute Get-Help about_Hash_Tables.

Datadisks <IDataDisk [] >: discos de dados usados pela imagem da plataforma.
  - `[Lun <Int32?>]`: Número de unidade lógica.
  - `[Uri <String>]`: Local do modelo de disco.

INPUTobject <IComputeAdminIdentity> : parâmetro de identidade
  - `[DiskId <String>]`: O GUID de disco como identidade.
  - `[Id <String>]`: Caminho de identidade do recurso
  - `[Location <String>]`: Local do recurso.
  - `[MigrationId <String>]`: O nome do GUID do trabalho de migração.
  - `[Offer <String>]`: Nome da oferta.
  - `[Publisher <String>]`: Nome do fornecedor.
  - `[QuotaName <String>]`: Nome da cota.
  - `[Sku <String>]`: Nome da SKU.
  - `[SubscriptionId <String>]`: Credenciais de assinatura que identificam exclusivamente a assinatura do Microsoft Azure. A ID da assinatura forma a parte do URI para cada chamada de serviço.
  - `[Type <String>]`: Tipo de extensão.
  - `[Version <String>]`: A versão do recurso.

NEWIMAGE <IPlatformImageParameters> : parâmetros usados para criar uma nova imagem de plataforma.
  - `[DataDisk <IDataDisk[]>]`: Discos de dados usados pela imagem da plataforma.
    - `[Lun <Int32?>]`: Número de unidade lógica.
    - `[Uri <String>]`: Local do modelo de disco.
  - `[DetailBillingPartNumber <String>]`: O número da peça é usado para cobrar pelos custos do software.
  - `[OSDiskOstype <OSType?>]`: Tipo de sistema operacional.
  - `[OSDiskUri <String>]`: Local do disco.
  - `[ProvisioningState <ProvisioningState?>]`: Status de provisionamento da imagem da plataforma.
