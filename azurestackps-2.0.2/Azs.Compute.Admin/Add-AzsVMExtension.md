# Add-AzsVMExtension

## Sinopse
Crie uma imagem de extensão da máquina virtual com o Publisher, versão.

## SYNTAX

### Createexpanded (padrão)
```
Add-AzsVMExtension -Publisher <String> -Type <String> -Version <String> [-Location <String>]
 [-SubscriptionId <String>] [-ComputeRole <String>] [-IsSystemExtension] [-PropertiesPublisher <String>]
 [-ProvisioningState <ProvisioningState>] [-SourceBlob <String>] [-SupportMultipleExtensions]
 [-VmOsType <OSType>] [-VMScaleSetEnabled] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### Criados
```
Add-AzsVMExtension -Publisher <String> -Type <String> -Version <String> -Extension <IVMExtensionParameters>
 [-Location <String>] [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### CreateViaIdentity
```
Add-AzsVMExtension -InputObject <IComputeAdminIdentity> -Extension <IVMExtensionParameters>
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CreateViaIdentityExpanded
```
Add-AzsVMExtension -InputObject <IComputeAdminIdentity> [-Publisher <String>] [-ComputeRole <String>]
 [-IsSystemExtension] [-ProvisioningState <ProvisioningState>] [-SourceBlob <String>]
 [-SupportMultipleExtensions] [-VmOsType <OSType>] [-VMScaleSetEnabled] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Crie uma imagem de extensão da máquina virtual com o Publisher, versão.

## INFORMA

Propriedades de parâmetros complexas para criar os parâmetros descritos abaixo, construa uma tabela de hash contendo as propriedades adequadas. Para obter informações sobre tabelas de hash, execute Get-Help about_Hash_Tables.

EXTENSÃO <IVMExtensionParameters> : parâmetros usados para criar uma nova imagem de extensão de máquina virtual.
  - `[ComputeRole <String>]`: Calcular função
  - `[IsSystemExtension <Boolean?>]`: Indica se a extensão é para o sistema.
  - `[ProvisioningState <ProvisioningState?>]`: Configurando o estado da extensão.
  - `[Publisher <String>]`: O fornecedor da extensão da VM
  - `[SourceBlobUri <String>]`: URI para blob do Azure ou AzureStack.
  - `[SupportMultipleExtension <Boolean?>]`: Verdadeiro se der suporte a várias extensões.
  - `[VMScaleSetEnabled <Boolean?>]`: Valor que indica se a extensão está habilitada para suporte ao conjunto de escala de máquina virtual.
  - `[VmosType <OSType?>]`: Destino do tipo de sistema operacional da máquina virtual necessária para implantar o manipulador de extensão.

INPUTobject <IComputeAdminIdentity> : parâmetro de identidade
  - `[DiskId <String>]`: O GUID de disco como identidade.
  - `[Id <String>]`: Caminho de identidade do recurso
  - `[Location <String>]`: Local do recurso.
  - `[MigrationId <String>]`: O nome do GUID do trabalho de migração.
  - `[Offer <String>]`: Nome da oferta.
  - `[Publisher <String>]`: Nome do fornecedor.
  - `[QuotaName <String>]`: Nome da cota.
  - `[Sku <String>]`: Nome da SKU.
  - `[SubscriptionId <String>]`: Credenciais de assinatura que identificam exclusivamente a assinatura do Microsoft Azure. A ID da assinatura forma a parte do URI para cada chamada de serviço.
  - `[Type <String>]`: Tipo de extensão.
  - `[Version <String>]`: A versão do recurso.
