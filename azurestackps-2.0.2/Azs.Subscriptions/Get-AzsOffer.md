# Get-AzsOffer

## Sinopse
Obtenha a lista de ofertas públicas para o provedor raiz.

## SYNTAX

```
Get-AzsOffer [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRITIVO
Obtenha a lista de ofertas públicas para o provedor raiz.

## EXEMPLOS
