# Test-AzsNameAvailability

## Sinopse
Obter a lista de assinaturas.

## SYNTAX

### CheckExpanded (padrão)
```
Test-AzsNameAvailability [-SubscriptionId <String>] [-Name <String>] [-ResourceType <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Selecionar
```
Test-AzsNameAvailability -NameAvailabilityDefinition <ICheckNameAvailabilityDefinition>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### CheckViaIdentity
```
Test-AzsNameAvailability -InputObject <ISubscriptionsAdminIdentity>
 -NameAvailabilityDefinition <ICheckNameAvailabilityDefinition> [-DefaultProfile <PSObject>] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### CheckViaIdentityExpanded
```
Test-AzsNameAvailability -InputObject <ISubscriptionsAdminIdentity> [-Name <String>] [-ResourceType <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Obter a lista de assinaturas.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Test-AzsNameAvailability -ResourceType "Microsoft.Subscriptions.Admin/offers" -Name "testoffer" | fl *

Message       : A resource of type 'Microsoft.Subscriptions.Admin/offers' with name 'testoffer' already exists. Please select a different name.
NameAvailable : False
Reason        : AlreadyExists
```

Retorna a disponibilidade do tipo e do nome do recurso de assinatura especificado

INPUTobject <ISubscriptionsAdminIdentity> : parâmetro de identidade
  - `[DelegatedProvider <String>]`: Identificador DelegatedProvider.
  - `[DelegatedProviderSubscriptionId <String>]`: Identificador de assinatura de provedor delegado.
  - `[Id <String>]`: Caminho de identidade do recurso
  - `[Location <String>]`: O local do AzureStack.
  - `[ManifestName <String>]`: O nome do manifesto.
  - `[Offer <String>]`: Nome de uma oferta.
  - `[OfferDelegationName <String>]`: Nome de uma delegação de oferta.
  - `[OperationsStatusName <String>]`: O nome do status da operação.
  - `[Plan <String>]`: Nome do plano.
  - `[PlanAcquisitionId <String>]`: O identificador de aquisição do plano
  - `[Quota <String>]`: Nome da cota.
  - `[ResourceGroupName <String>]`: O grupo de recursos no qual o recurso está localizado.
  - `[SubscriptionId <String>]`: Credenciais de assinatura que identificam exclusivamente a assinatura do Microsoft Azure. A ID da assinatura forma a parte do URI para cada chamada de serviço.
  - `[TargetSubscriptionId <String>]`: A ID da assinatura de destino.
  - `[Tenant <String>]`: Nome do locatário do diretório.

NAMEAVAILABILITYDEFINITION <ICheckNameAvailabilityDefinition> : a definição da ação verificar disponibilidade do nome.
  - `[Name <String>]`: O nome do recurso a ser verificado.
  - `[ResourceType <String>]`: O tipo de recurso a ser verificado.
