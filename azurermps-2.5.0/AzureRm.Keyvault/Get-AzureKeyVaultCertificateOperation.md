# Get-AzureKeyVaultCertificateOperation

## Sinopse
Obtém o status de uma operação de certificado.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzureKeyVaultCertificateOperation** Obtém o status de uma operação de certificado.
