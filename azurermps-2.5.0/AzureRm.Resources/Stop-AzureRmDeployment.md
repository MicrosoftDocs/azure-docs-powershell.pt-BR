# Stop-AzureRmDeployment

## Sinopse
Cancal uma implantação em execução

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### StopByDeploymentName (padrão)
```
Stop-AzureRmDeployment [-Name] <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzureRmDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzureRmDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Stop-AzureRmDeployment** cancela uma implantação em escopo de assinatura que foi iniciado, mas não foi concluído.
Para interromper uma implantação, a implantação deve ter um estado de provisionamento incompleto, como provisionamento, e não um estado concluído, como provisionado ou com falha.

Para criar uma implantação em escopo de assinatura, use o cmdlet New-AzureRmDeployment.

Este cmdlet interrompe apenas uma implantação em execução. Use o parâmetro *Name* para interromper uma implantação específica.

## EXEMPLOS

### Exemplo 1
```
PS C:\>Stop-AzureRmDeployment -Name "deployment01"
```

Esse comando cancela uma implantação em execução "deployment01" no escopo da assinatura atual.

### Exemplo 2
```
PS C:\>Get-AzureRmDeployment -Name "deployment01" | Stop-AzureRmDeployment
```

Esse comando obtém a implantação "deployment01" no escopo da assinatura atual e a cancela.
