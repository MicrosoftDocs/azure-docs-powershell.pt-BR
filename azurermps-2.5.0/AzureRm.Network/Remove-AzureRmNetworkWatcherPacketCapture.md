# Remove-AzureRmNetworkWatcherPacketCapture

## Sinopse
Remove um recurso de captura de pacote.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

### SetByResource (padrão)
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O Remove-AzureRmNetworkWatcherPacketCapture remove um recurso de captura de pacote. É recomendável chamar Stop-AzureRmNetworkWatcherPacketCapture antes de chamar remove-AzureRmNetworkWatcherPacketCapture. Se a sessão de captura de pacotes estiver em execução quando Remove-AzureRmNetworkWatcherPacketCapture for chamado, a captura de pacotes pode não ser salva. Se a sessão for interrompida antes da remoção, o arquivo. Cap contendo dados de captura não será removido. 

## EXEMPLOS

### ---Exemplo 1: remover uma sessão de captura de pacote--
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

Neste exemplo, removemos uma sessão de captura de pacote existente chamada "PacketCaptureTest".
