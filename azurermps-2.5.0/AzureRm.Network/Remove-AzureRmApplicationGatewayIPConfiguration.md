# Remove-AzureRmApplicationGatewayIPConfiguration

## Sinopse
Remove uma configuração de IP de um aplicativo gateway.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmApplicationGatewayIPConfiguration** remove uma configuração de IP de um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: remover uma configuração de IP de um gateway do aplicativo do Azure
```
PS C:\>$AppGw = Get-AzureRmApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzureRmApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

O primeiro comando obtém um gateway de aplicativo e o armazena na variável $AppGw.

O segundo comando Remove a configuração de IP denominada Subnet02 do gateway do aplicativo armazenado em $AppGw.
