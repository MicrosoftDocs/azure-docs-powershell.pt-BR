# Remove-AzureRmLoadBalancerProbeConfig

## Sinopse
Remove uma configuração de teste de um balanceador de carga.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Remove-AzureRmLoadBalancerProbeConfig [-Name <String>] -LoadBalancer <PSLoadBalancer>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzureRmLoadBalancerProbeConfig** remove uma configuração de teste de um balanceador de carga.

## EXEMPLOS

### Exemplo 1: remover uma configuração de teste de um balanceador de carga
```
PS C:\>$loadbalancer = Get-AzureRmLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzureRmLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $loadbalancer
```

O primeiro comando obtém o balanceador de carga chamado MyLoadBalancer e, em seguida, armazena-o na variável $loadbalancer.

O segundo comando exclui a configuração nomeada myprobe do balanceador de carga em $loadbalancer.
