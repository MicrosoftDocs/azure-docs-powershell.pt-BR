# Get-AzureRmNetworkUsage

## Sinopse
Lista os usos da rede para uma assinatura

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SYNTAX

```
Get-AzureRmNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet Get-AzureRmNetworkUsage Obtém limites e uso atual para recursos de rede.
