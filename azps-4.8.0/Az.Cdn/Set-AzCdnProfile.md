# Set-AzCdnProfile

## Sinopse
Atualiza um perfil CDN.

## SYNTAX

```
Set-AzCdnProfile -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **set-AzCdnProfile** atualiza um perfil da CDN (rede de distribuição de conteúdo) do Azure.
