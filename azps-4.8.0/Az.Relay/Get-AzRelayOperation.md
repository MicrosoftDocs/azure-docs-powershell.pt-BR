# Get-AzRelayOperation

## Sinopse
Lista de operações de retransmissão aceitas

## SYNTAX

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzRelayOperation** lista as operações com suporte para retransmissão.
