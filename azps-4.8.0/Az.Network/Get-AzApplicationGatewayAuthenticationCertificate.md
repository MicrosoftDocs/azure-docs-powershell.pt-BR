# Get-AzApplicationGatewayAuthenticationCertificate

## Sinopse
Obtém um certificado de autenticação para um gateway de aplicativo.

## SYNTAX

```
Get-AzApplicationGatewayAuthenticationCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzApplicationGatewayAuthenticationCertificate** Obtém um certificado de autenticação para um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1: obter um certificado de autenticação especificado
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $pool = Get-AzApplicationGatewayBackendAddressPool -Name "pool01" -ApplicationGateway $appgw
```

O primeiro comando obtém o gateway do aplicativo chamado appGwName e o armazena na variável $appgw.
O segundo comando obtém o certificado de autenticação chamado pool01 e o armazena na variável $pool.
