# New-AzVirtualApplianceSkuProperty

## Sinopse
Defina uma SKU de dispositivo virtual de rede para o recurso.

## SYNTAX

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O comando New-AzVirtualApplianceSkuProperties define um SKU para o recurso de dispositivo virtual de rede.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Crie um objeto de propriedades de SKU de aparelho virtual para ser usado com o comando New-AzNetworkVirtualAppliance.
