# Get-AzExpressRouteCircuit

## Sinopse
Obtém um circuito do Azure ExpressRoute do Azure.

## SYNTAX

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzExpressRouteCircuit** é usado para recuperar um objeto de circuito do ExpressRoute da sua assinatura. O objeto de circuito retornado pode ser usado como entrada para outros cmdlets que operam em circuitos do ExpressRoute.
