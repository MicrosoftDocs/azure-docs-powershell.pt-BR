# Remove-AzApplicationGatewayFrontendPort

## Sinopse
Remove uma porta de front-end de um aplicativo gateway.

## SYNTAX

```
Remove-AzApplicationGatewayFrontendPort -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApplicationGatewayFrontendPort** remove uma porta de front-end de um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo: remover uma porta de front-end de um aplicativo gateway
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort02"
```

O primeiro comando obtém um gateway do aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e armazena o gateway em $AppGw variável.
O segundo comando Remove a porta nomeada FrontEndPort02 do gateway do aplicativo.
