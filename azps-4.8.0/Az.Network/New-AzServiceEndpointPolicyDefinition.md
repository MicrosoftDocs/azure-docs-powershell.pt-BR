# New-AzServiceEndpointPolicyDefinition

## Sinopse
Cria uma definição de política de ponto de extremidade do serviço.

## SYNTAX

```
New-AzServiceEndpointPolicyDefinition -Name <String> [-Description <String>] [-ServiceResource <String[]>]
 [-Service <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzServiceEndpointPolicyDefinition** cria uma definição de política de ponto de extremidade de serviço.

## EXEMPLOS

### Exemplo 1: cria uma política de ponto de extremidade do serviço
```powershell
$policydef= New-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ResourceGroupName "ResourceGroup01" -Service "Microsoft.Storage" -ServiceResources "subscriptions/sub1" -Description "New Definition"
```

Esse comando cria a definição da política de ponto de extremidade do serviço com o nome ServiceEndpointPolicyDefinition1, serviço Microsoft. Storage, assinaturas de recursos de serviço/Sub1 e descrição "New Definition" que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $policydef.

### Exemplo 2

Cria uma definição de política de ponto de extremidade do serviço. (gerado automaticamente)

<!-- Aladdin Generated Example -->
```powershell
New-AzServiceEndpointPolicyDefinition -Description 'New Definition' -Name 'ServiceEndpointPolicyDefinition1' -Service 'Microsoft.Storage' -ServiceResource <String[]>
```
