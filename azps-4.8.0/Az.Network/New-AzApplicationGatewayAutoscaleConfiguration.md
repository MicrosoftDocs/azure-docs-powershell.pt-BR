# New-AzApplicationGatewayAutoscaleConfiguration

## Sinopse
Cria uma configuração de AutoEscala para o gateway do aplicativo.

## SYNTAX

```
New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity <Int32> [-MaxCapacity <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzApplicationGatewayAutoscaleConfiguration** cria uma configuração de AutoEscala para um gateway do aplicativo do Azure.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $autoscaleConfig = New-AzApplicationGatewayAutoscaleConfiguration -MinCapacity 3
PS C:\> $gw = New-AzApplicationGateway -Name $appgwName -ResourceGroupName $rgname ..  -AutoscaleConfiguration $autoscaleConfig
```

O primeiro comando cria uma configuração de dimensionamento automático com capacidade 3 mínima.
O segundo comando cria um gateway de aplicativo com a configuração de autoescala.
