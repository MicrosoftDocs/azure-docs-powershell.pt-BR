# Add-AzApplicationGatewayFrontendPort

## Sinopse
Adiciona uma porta de front-end a um gateway de aplicativo.

## SYNTAX

```
Add-AzApplicationGatewayFrontendPort -ApplicationGateway <PSApplicationGateway> -Name <String> -Port <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzApplicationGatewayFrontendPort** adiciona uma porta de front-end a um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1: adicionar uma porta de front-end a um gateway do aplicativo
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayFrontendPort -ApplicationGateway $AppGw -Name "FrontEndPort01" -Port 80
```

O primeiro comando obtém o gateway do aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O segundo comando adiciona a porta 80 como uma porta de front-end do gateway do aplicativo armazenado no $AppGw e nomeia a porta FrontEndPort01.
