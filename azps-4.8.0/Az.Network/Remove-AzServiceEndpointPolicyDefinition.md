# Remove-AzServiceEndpointPolicyDefinition

## Sinopse
Remove uma definição de política de ponto de extremidade do serviço.

## SYNTAX

### RemoveByNameParameterSet (padrão)
```
Remove-AzServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzServiceEndpointPolicyDefinition -ResourceId <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzServiceEndpointPolicyDefinition -InputObject <PSServiceEndpointPolicyDefinition>
 -ServiceEndpointPolicy <PSServiceEndpointPolicy> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzServiceEndpointPolicy** remove uma política de ponto de extremidade de serviço.

## EXEMPLOS

### Exemplo 1: Remove uma política de ponto de extremidade do serviço usando Name
```
Remove-AzServiceEndpointPolicyDefinition -Name "PolicyDef1"
```

Esse comando Remove uma política de ponto de extremidade de serviço com o nome PolicyDef1
