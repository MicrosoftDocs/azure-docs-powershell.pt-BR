# Add-AzApplicationGatewayPrivateLinkConfiguration

## Sinopse
Adiciona uma configuração de link particular a um gateway de aplicativo.

## SYNTAX

```
Add-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Add-AzApplicationGatewayPrivateLinkConfiguration** adiciona uma configuração de link privado a um gateway de aplicativo.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> $PrivateLinkIpConfiguration = New-AzApplicationGatewayPrivateLinkConfiguration -Name "ipConfig01" -Subnet $subnet -Primary
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $PrivateLinkIpConfiguration
```

O primeiro comando cria um privateLinkIpConfiguration e o armazena na variável $PrivateLinkIpConfiguration.
O segundo comando obtém o gateway do aplicativo chamado ApplicationGateway01 que pertence ao grupo de recursos chamado ResourceGroup01 e o armazena na variável $AppGw.
O terceiro comando adiciona a configuração do link particular chamado privateLinkConfig01, para o gateway em $AppGw
