# New-AzFirewallPolicyDnsSetting

## Sinopse
Cria uma nova configuração de DNS para a política de firewall do Azure

## SYNTAX

```
New-AzFirewallPolicyDnsSetting [-EnableProxy] [-ProxyNotRequiredForNetworkRule] [-Server <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **New-AzFirewallPolicyDnsSetting** cria um objeto de configuração de DNS para a política de firewall do Azure

## EXEMPLOS

### 1. criar uma política vazia
```powershell
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy
```

Este exemplo cria um objeto de configuração de DNS com a configuração habilitando o proxy DNS.

### 2. criar uma política vazia com o modo ThreatIntel
```powershell
PS C:\> $dnsServers = @("10.10.10.1", "20.20.20.2")
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy -Server $dnsServers
```
Este exemplo cria um objeto de configuração de DNS com a configuração habilitando o proxy DNS e definindo servidores DNS personalizados.
