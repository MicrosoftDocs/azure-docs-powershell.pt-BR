# Get-AzCognitiveServicesAccountNetworkRuleSet

## Sinopse
Obter a propriedade NetworkRule de uma conta de serviços cognitiva

## SYNTAX

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Get-AzCognitiveServicesAccountNetworkRuleSet** Obtém a propriedade NetworkRule de uma conta de serviços cognitivas

## EXEMPLOS

### Exemplo 1: obter a propriedade NetworkRule de uma conta de serviços cognitivas especificada
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

Este comando obtém a propriedade NetworkRule de uma conta de serviços cognitivas especificada
