# Get-AzBlockchainConsortium

## Sinopse
Lista os consórcio disponíveis para uma assinatura.

## SYNTAX

```
Get-AzBlockchainConsortium -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRITIVO
Lista os consórcio disponíveis para uma assinatura.

## EXEMPLOS

### Exemplo 1: obter consórcios Blockchain.
```powershell
PS C:\> Get-AzBlockchainConsortium -Location eastus

```

Esse comando lista os Consortiums em uma assinatura para um local específico.
