# Get-AzDataLakeStoreChildItemSummary

## Sinopse
Obtém o resumo do tamanho total, dos arquivos e das pastas contidos no caminho especificado

## SYNTAX

```
Get-AzDataLakeStoreChildItemSummary [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Concurrency <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
**Get-AzDataLakeStoreChildItemSummary** recupera o resumo de conteúdo para um determinado caminho. Calcula recursivamente o número total de arquivos, diretórios e tamanho total de todos os arquivos sob o caminho fornecido.

## EXEMPLOS

### Exemplo 1: obter o resumo de conteúdo de uma pasta
```
PS C:\> Get-AzDataLakeStoreChildItemSummary -Account ContosoADL -Path /a -Concurrency 128
```

Ele lista o número de diretórios, arquivos e o tamanho total contidos em/a.
