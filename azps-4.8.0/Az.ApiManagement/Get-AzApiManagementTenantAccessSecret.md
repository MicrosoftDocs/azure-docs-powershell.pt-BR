# Get-AzApiManagementTenantAccessSecret

## Sinopse
Obtém as chaves de configuração do Access para um locatário.

## SYNTAX

```
Get-AzApiManagementTenantAccessSecret -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRITIVO
Obtém as chaves de configuração do Access para um locatário.

## EXEMPLOS

### Exemplo 1: obter chaves de configuração de acesso ao locatário
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccessSecret -Context $apimContext
```

Este comando obtém as chaves de configuração de acesso de locatário para o contexto especificado.
