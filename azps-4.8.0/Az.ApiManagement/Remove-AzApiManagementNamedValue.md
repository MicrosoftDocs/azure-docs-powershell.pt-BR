# Remove-AzApiManagementNamedValue

## Sinopse
Remove um valor nomeado de gerenciamento de API.

## SYNTAX

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzApiManagementNamedValue** remove um **valor nomeado** de gerenciamento de API do Azure.

## EXEMPLOS

### Exemplo 1: remover o valor nomeado
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

Esse comando Remove o valor nomeado que tem a ID Property11.
