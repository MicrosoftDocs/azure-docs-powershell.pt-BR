# Remove-AzStreamAnalyticsFunction

## Sinopse
Exclui uma função de um trabalho do Stream Analytics.

## SYNTAX

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRITIVO
O cmdlet **Remove-AzStreamAnalyticsFunction** exclui uma função de forma assíncrona de um trabalho do Azure Stream Analytics.

## EXEMPLOS

### Exemplo 1: remover uma função do Stream Analytics
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Esse comando Remove a função denominada ScoreTweet do trabalho chamado StreamJob22.
