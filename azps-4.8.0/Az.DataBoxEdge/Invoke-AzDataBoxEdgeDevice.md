## Sinopse

## DESCRITIVO
O cmdlet **Invoke-AzDataBoxEdgeDevice** invoca ações para examinar, baixar e instalar as atualizações no dispositivo de borda da caixa de dados. Uma verificação automática é executada no dispositivo diariamente, você pode disparar a digitalização explicitamente executando esse cmdlet.

## EXEMPLOS

### Exemplo 1
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### Exemplo 2
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### Exemplo 3
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
