# Módulo AZ. KubernetesConfiguration
## Descritivo
Microsoft Azure PowerShell: cmdlets do KubernetesConfiguration

## Cmdlets AZ. KubernetesConfiguration
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Obtém detalhes da configuração de controle do código-fonte.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Crie uma nova configuração de controle de origem kubernetes.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Isso excluirá o arquivo YAML usado para configurar a configuração do controle do código-fonte, por isso interrompendo a sincronização futura do repositório de origem.
