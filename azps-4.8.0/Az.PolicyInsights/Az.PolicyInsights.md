# Módulo AZ. PolicyInsights
## Descritivo
Este tópico exibe ajuda para os cmdlets do Azure Policy insights.

## Cmdlets AZ. PolicyInsights
### [Get-AzPolicyEvent](Get-AzPolicyEvent.md)
Obtém eventos de avaliação de política gerados à medida que os recursos são criados ou atualizados.

### [Get-AzPolicyMetadata](Get-AzPolicyMetadata.md)
Obter recursos de metadados de política

### [Get-AzPolicyRemediation](Get-AzPolicyRemediation.md)
Obtém correções de política.

### [Get-AzPolicyState](Get-AzPolicyState.md)
Obtém os Estados de conformidade de política para recursos.

### [Get-AzPolicyStateSummary](Get-AzPolicyStateSummary.md)
Obtém o resumo dos Estados mais recentes de conformidade com a política para recursos.

### [Remove-AzPolicyRemediation](Remove-AzPolicyRemediation.md)
Exclui uma correção de política.

### [Start-AzPolicyRemediation](Start-AzPolicyRemediation.md)
Cria e inicia uma correção de política para uma atribuição de política.

### [Parar-AzPolicyRemediation](Stop-AzPolicyRemediation.md)
Cancela uma remediação de política em andamento.

### [Start-AzPolicyComplianceScan](Start-AzPolicyComplianceScan.md)
Dispara uma avaliação de conformidade de política para todos os recursos de uma assinatura ou grupo de recursos.
